## pwd, ls, cd

লিনাক্স টার্মিনালে কাজ করার সময় সবচেয়ে বেশি ব্যবহৃত হয় কিছু নেভিগেশন কমান্ড। এগুলোর মাধ্যমে আপনি ফাইল সিস্টেমে ঘুরে বেড়াতে, ফোল্ডার দেখতে ও অবস্থান পরিবর্তন করতে পারবেন। এখানে আমরা তিনটি গুরুত্বপূর্ণ কমান্ড নিয়ে বিস্তারিত আলোচনা করব: `pwd`, `ls`, এবং `cd`।

---

## ১. pwd: Present Working Directory

`pwd` কমান্ড দিয়ে আপনি জানতে পারবেন, বর্তমানে টার্মিনালে কোন ডিরেক্টরিতে (ফোল্ডারে) অবস্থান করছেন।

```bash
pwd
```

**উদাহরণ:**

```
/home/anis/Documents
```

এখানে `/home/anis/Documents` হচ্ছে আপনার বর্তমান ডিরেক্টরি।

**কেন দরকার?**
- কখনো কখনো আপনি এত ঘুরে বেড়ান যে, কোথায় আছেন ভুলে যান। তখন `pwd` দিয়ে চেক করতে পারেন।

---

## ২. ls: List Directory Contents

`ls` কমান্ড দিয়ে আপনি বর্তমান ডিরেক্টরির (বা নির্দিষ্ট ডিরেক্টরির) ফাইল ও ফোল্ডার দেখতে পারবেন।

```bash
ls
```

**উদাহরণ:**

```
Desktop  Documents  Downloads  Pictures
```

**কিছু গুরুত্বপূর্ণ অপশন:**
- `ls -l` : বিস্তারিত তথ্যসহ দেখাবে (পারমিশন, মালিক, সাইজ, তারিখ)
- `ls -a` : হিডেন ফাইলসহ দেখাবে
- `ls /path/to/dir` : নির্দিষ্ট ডিরেক্টরির কনটেন্ট দেখাবে

**উদাহরণ:**

```bash
ls -la /home/anis
```

---

## ৩. cd: Change Directory

`cd` কমান্ড দিয়ে আপনি এক ডিরেক্টরি থেকে অন্য ডিরেক্টরিতে যেতে পারবেন।

```bash
cd /path/to/directory
```

**কিছু গুরুত্বপূর্ণ ব্যবহার:**
- `cd ~` : হোম ডিরেক্টরিতে যাবে
- `cd ..` : এক ধাপ উপরে যাবে
- `cd -` : আগের ডিরেক্টরিতে ফিরে যাবে
- `cd` : শুধু লিখলে হোম ডিরেক্টরিতে যাবে

**উদাহরণ:**

```bash
cd Documents
cd ..
cd /var/log
```

---

## প্র্যাকটিক্যাল টিপস

- `ls` ও `cd` একসাথে ব্যবহার করে দ্রুত ফোল্ডার ঘুরে দেখতে পারেন।
- Tab key প্রেস করলে অটো-কমপ্লিট হবে (যেমন: `cd Doc` লিখে Tab চাপলে `Documents` হয়ে যাবে)।
- `ls -lh` দিয়ে ফাইল সাইজ human readable (KB, MB) ফরম্যাটে দেখতে পারবেন।

---

## চ্যালেঞ্জ: নিজে চেষ্টা করুন

১. আপনার হোম ডিরেক্টরিতে যান, তারপর Desktop-এ যান, আবার এক ধাপ উপরে আসুন।
২. `ls -la` দিয়ে হিডেন ফাইল দেখুন।
৩. `pwd` দিয়ে চেক করুন, কোথায় আছেন।

---

## উপসংহার

`pwd`, `ls`, ও `cd`—এই তিনটি কমান্ড ছাড়া লিনাক্স টার্মিনালে কাজ কল্পনাই করা যায় না। এগুলো ভালোভাবে আয়ত্ত করলে ফাইল সিস্টেমে ঘুরে বেড়ানো হবে অনেক সহজ ও দ্রুত।

**প্র্যাকটিস করুন, কমান্ড লাইনের জাদু অনুভব করুন!**
