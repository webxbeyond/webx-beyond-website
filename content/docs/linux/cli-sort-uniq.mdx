লিনাক্স টার্মিনালে ডেটা বিশ্লেষণ, গণনা, ও সাজানোর জন্য `wc`, `sort`, ও `uniq`—এই তিনটি কমান্ড খুবই গুরুত্বপূর্ণ। এগুলো দিয়ে আপনি দ্রুত লাইনের সংখ্যা, শব্দ, ক্যারেক্টার গণনা, ডেটা সাজানো, ও ডুপ্লিকেট ফিল্টার করতে পারবেন।

---

## ১. wc (word count): গণনা

`wc` দিয়ে ফাইল বা ইনপুটের লাইনের সংখ্যা, শব্দ, ও ক্যারেক্টার গণনা করা যায়।

**বেসিক ব্যবহার:**

```bash
wc filename.txt
```

**আউটপুট:**
- লাইনের সংখ্যা
- শব্দের সংখ্যা
- ক্যারেক্টারের সংখ্যা

**কিছু অপশন:**
- `-l` : শুধু লাইনের সংখ্যা
- `-w` : শুধু শব্দের সংখ্যা
- `-c` : শুধু ক্যারেক্টারের সংখ্যা

**উদাহরণ:**

```bash
wc -l file.txt   # লাইনের সংখ্যা
cat *.md | wc -w # সব md ফাইলের মোট শব্দ
```

---

## ২. sort: সাজানো

`sort` দিয়ে ফাইল বা ইনপুটের লাইনগুলো সাজানো যায় (alphabetical, numerical, reverse)।

**বেসিক ব্যবহার:**

```bash
sort filename.txt
```

**কিছু অপশন:**
- `-r` : উল্টো (reverse) সাজানো
- `-n` : সংখ্যাগত সাজানো
- `-u` : ডুপ্লিকেট বাদ দিয়ে সাজানো

**উদাহরণ:**

```bash
sort -n numbers.txt
sort -r names.txt
```

---

## ৩. uniq: ডুপ্লিকেট ফিল্টার

`uniq` দিয়ে ডুপ্লিকেট লাইন বাদ দেয়া যায় (শুধু কনসিকিউটিভ ডুপ্লিকেট)।

**বেসিক ব্যবহার:**

```bash
uniq filename.txt
```

**কিছু অপশন:**
- `-c` : প্রতিটি ইউনিক লাইনের সংখ্যা দেখাবে
- `-d` : শুধু ডুপ্লিকেট লাইন দেখাবে

**উদাহরণ:**

```bash
sort names.txt | uniq -c
```

---

## ৪. পাইপিং ও কম্বিনেশন

তিনটি কমান্ড একসাথে ব্যবহার করে শক্তিশালী ডেটা প্রসেসিং করা যায়।

**উদাহরণ:**

```bash
cat access.log | awk '{print $1}' | sort | uniq -c | sort -nr
```

এতে, access.log থেকে ইউনিক IP ও তাদের সংখ্যা descending order-এ দেখাবে।

## ৫. প্র্যাকটিক্যাল টিপস

- wc দিয়ে দ্রুত রিপোর্ট তৈরি করুন
- sort দিয়ে ডেটা সাজান, uniq দিয়ে ডুপ্লিকেট বাদ দিন
- পাইপিং দিয়ে একসাথে বিশ্লেষণ করুন
