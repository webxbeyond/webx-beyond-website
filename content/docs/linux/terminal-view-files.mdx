## cat, less, head, tail

লিনাক্সে ফাইলের কনটেন্ট দেখা খুবই গুরুত্বপূর্ণ। বিশেষ করে টেক্সট ফাইল, লগ, কনফিগারেশন ফাইল ইত্যাদি দ্রুত দেখতে হলে টার্মিনালের কিছু কমান্ড জানা দরকার। এখানে আমরা `cat`, `less`, `head`, ও `tail` নিয়ে বিস্তারিত আলোচনা করব।

---

## ১. cat: ফাইলের সম্পূর্ণ কনটেন্ট দেখুন

`cat` (concatenate) কমান্ড দিয়ে আপনি ফাইলের পুরো কনটেন্ট একসাথে দেখতে পারেন।

```bash
cat filename.txt
```

একাধিক ফাইল একসাথে দেখাতে:

```bash
cat file1.txt file2.txt
```

**কিছু টিপস:**
- বড় ফাইল হলে সব কনটেন্ট একসাথে স্ক্রলে চলে যাবে, তাই সাবধানে ব্যবহার করুন।
- `cat -n filename.txt` দিয়ে লাইনের নম্বরসহ দেখাতে পারবেন।

---

## ২. less: বড় ফাইল আরামসে দেখুন

`less` কমান্ড দিয়ে বড় ফাইল পেজ বাই পেজ দেখতে পারবেন।

```bash
less filename.txt
```

**ব্যবহার:**
- Up/Down arrow দিয়ে স্ক্রল করুন
- `/searchword` লিখে খুঁজুন
- `q` চাপলে বেরিয়ে আসবেন

**কেন দরকার?**
- বড় লগ ফাইল, কনফিগারেশন ফাইল ইত্যাদি পড়ার জন্য আদর্শ।

---

## ৩. head: ফাইলের শুরু দিকের অংশ দেখুন

`head` কমান্ড দিয়ে ফাইলের প্রথম ১০টি (ডিফল্ট) লাইন দেখতে পারবেন।

```bash
head filename.txt
```

নির্দিষ্ট সংখ্যক লাইন দেখতে:

```bash
head -20 filename.txt   # প্রথম ২০ লাইন
```

---

## ৪. tail: ফাইলের শেষ দিকের অংশ দেখুন

`tail` কমান্ড দিয়ে ফাইলের শেষ ১০টি (ডিফল্ট) লাইন দেখতে পারবেন।

```bash
tail filename.txt
```

নির্দিষ্ট সংখ্যক লাইন দেখতে:

```bash
tail -15 filename.txt   # শেষ ১৫ লাইন
```

লাইভ লগ মনিটর করতে:

```bash
tail -f /var/log/syslog
```

এতে নতুন লাইন আসলে সাথে সাথে দেখতে পাবেন।

---

## প্র্যাকটিক্যাল টিপস

- `less` দিয়ে বড় ফাইল পড়ুন, `cat` ছোট ফাইলের জন্য ব্যবহার করুন।
- `head` ও `tail` দিয়ে দ্রুত ফাইলের শুরু ও শেষ অংশ চেক করুন।
- `tail -f` দিয়ে লাইভ লগ মনিটর করুন (সার্ভার/ডেভেলপমেন্টে খুব কাজে লাগে)।
- `cat file | less` কম্বিনেশনেও ব্যবহার করা যায়, তবে সরাসরি `less file`-ই ভালো।

---

## চ্যালেঞ্জ: নিজে চেষ্টা করুন

১. একটি টেক্সট ফাইলের প্রথম ৫ লাইন দেখুন।
২. একটি লগ ফাইলের শেষ ২০ লাইন দেখুন।
৩. `less` দিয়ে একটি বড় ফাইল পড়ুন এবং `/` দিয়ে কিছু খুঁজুন।
৪. `tail -f` দিয়ে লাইভ লগ মনিটর করুন।

---

## উপসংহার

`cat`, `less`, `head`, ও `tail`—এই কমান্ডগুলো ছাড়া ফাইল কনটেন্ট দেখা, লগ মনিটরিং, ও কনফিগারেশন চেক করা অসম্ভব। এগুলো ভালোভাবে আয়ত্ত করলে আপনার টার্মিনাল দক্ষতা অনেক বাড়বে।

**প্র্যাকটিস করুন, ফাইল পড়ার জাদু শিখুন!**
