# শেল স্ক্রিপ্টে if স্টেটমেন্ট ও লুপ: কন্ডিশনাল ও রিপিটেটিভ লজিক

শেল স্ক্রিপ্টিংয়ে decision making (if) ও automation (loop) সবচেয়ে গুরুত্বপূর্ণ বিষয়। এখানে আমরা if, elif, else, for, while, এবং case—সবকিছু হাতে-কলমে, উদাহরণসহ শিখব।

---

## ১. if স্টেটমেন্ট: সিদ্ধান্ত নেওয়া

### ১.১. বেসিক if
```bash
num=10
if [ $num -gt 5 ]; then
  echo "$num পাঁচের চেয়ে বড়"
fi
```

### ১.২. if-else
```bash
if [ $num -lt 5 ]; then
  echo "$num পাঁচের চেয়ে ছোট"
else
  echo "$num পাঁচের চেয়ে বড় বা সমান"
fi
```

### ১.৩. if-elif-else
```bash
if [ $num -lt 5 ]; then
  echo "কম"
elif [ $num -eq 5 ]; then
  echo "সমান"
else
  echo "বেশি"
fi
```

## ২. শর্তের ধরন


- **সংখ্যা তুলনা (Numeric Comparison):**
  - `-eq`: সমান কিনা (equal to)। যেমন `[ $a -eq $b ]` মানে $a এবং $b সমান কি না।
  - `-ne`: সমান নয় কিনা (not equal)। যেমন `[ $a -ne $b ]` মানে $a এবং $b সমান নয় কি না।
  - `-gt`: বড় কিনা (greater than)। যেমন `[ $a -gt $b ]` মানে $a কি $b থেকে বড়?
  - `-lt`: ছোট কিনা (less than)। যেমন `[ $a -lt $b ]` মানে $a কি $b থেকে ছোট?
  - `-ge`: বড় বা সমান কিনা (greater or equal)। যেমন `[ $a -ge $b ]` মানে $a কি $b থেকে বড় বা সমান?
  - `-le`: ছোট বা সমান কিনা (less or equal)। যেমন `[ $a -le $b ]` মানে $a কি $b থেকে ছোট বা সমান?

- **স্ট্রিং তুলনা (String Comparison):**
  - `=`: দুইটি স্ট্রিং সমান কিনা। যেমন `[ "$str1" = "$str2" ]`
  - `!=`: দুইটি স্ট্রিং সমান নয় কিনা। যেমন `[ "$str1" != "$str2" ]`
  - `-z`: স্ট্রিং ফাঁকা কিনা (zero length)। যেমন `[ -z "$str" ]` মানে $str ফাঁকা কি না।
  - `-n`: স্ট্রিং ফাঁকা নয় কিনা (non-zero length)। যেমন `[ -n "$str" ]` মানে $str-এ কিছু আছে কি না।

- **ফাইল চেক (File Test):**
  - `-e`: ফাইল বা ডিরেক্টরি আছে কিনা। `[ -e file.txt ]`
  - `-f`: সাধারণ ফাইল কিনা (regular file)। `[ -f file.txt ]`
  - `-d`: ডিরেক্টরি কিনা। `[ -d myfolder ]`
  - `-r`: ফাইলটি পড়ার যোগ্য কিনা (readable)। `[ -r file.txt ]`
  - `-w`: ফাইলটি লেখার যোগ্য কিনা (writable)। `[ -w file.txt ]`
  - `-x`: ফাইলটি এক্সিকিউটেবল কিনা (executable)। `[ -x script.sh ]`

```bash
if [ -f myfile.txt ]; then
  echo "ফাইল আছে"
fi
```

## ৩. লজিক্যাল অপারেটর

- AND: `[ $a -gt 5 ] && [ $b -lt 10 ]`
- OR: `[ $a -gt 5 ] || [ $b -lt 10 ]`

## ৪. for লুপ: রিপিটেটিভ টাস্ক

### ৪.১. বেসিক for
```bash
for i in 1 2 3 4 5; do
  echo "নাম্বার: $i"
done
```

### ৪.২. ফাইল লিস্টিং
```bash
for file in *.txt; do
  echo "ফাইল: $file"
done
```

---

## ৫. while লুপ

```bash
count=1
while [ $count -le 5 ]; do
  echo "কাউন্ট: $count"
  count=$((count+1))
done
```

---

## ৬. until লুপ

```bash
num=1
until [ $num -gt 5 ]; do
  echo $num
  num=$((num+1))
done
```

---

## ৭. case স্টেটমেন্ট: মাল্টিপল অপশন

```bash
read -p "আপনার পছন্দ (a/b/c): " choice
case $choice in
  a) echo "আপনি a বেছে নিয়েছেন" ;;
  b) echo "আপনি b বেছে নিয়েছেন" ;;
  c) echo "আপনি c বেছে নিয়েছেন" ;;
  *) echo "ভুল ইনপুট" ;;
esac
```

## ৮. ব্রেক ও কন্টিনিউ

```bash
for i in {1..10}; do
  if [ $i -eq 5 ]; then
    break
  fi
  echo $i
done

for i in {1..5}; do
  if [ $i -eq 3 ]; then
    continue
  fi
  echo $i
done
```

## ৯. বাস্তব উদাহরণ: ফাইল এক্সটেনশন কাউন্ট
