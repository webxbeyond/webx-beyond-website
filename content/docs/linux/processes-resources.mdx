# free, df, du দিয়ে রিসোর্স মনিটর: RAM, ডিস্ক ও স্পেস চেক

লিনাক্সে সিস্টেমের RAM, ডিস্ক স্পেস, ও ফাইল/ডিরেক্টরি কতটা জায়গা নিচ্ছে—এসব মনিটর করার জন্য সবচেয়ে বেশি ব্যবহৃত টুল হলো `free`, `df`, ও `du`। এখানে আমরা এই তিনটি কমান্ডের ব্যবহার, আউটপুট বোঝা, ও প্র্যাকটিক্যাল টিপস জানব।

---

## ১. free: RAM ও Swap মনিটর

### বেসিক ব্যবহার:
```bash
free
```

### human readable (MB/GB):
```bash
free -h
```

### আউটপুট ব্যাখ্যা:
- total: মোট RAM
- used: ব্যবহৃত
- free: ফাঁকা
- shared: শেয়ার্ড মেমোরি
- buff/cache: ক্যাশড
- available: অ্যাপ্লিকেশনের জন্য ফ্রি

---

## ২. df: ডিস্ক স্পেস মনিটর

### বেসিক ব্যবহার:
```bash
df
```

### human readable (GB/MB):
```bash
df -h
```

### নির্দিষ্ট ফাইলসিস্টেম:
```bash
df -h /home
```

### আউটপুট ব্যাখ্যা:
- Filesystem: ডিস্ক/পার্টিশন
- Size: মোট স্পেস
- Used: ব্যবহৃত
- Avail: ফাঁকা
- Use%: পার্সেন্টেজ
- Mounted on: কোথায় মাউন্ট

---

## ৩. du: ফোল্ডার/ফাইল স্পেস মনিটর

### বেসিক ব্যবহার:
```bash
du
```

### human readable:
```bash
du -h
```

### নির্দিষ্ট ফোল্ডার:
```bash
du -sh ~/Downloads
```
- `-s`: summary
- `-h`: human readable

### ফোল্ডারের ভিতরের সাইজ:
```bash
du -h --max-depth=1 ~/Documents
```

---

## ৪. প্র্যাকটিক্যাল টিপস

- `free -h` ও `df -h` সবসময় human readable
- ডিস্ক ফুল হলে `du` দিয়ে কোন ফোল্ডার বেশি জায়গা নিচ্ছে দেখুন
- লজ ফাইল, ক্যাশ ক্লিন করুন
- man page পড়ুন: `man free`, `man df`, `man du`
