মাইক্রোটিক রাউটারে স্ট্যাটিক লিজ (Static Lease) হলো DHCP সার্ভার থেকে নির্দিষ্ট MAC Address-এর জন্য নির্দিষ্ট IP বরাদ্দ করার পদ্ধতি। এটি গুরুত্বপূর্ণ ডিভাইস (যেমন: সার্ভার, প্রিন্টার, ক্যামেরা) সবসময় একই IP পায় তা নিশ্চিত করে। এই টিউটোরিয়ালে আমরা স্ট্যাটিক লিজ কনফিগারেশন, বাস্তব উদাহরণ ও বেস্ট প্র্যাকটিস নিয়ে বিস্তারিত আলোচনা করব।

---

## স্ট্যাটিক লিজ কী ও কেন দরকার?

- গুরুত্বপূর্ণ ডিভাইসের জন্য নির্দিষ্ট IP নিশ্চিত করা
- নেটওয়ার্ক ম্যানেজমেন্ট সহজ করা
- Port Forwarding, Firewall, Monitoring ইত্যাদিতে নির্দিষ্ট IP দরকার হলে

---

## স্ট্যাটিক লিজ কনফিগারেশন স্টেপ

### ১. ডিভাইসের MAC Address সংগ্রহ করুন
- ডিভাইসের নেটওয়ার্ক সেটিংস বা DHCP Leases থেকে MAC Address নিন

### ২. DHCP Server-এ Static Lease অ্যাড করুন

```shell
/ip dhcp-server lease add mac-address=AA:BB:CC:DD:EE:FF address=192.168.88.50 server=dhcp1
```

- `mac-address`: টার্গেট ডিভাইসের MAC
- `address`: নির্দিষ্ট IP
- `server`: DHCP সার্ভারের নাম

### ৩. Winbox-এ Static Lease অ্যাড
1. **IP > DHCP Server > Leases** এ যান
2. ডিভাইস সিলেক্ট করে **Make Static** বাটনে ক্লিক করুন
3. চাইলে IP পরিবর্তন করুন

---

## স্ট্যাটিক লিজ ব্যবহারের কেস স্টাডি

### কেস ১: অফিস প্রিন্টার সবসময় একই IP পাবে
- Static Lease দিয়ে নির্দিষ্ট IP দিন

### কেস ২: সার্ভার/ক্যামেরার জন্য Port Forwarding
- Static Lease দিয়ে নির্দিষ্ট IP দিন, তারপর NAT/Firewall কনফিগার করুন

### কেস ৩: নির্দিষ্ট ইউজারকে নির্দিষ্ট ব্যান্ডউইথ
- Static Lease দিয়ে IP নির্ধারণ করুন, তারপর Queue/PCQ কনফিগার করুন

---

## টিপস ও বেস্ট প্র্যাকটিস
- Static Lease-এর জন্য IP Pool-এর বাইরে বা শেষে IP দিন
- MAC Address সঠিকভাবে সংগ্রহ করুন
- Static Lease লিস্ট নিয়মিত আপডেট রাখুন
- ডিভাইসের নেটওয়ার্ক সেটিংসে DHCP Enable রাখুন

---

## Static Lease বনাম Dynamic Lease

| বৈশিষ্ট্য         | Static Lease         | Dynamic Lease        |
|-------------------|---------------------|---------------------|
| IP নির্দিষ্ট      | হ্যাঁ                | না                  |
| MAC ভিত্তিক       | হ্যাঁ                | না                  |
| ম্যানুয়াল কনফিগার| হ্যাঁ                | না                  |
| ব্যবহারের ক্ষেত্র  | গুরুত্বপূর্ণ ডিভাইস  | সাধারণ ডিভাইস       |

---

## উপসংহার

স্ট্যাটিক লিজ কনফিগারেশন করলে গুরুত্বপূর্ণ ডিভাইস সবসময় নির্দিষ্ট IP পায়, ফলে নেটওয়ার্ক ম্যানেজমেন্ট, মনিটরিং ও সিকিউরিটি সহজ হয়। মাইক্রোটিকের DHCP Server-এ সহজেই Static Lease অ্যাড করা যায়।
