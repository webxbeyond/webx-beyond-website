মাইক্রোটিক রাউটার ব্যবহারের সময় অনেক সময় ডিভাইসকে রিসেট বা ডিফল্ট কনফিগারেশনে ফিরিয়ে আনার প্রয়োজন হয়। এতে পুরনো কনফিগারেশন মুছে যায় এবং ডিভাইস নতুন অবস্থায় ফিরে আসে। এখানে রিসেট ও ডিফল্ট কনফিগারেশন নিয়ে বিস্তারিত আলোচনা করা হলো।

## ১. কখন রিসেট করবেন?
- পাসওয়ার্ড ভুলে গেলে
- কনফিগারেশন সমস্যা হলে
- নতুনভাবে নেটওয়ার্ক সেটআপ করতে চাইলে
- ডিভাইস বিক্রির আগে

## ২. রিসেট করার পদ্ধতি

### ক. হার্ডওয়্যার বাটন দিয়ে রিসেট

১. রাউটারের পাওয়ার অন রাখুন
২. রিসেট বাটন খুঁজে বের করুন (সাধারণত ছোট হোল)
৩. বাটনটি প্রেস করে রাখুন (৫-১০ সেকেন্ড)
৪. LED ব্লিঙ্ক করলে ছেড়ে দিন
৫. ডিভাইস ডিফল্ট কনফিগারেশনে ফিরে আসবে

### খ. সফটওয়্যার/Winbox/WebFig দিয়ে রিসেট

- Winbox > System > Reset Configuration
- WebFig > System > Reset Configuration
- CLI:
  ```
  /system reset-configuration
  ```
- চাইলে "No Default Configuration" টিক দিন (একদম খালি কনফিগারেশন)

## ৩. ডিফল্ট কনফিগারেশন কী?

- মাইক্রোটিক রাউটারে সাধারণত একটি ডিফল্ট কনফিগারেশন থাকে
- সাধারণত IP: 192.168.88.1/24, DHCP Server, NAT, ব্রিজ ইত্যাদি থাকে
- ডিফল্ট কনফিগারেশন না চাইলে রিসেটের সময় "No Default Configuration" সিলেক্ট করুন

## ৪. ব্যাকআপ ও রিস্টোর

রিসেটের আগে কনফিগারেশন ব্যাকআপ রাখা জরুরি।

- Winbox: Files > Backup
- CLI:
  ```
  /system backup save name=backup
  /system backup load name=backup
  ```
- Export:
  ```
  /export file=config
  /import file=config
  ```

## ৫. সতর্কতা
- রিসেট করলে সব কনফিগারেশন মুছে যাবে
- লাইসেন্স, RouterOS ভার্সন অপরিবর্তিত থাকবে
- ব্যাকআপ ছাড়া রিসেট করবেন না

## ৬. সমস্যা ও সমাধান
- **রিসেটের পর Winbox-এ ডিভাইস দেখা যাচ্ছে না:**
  - IP কনফিগার দিন বা MAC দিয়ে সংযোগ করুন
- **ডিফল্ট কনফিগারেশন লোড হচ্ছে না:**
  - "No Default Configuration" টিক ছিল কিনা চেক করুন

## উপসংহার
রিসেট ও ডিফল্ট কনফিগারেশন জানা থাকলে যেকোনো সময় ডিভাইস নতুনভাবে কনফিগার করা সহজ হয়। পরবর্তী অধ্যায়ে রাউটার আইডেন্টিটি ও বেসিক সিকিউরিটি নিয়ে আলোচনা করা হবে।
