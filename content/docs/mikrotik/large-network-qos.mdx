## PCQ (Per Connection Queuing) কনফিগারেশন

### ১. PCQ Type Create
```shell
/queue type add name=pcq-download kind=pcq pcq-rate=2M pcq-classifier=dst-address
/queue type add name=pcq-upload kind=pcq pcq-rate=1M pcq-classifier=src-address
```

### ২. Simple Queue for Large User Base
```shell
/queue simple add name="User-Internet" target=192.168.0.0/16 max-limit=500M/500M queue=pcq-upload/pcq-download
```

---

## Queue Tree (Advanced Traffic Shaping)

- Parent Queue দিয়ে Bandwidth Divide
- Child Queue-এ PCQ/Other Type

### Example
```shell
/queue tree add name="Download" parent=global-in queue=pcq-download max-limit=500M
/queue tree add name="Upload" parent=global-out queue=pcq-upload max-limit=500M
```

---

## Layer 7 Filtering

- Application/Content Based Bandwidth Control

### Example
```shell
/ip firewall layer7-protocol add name=facebook regexp="^.+(facebook.com).*$"
/ip firewall mangle add chain=forward layer7-protocol=facebook action=mark-connection new-connection-mark=facebook_conn
/queue tree add name="Facebook" parent=global-in packet-mark=facebook_conn max-limit=50M queue=pcq-download
```
