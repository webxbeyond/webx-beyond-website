মাল্টি-ISP ফেইলওভার ও লোড ব্যালান্সিং নেটওয়ার্কের রিডান্ডেন্সি, আপটাইম, এবং ব্যান্ডউইথ অপ্টিমাইজেশনের জন্য অত্যন্ত গুরুত্বপূর্ণ। মাইক্রোটিক RouterOS-এ সহজেই একাধিক ISP সংযোগ, অটোমেটিক ফেইলওভার, এবং লোড ব্যালান্সিং কনফিগার করা যায়।

---

## মাল্টি-ISP নেটওয়ার্ক ডিজাইন

- **Primary ISP:** Main Internet Connection
- **Secondary ISP:** Backup/Failover
- **Load Balancing:** Bandwidth Optimization
- **Redundancy:** Network Stability

---

## Initial Setup

### ১. WAN Interfaces কনফিগার
```shell
/interface ethernet set [find default-name=ether1] name=ISP1
/interface ethernet set [find default-name=ether2] name=ISP2
```

### ২. IP Address ও Gateway
```shell
/ip address add address=100.100.100.2/30 interface=ISP1
/ip address add address=200.200.200.2/30 interface=ISP2
/ip route add gateway=100.100.100.1
/ip route add gateway=200.200.200.1 distance=2
```

---

## ফেইলওভার কনফিগারেশন

- **Primary Route:** distance=1
- **Secondary Route:** distance=2
- **Netwatch/Scripting:** Gateway Down হলে অটো-রুট পরিবর্তন

### Netwatch Example
```shell
/tool netwatch add host=8.8.8.8 down-script="/ip route disable [find gateway=100.100.100.1]" up-script="/ip route enable [find gateway=100.100.100.1]"
```

---

## লোড ব্যালান্সিং (ECMP)

- **Equal Cost Multi-Path (ECMP):** Multiple Gateway
- **Per Connection/Class Based Load Balancing**

### ECMP Example
```shell
/ip route add dst-address=0.0.0.0/0 gateway=100.100.100.1,200.200.200.1
```

---

## PCC (Per Connection Classifier) লোড ব্যালান্সিং

- PCC দিয়ে Connection/Class ভিত্তিক ট্রাফিক ভাগ করা যায়

### PCC Example
```shell
/ip firewall mangle add chain=prerouting dst-address-type=!local per-connection-classifier=src-address:2/0 action=mark-connection new-connection-mark=ISP1_conn passthrough=yes
/ip firewall mangle add chain=prerouting dst-address-type=!local per-connection-classifier=src-address:2/1 action=mark-connection new-connection-mark=ISP2_conn passthrough=yes
/ip route add gateway=100.100.100.1 routing-mark=ISP1_conn
/ip route add gateway=200.200.200.1 routing-mark=ISP2_conn
```
