ফাংশন ব্যবহারের সময় ইনপুট নেওয়ার জন্য প্যারামিটার ও আর্গুমেন্ট ব্যবহার করা হয়। এগুলো ফাংশনকে আরও জেনারেল ও পুনঃব্যবহারযোগ্য করে তোলে। এই অধ্যায়ে আমরা প্যারামিটার ও আর্গুমেন্টের পার্থক্য, ধরন, ব্যবহার, উদাহরণ ও ভালো অভ্যাস জানব।

---

## ১. প্যারামিটার ও আর্গুমেন্ট কী?

- **প্যারামিটার (Parameter):** ফাংশন ডিফিনিশনে উল্লেখিত ইনপুট ভেরিয়েবল
- **আর্গুমেন্ট (Argument):** ফাংশন কল করার সময় পাঠানো মান

### উদাহরণ (Python):
```python
def add(a, b):  # a, b হলো প্যারামিটার
    return a + b

result = add(3, 4)  # 3, 4 হলো আর্গুমেন্ট
```

## ২. প্যারামিটারের ধরন

### ১. পজিশনাল প্যারামিটার
- আর্গুমেন্টের ক্রম অনুযায়ী মান বসে

### ২. কিওয়ার্ড/নেমড প্যারামিটার
- আর্গুমেন্টের নাম উল্লেখ করে মান পাঠানো হয় (Python)

### ৩. ডিফল্ট প্যারামিটার
- প্যারামিটারের জন্য ডিফল্ট মান নির্ধারণ করা যায়

### ৪. ভ্যারিয়াডিক প্যারামিটার
- অজানা সংখ্যক আর্গুমেন্ট নিতে পারে (`*args`, `**kwargs`)

---

## ৩. প্যারামিটার ও আর্গুমেন্ট: বিভিন্ন ভাষায়

### Python:
```python
def greet(name, msg="Hello"):
    print(msg, name)

greet("Rahim")           # Hello Rahim
greet("Karim", "Hi")     # Hi Karim
```

### C:
```c
void greet(char name[], char msg[]) {
    printf("%s %s\n", msg, name);
}

greet("Rahim", "Hello");
```

## ৪. ভ্যারিয়াডিক ফাংশন

### Python:
```python
def add_all(*numbers):
    return sum(numbers)

print(add_all(1, 2, 3, 4))  # 10
```

### C (stdarg.h):
```c
#include <stdarg.h>
int add_all(int count, ...) {
    va_list args;
    va_start(args, count);
    int sum = 0;
    for (int i = 0; i < count; i++) {
        sum += va_arg(args, int);
    }
    va_end(args);
    return sum;
}
```

## ৫. প্যারামিটার ও আর্গুমেন্টের ভালো অভ্যাস

- অর্থবহ নাম ব্যবহার করুন
- ডিফল্ট মান যুক্ত করুন যেখানে প্রয়োজন
- ভ্যারিয়াডিক প্যারামিটার বুঝে ব্যবহার করুন
- ফাংশনের ইনপুট সীমিত রাখুন (অনেক বেশি প্যারামিটার এড়িয়ে চলুন)
