প্রোগ্রামিংয়ের মৌলিক ভিত্তি হলো ভেরিয়েবল ও ডাটা টাইপ। এই অধ্যায়ে আমরা জানব ভেরিয়েবল কী, কিভাবে কাজ করে, ডাটা টাইপের ধরন, বিভিন্ন ভাষায় ব্যবহারের নিয়ম, এবং ভালো অভ্যাস।

---

## ১. ভেরিয়েবল (Variable) কী?

**ভেরিয়েবল** হলো একটি নাম, যার মাধ্যমে মেমোরিতে সংরক্ষিত কোনো মানকে নির্দেশ করা হয়।

- ভেরিয়েবল মান পরিবর্তনশীল (variable) — মান পরিবর্তন করা যায়
- ভেরিয়েবলকে একটি নির্দিষ্ট ডাটা টাইপের মান দেওয়া হয়

### উদাহরণ:
```python
x = 10
name = "Rahim"
```
এখানে `x` ও `name` দুটি ভেরিয়েবল।

---

## ২. ভেরিয়েবল ডিক্লেয়ারেশন ও ইনিশিয়ালাইজেশন

- **ডিক্লেয়ারেশন:** ভেরিয়েবলের নাম ও টাইপ ঘোষণা করা
- **ইনিশিয়ালাইজেশন:** ভেরিয়েবলে প্রাথমিক মান দেওয়া

### উদাহরণ (C):
```c
int age;        // ডিক্লেয়ারেশন
age = 25;       // ইনিশিয়ালাইজেশন
int score = 90; // ডিক্লেয়ারেশন ও ইনিশিয়ালাইজেশন একসাথে
```

### উদাহরণ (Python):
```python
x = 5           # ডিক্লেয়ারেশন ও ইনিশিয়ালাইজেশন একসাথে
```

## ৩. ডাটা টাইপ (Data Type) কী?

**ডাটা টাইপ** নির্ধারণ করে ভেরিয়েবলে কী ধরনের মান রাখা যাবে এবং সেই মানের উপর কী কী অপারেশন করা যাবে।

### প্রধান ডাটা টাইপসমূহ:
- **ইন্টিজার (Integer):** পূর্ণসংখ্যা (যেমন: 5, -10)
- **ফ্লোট (Float/Double):** দশমিক সংখ্যা (যেমন: 3.14, -0.5)
- **স্ট্রিং (String):** টেক্সট (যেমন: "hello")
- **বুলিয়ান (Boolean):** সত্য/মিথ্যা (True/False)
- **চার (Char):** এক অক্ষরের টেক্সট (C, Java ইত্যাদিতে)

## ৪. ডাটা টাইপের ধরন

### ১. প্রিমিটিভ টাইপ
- ইন্টিজার, ফ্লোট, বুলিয়ান, চার ইত্যাদি

### ২. নন-প্রিমিটিভ/কমপ্লেক্স টাইপ
- অ্যারে, লিস্ট, অবজেক্ট, স্ট্রাকচার, ক্লাস ইত্যাদি

---

## ৫. ডাটা টাইপ: বিভিন্ন ভাষায়

### C ভাষা:
```c
int a = 10;
float b = 3.5;
char c = 'A';
```

### Python:
```python
x = 10        # int
y = 3.14      # float
name = "Ali"  # str
flag = True   # bool
```

### JavaScript:
```js
let n = 5;         // Number
typeof n;          // "number"
let s = "hello";   // String
let b = false;     // Boolean
```

---

## ৬. টাইপ কনভার্সন (Type Conversion)

ভেরিয়েবলের মান এক ধরনের ডাটা টাইপ থেকে অন্যটিতে রূপান্তর করা যায়।

- **ইম্প্লিসিট (Automatic):** ভাষা নিজে করে
- **এক্সপ্লিসিট (Manual):** প্রোগ্রামার করে

### উদাহরণ (Python):
```python
x = 5
f = float(x)   # এক্সপ্লিসিট কনভার্সন
```

---

## ৭. কনস্ট্যান্ট (Constant)

**কনস্ট্যান্ট** হলো এমন একটি নাম, যার মান একবার নির্ধারণ করার পর আর পরিবর্তন করা যায় না।

### উদাহরণ (C):
```c
const float PI = 3.1416;
```

### উদাহরণ (Python):
```python
PI = 3.1416  # কনভেনশন: সব অক্ষর বড় করে লিখলে কনস্ট্যান্ট বোঝায়
```

## ৮. ভালো অভ্যাস

- অর্থবহ নাম ব্যবহার করুন (যেমন: `totalMarks`, `userName`)
- টাইপ অনুযায়ী মান দিন
- প্রয়োজন ছাড়া টাইপ পরিবর্তন করবেন না
- কনস্ট্যান্টের জন্য বড় হাতের অক্ষর ব্যবহার করুন
