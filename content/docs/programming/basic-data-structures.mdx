## অ্যারে, লিস্ট, স্ট্যাক, কিউ

ডাটা স্ট্রাকচার (Data Structure) হলো ডাটা সংরক্ষণ ও ব্যবস্থাপনার পদ্ধতি। প্রোগ্রামিংয়ে ডাটা স্ট্রাকচার জানা অত্যন্ত গুরুত্বপূর্ণ, কারণ এগুলো ডাটাকে দক্ষভাবে সাজাতে ও ব্যবহার করতে সাহায্য করে। এই অধ্যায়ে আমরা অ্যারে, লিস্ট, স্ট্যাক ও কিউ নিয়ে বিস্তারিত জানব।

---

## ১. অ্যারে (Array)

- একই ধরনের ডাটার সংগ্রহ, নির্দিষ্ট আকারের
- ইনডেক্স দিয়ে অ্যাক্সেস করা যায় (০ থেকে শুরু)

### C:
```c
int arr[5] = {1, 2, 3, 4, 5};
printf("%d", arr[2]);  // 3
```

### Python:
```python
arr = [1, 2, 3, 4, 5]
print(arr[2])  # 3
```

### বৈশিষ্ট্য:
- ফিক্সড সাইজ (C, Java)
- Python-এ লিস্ট দিয়ে অ্যারে অনুকরণ করা হয়

---

## ২. লিস্ট (List)

- ডাটার অর্ডারড সংগ্রহ, সাইজ পরিবর্তনযোগ্য
- Python-এ বিল্ট-ইন, C-তে লিঙ্কড লিস্ট (linked list)

### Python:
```python
lst = [10, 20, 30]
lst.append(40)
print(lst)  # [10, 20, 30, 40]
```

### C (Linked List):
```c
struct Node {
    int data;
    struct Node* next;
};
// লিঙ্কড লিস্ট তৈরি ও ব্যবস্থাপনা একটু জটিল
```

---

## ৩. স্ট্যাক (Stack)

- LIFO (Last In, First Out) ডাটা স্ট্রাকচার
- শেষ যেটা ঢুকেছে, সেটাই আগে বের হয়

### Python:
```python
stack = []
stack.append(1)
stack.append(2)
print(stack.pop())  # 2
```

### C:
```c
#define MAX 100
int stack[MAX];
int top = -1;

void push(int val) {
    stack[++top] = val;
}
int pop() {
    return stack[top--];
}
```

### ব্যবহার:
- ফাংশন কল স্ট্যাক
- ব্রাউজার ব্যাক বাটন

---

## ৪. কিউ (Queue)

- FIFO (First In, First Out) ডাটা স্ট্রাকচার
- যেটা আগে ঢুকেছে, সেটাই আগে বের হয়

### Python:
```python
from collections import deque
queue = deque()
queue.append(1)
queue.append(2)
print(queue.popleft())  # 1
```

### C:
```c
#define MAX 100
int queue[MAX];
int front = 0, rear = -1;

void enqueue(int val) {
    queue[++rear] = val;
}
int dequeue() {
    return queue[front++];
}
```

### ব্যবহার:
- প্রিন্ট কিউ
- প্রসেসিং টাস্ক কিউ

## ৫. ডাটা স্ট্রাকচারের তুলনা

| স্ট্রাকচার | ইনসার্ট | ডিলিট | অ্যাক্সেস | অর্ডার |
|------------|---------|--------|----------|--------|
| অ্যারে     | দ্রুত   | ধীর    | দ্রুত    | আছে   |
| লিস্ট      | দ্রুত   | দ্রুত  | ধীর      | আছে   |
| স্ট্যাক    | দ্রুত   | দ্রুত  | শুধু টপ  | আছে   |
| কিউ       | দ্রুত   | দ্রুত  | শুধু ফ্রন্ট| আছে  |
