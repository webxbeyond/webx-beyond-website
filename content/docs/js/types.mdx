জাভাস্ক্রিপ্টে টাইপ সেফটি—বাগ কমানো, readable ও maintainable কোড লেখার জন্য অত্যন্ত গুরুত্বপূর্ণ। এখানে JSDoc ও TypeScript নিয়ে বিস্তারিত আলোচনা করা হয়েছে।

---

## ১. JSDoc

JSDoc—কোডে comment দিয়ে টাইপ, parameter, return value ইত্যাদি define করা যায়।

### উদাহরণ:
```js
/**
 * দুই সংখ্যার যোগফল
 * @param {number} a
 * @param {number} b
 * @returns {number}
 */
function add(a, b) {
  return a + b;
}
```

#### অ্যানালজি:
এটা অনেকটা—একটি রেসিপি, যেখানে প্রতিটি উপকরণের নাম, পরিমাণ, ও ফলাফল লেখা থাকে।

**ব্যবহার:**
- Editor-এ type hint
- Documentation generate
- Static analysis

---

## ২. TypeScript ইন্ট্রোডাকশন

TypeScript—জাভাস্ক্রিপ্টের superset, যেখানে static type, interface, enum, class ইত্যাদি define করা যায়।

### উদাহরণ:
```ts
function greet(name: string): string {
  return `Hello, ${name}`;
}
```

#### অ্যানালজি:
এটা অনেকটা—একটি চেকলিস্ট, যেখানে প্রতিটি কাজের জন্য নির্দিষ্ট নিয়ম আছে, ভুল করলে আগে থেকেই ধরা পড়ে।

**ব্যবহার:**
- Large scale app
- Early bug detection
- Better tooling

---

## ৩. Practical Tips
- JSDoc দিয়ে function/class-এর type define করুন
- TypeScript-এ type/interface ব্যবহার করুন
- TypeScript-এ any type এড়িয়ে চলুন

- TypeScript compile time error performance boost দেয়
