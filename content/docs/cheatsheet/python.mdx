## ২. মৌলিক ধারণা

### ২.১ ভেরিয়েবল এবং ডেটা টাইপ
| টাইপ       | বর্ণনা                                      | উদাহরণ                          |
|------------|---------------------------------------------|---------------------------------|
| `int`      | পূর্ণসংখ্যা।                               | `x = 10`                       |
| `float`    | দশমিক সংখ্যা।                              | `y = 3.14`                     |
| `str`      | টেক্সট।                                    | `name = "রহিম"`                |
| `bool`     | সত্য/মিথ্যা।                               | `is_true = True`               |
| `list`     | পরিবর্তনযোগ্য তালিকা।                      | `numbers = [1, 2, 3]`          |
| `tuple`    | অপরিবর্তনীয় তালিকা।                        | `coords = (10, 20)`            |
| `dict`     | কী-ভ্যালু জোড়া।                           | `person = {"name": "করিম"}`    |
| `set`      | অনন্য আইটেমের সংগ্রহ।                      | `unique = {1, 2, 3}`           |

**উদাহরণ:**
```python
x = 5
name = "পাইথন"
items = [1, "দুই", 3]
print(x, name, items)  # 5 পাইথন [1, 'দুই', 3]
```

## ৩. কন্ট্রোল স্ট্রাকচার

### ৩.১ শর্ত (Conditionals)
```python
x = 10
if x > 0:
    print("ধনাত্মক")
elif x == 0:
    print("শূন্য")
else:
    print("ঋণাত্মক")
```

### ৩.২ লুপ (Loops)
| ধরন        | বর্ণনা                                      | উদাহরণ                          |
|------------|---------------------------------------------|---------------------------------|
| `for`      | তালিকা বা রেঞ্জে লুপ।                       | `for i in range(3): print(i)`  |
| `while`    | শর্ত সত্য হলে চলে।                         | `while x > 0: x -= 1`          |

**উদাহরণ:**
```python
for i in range(1, 4):
    print(i)  # 1, 2, 3

x = 5
while x > 0:
    print(x)
    x -= 1  # 5, 4, 3, 2, 1
```

## ৫. ডেটা স্ট্রাকচার

### ৫.১ লিস্ট (List)
| মেথড       | বর্ণনা                                      | উদাহরণ                          |
|------------|---------------------------------------------|---------------------------------|
| `append`   | শেষে যোগ।                                  | `lst.append(4)`                |
| `pop`      | শেষ থেকে সরানো।                            | `lst.pop()`                    |

## ৬. স্ট্রিং ম্যানিপুলেশন
| মেথড       | বর্ণনা                                      | উদাহরণ                          |
|------------|---------------------------------------------|---------------------------------|
| `upper`    | বড় হাতের অক্ষর।                            | `"hello".upper()`  # HELLO     |
| `lower`    | ছোট হাতের অক্ষর।                            | `"HELLO".lower()`  # hello     |
| `split`    | তালিকায় ভাগ।                               | `"a,b".split(",")`  # ['a', 'b'] |
| `join`     | তালিকা থেকে স্ট্রিং।                        | `",".join(["a", "b"])`  # a,b  |
| `strip`    | ফাঁকা জায়গা সরানো।                         | `"  hi  ".strip()`  # hi       |

**উদাহরণ:**
```python
text = "পাইথন শিখছি"
print(text.split())  # ['পাইথন', 'শিখছি']
```

## ৮. উন্নত ধারণা

### ৮.১ ক্লাস এবং অবজেক্ট
```python
class Person:
    def __init__(self, name):
        self.name = name
    
    def greet(self):
        return f"হ্যালো, আমি {self.name}"

p = Person("রহিম")
print(p.greet())  # হ্যালো, আমি রহিম
```

---

### ৮.২ ইনহেরিটেন্স (Inheritance)
```python
class Student(Person):
    def study(self):
        return f"{self.name} পড়ছে"

s = Student("করিম")
print(s.greet())  # হ্যালো, আমি করিম
print(s.study())  # করিম পড়ছে
```

## ১০. ফাংশনাল প্রোগ্রামিং
| ফাংশন      | বর্ণনা                                      | উদাহরণ                          |
|------------|---------------------------------------------|---------------------------------|
| `map`      | প্রতিটি আইটেমে ফাংশন।                       | `map(lambda x: x*2, [1, 2])`  |
| `filter`   | শর্ত মেনে আইটেম।                            | `filter(lambda x: x>0, [-1, 1])` |
| `reduce`   | একটি মানে রূপান্তর।                         | `from functools import reduce; reduce(lambda x, y: x+y, [1, 2, 3])` |

**উদাহরণ:**
```python
nums = [1, 2, 3]
doubled = list(map(lambda x: x*2, nums))
print(doubled)  # [2, 4, 6]
```

## ১২. উদাহরণ: একটি সম্পূর্ণ প্রোগ্রাম
```python
class Calculator:
    def add(self, a, b):
        return a + b

calc = Calculator()
numbers = [1, 2, 3, 4, 5]

try:
    total = sum(numbers)
    result = calc.add(total, 10)
    print(f"মোট: {result}")
except Exception as e:
    print(f"এরর: {e}")
finally:
    print("কাজ শেষ!")
# মোট: 25
# কাজ শেষ!
```
