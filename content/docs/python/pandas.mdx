ডেটা অ্যানালাইসিস, ডেটা সায়েন্স, রিপোর্টিং—সবখানে পাইথনের Pandas লাইব্রেরি অপরিহার্য। Pandas দিয়ে ডেটা সহজে পড়া, প্রসেসিং, ক্লিনিং, ও বিশ্লেষণ করা যায়।

ভাবুন, আপনি একটি স্কুলের শিক্ষক। আপনার কাছে শত শত ছাত্রের নম্বর, নাম, ঠিকানা—সব ডেটা আছে। Excel-এ যেমন টেবিল থাকে, Pandas-এ ঠিক তেমন DataFrame থাকে, যেখানে সহজে ডেটা সাজানো, খোঁজা, পরিবর্তন করা যায়।

## Pandas কী?

Pandas হচ্ছে পাইথনের ওপেন-সোর্স লাইব্রেরি, যা ট্যাবুলার ডেটা (টেবিলের মতো) প্রসেসিংয়ের জন্য ব্যবহৃত হয়। এতে রয়েছে DataFrame, Series, ডেটা ক্লিনিং, ফিল্টারিং, গ্রুপিং, ও বিশ্লেষণের জন্য প্রচুর টুলস।

### ইনস্টলেশন

```bash
pip install pandas
```

---

## Series: এক কলামের ডেটা

Series হচ্ছে Pandas-এর এক ডাইমেনশনাল ডেটা স্ট্রাকচার। এটি মূলত একটি কলামের মতো।

```python
import pandas as pd

marks = pd.Series([80, 90, 75, 60])
print(marks)
```

### বাস্তব উদাহরণ:

ছাত্রদের নম্বরের একটি লিস্ট—এটি Series হিসেবে রাখা যায়।

---

## DataFrame: টেবিল ডেটা

DataFrame হচ্ছে Pandas-এর সবচেয়ে গুরুত্বপূর্ণ স্ট্রাকচার। এটি একাধিক কলাম ও সারি নিয়ে তৈরি হয়—একটি টেবিলের মতো।

```python
data = {
    'নাম': ['রাহুল', 'সুমন', 'রিমা', 'তানিয়া'],
    'নম্বর': [80, 45, 65, 30],
    'শ্রেণি': ['১০', '৯', '১০', '৮']
}
df = pd.DataFrame(data)
print(df)
```

### বাস্তব analogy:

DataFrame-কে ভাবুন Excel-এর শিটের মতো—যেখানে সারি ও কলাম আছে।

---

## ডেটা পড়া ও লেখা

Pandas দিয়ে CSV, Excel, JSON, SQL—সব ধরনের ডেটা পড়া ও লেখা যায়।

```python
df = pd.read_csv('students.csv')  # CSV ফাইল থেকে পড়া
df.to_csv('output.csv', index=False)  # CSV ফাইলে লেখা
```

---

## ডেটা অ্যাক্সেস ও ফিল্টারিং

DataFrame থেকে নির্দিষ্ট কলাম, সারি, বা কন্ডিশন অনুযায়ী ডেটা বের করা যায়।

```python
print(df['নাম'])  # শুধু নাম কলাম
print(df.loc[0])  # প্রথম সারি
print(df[df['নম্বর'] > 50])  # যাদের নম্বর ৫০-এর বেশি
```

---

## ডেটা ক্লিনিং

বাস্তব জীবনে ডেটা সবসময় পরিষ্কার থাকে না। Pandas দিয়ে Missing Value, Duplicate, Outlier, ডেটা টাইপ—সব সহজে ঠিক করা যায়।

### Missing Value

```python
df['নম্বর'] = df['নম্বর'].fillna(0)  # ফাঁকা জায়গায় ০ বসানো
```

### Duplicate

```python
df = df.drop_duplicates()  # ডুপ্লিকেট সারি বাদ
```

### ডেটা টাইপ পরিবর্তন

```python
df['নম্বর'] = df['নম্বর'].astype(int)
```

---

## ডেটা গ্রুপিং ও সামারি

Pandas দিয়ে ডেটা গ্রুপ করে গড়, যোগফল, সর্বোচ্চ, ইত্যাদি বের করা যায়।

```python
grouped = df.groupby('শ্রেণি')['নম্বর'].mean()
print(grouped)
```

---

## ডেটা ভিজুয়ালাইজেশন

Pandas-এর সাথে Matplotlib ব্যবহার করে সহজেই গ্রাফ আঁকা যায়।

```python
import matplotlib.pyplot as plt

df['নম্বর'].plot(kind='bar')
plt.show()
```

## বাস্তব জীবনের analogy

Pandas-কে ভাবুন, ডেটা-সংগ্রহের সুপার-পাওয়ার। যেমন, Excel-এ হাজার হাজার ডেটা ফিল্টার, ক্লিন, বিশ্লেষণ করা যায়—Pandas-এ আরও দ্রুত ও কোডে করা যায়।
