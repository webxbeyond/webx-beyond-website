লিস্ট হচ্ছে পাইথনের সবচেয়ে বহুল ব্যবহৃত ডাটা স্ট্রাকচার। এটি অনেকটা স্কুলের রোল কলের মতো—একটি তালিকায় একাধিক আইটেম রাখা যায়, এবং প্রয়োজনমতো আইটেম যোগ/বিয়োগ/পরিবর্তন করা যায়।

## লিস্ট তৈরি

লিস্ট তৈরি করতে স্কয়ার ব্র্যাকেট `[ ]` ব্যবহার হয়।
```python
fruits = ["apple", "banana", "mango"]
numbers = [1, 2, 3, 4, 5]
empty_list = []
```

## লিস্টে আইটেম অ্যাক্সেস

লিস্টের আইটেম অ্যাক্সেস করতে ইনডেক্স ব্যবহার হয় (০ থেকে শুরু)।
```python
print(fruits[0])  # আউটপুট: apple
print(fruits[2])  # আউটপুট: mango
```

## লিস্টে আইটেম পরিবর্তন
```python
fruits[1] = "orange"
print(fruits)  # ['apple', 'orange', 'mango']
```

## লিস্টে আইটেম যোগ/বিয়োগ
```python
fruits.append("grape")      # শেষে যোগ
fruits.insert(1, "lemon")   # নির্দিষ্ট স্থানে যোগ
print(fruits)
fruits.remove("apple")      # নির্দিষ্ট আইটেম বাদ
print(fruits)
```

## লিস্ট স্লাইসিং

লিস্টের নির্দিষ্ট অংশ বের করতে স্লাইসিং ব্যবহার হয়।
```python
numbers = [10, 20, 30, 40, 50, 60]
print(numbers[1:4])    # [20, 30, 40]
print(numbers[:3])     # [10, 20, 30]
print(numbers[3:])     # [40, 50, 60]
print(numbers[-2:])    # [50, 60]
```

## লিস্ট কম্প্রিহেনশন

লিস্ট কম্প্রিহেনশন হচ্ছে সংক্ষিপ্তভাবে নতুন লিস্ট তৈরি করার উপায়।
```python
squares = [x**2 for x in range(1, 6)]
print(squares)  # [1, 4, 9, 16, 25]

# শুধু জোড় সংখ্যা
evens = [x for x in range(1, 11) if x % 2 == 0]
print(evens)  # [2, 4, 6, 8, 10]
```

## বাস্তব উদাহরণ

### ১. ইউজার থেকে নাম নিয়ে লিস্ট তৈরি
```python
names = []
for i in range(3):
    name = input("নাম দিন: ")
    names.append(name)
print(names)
```

### ২. লিস্ট থেকে সর্বোচ্চ/সর্বনিম্ন বের করা
```python
scores = [88, 92, 79, 95]
print(max(scores))  # 95
print(min(scores))  # 79
```

### ৩. লিস্ট কম্প্রিহেনশন দিয়ে ফিল্টার
```python
marks = [55, 67, 80, 45, 90]
passed = [m for m in marks if m >= 60]
print(passed)  # [67, 80, 90]
```

## ভুল ও সমাধান

- ভুল: ইনডেক্স ভুল হলে
  ```python
  print(fruits[5])  # IndexError
  ```
- ভুল: remove() দিয়ে না থাকা আইটেম বাদ দিতে চাইলে
  ```python
  fruits.remove("watermelon")  # ValueError
  ```

## অনুশীলন

১. ৫টি সংখ্যা নিয়ে একটি লিস্ট তৈরি করুন এবং যোগফল বের করুন।
২. একটি লিস্ট থেকে শুধু বেজোড় সংখ্যা বের করুন (লিস্ট কম্প্রিহেনশন)।
৩. লিস্টের প্রথম ৩টি ও শেষ ২টি আইটেম প্রিন্ট করুন (স্লাইসিং)।

## উপসংহার

লিস্ট দিয়ে পাইথনে ডাটা সংগ্রহ, প্রসেসিং, এবং ফিল্টারিং সহজ হয়। লিস্ট কম্প্রিহেনশন কোডকে সংক্ষিপ্ত ও শক্তিশালী করে। পরবর্তী পাঠে টুপল, সেট ও ডিকশনারি নিয়ে আলোচনা করব।
