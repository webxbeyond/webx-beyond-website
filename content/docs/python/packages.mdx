পাইথনে বড় প্রজেক্টে কোডকে সংগঠিত ও পুনঃব্যবহারযোগ্য রাখতে কাস্টম মডিউল ও প্যাকেজ তৈরি করা অত্যন্ত গুরুত্বপূর্ণ। এই পাঠে আমরা জানবো:
- কাস্টম মডিউল কী, কেন দরকার
- কিভাবে নিজের মডিউল ও প্যাকেজ তৈরি করবেন
- রিয়েল-লাইফ উদাহরণ ও analogies
- Best practices

## ১. কাস্টম মডিউল কী?

মডিউল হলো পাইথন ফাইল (যেমন: `calculator.py`) যাতে ফাংশন, ক্লাস, ভ্যারিয়েবল থাকে। আপনি নিজের প্রয়োজন অনুযায়ী মডিউল লিখে অন্য ফাইলে ইম্পোর্ট করতে পারেন।

### উদাহরণ:
`calculator.py`
```python
def add(a, b):
    return a + b

def subtract(a, b):
    return a - b
```

`main.py`
```python
import calculator
print(calculator.add(5, 3))  # 8
```

## ২. কাস্টম প্যাকেজ কী?

প্যাকেজ হলো ফোল্ডার, যাতে একাধিক মডিউল থাকে এবং একটি `__init__.py` ফাইল থাকে (পাইথন 3.3+ এ ঐচ্ছিক)।

### উদাহরণ:
```
math_tools/
    __init__.py
    basic.py
    advanced.py
```

`basic.py`
```python
def add(a, b):
    return a + b
```

`advanced.py`
```python
def power(a, b):
    return a ** b
```

`main.py`
```python
from math_tools import basic, advanced
print(basic.add(2, 3))      # 5
print(advanced.power(2, 4)) # 16
```

## ৩. প্যাকেজে `__init__.py` এর ভূমিকা

`__init__.py` ফাইল প্যাকেজকে পাইথনকে জানায়—এটি একটি প্যাকেজ। এখানে চাইলে ডিফল্ট ইম্পোর্ট, কনফিগারেশন, বা ইনিশিয়ালাইজেশন কোড রাখতে পারেন।

## ৪. রিয়েল-লাইফ উদাহরণ: স্কুল ম্যানেজমেন্ট সফটওয়্যার

ধরা যাক, আপনি একটি স্কুল সফটওয়্যার বানাচ্ছেন।
```
school/
    __init__.py
    students.py
    teachers.py
    exams.py
```

`students.py`
```python
def get_student(id):
    # ডাটাবেস থেকে ছাত্রের তথ্য ফেরত দেয়
    pass
```

`main.py`
```python
from school import students
students.get_student(101)
```

## ৫. Analogies: ফাইল ক্যাবিনেট

মডিউল ও প্যাকেজকে ভাবুন ফাইল ক্যাবিনেটের মতো। প্রতিটি ড্রয়ার (প্যাকেজ)–এর ভিতরে ফাইল (মডিউল) থাকে। প্রয়োজন অনুযায়ী নির্দিষ্ট ফাইল বের করে কাজ করেন। এতে কাজ সহজ ও সংগঠিত হয়।

## ৬. Best Practices

- মডিউল ও প্যাকেজের নাম অর্থবহ ও ছোট রাখুন
- এক ফাইলে এক/দুইটি রিলেটেড ফাংশন/ক্লাস রাখুন
- বড় প্রজেক্টে প্যাকেজ ব্যবহার করুন
- `__init__.py` ফাইল ব্যবহার করুন
- ডকুমেন্টেশন লিখুন

## ৭. সংক্ষেপে

- কাস্টম মডিউল ও প্যাকেজ কোডকে সংগঠিত ও পুনঃব্যবহারযোগ্য করে
- বড় প্রজেক্টে প্যাকেজ ব্যবহার করা ভালো অভ্যাস
- Analogies ও উদাহরণ দিয়ে সহজে বুঝতে পারবেন

এভাবেই পাইথনে কাস্টম মডিউল ও প্যাকেজ তৈরি করে আপনার কোডকে আরও শক্তিশালী, পরিষ্কার ও মেইনটেনেবল করতে পারেন।
