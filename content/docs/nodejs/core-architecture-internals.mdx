Node.js-এর কোর আর্কিটেকচার হচ্ছে ইভেন্ট-ড্রিভেন, নন-ব্লকিং I/O, এবং V8 ইঞ্জিনের উপর ভিত্তি করে তৈরি। এর ইন্টারনালস বুঝলে পারফরম্যান্স, স্কেল, এবং ডিপ বাগ ফিক্সিং সহজ হয়।

## Node.js কোর আর্কিটেকচার

### ১. V8 JavaScript Engine
- Google Chrome-এর V8 ইঞ্জিন Node.js-এ ব্যবহৃত হয়
- JavaScript কে দ্রুত মেশিন কোডে রূপান্তর করে

### ২. libuv
- C-তে লেখা লাইব্রেরি
- ইভেন্ট-ড্রিভেন, নন-ব্লকিং I/O, File System, TCP/UDP, DNS ইত্যাদি হ্যান্ডল করে

### ৩. Event Loop
- অ্যাসিনক্রোনাস টাস্ক, Callback, Timer, I/O ইভেন্ট প্রসেস করে
- Node.js-এর "হৃদপিণ্ড"

### ৪. C++ Bindings & Native APIs
- Node.js-এর কোর ফিচার (File System, Network) C++ দিয়ে লেখা
- JavaScript API-এর মাধ্যমে এক্সেস করা যায়

### ৫. Internal Modules
- fs, http, net, stream, crypto, path ইত্যাদি
- Efficiently written in C++/JavaScript

## Execution Flow

1. User request (HTTP, TCP, etc.)
2. Event Loop request গ্রহণ করে
3. I/O হলে libuv অ্যাসিনক্রোনাসভাবে প্রসেস করে
4. Callback/Event Trigger হয়
5. Response user-কে পাঠানো হয়

## Real Life Analogy

Node.js হচ্ছে "বহুমুখী অফিস"—V8 হচ্ছে "এক্সিকিউটিভ", libuv হচ্ছে "অপারেশন টিম", Event Loop হচ্ছে "ম্যানেজার", Internal Modules হচ্ছে "ডিপার্টমেন্ট"।

## Internal APIs

- **process:** প্রসেস info, env, args
- **buffer:** Binary data
- **stream:** Data flow
- **cluster:** Multi-process
- **child_process:** External command

## Performance Tips

- **Avoid Blocking Main Thread:** Heavy computation worker thread/child process-এ দিন
- **Use Streams for Large Data**
- **Use Buffer for Binary Data**
- **Profile with --inspect, DevTools**

## Practical Example: Custom Native Addon

```cpp
// addon.cc
#include <napi.h>
Napi::String Hello(const Napi::CallbackInfo& info) {
  return Napi::String::New(info.Env(), "Hello from C++!");
}
Napi::Object Init(Napi::Env env, Napi::Object exports) {
  exports.Set("hello", Napi::Function::New(env, Hello));
  return exports;
}
NODE_API_MODULE(addon, Init)
```

```js
// index.js
const addon = require('./build/Release/addon');
console.log(addon.hello());
```

## Best Practices

- **Use Internal Modules for efficiency**
- **Profile regularly**
- **Avoid memory leaks**

## উপসংহার

Node.js-এর কোর আর্কিটেকচার—V8, libuv, Event Loop, Internal Modules—সবকিছু মিলিয়ে এটি দ্রুত, স্কেলেবল ও API-friendly। ইন্টারনালস বুঝলে efficient, scalable অ্যাপ তৈরি করা যায়। পরবর্তী পাঠে libuv ও C++ Bindings নিয়ে আলোচনা হবে।
