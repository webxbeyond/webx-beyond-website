## WebAssembly (WASM) কী?
- Browser-এ শুরু, এখন server-side
- Binary format, near-native speed
- C/C++/Rust কোড run করা যায়

### Analogy
"যেন আপনি Node.js-এ সুপারফাস্ট বাইরের language-এর engine লাগিয়ে দিলেন—একটা গাড়িতে জেট ইঞ্জিন!"

## Node.js-এ WASM কেন?
- Performance-critical task (image/video processing, encryption)
- Legacy code reuse (C/C++/Rust)
- Cross-language ecosystem

## Example: Node.js-এ WASM ব্যবহার
```js
const fs = require('fs');
const wasmBuffer = fs.readFileSync('./add.wasm');
WebAssembly.instantiate(wasmBuffer).then(wasmModule => {
  const { add } = wasmModule.instance.exports;
  console.log(add(5, 7)); // 12
});
```
