## কাগজ, মেটাল সিড প্লেট, অফলাইন স্টোরেজ

ক্রিপ্টো ওয়ালেটের সিড ফ্রেইজ, প্রাইভেট কী, এবং গুরুত্বপূর্ণ তথ্য নিরাপদে সংরক্ষণ করা—নিরাপত্তার মূল স্তম্ভ। এই পাঠে জানবো—কেন ব্যাকআপ দরকার, কোন কোন ব্যাকআপ মেথড সবচেয়ে নিরাপদ, এবং বাস্তব উদাহরণ ও অ্যানালজি দিয়ে বিষয়টি পরিষ্কার করবো।

## কেন ব্যাকআপ দরকার?
- **ওয়ালেট হারালে/ডিভাইস নষ্ট হলে:** সিড ফ্রেইজ ছাড়া ফান্ড পুনরুদ্ধার সম্ভব নয়।
- **হ্যাক/স্ক্যাম হলে:** নতুন ডিভাইসে ফান্ড restore করতে হবে।
- **পারিবারিক/জরুরি পরিস্থিতি:** ফান্ড/অ্যাক্সেস পরিবারের কেউ পেতে পারে (বিশ্বাসযোগ্য হলে)।

## ব্যাকআপ মেথডস
### ১. কাগজে লিখে রাখা
- সিড ফ্রেইজ/প্রাইভেট কী পরিষ্কারভাবে কাগজে লিখুন।
- জল/আগুন/পোকা থেকে নিরাপদ স্থানে রাখুন (লকার, সেফ)।
- একাধিক কপি রাখুন—ভিন্ন স্থানে।

### ২. মেটাল সিড প্লেট
- সিড ফ্রেইজ/কী মেটাল প্লেটে খোদাই করুন (Steel, Titanium)।
- আগুন, পানি, পোকা, সময়—সবকিছু থেকে নিরাপদ।
- বাজারে Cryptosteel, Billfodl, Cobo Tablet ইত্যাদি পাওয়া যায়।

### ৩. অফলাইন স্টোরেজ
- USB Drive, External HDD—কিন্তু কখনোই ইন্টারনেট কানেক্ট করবেন না।
- এনক্রিপ্টেড ফাইল/ডকুমেন্টে সংরক্ষণ করুন।
- TrueCrypt, VeraCrypt, BitLocker ইত্যাদি ব্যবহার করুন।

## বাস্তব উদাহরণ ও অ্যানালজি
- **পাসপোর্ট/জন্ম সনদের কপি:** যেমন গুরুত্বপূর্ণ ডকুমেন্টের কপি আলাদা স্থানে রাখেন, তেমনি সিড ফ্রেইজের ব্যাকআপ রাখুন।
- **লকারের চাবি:** চাবি হারালে লকার খুলতে পারবেন না—তেমনি সিড ফ্রেইজ ছাড়া ফান্ড পাবেন না।

## সাধারণ ভুল
- **ডিজিটাল কপি/স্ক্রিনশট/ক্লাউডে রাখা:** হ্যাকিং/ফিশিংয়ের ঝুঁকি বাড়ে।
- **একই স্থানে একাধিক কপি রাখা:** আগুন/জল/চুরি হলে সব হারাতে পারেন।
- **কাউকে শেয়ার করা:** বিশ্বাসযোগ্য না হলে শেয়ার করবেন না।

## নিরাপত্তা টিপস
- **ব্যাকআপ কপি নিয়মিত চেক করুন—পড়া যায় কিনা।**
- **কাগজ/মেটাল প্লেটের অবস্থান পরিবারের বিশ্বাসযোগ্য সদস্যকে জানান।**
- **ডিজিটাল ব্যাকআপে শক্তিশালী এনক্রিপশন ব্যবহার করুন।**

## Python Demo: এনক্রিপ্টেড ব্যাকআপ
```python
from cryptography.fernet import Fernet

key = Fernet.generate_key()
cipher = Fernet(key)
seed_phrase = b"cat mango river moon tiger apple cloud drum fish lamp tree sun"
encrypted = cipher.encrypt(seed_phrase)
print("Encrypted backup:", encrypted)
# key নিরাপদ স্থানে রাখুন
```
