ক্রিপ্টো ওয়ালেটের তিনটি মূল উপাদান—পাবলিক অ্যাড্রেস, প্রাইভেট কী, এবং সিড ফ্রেইজ। এগুলো বুঝলে ওয়ালেটের নিরাপত্তা ও ব্যবহার সহজ হবে।

## পাবলিক অ্যাড্রেস

- **সবার জন্য উন্মুক্ত:** এটি আপনার ক্রিপ্টোকারেন্সি গ্রহণের ঠিকানা।
- **ব্যবহার:** কেউ আপনাকে ফান্ড পাঠাতে চাইলে এই অ্যাড্রেস ব্যবহার করবে।
- **উদাহরণ:** Ethereum: `0x1234abcd...`, Bitcoin: `bc1qxyz...`

## প্রাইভেট কী

- **গোপন চাবি:** এটি দিয়ে আপনার ফান্ড নিয়ন্ত্রণ হয়।
- **ব্যবহার:** ফান্ড পাঠানো, dApp-এ সাইনিং, ট্রানজ্যাকশন অথেন্টিকেট করা।
- **নিরাপত্তা:** কখনো শেয়ার করবেন না, হারালে ফান্ড চিরতরে হারাতে পারেন।

## সিড ফ্রেইজ (Seed Phrase)

- **Backup/Recovery:** ১২/২৪ শব্দের একটি বাক্য, যা দিয়ে আপনার ওয়ালেট পুনরুদ্ধার করা যায়।
- **ব্যবহার:** নতুন ডিভাইসে ওয়ালেট রিকভার, হার্ডওয়্যার ওয়ালেট সেটআপ।
- **নিরাপত্তা:** কাগজে/মেটাল প্লেটে লিখে অফলাইনে রাখুন।

## বাস্তব উদাহরণ ও Analogies

- **পাবলিক অ্যাড্রেস = ব্যাংক অ্যাকাউন্ট নম্বর:** সবাইকে দিতে পারেন, কেউ টাকা পাঠাতে চাইলে ব্যবহার করবে।
- **প্রাইভেট কী = ব্যাংকের চেক বই/পিন:** শুধু আপনার কাছে, চুরি হলে ফান্ড হারাতে পারেন।
- **সিড ফ্রেইজ = মাস্টার চাবি:** হারালে সবকিছু পুনরুদ্ধার করা যায়, চুরি হলে সব হারাতে পারেন।

## নিরাপত্তা টিপস

- **প্রাইভেট কী/সিড ফ্রেইজ কখনো অনলাইনে রাখবেন না।**
- **কাগজ/মেটাল প্লেটে ব্যাকআপ রাখুন।**
- **ফিশিং/স্ক্যাম থেকে সতর্ক থাকুন।**

## Python-অনুরূপ কোড: সিড ফ্রেইজ থেকে অ্যাড্রেস জেনারেশন (ডেমো)

```python
import hashlib

def seed_to_address(seed):
    pk = hashlib.sha256(seed.encode()).hexdigest()
    address = '0x' + pk[:40]
    return address

print(seed_to_address('correct horse battery staple'))
```

## ভবিষ্যৎ

ওয়ালেটের নিরাপত্তা, মালিকানা, ও রিকভারি—সবকিছু নির্ভর করে এই তিনটি উপাদানের ওপর। এগুলো বুঝে, নিরাপদে সংরক্ষণ করুন।
