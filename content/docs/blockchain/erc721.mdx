# ইউনিক ডিজিটাল অ্যাসেটের স্ট্যান্ডার্ড

ERC-721 হচ্ছে Ethereum ব্লকচেইনে Non-Fungible Token (NFT) তৈরির জন্য স্ট্যান্ডার্ড। এটি এমন ডিজিটাল অ্যাসেট তৈরি করতে দেয়, যা একে অপরের থেকে সম্পূর্ণ আলাদা ও ইউনিক।

## ERC-721 NFT কী?

- **Non-Fungible:** প্রতিটি NFT ইউনিক, একে অপরের সাথে বিনিময়যোগ্য নয়।
- **স্ট্যান্ডার্ড:** নির্দিষ্ট ফাংশন ও ইভেন্টের মাধ্যমে NFT তৈরি, ট্রান্সফার, মালিকানা চেক ইত্যাদি করা যায়।
- **ইউনিক অ্যাসেট:** আর্ট, মিউজিক, গেম আইটেম, টিকিট, সার্টিফিকেট ইত্যাদি NFT হিসেবে তৈরি করা যায়।

## বাস্তব উদাহরণ ও এনালজি

- **আর্ট গ্যালারি:** প্রতিটি চিত্রকর্ম ইউনিক, একে অপরের সাথে বিনিময়যোগ্য নয়।
- **গেমের ইউনিক আইটেম:** যেমন "Legendary Sword"—শুধু একজনের কাছে থাকতে পারে।
- **CryptoPunks, Bored Ape Yacht Club:** জনপ্রিয় NFT কালেকশন, প্রতিটি NFT আলাদা।

## ERC-721 টোকেনের প্রধান ফাংশন

- `ownerOf(tokenId)`: নির্দিষ্ট NFT-এর মালিক কে
- `balanceOf(address)`: অ্যাড্রেসে কত NFT আছে
- `transferFrom(from, to, tokenId)`: NFT ট্রান্সফার
- `approve(address, tokenId)`: অন্য অ্যাড্রেসকে NFT ব্যবহারের অনুমতি
- `getApproved(tokenId)`: কোন অ্যাড্রেস অনুমোদিত
- `safeTransferFrom(from, to, tokenId)`: নিরাপদ ট্রান্সফার

## ERC-721 NFT-এর সুবিধা

- **ইউনিকনেস:** প্রতিটি অ্যাসেট আলাদা, মালিকানা ট্র্যাক করা যায়
- **ডিজিটাল প্রমাণ:** আর্ট, সার্টিফিকেট, টিকিট ইত্যাদির ডিজিটাল মালিকানা
- **ইন্টারঅপারেবিলিটি:** একাধিক মার্কেটপ্লেস ও অ্যাপ সাপোর্ট করে

## চ্যালেঞ্জ ও সীমাবদ্ধতা

- **ফি:** Ethereum নেটওয়ার্কে NFT ট্রান্সফার/মিন্টিং ফি বেশি হতে পারে
- **স্টোরেজ:** বড় ফাইল (আর্ট, ভিডিও) ব্লকচেইনে রাখা যায় না, IPFS/Arweave ব্যবহার করতে হয়
- **ভুল কোডিং:** ভুল ইমপ্লিমেন্টেশন হলে NFT হারাতে পারেন

## Solidity কোড ডেমো: ERC-721 NFT

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

import "@openzeppelin/contracts/token/ERC721/ERC721.sol";

contract MyNFT is ERC721 {
    uint256 public nextTokenId;
    address public admin;

    constructor() ERC721("MyNFT", "MNFT") {
        admin = msg.sender;
    }

    function mint(address to) external {
        require(msg.sender == admin, "only admin");
        _safeMint(to, nextTokenId);
        nextTokenId++;
    }
}
```

## পারফরম্যান্স ও সিকিউরিটি টিপস

- OpenZeppelin লাইব্রেরি ব্যবহার করুন—স্ট্যান্ডার্ড ও নিরাপদ ইমপ্লিমেন্টেশন
- বড় ফাইলের জন্য IPFS/Arweave ব্যবহার করুন
- NFT মিন্ট/ট্রান্সফার করার আগে ভালোভাবে পরীক্ষা করুন
