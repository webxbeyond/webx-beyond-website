গিটে সাইনড কমিট ও ট্যাগ যাচাই করা নিরাপত্তা ও অথেন্টিসিটি নিশ্চিত করতে গুরুত্বপূর্ণ। এই পাঠে দেখানো হয়েছে কীভাবে সাইনড কমিট/ট্যাগ যাচাই করবেন এবং কোন ইন্ডিকেটর দেখে বুঝবেন কমিটটি সত্যিই সাইনড ও ভেরিফায়েড।

## ১. সাইনড কমিট যাচাই (লোকাল)

- সাইনড কমিট দেখতে:

```sh
git log --show-signature
```

- আউটপুটে দেখাবে:
  - `Good signature from ...` → সাইনড ও ভেরিফায়েড
  - `BAD signature from ...` → সাইনড, কিন্তু ভেরিফায়েড নয়
  - `No signature` → সাইনড নয়

- নির্দিষ্ট কমিট যাচাই:

```sh
git show --show-signature <commit-hash>
```

## ২. সাইনড ট্যাগ যাচাই

```sh
git tag -v <tagname>
```

- আউটপুটে সিগনেচার ও অথেন্টিসিটি দেখাবে।

## ৩. GitHub/GitLab-এ সাইনড কমিট যাচাই

- GitHub-এ সাইনড কমিট/ট্যাগের পাশে "Verified" ব্যাজ দেখাবে।
- কমিট ডিটেইলে ক্লিক করলে সিগনেচার ও কী ইনফো পাওয়া যায়।
- GitLab-এও একইভাবে "Verified" বা "Unverified" দেখায়।

## ৪. সাধারণ সমস্যা ও সমাধান

- **BAD signature:**
  - GPG/SSH কী GitHub/GitLab-এ অ্যাড করা আছে কিনা চেক করুন।
  - লোকাল GPG কী ট্রাস্টেড কিনা চেক করুন: `gpg --edit-key <key-id>`, তারপর `trust` কমান্ড।
- **No signature:**
  - কমিট/ট্যাগ সাইনড হয়নি।

## ৫. বেস্ট প্র্যাকটিস
- সব গুরুত্বপূর্ণ কমিট ও ট্যাগ সাইন করুন।
- সাইনড কমিট/ট্যাগ রিভিউ করার সময় যাচাই করুন।
- কী নিরাপদে রাখুন ও GitHub/GitLab-এ আপডেট রাখুন।

---

**সংক্ষেপে:**
- `git log --show-signature` ও `git tag -v` দিয়ে সাইনড কমিট/ট্যাগ যাচাই করুন।
- GitHub/GitLab-এ "Verified" ব্যাজ দেখে নিশ্চিত হন।
