গিট রিপোজিটরি কখনো কখনো দুর্ভাগ্যবশত "ব্রোকেন" বা ক্ষতিগ্রস্ত হতে পারে—যেমন `.git` ডিরেক্টরি নষ্ট হওয়া, কমিট/ব্রাঞ্চ হারিয়ে যাওয়া, বা ফাইল সিস্টেম সমস্যা। এই পাঠে আমরা ব্রোকেন রিপোজিটরি চিহ্নিত ও মেরামতের কৌশল শিখব।

## ১. সমস্যা চিহ্নিতকরণ

- **গিট কমান্ডে অদ্ভুত এরর:** যেমন `fatal: bad object`, `corrupt loose object`, `unresolved merge` ইত্যাদি।
- **.git ডিরেক্টরি মিসিং/করাপ্টেড:** `.git` ফোল্ডার ডিলিট বা ক্ষতিগ্রস্ত হলে গিট কাজ করবে না।
- **ব্রাঞ্চ/কমিট হারিয়ে যাওয়া:** হঠাৎ করে ইতিহাস বা ব্রাঞ্চ দেখা যাচ্ছে না।

## ২. ডায়াগনোসিস টুল

- `git fsck` : রিপোতে কোনো সমস্যা আছে কিনা চেক করে।

```sh
git fsck
```

- `git status` : রিপোর বর্তমান অবস্থা দেখায়।
- `git reflog` : হারানো কমিট বা ব্রাঞ্চ খুঁজে পেতে সাহায্য করে।

## ৩. সাধারণ ফিক্স

### (ক) `.git` ডিরেক্টরি মিসিং

- ব্যাকআপ থেকে `.git` ফোল্ডার রিস্টোর করুন (যদি থাকে)।
- না থাকলে, নতুন রিপো ইনিশিয়ালাইজ করে ফাইলগুলো নতুনভাবে অ্যাড ও কমিট করতে হবে।

### (খ) করাপ্টেড অবজেক্ট ফিক্স

- `git fsck` চালিয়ে কোন অবজেক্ট নষ্ট হয়েছে দেখুন।
- অন্য কোনো ক্লিন রিপো থেকে সেই অবজেক্ট কপি করে `.git/objects/`-এ পেস্ট করুন।

### (গ) হারানো ব্রাঞ্চ/কমিট উদ্ধার

- `git reflog` চালিয়ে পুরনো HEAD/ব্রাঞ্চ লোকেট করুন:

```sh
git reflog
```

- হারানো কমিটে যেতে:

```sh
git checkout <commit-hash>
```

- নতুন ব্রাঞ্চ তৈরি করুন:

```sh
git checkout -b recovered-branch <commit-hash>
```

### (ঘ) মার্জ/রিবেস কনফ্লিক্টে আটকে গেলে

- কনফ্লিক্ট ফাইল ঠিক করে `git add` দিন, তারপর `git merge --continue` বা `git rebase --continue`।
- প্রয়োজনে `git merge --abort` বা `git rebase --abort` দিয়ে আগের অবস্থায় ফিরে যান।

## ৪. রিপো ক্লিনআপ

- অপ্রয়োজনীয় অবজেক্ট ও রেফ ক্লিন করতে:

```sh
git gc --prune=now
```

- রিফলগ ক্লিন করতে:

```sh
git reflog expire --expire=now --all
git gc --prune=now --aggressive
```

## ৫. ভবিষ্যতে সমস্যা এড়াতে

- নিয়মিত ব্যাকআপ নিন (রিমোটে পুশ করুন)।
- `.git` ফোল্ডার কখনো ডিলিট/এডিট করবেন না।
- বড় পরিবর্তনের আগে ব্রাঞ্চ ও ট্যাগ ব্যবহার করুন।

---

**সংক্ষেপে:**
- `git fsck`, `git reflog`, ব্যাকআপ, ও ক্লিন রিপো থেকে অবজেক্ট কপি—এসব কৌশল দিয়ে বেশিরভাগ ব্রোকেন রিপোজিটরি ফিক্স করা যায়।
- সমস্যা হলে আতঙ্কিত না হয়ে ধাপে ধাপে ডায়াগনোসিস ও ফিক্স করুন।
