## অটোমেশনের জন্য গিট হুক পরিচিতি

গিট হুক (Git Hooks) হল গিটের বিল্ট-ইন ফিচার, যা নির্দিষ্ট ইভেন্টে (যেমন commit, push, merge) স্বয়ংক্রিয়ভাবে স্ক্রিপ্ট চালাতে দেয়। এতে কোড কোয়ালিটি, টেস্ট, লিন্ট, ডিপ্লয় ইত্যাদি অটোমেট করা যায়।

### গিট হুক কীভাবে কাজ করে?
- `.git/hooks/` ডিরেক্টরিতে বিভিন্ন হুক স্ক্রিপ্ট থাকে (যেমন `pre-commit`, `pre-push`)
- নির্দিষ্ট ইভেন্টে সংশ্লিষ্ট স্ক্রিপ্ট অটো-রান হয়

### জনপ্রিয় গিট হুক
- **pre-commit**: কমিটের আগে চলে (লিন্ট, টেস্ট, ফরম্যাট)
- **commit-msg**: কমিট মেসেজ চেক
- **pre-push**: পুশের আগে টেস্ট/বিল্ড
- **post-merge**: মার্জের পর টাস্ক

### হুক সক্রিয় করার নিয়ম
1. `.git/hooks/`-এ স্ক্রিপ্ট তৈরি করুন (যেমন `pre-commit`)
2. স্ক্রিপ্ট এক্সিকিউটেবল করুন (Linux/macOS: `chmod +x pre-commit`)

#### উদাহরণ: pre-commit
```bash
#!/bin/sh
npm run lint
if [ $? -ne 0 ]; then
  echo "লিন্টিং ফেইল! কমিট বাতিল।"
  exit 1
fi
```

### টুল
- [Husky](https://typicode.github.io/husky) দিয়ে হুক সহজে শেয়ার ও ম্যানেজ
- [lefthook](https://evilmartians.com/chronicles/lefthook) মাল্টিপ্ল্যাটফর্ম হুক টুল

### উপকারিতা
- কোড কোয়ালিটি ও সিকিউরিটি নিশ্চিত
- টিমে কনসিসটেন্ট স্ট্যান্ডার্ড
- অটোমেটেড টেস্ট, বিল্ড, ফরম্যাট

### উপসংহার
গিট হুক দিয়ে অটোমেশন সহজ ও কার্যকর হয়। টিমে কোড কোয়ালিটি ও প্রফেশনাল ওয়ার্কফ্লো নিশ্চিত করতে গিট হুক ব্যবহার করুন।
