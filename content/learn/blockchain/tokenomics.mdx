# ডিজিটাল অ্যাসেটের অর্থনীতি

টোকেনমিকস (Tokenomics) হচ্ছে ব্লকচেইন বা ক্রিপ্টো প্রকল্পে টোকেনের ডিজাইন, ডিস্ট্রিবিউশন, ইউটিলিটি, এবং ভ্যালু তৈরির কৌশল। এটি একটি টোকেনের অর্থনৈতিক কাঠামো, যা ইউজার, ডেভেলপার, এবং ইনভেস্টরদের জন্য গুরুত্বপূর্ণ।

## টোকেনমিকস কী?

- **ডিজাইন:** টোকেনের ধরন (ফাংজিবল, নন-ফাংজিবল), টোটাল সাপ্লাই, ডিভিশন
- **ডিস্ট্রিবিউশন:** টোকেন কিভাবে ও কারা পাবে (মাইনিং, ICO, এয়ারড্রপ, স্টেকিং)
- **ইউটিলিটি:** টোকেনের ব্যবহার (পেমেন্ট, ভোটিং, অ্যাক্সেস, রিওয়ার্ড)
- **ভ্যালু:** টোকেনের মূল্য নির্ধারণের ফ্যাক্টর (ডিমান্ড, সাপ্লাই, ইউটিলিটি)

## বাস্তব উদাহরণ ও এনালজি

- **গেমের কয়েন:** গেমে কয়েন দিয়ে আইটেম কেনা, লেভেল আনলক, রিওয়ার্ড—সবই টোকেনমিকসের অংশ
- **লয়ালটি পয়েন্ট:** শপিং অ্যাপে পয়েন্ট দিয়ে ডিসকাউন্ট, গিফট—এটাও টোকেনমিকস
- **Bitcoin:** সীমিত সাপ্লাই (২১ মিলিয়ন), মাইনিং রিওয়ার্ড, ইউটিলিটি—পেমেন্ট
- **Ethereum:** গ্যাস ফি, স্মার্ট কন্ট্র্যাক্ট এক্সিকিউশন, স্টেকিং

## টোকেনমিকসের মূল উপাদান

- **সাপ্লাই:** মোট কত টোকেন থাকবে, কিভাবে রিলিজ হবে
- **ডিমান্ড:** ইউজার কেন টোকেন কিনবে বা ব্যবহার করবে
- **ইনসেনটিভ:** ইউজার, ডেভেলপার, নোড—সবাইকে কীভাবে উৎসাহিত করা হবে
- **লকআপ/ভেস্টিং:** টোকেন কখন, কিভাবে আনলক হবে
- **বার্নিং:** টোকেন ডেস্ট্রয় করে সাপ্লাই কমানো

## টোকেনমিকস ডিজাইনের চ্যালেঞ্জ

- **কেন্দ্রীকরণ:** কিছু হাতে বেশি টোকেন থাকলে ক্ষমতা কেন্দ্রীভূত হয়
- **ইনফ্লেশন:** বেশি টোকেন রিলিজ হলে মূল্য কমে যেতে পারে
- **ইউটিলিটি:** টোকেনের ব্যবহার না থাকলে মূল্য কমে যায়
- **রেগুলেশন:** আইনগত বাধা, KYC/AML

## পারফরম্যান্স ও সিকিউরিটি টিপস

- টোকেনমিকস ডিজাইন করার সময় সাপ্লাই, ডিমান্ড, ইনসেনটিভ, এবং ইউটিলিটি ব্যালান্স করুন
- লকআপ/ভেস্টিং প্ল্যান রাখুন—ইনভেস্টরদের জন্য নিরাপত্তা
- কমিউনিটি ও ডেভেলপারদের জন্য ফেয়ার ডিস্ট্রিবিউশন নিশ্চিত করুন

## Python কোড ডেমো: টোকেনমিকস সিমুলেশন

```python
class Token:
    def __init__(self, supply):
        self.supply = supply
        self.burned = 0

    def burn(self, amount):
        self.supply -= amount
        self.burned += amount

    def get_supply(self):
        return self.supply

# উদাহরণ
token = Token(1000000)
token.burn(50000)
print("Current supply:", token.get_supply())
```
