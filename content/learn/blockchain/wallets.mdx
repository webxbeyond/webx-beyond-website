বিটকয়েন ও অন্যান্য ব্লকচেইনে "ওয়ালেট" ও "অ্যাড্রেস" হল ফান্ড ব্যবস্থাপনার মূল মাধ্যম। এই পাঠে আমরা জানব কীভাবে ওয়ালেট কাজ করে, কীভাবে অ্যাড্রেস তৈরি হয়, এবং নিরাপদে ফান্ড সংরক্ষণ ও পাঠানো যায়।

## ওয়ালেট কী?

ওয়ালেট হল একটি সফটওয়্যার বা হার্ডওয়্যার, যা আপনার প্রাইভেট কী সংরক্ষণ করে এবং লেনদেন সাইন করতে সাহায্য করে।

- **প্রাইভেট কী:** গোপন সংখ্যা, যা দিয়ে ফান্ড খরচ করা যায়
- **পাবলিক কী:** প্রাইভেট কী থেকে তৈরি, যা দিয়ে অ্যাড্রেস তৈরি হয়
- **অ্যাড্রেস:** পাবলিক কী থেকে হ্যাশ করে তৈরি, যেখানে ফান্ড পাঠানো যায়

## ওয়ালেটের ধরন

1. **হার্ডওয়্যার ওয়ালেট:** Ledger, Trezor (সবচেয়ে নিরাপদ)
2. **সফটওয়্যার ওয়ালেট:** Electrum, Exodus, Trust Wallet
3. **পেপার ওয়ালেট:** প্রাইভেট কী কাগজে লিখে রাখা
4. **মাল্টিসিগ ওয়ালেট:** একাধিক কী দিয়ে ফান্ড খরচ

## অ্যাড্রেস কীভাবে তৈরি হয়?

1. প্রাইভেট কী জেনারেট
2. পাবলিক কী তৈরি (Elliptic Curve Cryptography)
3. পাবলিক কী-কে হ্যাশ করে অ্যাড্রেস তৈরি (Base58, Bech32 ইত্যাদি)

## বাস্তব উদাহরণ

ধরা যাক, আপনার ব্যাংক অ্যাকাউন্ট নম্বর হল "অ্যাড্রেস" আর ATM PIN হল "প্রাইভেট কী"। PIN ছাড়া টাকা তুলতে পারবেন না, কিন্তু অ্যাকাউন্ট নম্বর দিয়ে কেউ আপনাকে টাকা পাঠাতে পারে।

## কোড উদাহরণ: বিটকয়েন অ্যাড্রেস জেনারেশন

```python
import os
import hashlib
import base58

# প্রাইভেট কী জেনারেট
private_key = os.urandom(32)

# পাবলিক কী তৈরি (Elliptic Curve ব্যবহার করে)
# এখানে সহজভাবে দেখানো হয়েছে
public_key = hashlib.sha256(private_key).digest()

# অ্যাড্রেস তৈরি
pubkey_hash = hashlib.new('ripemd160', public_key).digest()
address = base58.b58encode(pubkey_hash)
print(f"Bitcoin Address: {address.decode()}")
```

## নিরাপত্তা টিপস

- প্রাইভেট কী কখনো অনলাইনে শেয়ার করবেন না
- হার্ডওয়্যার ওয়ালেট ব্যবহার করুন
- Seed phrase নিরাপদ স্থানে লিখে রাখুন
- Multisig ওয়ালেট ব্যবহার করলে নিরাপত্তা বাড়ে

- ব্যাকআপ ওয়ালেট রাখুন
