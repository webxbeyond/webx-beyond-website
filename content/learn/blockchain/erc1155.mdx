# ফাংজিবল ও নন-ফাংজিবল একসাথে

ERC-1155 হচ্ছে Ethereum ব্লকচেইনে মাল্টি-টোকেন (ফাংজিবল ও নন-ফাংজিবল) তৈরি ও ব্যবহারের জন্য আধুনিক ও শক্তিশালী স্ট্যান্ডার্ড। এটি গেম, NFT কালেকশন, এবং কমপ্লেক্স অ্যাপ্লিকেশনের জন্য আদর্শ।

## ERC-1155 কী?

- **মাল্টি-টোকেন:** একই কন্ট্র্যাক্টে একাধিক ফাংজিবল (ERC-20 টাইপ) ও নন-ফাংজিবল (ERC-721 টাইপ) টোকেন তৈরি ও ব্যবহারের সুযোগ।
- **ইফিশিয়েন্ট:** একাধিক টোকেন একসাথে ট্রান্সফার, ব্যালেন্স চেক, ও অপারেশন করা যায়।
- **গ্যাস সেভিং:** একাধিক ট্রান্সফার/অপারেশন একসাথে করলে গ্যাস ফি কম লাগে।

## বাস্তব উদাহরণ ও এনালজি

- **গেম ইনভেন্টরি:** একটি গেমে কয়েন, অস্ত্র, পোশাক—সবই ERC-1155 দিয়ে একসাথে তৈরি ও ব্যবহৃত হয়।
- **NFT কালেকশন:** একাধিক ইউনিক ও ফাংজিবল আইটেম একসাথে ম্যানেজ করা যায়।
- **Enjin, OpenSea:** জনপ্রিয় ERC-1155 ভিত্তিক প্ল্যাটফর্ম।

## ERC-1155-এর প্রধান ফাংশন

- `balanceOf(address, id)`: নির্দিষ্ট অ্যাড্রেসে নির্দিষ্ট টোকেনের সংখ্যা
- `safeTransferFrom(from, to, id, amount, data)`: নির্দিষ্ট টোকেন ট্রান্সফার
- `batchTransferFrom(from, to, ids, amounts, data)`: একাধিক টোকেন একসাথে ট্রান্সফার
- `setApprovalForAll(operator, approved)`: অপারেটরকে অনুমতি
- `isApprovedForAll(owner, operator)`: অপারেটর অনুমোদিত কিনা

## ERC-1155-এর সুবিধা

- **ইফিশিয়েন্সি:** একাধিক টোকেন একসাথে ম্যানেজ ও ট্রান্সফার
- **গ্যাস সেভিং:** ব্যাচ অপারেশনে গ্যাস ফি কম
- **ফ্লেক্সিবিলিটি:** ফাংজিবল ও নন-ফাংজিবল টোকেন একসাথে

## চ্যালেঞ্জ ও সীমাবদ্ধতা

- **কোড কমপ্লেক্সিটি:** ERC-20/721-এর তুলনায় বেশি কমপ্লেক্স
- **মার্কেটপ্লেস সাপোর্ট:** সব মার্কেটপ্লেস ERC-1155 সাপোর্ট করে না
- **স্টোরেজ:** বড় NFT/ফাইলের জন্য IPFS/Arweave ব্যবহার করতে হয়

## Solidity কোড ডেমো: ERC-1155

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

import "@openzeppelin/contracts/token/ERC1155/ERC1155.sol";

contract MyMultiToken is ERC1155 {
    constructor() ERC1155("https://api.example.com/metadata/{id}.json") {}

    function mint(address to, uint256 id, uint256 amount) external {
        _mint(to, id, amount, "");
    }

    function mintBatch(address to, uint256[] memory ids, uint256[] memory amounts) external {
        _mintBatch(to, ids, amounts, "");
    }
}
```

## পারফরম্যান্স ও সিকিউরিটি টিপস

- OpenZeppelin লাইব্রেরি ব্যবহার করুন—স্ট্যান্ডার্ড ও নিরাপদ ইমপ্লিমেন্টেশন
- ব্যাচ অপারেশন ব্যবহার করুন—গ্যাস সেভিং
- বড় ফাইলের জন্য IPFS/Arweave ব্যবহার করুন
