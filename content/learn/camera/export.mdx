## পরিচিতি

ফটোগ্রাফির শেষ ধাপ হলো **এক্সপোর্ট বা আউটপুট প্রস্তুতি**।  
এক্সপোর্ট ঠিকভাবে না করলে, ছবি বা ভিডিও যতই সুন্দর হোক, Final Output এ মান বজায় থাকবে না।  

> কল্পনা করুন, আপনি একটি সুন্দর কেক বানিয়েছেন। Presentation বা প্যাকেজিং না করলে, কেউ সঠিকভাবে উপভোগ করতে পারবে না। এক্সপোর্ট হলো সেই Presentation।  

---

## ১. ওয়েব এক্সপোর্ট

**উদ্দেশ্য:** Social Media, Website বা Online Portfolio  

**Lightroom:**  
- File → Export → Export Location নির্বাচন  
- File Settings: JPEG, sRGB, Quality 70–100  
- Image Sizing: Resize to Fit → Width/Height  
- Output Sharpening: Screen  

**Capture One:**  
- File → Export → Variants  
- Format: JPEG, sRGB  
- Quality ও Resolution সেট করুন  

**টিপস:**  
- ওয়েবের জন্য File Size হালকা রাখুন  
- sRGB Color Profile ব্যবহার করুন  

---

## ২. প্রিন্ট এক্সপোর্ট

**উদ্দেশ্য:** Physical Prints, Galleries, Portfolio  

**Lightroom:**  
- File → Export → File Settings: TIFF বা High Quality JPEG  
- Color Space: Adobe RGB বা ProPhoto RGB  
- Resolution: 300ppi  
- Output Sharpening: Matte/Glossy Paper  

**Capture One:**  
- Format: TIFF/High Quality JPEG  
- Color Profile: Printer-specific  
- Resolution & Sharpening adjust করুন  

**টিপস:**  
- Print Proofing ব্যবহার করুন  
- Crop, Bleed এবং Margins লক্ষ্য করুন  

---

## ৩. ভিডিও এক্সপোর্ট (যদি ছবির ভিডিও তৈরি করা হয়)

**Lightroom (Slideshow Export)**  
- Export Video → Resolution, Frame Rate নির্বাচন  
- Codec: H.264  
- Quality adjust করুন  

**Capture One**  
- Time-lapse বা Sequence → Export Frames  
- After Effects/ Premiere Pro-এ Import  

**টিপস:**  
- ভিডিওর জন্য sRGB/Rec.709 color space ব্যবহার করুন  
- File Size এবং Resolution লক্ষ্য করুন

## প্র্যাকটিক্যাল এক্সারসাইজ

1. **ওয়েব এক্সপোর্ট প্র্যাকটিস:**  
   - একটি ছবি ওয়েব রেজোলিউশনে Export করুন  
   - File Size এবং Quality লক্ষ্য করুন  

2. **প্রিন্ট এক্সপোর্ট চ্যালেঞ্জ:**  
   - একটি ছবি 300ppi তে TIFF হিসেবে Export করুন  
   - Color Profile এবং Output Sharpening পরীক্ষা করুন  

3. **ভিডিও এক্সপোর্ট প্র্যাকটিস:**  
   - Slideshow বা Timelapse Sequence Export করুন  
   - Codec ও Resolution adjust করে Video Output চেক করুন
