## টেনসর অপারেশনস কী?

PyTorch-এ টেনসর মানে শুধু ডেটা স্টোর নয়—এটি দিয়ে নানা ধরনের গাণিতিক ও লজিক্যাল অপারেশন করা যায়। যেমন: যোগ, বিয়োগ, গুণ, ভাগ, ম্যাট্রিক্স মাল্টিপ্লিকেশন, রি-শেইপ, ট্রান্সপোজ, ইত্যাদি।

_অ্যানালজি:_ টেনসরকে ভাবুন ক্যালকুলেটরের মতো—যেখানে আপনি একাধিক সংখ্যার উপর একসাথে অপারেশন করতে পারেন।

## সাধারণ টেনসর অপারেশন

### ১. যোগ, বিয়োগ, গুণ, ভাগ
```python
import torch
x = torch.tensor([1, 2, 3])
y = torch.tensor([4, 5, 6])
print(x + y)      # tensor([5, 7, 9])
print(x - y)      # tensor([-3, -3, -3])
print(x * y)      # tensor([4, 10, 18])
print(x / y)      # tensor([0.25, 0.4, 0.5])
```

### ২. ম্যাট্রিক্স মাল্টিপ্লিকেশন
```python
A = torch.tensor([[1, 2], [3, 4]])
B = torch.tensor([[5, 6], [7, 8]])
print(torch.matmul(A, B))
```

### ৩. রি-শেইপ ও ট্রান্সপোজ
```python
x = torch.arange(0, 6)
x = x.reshape(2, 3)
print(x)
print(x.T)  # ট্রান্সপোজ
```

### ৪. স্লাইসিং ও ইনডেক্সিং
```python
x = torch.tensor([[1, 2, 3], [4, 5, 6]])
print(x[0, :])  # প্রথম সারি
print(x[:, 1])  # দ্বিতীয় কলাম
```

## ব্রডকাস্টিং কী?

ব্রডকাস্টিং হচ্ছে এমন একটি কনসেপ্ট, যেখানে ছোট টেনসরকে বড় টেনসরের শেইপে অটোমেটিকভাবে ম্যাচ করানো হয়—যাতে অপারেশন করা যায়।

_অ্যানালজি:_ ধরুন, আপনি ছোট কাপড় বড় বিছানায় বিছাতে চান—ব্রডকাস্টিং সেই কাপড়কে বড় করে বিছানার মতো করে দেয়!

### ব্রডকাস্টিং উদাহরণ
```python
x = torch.tensor([1, 2, 3])
y = torch.tensor([[10], [20], [30]])
print(x + y)
# Output:
# tensor([[11, 12, 13],
#         [21, 22, 23],
#         [31, 32, 33]])
```

এখানে x-এর শেইপ (3,) এবং y-এর শেইপ (3,1)। যোগ করার সময় PyTorch x-কে (3,1) থেকে (3,3)-এ ব্রডকাস্ট করে।

## রিয়েল-ওয়ার্ল্ড উদাহরণ

ধরা যাক, আপনি ইমেজ প্রসেসিং করছেন—একটি ইমেজ টেনসরের প্রতিটি পিক্সেলে একটি কনস্ট্যান্ট যোগ করতে চান। ব্রডকাস্টিং দিয়ে সহজেই করা যায়।

## কমন টেনসর ফাংশন
- `torch.sum(x)`: যোগফল
- `torch.mean(x)`: গড়
- `torch.max(x)`, `torch.min(x)`: সর্বোচ্চ/সর্বনিম্ন
- `torch.exp(x)`, `torch.log(x)`: এক্সপোনেনশিয়াল/লগ
- `torch.abs(x)`: অ্যাবসোলিউট

## টিপস ও সতর্কতা
- ব্রডকাস্টিং সবসময় অটোমেটিক হয় না—শেইপ mismatch হলে error আসতে পারে।
- `.reshape()` দিয়ে শেইপ ঠিক করুন।
- ম্যাট্রিক্স মাল্টিপ্লিকেশনে `@` বা `torch.matmul()` ব্যবহার করুন।

## সংক্ষিপ্ত চেকলিস্ট
- টেনসর অপারেশন: যোগ, গুণ, রি-শেইপ, ট্রান্সপোজ
- ব্রডকাস্টিং: ছোট টেনসরকে বড় টেনসরের সাথে অপারেশন
- রিয়েল-ওয়ার্ল্ড: ইমেজ, টেক্সট, অডিও প্রসেসিং
