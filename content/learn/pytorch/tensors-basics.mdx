## টেনসর কী?

টেনসর হচ্ছে মাল্টি-ডাইমেনশনাল ডেটা স্ট্রাকচার, যা PyTorch-এর মূল ভিত্তি। একে ভাবুন—স্কেলার (একটি সংখ্যা), ভেক্টর (একটি সংখ্যা সারি), ম্যাট্রিক্স (দুই ডাইমেনশন), আর টেনসর—যা আরও বেশি ডাইমেনশন ধরে রাখতে পারে।

_অ্যানালজি:_ টেনসরকে একটি ডেটা কনটেইনার হিসেবে ভাবুন, যেখানে একাধিক ডাইমেনশন বা স্তর আছে—যেমন, একটি বইয়ের পৃষ্ঠা, অধ্যায়, বাক্য—সব একসাথে!

## টেনসর তৈরি

PyTorch-এ টেনসর তৈরি করা খুব সহজ।

```python
import torch
# স্কেলার টেনসর
x = torch.tensor(5)
# ভেক্টর টেনসর
y = torch.tensor([1, 2, 3])
# ম্যাট্রিক্স টেনসর
z = torch.tensor([[1, 2], [3, 4]])
# 3D টেনসর
a = torch.zeros((2, 3, 4))
```

## টেনসর শেইপ (Shape)

শেইপ মানে টেনসরের ডাইমেনশন ও প্রতিটি ডাইমেনশনে কতগুলো এলিমেন্ট আছে।

```python
print(y.shape)  # torch.Size([3])
print(z.shape)  # torch.Size([2, 2])
print(a.shape)  # torch.Size([2, 3, 4])
```

_রিয়েল লাইফ উদাহরণ:_
- ইমেজ প্রসেসিংয়ে একটি কালার ইমেজের শেইপ হয় (উচ্চতা, প্রস্থ, চ্যানেল) যেমন (224, 224, 3)
- NLP-তে টেক্সট ডেটার শেইপ হয় (বাক্য সংখ্যা, শব্দ সংখ্যা)

## টেনসর টাইপ (dtype)

টেনসরের ডেটা টাইপ—যেমন, float, int, bool ইত্যাদি।

```python
b = torch.tensor([1.5, 2.3, 3.7], dtype=torch.float32)
c = torch.tensor([1, 0, 1], dtype=torch.bool)
print(b.dtype)  # torch.float32
print(c.dtype)  # torch.bool
```

## টেনসর তৈরি করার অন্যান্য পদ্ধতি

- `torch.zeros(shape)`: সব এলিমেন্ট ০
- `torch.ones(shape)`: সব এলিমেন্ট ১
- `torch.rand(shape)`: র‍্যান্ডম ভ্যালু
- `torch.arange(start, end)`: সিরিজ

```python
zeros = torch.zeros(4)
ones = torch.ones((2, 2))
rand = torch.rand((3, 3))
series = torch.arange(0, 10, 2)
```

## টেনসর ডিভাইস: CPU vs GPU

PyTorch-এ টেনসর CPU বা GPU-তে রাখা যায়।

```python
x = torch.tensor([1, 2, 3])
x_gpu = x.to('cuda')  # GPU-তে পাঠানো
print(x_gpu.device)   # cuda:0
```

_অ্যানালজি:_ CPU হচ্ছে আপনার কম্পিউটারের সাধারণ মেমরি, আর GPU হচ্ছে সুপারফাস্ট মেমরি—যেখানে ডিপ লার্নিং মডেল দ্রুত রান হয়।

## টেনসর অ্যাট্রিবিউটস

- `.shape`: শেইপ
- `.dtype`: ডেটা টাইপ
- `.device`: CPU/GPU
- `.ndim`: ডাইমেনশন সংখ্যা

```python
x = torch.rand((2, 3, 4))
print(x.ndim)  # 3
```

## টেনসর ইমিউটেবিলিটি ও কপি

টেনসর সাধারণত ইমিউটেবল, তবে `.clone()` দিয়ে কপি করা যায়।

```python
x = torch.tensor([1, 2, 3])
y = x.clone()
```

## রিয়েল-ওয়ার্ল্ড উদাহরণ

ধরা যাক, আপনি একটি ইমেজ ক্লাসিফিকেশন মডেল বানাচ্ছেন। ইমেজ ডেটা টেনসর হিসেবে GPU-তে পাঠিয়ে দ্রুত প্রসেস করতে পারবেন।

## সংক্ষিপ্ত টিপস

- টেনসর তৈরি, শেইপ, টাইপ—PyTorch-এর বেসিক, এগুলো ভালোভাবে আয়ত্ত করুন।
- `.to('cuda')` দিয়ে GPU-তে পাঠান, `.cpu()` দিয়ে আবার CPU-তে আনুন।
- ডেটা টাইপ ভুল হলে মডেল ট্রেইনিংয়ে সমস্যা হতে পারে—সবসময় `.dtype` চেক করুন।

## উপসংহার

টেনসর PyTorch-এর মূল ভিত্তি—ডিপ লার্নিং, ইমেজ, টেক্সট, অডিও—সবকিছু টেনসর দিয়ে প্রসেস হয়। তৈরি, শেইপ, টাইপ—এই তিনটি কনসেপ্ট ভালোভাবে বুঝলে PyTorch শেখার পথ অনেক সহজ হবে।
