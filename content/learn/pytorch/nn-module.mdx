## নিউরাল নেটওয়ার্ক কী?

নিউরাল নেটওয়ার্ক হচ্ছে এমন একটি গাণিতিক মডেল, যা মানুষের মস্তিষ্কের মতো তথ্য প্রসেস করে। এতে থাকে লেয়ার—যা ইনপুট থেকে আউটপুটে তথ্য পাঠায়, ওজন (weights) ও বায়াস (bias) আপডেট করে শেখে।

_অ্যানালজি:_ নিউরাল নেটওয়ার্ককে ভাবুন—একটি ফ্যাক্টরি, যেখানে কাঁচামাল (ইনপুট) বিভিন্ন মেশিন (লেয়ার) দিয়ে প্রসেস হয়ে চূড়ান্ত পণ্য (আউটপুট) হয়।

## PyTorch-এ nn.Module

PyTorch-এ নিউরাল নেটওয়ার্ক বানানোর জন্য সবচেয়ে গুরুত্বপূর্ণ ক্লাস হলো `nn.Module`। এটি হচ্ছে সব লেয়ার, মডেল, ও কাস্টম নেটওয়ার্কের বেস ক্লাস।

### nn.Module-এর বৈশিষ্ট্য
- লেয়ার ও মডেল সংজ্ঞায়িত করা যায়
- ফরওয়ার্ড পাস (forward method) ডিফাইন করা যায়
- ওজন ও বায়াস অটোমেটিকভাবে ট্র্যাক হয়

## কিভাবে nn.Module দিয়ে মডেল বানাবেন?

### ১. nn.Module থেকে ইনহেরিট করুন
### ২. `__init__`-এ লেয়ার ডিফাইন করুন
### ৩. `forward`-এ ডেটা প্রসেসিং লিখুন

```python
import torch
import torch.nn as nn

class SimpleNet(nn.Module):
    def __init__(self):
        super(SimpleNet, self).__init__()
        self.fc1 = nn.Linear(4, 8)  # ইনপুট ৪, আউটপুট ৮
        self.relu = nn.ReLU()
        self.fc2 = nn.Linear(8, 2)  # আউটপুট ২
    def forward(self, x):
        x = self.fc1(x)
        x = self.relu(x)
        x = self.fc2(x)
        return x
```

## লেয়ার কী?

লেয়ার হচ্ছে নিউরাল নেটওয়ার্কের বিল্ডিং ব্লক—যেমন, Linear, Convolutional, Activation, Dropout ইত্যাদি।

### সাধারণ PyTorch লেয়ার
- `nn.Linear`: ফিড-ফরওয়ার্ড লেয়ার
- `nn.Conv2d`: কনভোলিউশনাল লেয়ার (ইমেজ)
- `nn.ReLU`, `nn.Sigmoid`, `nn.Tanh`: অ্যাক্টিভেশন
- `nn.Dropout`: রেগুলারাইজেশন

## লেয়ার ব্যবহার উদাহরণ
```python
fc = nn.Linear(10, 5)  # ১০ ইনপুট, ৫ আউটপুট
x = torch.rand(2, 10)
out = fc(x)
print(out.shape)  # torch.Size([2, 5])
```

## ফরওয়ার্ড পাস কী?

ফরওয়ার্ড পাস মানে—ইনপুট ডেটা লেয়ার দিয়ে প্রসেস করে আউটপুট পাওয়া।

_অ্যানালজি:_ ফ্যাক্টরিতে কাঁচামাল ঢুকিয়ে একের পর এক মেশিনে পাঠিয়ে চূড়ান্ত পণ্য তৈরি।

## রিয়েল-ওয়ার্ল্ড উদাহরণ

ধরা যাক, আপনি একটি ফ্লাওয়ার ক্লাসিফিকেশন মডেল বানাচ্ছেন। ইনপুট: ফুলের বৈশিষ্ট্য, লেয়ার: Linear, Activation, আউটপুট: ফুলের ক্যাটাগরি।

## nn.Sequential

একাধিক লেয়ার সিরিয়ালভাবে সাজাতে `nn.Sequential` ব্যবহার করা যায়।

```python
model = nn.Sequential(
    nn.Linear(4, 8),
    nn.ReLU(),
    nn.Linear(8, 2)
)
```

## মডেল ইনস্ট্যান্স ও ফরওয়ার্ড
```python
net = SimpleNet()
x = torch.rand(1, 4)
out = net(x)
print(out)
```

## সংক্ষিপ্ত চেকলিস্ট
- nn.Module থেকে ইনহেরিট করুন
- `__init__`-এ লেয়ার ডিফাইন করুন
- `forward`-এ ডেটা প্রসেসিং লিখুন
- nn.Sequential দিয়ে সহজ মডেল বানান
