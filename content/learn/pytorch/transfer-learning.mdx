## ট্রান্সফার লার্নিং কী?

ট্রান্সফার লার্নিং মানে—একটি বড় ডেটাসেটে ট্রেইন করা মডেলের জ্ঞান (ফিচার) নিজের ছোট ডেটাসেটে ব্যবহার করা। এতে কম ডেটা, কম সময়, ও কম কম্পিউটেশনে ভালো ফলাফল পাওয়া যায়।

_অ্যানালজি:_ ধরুন, আপনি একজন দক্ষ চিত্রশিল্পীর কাছ থেকে আঁকার কৌশল শিখলেন—এখন নিজের ছবিতে সেই কৌশল ব্যবহার করতে পারবেন।

## ফাইন-টিউনিং কী?

ফাইন-টিউনিং মানে—প্রি-ট্রেইনড মডেলের কিছু অংশ (সাধারণত শেষের লেয়ার) নিজের ডেটাসেটে আবার ট্রেইন করা। এতে মডেল আপনার ডেটার জন্য আরও কাস্টমাইজড হয়।

## কেন দরকার?
- কম ডেটা থাকলেও ভালো accuracy
- Complex ফিচার অটোমেটিক পাওয়া যায়
- কম সময় ও কম GPU-তে ট্রেইনিং

## PyTorch-এ ট্রান্সফার লার্নিং ও ফাইন-টিউনিং

### ১. প্রি-ট্রেইনড মডেল লোড
```python
from torchvision import models
resnet = models.resnet18(pretrained=True)
```

### ২. ফিচার এক্সট্রাক্ট (ফ্রিজ) করুন
```python
for param in resnet.parameters():
    param.requires_grad = False  # ফিচার লেয়ার ফ্রিজ
```

### ৩. ক্লাসিফায়ার/হেড পরিবর্তন করুন
```python
import torch.nn as nn
num_classes = 5  # নিজের ডেটার ক্লাস সংখ্যা
resnet.fc = nn.Linear(resnet.fc.in_features, num_classes)
```

### ৪. শুধু নতুন লেয়ার ট্রেইন করুন
```python
optimizer = torch.optim.Adam(resnet.fc.parameters(), lr=0.001)
```

### ৫. ট্রেনিং লুপ
```python
for images, labels in train_loader:
    optimizer.zero_grad()
    output = resnet(images)
    loss = loss_fn(output, labels)
    loss.backward()
    optimizer.step()
```

## রিয়েল-ওয়ার্ল্ড উদাহরণ

ধরা যাক, আপনার কাছে ৫০০টি ফুলের ছবি আছে—ImageNet-এ ট্রেইন করা ResNet ফ্রিজ করে, শুধু শেষের লেয়ার ফাইন-টিউন করলে, অল্প ডেটাতেও ভালো accuracy পাবেন।
