PyTorch Lightning হচ্ছে PyTorch-এর উপর তৈরি একটি ফ্রেমওয়ার্ক, যা গবেষণা ও এক্সপেরিমেন্টকে সহজ, দ্রুত, ও স্কেলেবল করে তোলে। Boilerplate কোড কমিয়ে, মডেল, ট্রেনিং, ও লজিককে আলাদা করে—Lightning দিয়ে আপনি সহজেই বড় এক্সপেরিমেন্ট, GPU/TPU ট্রেনিং, ও প্রোডাকশন-রেডি কোড লিখতে পারেন।

## বাস্তব উদাহরণ ও অ্যানালজি

ভাবুন, আপনি একটি বড় রান্নাঘরে কাজ করছেন—সব উপকরণ, সরঞ্জাম, ও রেসিপি আলাদা করে রাখা। PyTorch Lightning হচ্ছে সেই "রান্নাঘরের অর্গানাইজার"—যা আপনার রিসার্চ ও এক্সপেরিমেন্টকে গুছিয়ে, দ্রুত, ও স্কেলেবল করে তোলে।

## PyTorch Lightning-এর মূল সুবিধা
- **Boilerplate কমানো:** Training loop, validation, logging—সব Lightning handle করে
- **GPU/TPU সাপোর্ট:** এক লাইনে multi-GPU/TPU ট্রেনিং
- **Experiment Management:** Logging, checkpoint, early stopping
- **Reproducibility:** কোড সহজে রি-ইউজ, শেয়ার, ও reproduce করা যায়
- **Production Ready:** Deployment, serving, ONNX/TorchScript export

## Lightning দিয়ে Model লেখা

### ইনস্টলেশন
```bash
pip install lightning
```

### Simple Classification Model
```python
import torch
import torch.nn as nn
import lightning as L

class LitModel(L.LightningModule):
    def __init__(self):
        super().__init__()
        self.layer = nn.Linear(10, 2)
    def forward(self, x):
        return self.layer(x)
    def training_step(self, batch, batch_idx):
        x, y = batch
        y_hat = self(x)
        loss = nn.functional.cross_entropy(y_hat, y)
        self.log('train_loss', loss)
        return loss
    def configure_optimizers(self):
        return torch.optim.Adam(self.parameters(), lr=1e-3)
```

### DataLoader ও Trainer
```python
from torch.utils.data import DataLoader, TensorDataset

x = torch.randn(100, 10)
y = torch.randint(0, 2, (100,))
dataset = TensorDataset(x, y)
loader = DataLoader(dataset, batch_size=16)

model = LitModel()
trainer = L.Trainer(max_epochs=10, accelerator='cpu')
trainer.fit(model, loader)
```

### কোড ব্যাখ্যা
- `LitModel`: Model, training_step, optimizer—সব LightningModule-এ
- `Trainer`: Training loop, logging, checkpoint—সব handle করে
- `accelerator`: GPU/TPU/CPU—এক লাইনে সাপোর্ট

## Advanced Features
- **Callbacks:** Early stopping, model checkpoint
- **Logging:** TensorBoard, CSV, MLFlow
- **Distributed Training:** Multi-GPU/TPU
- **Hyperparameter Tuning:** Integration with Optuna/Ray
- **Export:** TorchScript, ONNX
