বড় ডেটা নিয়ে কাজ করতে গেলে পাইথনের সাধারণ লিস্ট বা টুপল যথেষ্ট নয়। বিজ্ঞান, ইঞ্জিনিয়ারিং, ডেটা অ্যানালাইসিস—সবখানে NumPy (নামপাই) ব্যবহার হয়, কারণ এটি দ্রুত, শক্তিশালী এবং সহজে মাল্টিডাইমেনশনাল ডেটা প্রসেস করতে পারে।

ভাবুন, আপনি একসাথে হাজার হাজার ছাত্রের নম্বর যোগ করতে চান। সাধারণ লিস্ট দিয়ে করলে সময় বেশি লাগবে, কিন্তু NumPy দিয়ে এক লাইনেই দ্রুত কাজ হয়ে যাবে।

---

## NumPy কী?

NumPy হচ্ছে পাইথনের একটি ওপেন-সোর্স লাইব্রেরি, যা মূলত অ্যারে (array) এবং ম্যাট্রিক্স (matrix) ডেটা প্রসেসিংয়ের জন্য ব্যবহৃত হয়। এতে রয়েছে প্রচুর ম্যাথ ফাংশন, ভেক্টর অপারেশন, এবং ডেটা ম্যানিপুলেশন টুলস।

### ইনস্টলেশন

```bash
pip install numpy
```

## NumPy অ্যারে: বেসিক ধারণা

NumPy-এর মূল ডেটা স্ট্রাকচার হচ্ছে ndarray (n-dimensional array)। এটি সাধারণ লিস্টের চেয়ে অনেক দ্রুত এবং কম মেমোরি ব্যবহার করে।

### উদাহরণ:

```python
import numpy as np

arr = np.array([1, 2, 3, 4, 5])
print(arr)
```

### লিস্ট বনাম NumPy অ্যারে

- **লিস্ট:** বিভিন্ন ডেটা টাইপ রাখতে পারে, কিন্তু ম্যাথ অপারেশন ধীরগতির।
- **NumPy অ্যারে:** এক ধরনের ডেটা টাইপ, দ্রুত ম্যাথ অপারেশন।

---

## মাল্টিডাইমেনশনাল অ্যারে

NumPy দিয়ে 2D, 3D এমনকি আরও বেশি ডাইমেনশনাল অ্যারে তৈরি করা যায়।

```python
matrix = np.array([[1, 2], [3, 4]])
print(matrix)
```

---

## অ্যারে শেপ, টাইপ, ও রিসাইজ

- **shape:** অ্যারেটির ডাইমেনশন দেখায়
- **dtype:** ডেটা টাইপ
- **reshape:** অ্যারেকে নতুন শেপে রূপান্তর

```python
arr = np.array([1, 2, 3, 4, 5, 6])
reshaped = arr.reshape((2, 3))
print(reshaped)
```

---

## ভেক্টর অপারেশন

NumPy-এর সবচেয়ে বড় সুবিধা—ভেক্টরাইজড অপারেশন। অর্থাৎ, একসাথে পুরো অ্যারেতে ম্যাথ অপারেশন করা যায়।

### উদাহরণ:

```python
arr1 = np.array([1, 2, 3])
arr2 = np.array([4, 5, 6])

sum_arr = arr1 + arr2  # [5, 7, 9]
mul_arr = arr1 * arr2  # [4, 10, 18]
print(sum_arr, mul_arr)
```

### বাস্তব উদাহরণ:

ধরা যাক, ১০০ জন ছাত্রের নম্বর ৫% বাড়াতে হবে।

```python
marks = np.array([50, 60, 70, 80, 90])
updated_marks = marks * 1.05
print(updated_marks)
```

---

## ম্যাথ ফাংশন ও স্ট্যাটিস্টিক্স

NumPy-তে রয়েছে প্রচুর ম্যাথ ফাংশন—sum, mean, std, min, max ইত্যাদি।

```python
arr = np.array([1, 2, 3, 4, 5])
print(np.sum(arr))      # যোগফল
print(np.mean(arr))     # গড়
print(np.std(arr))      # স্ট্যান্ডার্ড ডিভিয়েশন
print(np.min(arr))      # সর্বনিম্ন
print(np.max(arr))      # সর্বোচ্চ
```

---

## স্লাইসিং ও ইন্ডেক্সিং

NumPy অ্যারে থেকে নির্দিষ্ট অংশ বের করা যায় সহজেই।

```python
arr = np.array([10, 20, 30, 40, 50])
print(arr[1:4])  # [20, 30, 40]
```

2D অ্যারেতে:

```python
matrix = np.array([[1, 2, 3], [4, 5, 6]])
print(matrix[:, 1])  # দ্বিতীয় কলাম
```

---

## ব্রডকাস্টিং

NumPy-এর আরেকটি শক্তিশালী ফিচার—broadcasting। ছোট অ্যারে বড় অ্যারেতে অটোমেটিক অপারেশন করতে পারে।

```python
arr = np.array([1, 2, 3])
print(arr + 10)  # [11, 12, 13]
```

## বাস্তব জীবনের analogy
