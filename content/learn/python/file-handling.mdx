# ফাইল পড়া ও লেখা (`open`, `read`, `write`)

ফাইল হ্যান্ডলিং হচ্ছে পাইথনের অন্যতম গুরুত্বপূর্ণ স্কিল। এটি অনেকটা ডায়েরি বা খাতা থেকে তথ্য পড়া ও লেখা শেখার মতো—কম্পিউটারে ডাটা সংরক্ষণ, প্রসেসিং, ও রিপোর্টিংয়ের জন্য ফাইল হ্যান্ডলিং অপরিহার্য।

## ফাইল কী?

ফাইল হচ্ছে ডাটা সংরক্ষণের একটি মাধ্যম—যেখানে টেক্সট, সংখ্যা, বা অন্য কোনো তথ্য রাখা যায়। যেমন: `.txt`, `.csv`, `.json` ইত্যাদি।

## ফাইল ওপেন করা

পাইথনে ফাইল ওপেন করতে `open()` ফাংশন ব্যবহার হয়।
```python
file = open("data.txt", "r")  # "r" মানে read mode
```

### মোড
- "r" : পড়ার জন্য
- "w" : লেখার জন্য (পুরনো ডাটা মুছে যাবে)
- "a" : যোগ করার জন্য (append)
- "b" : বাইনারি মোড

## ফাইল থেকে পড়া

### ১. পুরো ফাইল পড়া
```python
file = open("data.txt", "r")
content = file.read()
print(content)
file.close()
```

### ২. লাইনে লাইনে পড়া
```python
file = open("data.txt", "r")
for line in file:
    print(line.strip())
file.close()
```

## ফাইল লেখা

### ১. নতুন ফাইল তৈরি ও লেখা
```python
file = open("output.txt", "w")
file.write("Hello, World!\n")
file.write("Python file handling is easy.\n")
file.close()
```

### ২. ফাইলে যোগ (append)
```python
file = open("output.txt", "a")
file.write("New line added.\n")
file.close()
```

## with স্টেটমেন্ট (বেস্ট প্র্যাকটিস)

ফাইল হ্যান্ডলিংয়ে `with` স্টেটমেন্ট ব্যবহার করা ভালো, কারণ এতে ফাইল অটো-ক্লোজ হয়।
```python
with open("data.txt", "r") as file:
    content = file.read()
    print(content)
# এখানে file অটো-ক্লোজ হয়ে যাবে
```

## বাস্তব উদাহরণ

### ১. ইউজার থেকে ইনপুট নিয়ে ফাইলে লেখা
```python
name = input("আপনার নাম লিখুন: ")
with open("names.txt", "a") as file:
    file.write(name + "\n")
```

### ২. ফাইল থেকে সংখ্যা পড়ে যোগফল বের করা
```python
with open("numbers.txt", "r") as file:
    total = 0
    for line in file:
        total += int(line.strip())
print("যোগফল:", total)
```

## ভুল ও সমাধান

- ভুল: ফাইল ক্লোজ না করলে মেমরি লিক বা ডাটা সেভ না হতে পারে
- ভুল: "w" মোডে ফাইল ওপেন করলে পুরনো ডাটা মুছে যাবে
- ভুল: ফাইল না থাকলে FileNotFoundError

## অনুশীলন

১. একটি ফাইলে ৫টি নাম লিখুন (ইউজার ইনপুট)।
২. একটি ফাইল থেকে সব লাইন পড়ে, বড় হাতের অক্ষরে প্রিন্ট করুন।
৩. একটি ফাইলে সংখ্যা লিখে, পরে সেই সংখ্যা যোগফল বের করুন।

## উপসংহার

ফাইল হ্যান্ডলিং দিয়ে পাইথনে ডাটা সংরক্ষণ, প্রসেসিং, ও রিপোর্টিং সহজ হয়। `open`, `read`, `write`, এবং `with`—এই টুলগুলো ভালোভাবে আয়ত্ত করলে বাস্তব প্রজেক্টে দক্ষতা বাড়বে। পরবর্তী পাঠে CSV ও JSON ফাইল হ্যান্ডলিং নিয়ে আলোচনা করব।
