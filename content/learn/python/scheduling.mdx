# শিডিউলিং ও ক্রন জবস পাইথনে

## ভূমিকা

শিডিউলিং মানে—নির্দিষ্ট সময় বা ইন্টারভ্যালে কোনো কাজ স্বয়ংক্রিয়ভাবে চালানো। পাইথন দিয়ে শিডিউলিং ও ক্রন জবস—অর্থাৎ, রিপোর্ট তৈরি, ডেটা সংগ্রহ, ইমেইল পাঠানো—সবকিছু নির্দিষ্ট সময় অনুযায়ী অটোমেট করা যায়।

ভাবুন, আপনি প্রতিদিন সকাল ৮টায় রিপোর্ট তৈরি করতে চান, বা প্রতি ঘণ্টায় ওয়েবসাইট মনিটর করতে চান—পাইথন দিয়ে একবার কোড লিখে, বাকিটা সময় অটোমেটেড।

---

## শিডিউলিং কী?

শিডিউলিং হচ্ছে—কোনো কাজ নির্দিষ্ট সময়, দিন, বা ইন্টারভ্যালে চালানো।

- **Daily:** প্রতিদিন
- **Hourly:** প্রতি ঘণ্টা
- **Weekly:** প্রতি সপ্তাহে

### Analogy

শিডিউলিং-কে ভাবুন, অ্যালার্ম ঘড়ি—নির্দিষ্ট সময় হলে নিজে নিজে বাজে।

---

## পাইথনে শিডিউলিং: schedule লাইব্রেরি

`schedule` লাইব্রেরি দিয়ে সহজেই শিডিউলিং করা যায়।

### ইনস্টলেশন

```bash
pip install schedule
```

### উদাহরণ: প্রতিদিন রিপোর্ট তৈরি

```python
import schedule
import time

def job():
    print("রিপোর্ট তৈরি হচ্ছে...")

schedule.every().day.at("08:00").do(job)

while True:
    schedule.run_pending()
    time.sleep(1)
```

## ক্রন জবস: Unix/Linux-এ শিডিউলিং

ক্রন (cron) হচ্ছে Unix/Linux-এ বিল্ট-ইন শিডিউলার। পাইথন স্ক্রিপ্ট নির্দিষ্ট সময় অনুযায়ী চালাতে ক্রনজব ব্যবহার হয়।

### ক্রনজব যুক্ত করা

```bash
crontab -e
```

ক্রনজব ফরম্যাট:
```
মিনিট ঘণ্টা দিন মাস সপ্তাহ /path/to/python /path/to/script.py
```

### উদাহরণ: প্রতিদিন সকাল ৮টায় স্ক্রিপ্ট চালানো
```
0 8 * * * /usr/bin/python3 /home/user/report.py
```

## পাইথনে advanced শিডিউলিং: APScheduler

APScheduler দিয়ে interval, date, cron—সব ধরনের শিডিউলিং করা যায়।

### ইনস্টলেশন

```bash
pip install apscheduler
```

### উদাহরণ

```python
from apscheduler.schedulers.blocking import BlockingScheduler

scheduler = BlockingScheduler()

def job():
    print("রিপোর্ট তৈরি হচ্ছে...")

scheduler.add_job(job, 'interval', minutes=60)
scheduler.start()
```

## বাস্তব উদাহরণ: ইমেইল অটোমেশন

ধরা যাক, প্রতি সোমবার সকাল ৯টায় রিপোর্ট ইমেইল পাঠাতে চান—schedule বা APScheduler দিয়ে সহজেই করা যায়।
