# অপারেটরস: অ্যারিথমেটিক, লজিকাল, তুলনা অপারেটর

পাইথনে অপারেটরস হচ্ছে এমন কিছু চিহ্ন বা শব্দ, যা দিয়ে ভ্যারিয়েবল বা ডাটার উপর বিভিন্ন ধরনের কাজ করা যায়। যেমন যোগ, বিয়োগ, তুলনা, এবং লজিক্যাল অপারেশন।

## ১. অ্যারিথমেটিক অপারেটর

এগুলো দিয়ে সংখ্যা নিয়ে গণিতের কাজ করা হয়।

| অপারেটর | কাজ              | উদাহরণ         |
|----------|------------------|----------------|
| +        | যোগ              | 5 + 3 = 8      |
| -        | বিয়োগ            | 5 - 2 = 3      |
| *        | গুণ              | 4 * 2 = 8      |
| /        | ভাগ              | 10 / 2 = 5.0   |
| %        | ভাগশেষ           | 7 % 3 = 1      |
| **       | ঘাত              | 2 ** 3 = 8     |
| //       | পূর্ণ সংখ্যা ভাগ | 7 // 2 = 3     |

### উদাহরণ:
```python
x = 10
y = 3
print(x + y)    # 13
print(x - y)    # 7
print(x * y)    # 30
print(x / y)    # 3.333...
print(x % y)    # 1
print(x ** y)   # 1000
print(x // y)   # 3
```

## ২. তুলনা (Comparison) অপারেটর

এগুলো দিয়ে দুইটি ভ্যালু তুলনা করা হয়।
```
| অপারেটর | কাজ                | উদাহরণ         |
|----------|--------------------|----------------|
| ==       | সমান কিনা          | 5 == 5 → True  |
| !=       | সমান নয় কিনা       | 5 != 3 → True  |
| >        | বড় কিনা            | 7 > 2 → True   |
| <        | ছোট কিনা           | 2 < 5 → True   |
| >=       | বড় বা সমান কিনা    | 5 >= 5 → True  |
| <=       | ছোট বা সমান কিনা   | 3 <= 4 → True  |
```

### উদাহরণ:
```python
a = 5
b = 7
print(a == b)   # False
print(a != b)   # True
print(a > b)    # False
print(a < b)    # True
print(a >= b)   # False
print(a <= b)   # True
```

## ৩. লজিকাল অপারেটর

এগুলো দিয়ে একাধিক শর্ত একসাথে চেক করা হয়।

| অপারেটর | কাজ                | উদাহরণ         |
|----------|--------------------|----------------|
| and      | উভয় শর্ত সত্য হলে | True and False → False |
| or       | যেকোনো একটি সত্য  | True or False → True   |
| not      | বিপরীত মান         | not True → False       |

### উদাহরণ:
```python
x = 10
y = 5
print(x > 5 and y < 10)   # True
print(x < 5 or y < 10)    # True
print(not(x == y))        # True
```

## বাস্তব উদাহরণ

### বয়স যাচাই
```python
age = 18
if age >= 18:
    print("আপনি ভোট দিতে পারবেন।")
else:
    print("আপনি এখনও ছোট।")
```

### একাধিক শর্ত
```python
score = 85
passed = score >= 60 and score <= 100
print(passed)  # True
```

## ভুল ও সমাধান

- ভুল: = (assignment) ও == (comparison) গুলিয়ে ফেলা
  ```python
  if x = 5:   # ভুল
  ```
  সমাধান:
  ```python
  if x == 5:  # সঠিক
  ```

## অনুশীলন

১. দুটি সংখ্যা যোগ, বিয়োগ, গুণ, ভাগ, ভাগশেষ বের করুন।
২. দুটি ভ্যারিয়েবল তুলনা করুন `(==, !=, >, <)`।
৩. and, or, not অপারেটর ব্যবহার করে শর্ত চেক করুন।

## উপসংহার

অপারেটরস দিয়ে পাইথনে ডাটা প্রসেসিং, শর্ত যাচাই, এবং লজিক তৈরি করা হয়। এগুলো ভালোভাবে আয়ত্ত করলে পরবর্তী কনসেপ্ট সহজ হবে। পরবর্তী পাঠে কন্ট্রোল ফ্লো নিয়ে আলোচনা করব।
