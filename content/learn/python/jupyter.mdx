# Jupyter Notebook ও Exploratory Data Analysis

## Jupyter Notebook: ইন্টারেক্টিভ পাইথন

Jupyter Notebook হচ্ছে একটি ইন্টারেক্টিভ প্ল্যাটফর্ম, যেখানে কোড, টেক্সট, গ্রাফ, ও ডেটা একসাথে দেখা ও চালানো যায়।

ভাবুন, আপনি একজন গবেষক—ডেটা বিশ্লেষণ করছেন, কোড লিখছেন, সাথে সাথে ফলাফল দেখছেন। Jupyter Notebook-এ কোডের প্রতিটি অংশ আলাদা "সেল"-এ রাখা যায়, ফলে সহজে পরীক্ষা, পরিবর্তন, ও বিশ্লেষণ করা যায়।

<VSCode.Cell language="markdown">
# Jupyter Notebook কীভাবে কাজ করে?

<VSCode.Cell language="markdown">
## Exploratory Data Analysis (EDA) কী?

EDA হচ্ছে ডেটা বিশ্লেষণের প্রথম ধাপ—ডেটা দেখা, বুঝা, ও প্রাথমিক বিশ্লেষণ করা।

ভাবুন, আপনি একটি স্কুলের ছাত্রদের নম্বর বিশ্লেষণ করছেন—কতজন ভালো করছে, কারা পিছিয়ে, কোন সাবজেক্টে বেশি নম্বর।
</VSCode.Cell>

<VSCode.Cell language="python">
# প্রয়োজনীয় লাইব্রেরি ইম্পোর্ট
```
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns
```
</VSCode.Cell>

<VSCode.Cell language="markdown">
## ডেটা লোড ও দেখা

প্রথমে ডেটা লোড করুন এবং কয়েকটি সারি দেখে নিন।
</VSCode.Cell>

<VSCode.Cell language="python">
# ডেটা লোড
```
students = pd.DataFrame({
    'নাম': ['রাহুল', 'সুমন', 'রিমা', 'তানিয়া'],
    'নম্বর': [80, 45, 65, 30],
    'শ্রেণি': ['১০', '৯', '১০', '৮']
})
```
print(students.head())
</VSCode.Cell>

<VSCode.Cell language="markdown">
## ডেটার সারাংশ ও বেসিক স্ট্যাটিস্টিক্স

ডেটার সারাংশ, গড়, সর্বোচ্চ, সর্বনিম্ন, স্ট্যান্ডার্ড ডিভিয়েশন বের করুন।
</VSCode.Cell>

<VSCode.Cell language="python">
print(students.describe())
print("গড় নম্বর:", students['নম্বর'].mean())
print("সর্বোচ্চ নম্বর:", students['নম্বর'].max())
print("সর্বনিম্ন নম্বর:", students['নম্বর'].min())
</VSCode.Cell>

<VSCode.Cell language="markdown">
## ডেটা ভিজুয়ালাইজেশন

বার চার্ট, হিস্টোগ্রাম, ও বক্সপ্লট দিয়ে ডেটা সহজে বিশ্লেষণ করা যায়।
</VSCode.Cell>

<VSCode.Cell language="python">
# বার চার্ট
plt.bar(students['নাম'], students['নম্বর'])
plt.xlabel('নাম')
plt.ylabel('নম্বর')
plt.title('ছাত্রদের নম্বর')
plt.show()

# হিস্টোগ্রাম
plt.hist(students['নম্বর'], bins=5)
plt.title('নম্বরের হিস্টোগ্রাম')
plt.show()

# বক্সপ্লট
sns.boxplot(y=students['নম্বর'])
plt.title('নম্বরের বিস্তার')
plt.show()
</VSCode.Cell>

<VSCode.Cell language="markdown">
## Missing Value ও Data Cleaning

ডেটা বিশ্লেষণের আগে Missing Value, Duplicate, Outlier ঠিক করা জরুরি।
</VSCode.Cell>

<VSCode.Cell language="python">
# Missing Value চেক
print(students.isnull().sum())

# Missing Value পূরণ
students['নম্বর'] = students['নম্বর'].fillna(0)

# Duplicate চেক ও বাদ
students = students.drop_duplicates()
</VSCode.Cell>

<VSCode.Cell language="markdown">
## গ্রুপিং ও সামারি

শ্রেণি অনুযায়ী গড় নম্বর বের করুন।
</VSCode.Cell>

<VSCode.Cell language="python">
grouped = students.groupby('শ্রেণি')['নম্বর'].mean()
print(grouped)
</VSCode.Cell>

<VSCode.Cell language="markdown">
## বাস্তব উদাহরণ ও analogy

Jupyter Notebook-কে ভাবুন, ডেটা বিশ্লেষণের খাতা—যেখানে কোড, টেক্সট, ও গ্রাফ একসাথে রাখা যায়।

EDA-কে ভাবুন, ডেটার "হেলথ চেক"—প্রথমে ডেটা দেখে, সমস্যা খুঁজে, বিশ্লেষণ শুরু করা।
</VSCode.Cell>
