# ফাংশন ডিফাইন ও কল করা, প্যারামিটার ও রিটার্ন ভ্যালু

ফাংশন হচ্ছে প্রোগ্রামিংয়ের "রেসিপি"—একবার লিখে বারবার ব্যবহার করা যায়। যেমন রান্নার রেসিপি বারবার কাজে লাগে, তেমনি ফাংশন কোডকে পুনরায় ব্যবহারযোগ্য ও সংগঠিত করে।

## ফাংশন কী?

ফাংশন হচ্ছে কোডের একটি ব্লক, যা নির্দিষ্ট কাজ সম্পাদন করে। এতে ইনপুট (প্যারামিটার) নেওয়া যায়, আউটপুট (রিটার্ন ভ্যালু) দেওয়া যায়।

## ফাংশন ডিফাইন করা

পাইথনে ফাংশন ডিফাইন করতে `def` কীওয়ার্ড ব্যবহার হয়।

### সাধারণ ফাংশন:
```python
def greet():
    print("স্বাগতম!")
```

এখানে `greet` নামের ফাংশনটি কোনো ইনপুট নেয় না, শুধু "স্বাগতম!" প্রিন্ট করে।

## ফাংশন কল করা

ফাংশন ডিফাইন করার পর, সেটি "কল" করতে হয়:
```python
greet()  # আউটপুট: স্বাগতম!
```

## প্যারামিটার ও আর্গুমেন্ট

ফাংশনে ইনপুট নেওয়ার জন্য প্যারামিটার ব্যবহার হয়।

### উদাহরণ:
```python
def greet(name):
    print("হ্যালো,", name)

greet("Anis")  # আউটপুট: হ্যালো, Anis
```

এখানে `name` হচ্ছে প্যারামিটার, "Anis" হচ্ছে আর্গুমেন্ট।

### একাধিক প্যারামিটার:
```python
def add(a, b):
    print(a + b)

add(5, 7)  # আউটপুট: 12
```

## রিটার্ন ভ্যালু

ফাংশন কোনো মান "রিটার্ন" করতে পারে, যাতে সেই মান পরে ব্যবহার করা যায়।

### উদাহরণ:
```python
def add(a, b):
    return a + b

result = add(10, 20)
print(result)  # আউটপুট: 30
```

## বাস্তব উদাহরণ

### ১. ইউজার থেকে নাম নিয়ে শুভেচ্ছা
```python
def personalized_greet(name):
    return f"হ্যালো, {name}!"

user_name = input("আপনার নাম লিখুন: ")
print(personalized_greet(user_name))
```

### ২. সংখ্যা যোগফল বের করা
```python
def sum_list(numbers):
    total = 0
    for n in numbers:
        total += n
    return total

print(sum_list([1, 2, 3, 4]))  # আউটপুট: 10
```

## ডিফল্ট প্যারামিটার

ফাংশনে ডিফল্ট মান দেওয়া যায়:
```python
def greet(name="বন্ধু"):
    print(f"হ্যালো, {name}!")

greet()         # আউটপুট: হ্যালো, বন্ধু!
greet("Anis")   # আউটপুট: হ্যালো, Anis!
```

## কিওয়ার্ড আর্গুমেন্ট

প্যারামিটার নাম দিয়ে মান পাঠানো যায়:
```python
def info(name, age):
    print(f"নাম: {name}, বয়স: {age}")

info(age=25, name="Anis")
```

## ভুল ও সমাধান

- ভুল: ফাংশন কল না করা
  ```python
  def show():
      print("Hello")
  # show() না দিলে কিছু হবে না
  ```
- ভুল: প্যারামিটার না পাঠানো
  ```python
  def greet(name):
      print(name)
  greet()  # ভুল, আর্গুমেন্ট লাগবে
  ```

## অনুশীলন

১. দুটি সংখ্যা যোগ করে রিটার্ন করে এমন ফাংশন লিখুন।
২. একটি লিস্টের সর্বোচ্চ সংখ্যা বের করে রিটার্ন করে এমন ফাংশন লিখুন।
৩. ডিফল্ট প্যারামিটারসহ একটি ফাংশন লিখুন।

## উপসংহার

ফাংশন কোডকে সংগঠিত, পুনরায় ব্যবহারযোগ্য ও সহজবোধ্য করে। প্যারামিটার ও রিটার্ন ভ্যালু দিয়ে ফাংশন আরও শক্তিশালী হয়। পরবর্তী পাঠে স্কোপ ও ল্যাম্বডা ফাংশন নিয়ে আলোচনা করব।
