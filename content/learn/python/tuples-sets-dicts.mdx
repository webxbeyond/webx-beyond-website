# টুপল, সেট ও ডিকশনারি: বেসিক ও ব্যবহার ক্ষেত্র

পাইথনে লিস্ট ছাড়াও আরও তিনটি গুরুত্বপূর্ণ কালেকশন ডাটা স্ট্রাকচার আছে—টুপল, সেট, ও ডিকশনারি। এগুলো বিভিন্ন পরিস্থিতিতে ডাটা সংরক্ষণ ও প্রসেসিং সহজ করে।

## টুপল (Tuple)

টুপল হচ্ছে অপরিবর্তনীয় (immutable) লিস্ট। একবার তৈরি হলে এর আইটেম পরিবর্তন করা যায় না।

### টুপল তৈরি
```python
tuple1 = (1, 2, 3)
tuple2 = ("apple", "banana", "mango")
empty_tuple = ()
```

### টুপল অ্যাক্সেস
```python
print(tuple1[0])  # 1
print(tuple2[-1]) # mango
```

### টুপলের ব্যবহার ক্ষেত্র
- ডাটা পরিবর্তন না হলে
- ফাংশন থেকে একাধিক মান রিটার্ন করতে

### উদাহরণ:
```python
def get_info():
    return ("Anis", 25)
name, age = get_info()
print(name, age)
```

## সেট (Set)

সেট হচ্ছে ইউনিক আইটেমের কালেকশন—একই আইটেম একাধিকবার রাখা যায় না।

### সেট তৈরি
```python
set1 = {1, 2, 3, 2}
print(set1)  # {1, 2, 3}
empty_set = set()
```

### সেটে আইটেম যোগ/বিয়োগ
```python
set1.add(4)
set1.remove(2)
print(set1)
```

### সেটের ব্যবহার ক্ষেত্র
- ডুপ্লিকেট বাদ দিতে
- গণিতের সেট অপারেশন (union, intersection)

### উদাহরণ:
```python
a = {1, 2, 3}
b = {3, 4, 5}
print(a | b)  # union: {1, 2, 3, 4, 5}
print(a & b)  # intersection: {3}
```

## ডিকশনারি (Dictionary)

ডিকশনারি হচ্ছে কী-ভ্যালু পেয়ার—যেখানে প্রতিটি আইটেমের জন্য একটি "নাম" (key) ও "মান" (value) থাকে।

### ডিকশনারি তৈরি
```python
dict1 = {"name": "Anis", "age": 25}
empty_dict = {}
```

### ডিকশনারি অ্যাক্সেস
```python
print(dict1["name"])  # Anis
```

### ডিকশনারিতে আইটেম যোগ/পরিবর্তন
```python
dict1["city"] = "Dhaka"
dict1["age"] = 26
print(dict1)
```

### ডিকশনারির ব্যবহার ক্ষেত্র
- ডাটা ম্যাপিং (নাম-রোল, প্রোডাক্ট-প্রাইস)
- কনফিগারেশন, JSON ডাটা

### উদাহরণ:
```python
students = {
    "Anis": 90,
    "Bithi": 85
}
for name, score in students.items():
    print(f"{name}: {score}")
```

## ভুল ও সমাধান

- টুপল পরিবর্তন করতে চাইলে TypeError
- সেটে remove() দিয়ে না থাকা আইটেম বাদ দিলে KeyError
- ডিকশনারিতে ভুল key দিলে KeyError

## অনুশীলন

১. একটি টুপল তৈরি করুন এবং প্রথম ও শেষ আইটেম প্রিন্ট করুন।
২. একটি সেটে ডুপ্লিকেট সংখ্যা যোগ করুন এবং ইউনিক সংখ্যা দেখান।
৩. একটি ডিকশনারিতে নাম-রোল নম্বর সংরক্ষণ করুন এবং সব নাম ও রোল প্রিন্ট করুন।

## উপসংহার

টুপল, সেট ও ডিকশনারি—এই তিনটি ডাটা স্ট্রাকচার পাইথনে ডাটা সংগঠন, প্রসেসিং ও ফিল্টারিং সহজ করে। সঠিক পরিস্থিতিতে এগুলো ব্যবহার করলে কোড আরও কার্যকর ও পরিষ্কার হয়। পরবর্তী পাঠে ইটারেশন নিয়ে আলোচনা করব।
