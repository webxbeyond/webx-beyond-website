কন্ট্রোল ফ্লো হচ্ছে প্রোগ্রামের সেই অংশ, যা নির্ধারণ করে কোন কোড কখন চলবে। এটি ঠিক যেমন ট্রাফিক সিগনাল রাস্তার গাড়ি নিয়ন্ত্রণ করে, তেমনি কন্ট্রোল ফ্লো কোডের "গতি" নিয়ন্ত্রণ করে।

## শর্ত (Conditionals): if, else, elif

### if
`if` দিয়ে কোনো শর্ত সত্য হলে নির্দিষ্ট কোড চালানো হয়।
```python
age = 18
if age >= 18:
    print("আপনি ভোট দিতে পারবেন।")
```

### else
`else` দিয়ে if-এর শর্ত মিথ্যা হলে বিকল্প কোড চালানো হয়।
```python
age = 15
if age >= 18:
    print("আপনি ভোট দিতে পারবেন।")
else:
    print("আপনি এখনও ছোট।")
```

### elif
`elif` (else if) দিয়ে একাধিক শর্ত চেক করা যায়।
```python
score = 75
if score >= 80:
    print("A grade")
elif score >= 60:
    print("B grade")
else:
    print("C grade")
```

## লুপ (Loops)

লুপ দিয়ে একাধিক বার কোড চালানো যায়।

### for loop
`for` লুপ দিয়ে লিস্ট, রেঞ্জ, বা অন্য iterable-এর উপর ইটারেট করা যায়।
```python
fruits = ["apple", "banana", "mango"]
for fruit in fruits:
    print(fruit)
```

### range() ব্যবহার
```python
for i in range(1, 6):
    print(i)
```

### while loop
`while` লুপ দিয়ে কোনো শর্ত সত্য থাকলে বারবার কোড চালানো যায়।
```python
count = 1
while count <= 5:
    print(count)
    count += 1
```

## লুপ কন্ট্রোল: break, continue

### break
লুপের মধ্যে কোনো শর্তে "break" দিলে লুপ থেমে যায়।
```python
for i in range(1, 10):
    if i == 5:
        break
    print(i)
```

### continue
"continue" দিলে ওই ইটারেশন স্কিপ হয়, পরের ইটারেশনে চলে যায়।
```python
for i in range(1, 6):
    if i == 3:
        continue
    print(i)
```

## বাস্তব উদাহরণ

### ইউজার থেকে ইনপুট নিয়ে শর্ত
```python
password = input("পাসওয়ার্ড দিন: ")
if password == "1234":
    print("অ্যাক্সেস অনুমোদিত!")
else:
    print("ভুল পাসওয়ার্ড!")
```

### লিস্ট থেকে শুধু জোড় সংখ্যা প্রিন্ট
```python
numbers = [1, 2, 3, 4, 5, 6]
for n in numbers:
    if n % 2 == 0:
        print(n)
```

## ভুল ও সমাধান

- ভুল: ইন্ডেন্টেশন ভুল হলে
  ```python
  if x > 5:
  print(x)  # ভুল
  ```
  সমাধান:
  ```python
  if x > 5:
      print(x)  # সঠিক
  ```

## অনুশীলন

১. ইউজার থেকে সংখ্যা নিয়ে, যদি সংখ্যা ১০-এর বেশি হয় "বড় সংখ্যা" প্রিন্ট করুন, না হলে "ছোট সংখ্যা"।
২. ১ থেকে ১০ পর্যন্ত সংখ্যা প্রিন্ট করুন (for loop)।
৩. ১ থেকে ২০ পর্যন্ত শুধু জোড় সংখ্যা প্রিন্ট করুন (while loop)।

## উপসংহার

কন্ট্রোল ফ্লো দিয়ে প্রোগ্রামের লজিক, সিদ্ধান্ত, এবং পুনরাবৃত্তি নিয়ন্ত্রণ করা যায়। এগুলো ভালোভাবে আয়ত্ত করলে পাইথনের বেসিক সিনট্যাক্সে দক্ষতা আসবে। পরবর্তী মডিউলে ফাংশন ও কালেকশন নিয়ে আলোচনা করব।
