প্রোগ্রামিংয়ে এরর বা ভুল হওয়া স্বাভাবিক। পাইথনে এরর হ্যান্ডলিং শেখা মানে হচ্ছে—কিভাবে ভুল হলে প্রোগ্রাম ক্র্যাশ না করে সুন্দরভাবে সমস্যার সমাধান করা যায়।

## এরর ও এক্সসেপশন কী?

- **এরর:** কোডে ভুল হলে, যেমন: ভাগে শূন্য, ফাইল না পাওয়া, টাইপ মিসম্যাচ ইত্যাদি।
- **এক্সসেপশন:** পাইথন যখন এরর পায়, তখন "এক্সসেপশন" তৈরি হয়।

## try, except

`try` ব্লকে কোড লিখলে, যদি কোনো এরর হয়, তাহলে `except` ব্লকে চলে যায়।

### সাধারণ উদাহরণ:
```python
try:
    x = 10 / 0
except ZeroDivisionError:
    print("শূন্য দিয়ে ভাগ করা যায় না!")
```

### একাধিক except
```python
try:
    num = int(input("সংখ্যা দিন: "))
    print(100 / num)
except ValueError:
    print("শুধু সংখ্যা দিন!")
except ZeroDivisionError:
    print("শূন্য দিয়ে ভাগ করা যাবে না!")
```

## finally

`finally` ব্লক সবসময় চলে—এরর হোক বা না হোক। সাধারণত রিসোর্স ক্লোজ করার জন্য ব্যবহার হয়।
```python
try:
    file = open("data.txt", "r")
    print(file.read())
except FileNotFoundError:
    print("ফাইল পাওয়া যায়নি!")
finally:
    print("প্রোগ্রাম শেষ!")
```

## else

`else` ব্লক তখনই চলে, যখন কোনো এরর হয় না।
```python
try:
    print("সব ঠিক আছে!")
except:
    print("এরর!")
else:
    print("কোনো এরর হয়নি!")
```

## বাস্তব উদাহরণ

### ১. ইউজার ইনপুটে এরর হ্যান্ডলিং
```python
try:
    age = int(input("বয়স লিখুন: "))
    print("আপনার বয়স:", age)
except ValueError:
    print("শুধু সংখ্যা দিন!")
```

### ২. ফাইল ওপেন করার সময়
```python
try:
    with open("info.txt", "r") as file:
        print(file.read())
except FileNotFoundError:
    print("ফাইল পাওয়া যায়নি!")
```

## কমন এক্সসেপশন

| এক্সসেপশন           | কখন হয়                |
|---------------------|-----------------------|
| ValueError          | ভুল টাইপ ইনপুট        |
| ZeroDivisionError   | শূন্য দিয়ে ভাগ        |
| FileNotFoundError   | ফাইল না পাওয়া         |
| TypeError           | ভুল টাইপ অপারেশন      |
| IndexError          | ভুল ইনডেক্স           |
| KeyError            | ভুল ডিকশনারি কী       |

## ভুল ও সমাধান

- ভুল: except ব্লকে নির্দিষ্ট এক্সসেপশন না ধরলে সব এরর ধরা পড়ে, কিন্তু কারণ জানা যায় না
- ভুল: finally ব্লকে রিসোর্স ক্লোজ না করলে মেমরি লিক

## অনুশীলন

১. ইউজার থেকে সংখ্যা নিয়ে, শূন্য দিয়ে ভাগ করলে সুন্দরভাবে মেসেজ দেখান।
২. ফাইল না থাকলে except দিয়ে "ফাইল পাওয়া যায়নি" দেখান।
৩. একটি লিস্টে ভুল ইনডেক্স দিলে except দিয়ে "ইনডেক্স ভুল" দেখান।

## উপসংহার

এরর ও এক্সসেপশন হ্যান্ডলিং—প্রোগ্রামকে আরও শক্তিশালী ও ইউজার-ফ্রেন্ডলি করে। `try`, `except`, `finally`, `else`—এই টুলগুলো ভালোভাবে আয়ত্ত করলে বাস্তব প্রজেক্টে সমস্যা সমাধান সহজ হবে। পরবর্তী পাঠে কাস্টম এক্সসেপশন নিয়ে আলোচনা করব।
