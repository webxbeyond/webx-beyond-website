পাইথনের শক্তি তার বিশাল লাইব্রেরি ও মডিউল সিস্টেমে। মডিউল ব্যবহার করে কোডকে ছোট ছোট অংশে ভাগ করা যায়, পুনঃব্যবহারযোগ্য ও সংগঠিত রাখা যায়। এই পাঠে আমরা জানবো:
- মডিউল কী, কেন দরকার
- কিভাবে ইম্পোর্ট ও ব্যবহার করবেন
- জনপ্রিয় বিল্ট-ইন মডিউল: `math`, `datetime`, `os`
- রিয়েল-লাইফ উদাহরণ ও analogies

## ১. মডিউল কী?

মডিউল হলো পাইথন ফাইল (যেমন: `math.py`, `os.py`) যাতে ফাংশন, ক্লাস, ভ্যারিয়েবল থাকে। আপনি নিজের কোডকে মডিউল আকারে লিখতে পারেন, আবার পাইথনের বিল্ট-ইন/থার্ড-পার্টি মডিউলও ব্যবহার করতে পারেন।

## ২. মডিউল ইম্পোর্ট করা

মডিউল ইম্পোর্ট করতে `import` বা `from ... import ...` ব্যবহার হয়।

```python
import math
print(math.sqrt(16))  # 4.0

from datetime import datetime
print(datetime.now())
```

## ৩. বিল্ট-ইন মডিউল: math

`math` মডিউল দিয়ে গণিতের নানা ফাংশন পাওয়া যায়।

```python
import math
print(math.pi)           # 3.141592...
print(math.sin(math.pi/2)) # 1.0
print(math.factorial(5)) # 120
```

### রিয়েল-লাইফ উদাহরণ:
- ক্যালকুলেটর অ্যাপ
- জ্যামিতিক হিসাব

## ৪. বিল্ট-ইন মডিউল: datetime

`datetime` দিয়ে তারিখ, সময়, টাইমজোন ইত্যাদি হ্যান্ডল করা যায়।

```python
from datetime import datetime, timedelta
now = datetime.now()
print("এখন:", now)
future = now + timedelta(days=5)
print("৫ দিন পর:", future)
```

### রিয়েল-লাইফ উদাহরণ:
- ইভেন্ট শিডিউলিং
- লগিং

## ৫. বিল্ট-ইন মডিউল: os

`os` দিয়ে অপারেটিং সিস্টেম সংক্রান্ত কাজ (ফাইল, ফোল্ডার, এনভায়রনমেন্ট) করা যায়।

```python
import os
print(os.getcwd())  # বর্তমান ডিরেক্টরি
os.mkdir('test_folder')  # নতুন ফোল্ডার
print(os.listdir('.'))   # সব ফাইল/ফোল্ডার
```

### রিয়েল-লাইফ উদাহরণ:
- ফাইল অটোমেশন
- ডিরেক্টরি ম্যানেজমেন্ট

## ৬. Analogies: লাইব্রেরি ও বই

মডিউলকে ভাবুন লাইব্রেরির বইয়ের মতো। প্রয়োজন হলে নির্দিষ্ট বই (মডিউল) বের করে পড়েন, আবার নিজের বইও যোগ করতে পারেন। এতে কাজ সহজ ও দ্রুত হয়।

## ৭. Best Practices

- প্রয়োজন অনুযায়ী নির্দিষ্ট ফাংশন ইম্পোর্ট করুন: `from math import sqrt`
- মডিউল নাম অর্থবহ রাখুন
- কোডকে ছোট ছোট মডিউলে ভাগ করুন
- বিল্ট-ইন মডিউল আগে ব্যবহার করুন, পরে থার্ড-পার্টি

## ৮. সংক্ষেপে

- মডিউল কোডকে সংগঠিত ও পুনঃব্যবহারযোগ্য করে
- `import` দিয়ে সহজে ব্যবহার করা যায়
- `math`, `datetime`, `os`—সবচেয়ে বেশি ব্যবহৃত বিল্ট-ইন মডিউল

এভাবেই পাইথনে মডিউল ইম্পোর্ট ও বিল্ট-ইন মডিউল ব্যবহার করে আপনার কোডকে আরও শক্তিশালী, পরিষ্কার ও মেইনটেনেবল করতে পারেন।
