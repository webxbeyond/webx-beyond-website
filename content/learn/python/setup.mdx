পাইথন শেখার প্রথম ধাপ হলো আপনার কম্পিউটারে পাইথন ইন্সটল করা এবং কোড লেখার জন্য একটি উপযুক্ত পরিবেশ (IDE) তৈরি করা। এই পাঠে আমরা পাইথন ইন্সটলেশন, জনপ্রিয় IDE (VS Code, PyCharm, Jupyter Notebook) সেটআপ, এবং বাস্তব উদাহরণ দেখব।

## পাইথন ইন্সটলেশন

### Windows-এ পাইথন ইন্সটল
1. [python.org](https://www.python.org/downloads/) থেকে Windows Installer ডাউনলোড করুন।
2. ইনস্টলার চালান এবং "Add Python to PATH" চেকবক্সটি অবশ্যই সিলেক্ট করুন।
3. "Install Now" ক্লিক করুন।
4. ইন্সটল শেষে Command Prompt-এ লিখুন:
   ```bash
   python --version
   ```
   যদি পাইথনের ভার্সন দেখায়, ইন্সটল সফল।

### macOS-এ পাইথন ইন্সটল
- macOS-এ সাধারণত পাইথন প্রি-ইন্সটল থাকে, তবে নতুন ভার্সন চাইলে Homebrew ব্যবহার করুন:
  ```bash
  brew install python
  ```
- ইন্সটল শেষে Terminal-এ লিখুন:
  ```bash
  python3 --version
  ```

### Linux-এ পাইথন ইন্সটল
- বেশিরভাগ Linux ডিস্ট্রিবিউশনে পাইথন থাকে। নতুন ভার্সন চাইলে:
  ```bash
  sudo apt update
  sudo apt install python3
  ```
- চেক করুন:
  ```bash
  python3 --version
  ```

## IDE কী ও কেন?

IDE (Integrated Development Environment) হলো এমন একটি সফটওয়্যার যেখানে কোড লেখা, রান করা, ডিবাগিং, এবং প্রজেক্ট ম্যানেজমেন্ট সহজ হয়।

### VS Code
- **বৈশিষ্ট্য:** ফ্রি, লাইটওয়েট, এক্সটেনশন সমৃদ্ধ।
- **ইন্সটল:**
  1. [VS Code Download](https://code.visualstudio.com/) থেকে ডাউনলোড করুন।
  2. "Python" এক্সটেনশন ইন্সটল করুন (Extension Marketplace থেকে)।
- **ব্যবহার:**
  - নতুন ফাইল খুলুন: `hello.py`
  - কোড লিখুন:
    ```python
    print("Hello, World!")
    ```
  - রান করুন: Run বাটন বা Terminal-এ `python hello.py`

### PyCharm
- **বৈশিষ্ট্য:** শক্তিশালী Python IDE, অটো-কমপ্লিশন, ডিবাগার, ভার্চুয়াল এনভায়রনমেন্ট সাপোর্ট।
- **ইন্সটল:**
  1. [PyCharm Download](https://www.jetbrains.com/pycharm/) থেকে ডাউনলোড করুন।
  2. ইনস্টল শেষে "New Project" খুলুন।
- **ব্যবহার:**
  - নতুন Python ফাইল খুলুন, কোড লিখুন, রান করুন।

### Jupyter Notebook
- **বৈশিষ্ট্য:** ইন্টারেক্টিভ কোডিং, ডাটা অ্যানালাইসিস, ভিজুয়ালাইজেশন, Markdown সাপোর্ট।
- **ইন্সটল:**
  1. পাইথন ইন্সটল হলে Terminal-এ লিখুন:
     ```bash
     pip install notebook
     ```
  2. রান করুন:
     ```bash
     jupyter notebook
     ```
  3. ব্রাউজারে নতুন Notebook খুলুন, কোড লিখুন ও রান করুন।

## বাস্তব উদাহরণ

### VS Code-এ "Hello, World!" রান করা
```python
print("Hello, World!")
```

### Jupyter Notebook-এ কোড ও Markdown
```python
# কোড সেল
x = 5
y = 10
print(x + y)
```

Markdown সেল:
```
## পাইথন যোগফল উদাহরণ
উপরের কোডে x ও y যোগ করা হয়েছে।
```

## IDE বাছাইয়ের টিপস
- **VS Code:** যারা লাইটওয়েট, কাস্টমাইজেবল, এবং একাধিক ভাষা নিয়ে কাজ করতে চান।
- **PyCharm:** যারা শুধুমাত্র পাইথন নিয়ে কাজ করেন এবং বড় প্রজেক্টে ডিবাগিং, টেস্টিং দরকার।
- **Jupyter:** ডাটা সায়েন্স, এক্সপ্লোরেটরি অ্যানালাইসিস, ও রিপোর্টিং-এর জন্য আদর্শ।

## উপসংহার

পাইথন শেখার জন্য সঠিক পরিবেশ তৈরি করা অত্যন্ত গুরুত্বপূর্ণ। আপনার কাজের ধরন ও পছন্দ অনুযায়ী IDE বেছে নিন এবং অনুশীলন শুরু করুন। পরবর্তী পাঠে আমরা প্রথম পাইথন প্রোগ্রাম নিয়ে আলোচনা করব।
