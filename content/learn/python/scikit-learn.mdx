# Scikit-learn দিয়ে মেশিন লার্নিং বেসিকস

## ভূমিকা

মেশিন লার্নিং (Machine Learning) হচ্ছে এমন একটি প্রযুক্তি, যেখানে কম্পিউটার নিজে নিজে ডেটা থেকে শিখে সিদ্ধান্ত নিতে পারে। Scikit-learn (স্কিকিট-লার্ন) পাইথনের সবচেয়ে জনপ্রিয় ও সহজ লাইব্রেরি, যা দিয়ে মেশিন লার্নিংয়ের বিভিন্ন টাস্ক—যেমন ক্লাসিফিকেশন, রিগ্রেশন, ক্লাস্টারিং—সহজে করা যায়।

ভাবুন, আপনি একজন শিক্ষক। প্রতিদিন ছাত্রদের নম্বর দেখে অনুমান করেন, কে ভালো করবে, কে একটু পিছিয়ে আছে। ঠিক একইভাবে, মেশিন লার্নিং মডেল ডেটা দেখে ভবিষ্যৎ অনুমান করতে পারে।

## Scikit-learn পরিচিতি

Scikit-learn হচ্ছে ওপেন-সোর্স পাইথন লাইব্রেরি, যা ডেটা মাইনিং ও ডেটা অ্যানালাইসিসের জন্য ব্যবহৃত হয়। এতে রয়েছে প্রচুর মেশিন লার্নিং অ্যালগরিদম—যেমন Decision Tree, Random Forest, SVM, KNN, Linear Regression ইত্যাদি।

### ইনস্টলেশন

```bash
pip install scikit-learn
```

## মেশিন লার্নিংয়ের ধাপসমূহ

১. **ডেটা সংগ্রহ ও প্রস্তুতি:** ডেটা সংগ্রহ, ক্লিনিং, ও ফিচার সিলেকশন।
২. **মডেল নির্বাচন:** কোন অ্যালগরিদম ব্যবহার করবেন—ক্লাসিফিকেশন, রিগ্রেশন, ক্লাস্টারিং।
৩. **মডেল ট্রেনিং:** ডেটা দিয়ে মডেলকে "শিক্ষা" দেওয়া।
৪. **মডেল টেস্টিং ও ইভালুয়েশন:** নতুন ডেটা দিয়ে মডেল কতটা ভালো কাজ করছে, তা যাচাই।
৫. **প্রেডিকশন:** মডেল দিয়ে ভবিষ্যৎ অনুমান করা।

## বাস্তব উদাহরণ: ছাত্রদের নম্বর থেকে পাশ/ফেল প্রেডিকশন

ধরা যাক, আমাদের কাছে ছাত্রদের নম্বর আছে, এবং আমরা জানতে চাই, কে পাশ করবে, কে ফেল করবে।

### ডেটাসেট

| নাম   | নম্বর | পাশ/ফেল |
|-------|------|---------|
| রাহুল | ৮০   | পাশ     |
| সুমন  | ৪৫   | ফেল     |
| রিমা  | ৬৫   | পাশ     |
| তানিয়া| ৩০   | ফেল     |

### কোড উদাহরণ

```python
from sklearn.tree import DecisionTreeClassifier

# ডেটা প্রস্তুতি
X = [[80], [45], [65], [30]]  # নম্বর
y = ['পাশ', 'ফেল', 'পাশ', 'ফেল']  # টার্গেট

# মডেল তৈরি ও ট্রেনিং
model = DecisionTreeClassifier()
model.fit(X, y)

# নতুন ছাত্রের নম্বর দিয়ে প্রেডিকশন
result = model.predict([[70]])
print("৭০ নম্বর পেলে:", result[0])  # Output: 'পাশ'
```

### ব্যাখ্যা

এখানে DecisionTreeClassifier ব্যবহার করা হয়েছে। মডেল নম্বর দেখে শিখেছে, ৫০-এর বেশি নম্বর পেলে সাধারণত পাশ হয়। এরপর নতুন নম্বর দিয়ে মডেলকে প্রশ্ন করা হয়েছে—৭০ নম্বর পেলে কী হবে?

## ক্লাসিফিকেশন বনাম রিগ্রেশন

- **ক্লাসিফিকেশন:** ডেটাকে ক্যাটাগরিতে ভাগ করা (পাশ/ফেল, কুকুর/বিড়াল)।
- **রিগ্রেশন:** সংখ্যাগত মান অনুমান করা (বাড়ির দাম, ছাত্রের নম্বর)।

### রিগ্রেশন উদাহরণ

```python
from sklearn.linear_model import LinearRegression

# ডেটা
X = [[1], [2], [3], [4]]  # দিন
y = [100, 200, 300, 400]  # বিক্রি

model = LinearRegression()
model.fit(X, y)

# ৫ম দিনে বিক্রি কত হতে পারে?
prediction = model.predict([[5]])
print("৫ম দিনে সম্ভাব্য বিক্রি:", prediction[0])
```

## ডেটা ক্লিনিং ও ফিচার সিলেকশন

বাস্তব জীবনে ডেটা সবসময় পরিষ্কার থাকে না। Scikit-learn-এর সাথে Pandas ব্যবহার করে ডেটা ক্লিনিং করা যায়। যেমন, Missing Value, Outlier, ডেটা স্কেলিং ইত্যাদি।

```python
from sklearn.preprocessing import StandardScaler

data = [[10], [20], [30], [40]]
scaler = StandardScaler()
scaled_data = scaler.fit_transform(data)
print(scaled_data)
```

## মডেল ইভালুয়েশন

মডেল কতটা ভালো কাজ করছে, তা যাচাই করতে Accuracy, Precision, Recall, F1 Score ইত্যাদি ব্যবহার করা হয়।

```python
from sklearn.metrics import accuracy_score

y_true = ['পাশ', 'ফেল', 'পাশ']
y_pred = ['পাশ', 'ফেল', 'ফেল']

print("Accuracy:", accuracy_score(y_true, y_pred))
```
