# ভ্যারিয়েবল স্কোপ ও ল্যাম্বডা ফাংশন

পাইথনে ভ্যারিয়েবল স্কোপ ও ল্যাম্বডা ফাংশন—দুইটি গুরুত্বপূর্ণ কনসেপ্ট, যা কোডের গঠন ও কার্যকারিতা বোঝার জন্য অপরিহার্য।

## ভ্যারিয়েবল স্কোপ

স্কোপ মানে "পরিসর"—কোথায় কোন ভ্যারিয়েবল অ্যাক্সেস করা যাবে।

### ১. গ্লোবাল স্কোপ
গ্লোবাল ভ্যারিয়েবল পুরো ফাইলে, সব ফাংশনে অ্যাক্সেসযোগ্য।
```python
x = 10  # গ্লোবাল

def show():
    print(x)

show()  # আউটপুট: 10
```

### ২. লোকাল স্কোপ
লোকাল ভ্যারিয়েবল শুধু ফাংশনের ভিতরে অ্যাক্সেসযোগ্য।
```python
def demo():
    y = 5  # লোকাল
    print(y)

demo()  # আউটপুট: 5
#print(y)  # ভুল, বাইরে অ্যাক্সেস করা যাবে না
```

### ৩. গ্লোবাল কীওয়ার্ড
ফাংশনের ভিতরে গ্লোবাল ভ্যারিয়েবল পরিবর্তন করতে চাইলে `global` ব্যবহার করতে হয়।
```python
count = 0

def increase():
    global count
    count += 1

increase()
print(count)  # আউটপুট: 1
```

### ৪. ননলোকাল স্কোপ (nested function)
নেস্টেড ফাংশনে বাইরের লোকাল ভ্যারিয়েবল পরিবর্তন করতে `nonlocal` ব্যবহার হয়।
```python
def outer():
    msg = "Hello"
    def inner():
        nonlocal msg
        msg = "Hi"
    inner()
    print(msg)

outer()  # আউটপুট: Hi
```

## ল্যাম্বডা ফাংশন

ল্যাম্বডা ফাংশন হচ্ছে "ছোট ফাংশন"—এক লাইনে সংক্ষিপ্তভাবে লেখা যায়।

### সাধারণ ফর্ম
```python
lambda arguments: expression
```

### উদাহরণ:
```python
add = lambda a, b: a + b
print(add(5, 7))  # আউটপুট: 12
```

### লিস্টে ল্যাম্বডা ব্যবহার (map, filter)
```python
numbers = [1, 2, 3, 4]
squared = list(map(lambda x: x**2, numbers))
print(squared)  # আউটপুট: [1, 4, 9, 16]

# filter
evens = list(filter(lambda x: x % 2 == 0, numbers))
print(evens)  # আউটপুট: [2, 4]
```

## বাস্তব উদাহরণ

### ১. স্কোপ ভুল হলে
```python
def test():
    a = 5
print(a)  # ভুল, কারণ a লোকাল
```

### ২. ল্যাম্বডা দিয়ে sort
```python
students = [
    {"name": "Anis", "score": 90},
    {"name": "Bithi", "score": 85}
]
sorted_students = sorted(students, key=lambda s: s["score"], reverse=True)
print(sorted_students)
```

## ভুল ও সমাধান

- ভুল: গ্লোবাল ভ্যারিয়েবল ফাংশনের ভিতরে পরিবর্তন করতে চাইলে global ব্যবহার না করা
- ভুল: লোকাল ভ্যারিয়েবল বাইরে অ্যাক্সেস করা
- ভুল: ল্যাম্বডা ফাংশনে একাধিক স্টেটমেন্ট লেখা (ল্যাম্বডা শুধু এক্সপ্রেশন)

## অনুশীলন

১. একটি ফাংশনের ভিতরে লোকাল ও গ্লোবাল ভ্যারিয়েবল ব্যবহার করুন।
২. ল্যাম্বডা দিয়ে দুটি সংখ্যা গুণ করুন।
৩. লিস্ট থেকে শুধু জোড় সংখ্যা বের করতে filter ও lambda ব্যবহার করুন।

## উপসংহার

স্কোপ ও ল্যাম্বডা ফাংশন—দুইটি কনসেপ্ট পাইথনের কোড সংগঠন ও কার্যকারিতা বোঝার জন্য অত্যন্ত গুরুত্বপূর্ণ। এগুলো ভালোভাবে আয়ত্ত করলে কোড আরও পরিষ্কার ও শক্তিশালী হবে। পরবর্তী পাঠে লিস্ট নিয়ে আলোচনা করব।
