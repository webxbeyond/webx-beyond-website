Flask হচ্ছে পাইথনের একটি জনপ্রিয় ওয়েব ফ্রেমওয়ার্ক, যা দিয়ে খুব সহজে ওয়েব অ্যাপ্লিকেশন তৈরি করা যায়। Django-এর তুলনায় Flask অনেক হালকা, সহজ, এবং দ্রুত প্রোটোটাইপিংয়ের জন্য আদর্শ।

ভাবুন, আপনি একটি স্কুলের ওয়েবসাইট বানাতে চান—ছাত্রদের তথ্য দেখানো, ফর্মে ডেটা নেওয়া, বা রেজাল্ট প্রকাশ করা। Flask দিয়ে কয়েক লাইনের কোডেই এসব করা যায়।

---

## Flask কী?

Flask হচ্ছে পাইথনের মাইক্রো-ফ্রেমওয়ার্ক, যা ওয়েব সার্ভার, রাউটিং, টেমপ্লেট, ফর্ম হ্যান্ডলিং—সবকিছু সহজে করতে দেয়।

### ইনস্টলেশন

```bash
pip install flask
```

---

## Flask অ্যাপের বেসিক স্ট্রাকচার

```python
from flask import Flask
app = Flask(__name__)

@app.route('/')
def home():
    return "হ্যালো, Flask!"

if __name__ == '__main__':
    app.run(debug=True)
```

### ব্যাখ্যা
- `Flask(__name__)`: অ্যাপ তৈরি
- `@app.route('/')`: হোমপেজের URL
- `home()`: ফাংশনটি হোমপেজে কল হয়
- `app.run(debug=True)`: সার্ভার চালু

## রাউটিং: URL-এর সাথে ফাংশন যুক্ত করা

Flask-এ রাউটিং মানে—নির্দিষ্ট URL-এ গেলে কোন ফাংশন চলবে তা নির্ধারণ করা।

```python
@app.route('/about')
def about():
    return "এটি অ্যাবাউট পেজ"
```

### বাস্তব উদাহরণ

স্কুল ওয়েবসাইটে `/students`, `/results`, `/contact`—প্রতিটি URL-এ আলাদা ফাংশন চলবে।

## টেমপ্লেটস: HTML ও ডাইনামিক কনটেন্ট

Flask-এ Jinja2 টেমপ্লেট ইঞ্জিন ব্যবহার হয়, যাতে HTML ফাইলের মধ্যে পাইথন ভ্যারিয়েবল, লুপ, কন্ডিশন ব্যবহার করা যায়।

### টেমপ্লেট ফোল্ডার
- `templates/` ফোল্ডারে HTML ফাইল রাখতে হয়।

### উদাহরণ:

`templates/home.html`:
```html
<!DOCTYPE html>
<html>
<head><title>হোম</title></head>
<body>
    <h1>স্বাগতম, {{ name }}!</h1>
</body>
</html>
```

`app.py`:
```python
from flask import Flask, render_template
app = Flask(__name__)

@app.route('/user/<name>')
def user(name):
    return render_template('home.html', name=name)
```

## ফর্ম হ্যান্ডলিং: ইউজার ইনপুট নেওয়া

Flask দিয়ে HTML ফর্ম থেকে ডেটা নেওয়া, প্রসেস করা, ও রেসপন্স দেখানো যায়।

### উদাহরণ:

`templates/form.html`:
```html
<form method="POST">
    নাম: <input type="text" name="username">
    <input type="submit" value="জমা দিন">
</form>
```

`app.py`:
```python
from flask import Flask, request, render_template
app = Flask(__name__)

@app.route('/form', methods=['GET', 'POST'])
def form():
    if request.method == 'POST':
        username = request.form['username']
        return f"স্বাগতম, {username}!"
    return render_template('form.html')
```

## Analogy

Flask-কে ভাবুন, ওয়েব অ্যাপের জন্য "ব্ল্যাঙ্ক ক্যানভাস"—আপনি নিজের মতো করে সবকিছু আঁকতে পারেন। Django-তে অনেক কিছু আগে থেকেই থাকে, Flask-এ আপনি নিজের মতো করে গড়তে পারেন।
