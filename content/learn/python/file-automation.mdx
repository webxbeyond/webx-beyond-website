## ভূমিকা

পাইথন দিয়ে ফাইল ও ফোল্ডার অটোমেশন মানে—কম্পিউটারে ফাইল তৈরি, কপি, মুভ, ডিলিট, ফোল্ডার ম্যানেজমেন্ট—সবকিছু কোডে করা যায়। অফিস, স্কুল, ডেটা অ্যানালাইসিস—সবখানে ফাইল অটোমেশন সময় ও শ্রম বাঁচায়।

ভাবুন, আপনি প্রতিদিন শত শত রিপোর্ট ফোল্ডারে রাখতে চান, পুরনো ফাইল ডিলিট করতে চান, বা নতুন ফোল্ডার বানাতে চান—পাইথন দিয়ে এক ক্লিকে সব সম্ভব।

---

## os ও shutil মডিউল

- **os:** ফাইল, ফোল্ডার, সিস্টেম অপারেশন
- **shutil:** ফাইল/ফোল্ডার কপি, মুভ, ডিলিট

### ইনস্টলেশন

os ও shutil পাইথনের বিল্ট-ইন, আলাদা ইনস্টল দরকার নেই।

---

## ফাইল তৈরি, পড়া, লেখা

```python
# ফাইল তৈরি
with open('report.txt', 'w') as f:
    f.write('আজকের রিপোর্ট')

# ফাইল পড়া
with open('report.txt', 'r') as f:
    print(f.read())
```

---

## ফোল্ডার তৈরি ও চেক

```python
import os

# ফোল্ডার তৈরি
os.makedirs('reports', exist_ok=True)

# ফোল্ডার আছে কিনা চেক
if os.path.exists('reports'):
    print('ফোল্ডার আছে')
```

---

## ফাইল/ফোল্ডার কপি, মুভ, ডিলিট

```python
import shutil

# ফাইল কপি
shutil.copy('report.txt', 'reports/report.txt')

# ফোল্ডার কপি
shutil.copytree('reports', 'backup_reports')

# ফাইল মুভ
shutil.move('report.txt', 'reports/report.txt')

# ফাইল/ফোল্ডার ডিলিট
os.remove('reports/report.txt')
shutil.rmtree('backup_reports')
```

---

## ফাইল লিস্টিং ও ফিল্টার

```python
files = os.listdir('reports')
for file in files:
    if file.endswith('.txt'):
        print(file)
```

---

## ফাইল সাইজ, টাইম, পারমিশন

```python
file = 'reports/report.txt'
print('সাইজ:', os.path.getsize(file))
print('লাস্ট মডিফাই:', os.path.getmtime(file))
print('পারমিশন:', os.access(file, os.R_OK))
```

## বাস্তব উদাহরণ: মাসিক রিপোর্ট অটোমেশন

ধরা যাক, প্রতি মাসে নতুন ফোল্ডার তৈরি, রিপোর্ট কপি, পুরনো রিপোর্ট ডিলিট—সবকিছু পাইথন দিয়ে অটোমেট করা যায়।

---

## Analogy

os ও shutil-কে ভাবুন, কম্পিউটারের "রোবট হেল্পার"—আপনার নির্দেশে ফাইল/ফোল্ডার ম্যানেজ করে।

---

## os ও shutil-এর সুবিধা

- দ্রুত ফাইল/ফোল্ডার ম্যানেজমেন্ট
- বড় ডেটা প্রসেসিং সহজ
- ব্যাচ অপারেশন ও অটোমেশন
- ক্রস-প্ল্যাটফর্ম (Windows, Mac, Linux)

## উপসংহার

os ও shutil দিয়ে ফাইল ও ফোল্ডার অটোমেশন সহজ, দ্রুত, এবং শক্তিশালী। অফিস, স্কুল, ডেটা অ্যানালাইসিস—সবখানে অটোমেশন সময় ও শ্রম বাঁচায়। পরবর্তী পাঠে ওয়েব স্ক্র্যাপিং নিয়ে আলোচনা হবে।
