# ভ্যারিয়েবল, ডাটা টাইপ ও টাইপ কনভার্সন

পাইথনের বেসিক সিনট্যাক্স শেখার অন্যতম গুরুত্বপূর্ণ অংশ হলো ভ্যারিয়েবল, ডাটা টাইপ এবং টাইপ কনভার্সন। এই পাঠে আমরা এগুলো নিয়ে বিস্তারিত আলোচনা করব, বাস্তব উদাহরণ ও সহজ ভাষায়।

## ভ্যারিয়েবল কী?

ভ্যারিয়েবল হচ্ছে একটি "বাক্স" যেখানে আপনি ডাটা রাখতে পারেন। যেমন স্কুলব্যাগে বই রাখার মতো, ভ্যারিয়েবলে আপনি সংখ্যা, টেক্সট, বা অন্য কোনো ডাটা রাখতে পারেন।

### ভ্যারিয়েবল ডিক্লেয়ার ও অ্যাসাইন
```python
name = "Anis"      # স্ট্রিং টাইপ
age = 25            # ইন্টিজার টাইপ
height = 5.7        # ফ্লোট টাইপ
is_student = True   # বুলিয়ান টাইপ
```

- **name, age, height, is_student** — এগুলো ভ্যারিয়েবল
- **=** — অ্যাসাইনমেন্ট অপারেটর

## পাইথনের ডাটা টাইপ

পাইথনে বিভিন্ন ধরনের ডাটা টাইপ আছে:

| টাইপ        | উদাহরণ         | ব্যাখ্যা                |
|-------------|----------------|------------------------|
| int         | 10, -5         | পূর্ণ সংখ্যা           |
| float       | 3.14, -0.5     | দশমিক সংখ্যা           |
| str         | "Hello"        | টেক্সট/স্ট্রিং         |
| bool        | True, False    | সত্য/মিথ্যা           |
| list        | [1, 2, 3]      | একাধিক আইটেমের তালিকা |
| tuple       | (1, 2, 3)      | অপরিবর্তনীয় তালিকা    |
| dict        | `{"a": 1}`      | কী-ভ্যালু পেয়ার        |
| set         | {1, 2, 3}      | ইউনিক আইটেমের সেট     |

### উদাহরণ:
```python
score = 90              # int
price = 99.99           # float
message = "Hi!"         # str
passed = True           # bool
fruits = ["apple", "banana"]  # list
tuple_example = (1, 2)  # tuple
info = {"name": "Anis", "age": 25}  # dict
unique_numbers = {1, 2, 3}  # set
```

## টাইপ চেক করা

`type()` ফাংশন দিয়ে ভ্যারিয়েবলের টাইপ চেক করা যায়:
```python
x = 5
print(type(x))  # <class 'int'>
```

## টাইপ কনভার্সন

কখনো কখনো এক ধরনের ডাটা অন্য ধরনের ডাটায় রূপান্তর করতে হয়। একে বলে টাইপ কনভার্সন।

### উদাহরণ:
```python
# int থেকে float
num = 10
num_float = float(num)  # 10.0

# str থেকে int
s = "25"
num2 = int(s)           # 25

# float থেকে str
f = 3.14
f_str = str(f)          # "3.14"
```

### বাস্তব উদাহরণ:
ধরা যাক, ইউজার থেকে ইনপুট নিচ্ছেন:
```python
age = input("আপনার বয়স লিখুন: ")  # str টাইপ
age_int = int(age)                # int টাইপে কনভার্ট
print(age_int + 5)
```

## ভুল ও সমাধান

- ভুল: টাইপ কনভার্সন না করলে
  ```python
  age = input("বয়স?")
  print(age + 5)  # ভুল, কারণ age str
  ```
  সমাধান:
  ```python
  age = input("বয়স?")
  print(int(age) + 5)
  ```

## অনুশীলন

১. নিজের নাম, বয়স, এবং পছন্দের সংখ্যা ভ্যারিয়েবল হিসেবে সংরক্ষণ করুন।
২. একটি স্ট্রিংকে ইন্টিজারে কনভার্ট করুন এবং যোগফল বের করুন।
৩. একটি লিস্ট, টুপল, ডিকশনারি ও সেট তৈরি করুন।

## উপসংহার

ভ্যারিয়েবল, ডাটা টাইপ ও টাইপ কনভার্সন—এই তিনটি কনসেপ্ট পাইথনের ভিত্তি। এগুলো ভালোভাবে আয়ত্ত করলে পরবর্তী সিনট্যাক্স ও লজিক সহজ হবে। পরবর্তী পাঠে আমরা অপারেটরস নিয়ে আলোচনা করব।
