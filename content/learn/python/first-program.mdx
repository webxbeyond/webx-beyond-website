# প্রথম পাইথন প্রোগ্রাম: `print("Hello, World!")`

পাইথন শেখার যাত্রা শুরু হয় একটি ছোট্ট, কিন্তু ঐতিহাসিক প্রোগ্রাম দিয়ে—`print("Hello, World!")`। এটি শুধু একটি টেক্সট স্ক্রিনে দেখানোর কোড নয়, বরং প্রোগ্রামিংয়ের জগতে প্রথম পদক্ষেপ।

## কেন "Hello, World!"?

"Hello, World!" প্রোগ্রামটি বিশ্বের সব প্রোগ্রামিং ভাষার জন্য একটি রীতি। এটি নতুন ভাষা শেখার সময় পরিবেশ, সিনট্যাক্স, এবং কোড রান করার প্রক্রিয়া যাচাই করার সহজ উপায়।

### উদাহরণ ও ব্যাখ্যা

```python
print("Hello, World!")
```

- **print:** পাইথনের বিল্ট-ইন ফাংশন, যা স্ক্রিনে টেক্সট দেখায়।
- **"Hello, World!":** ডাবল কোটেশনের মধ্যে থাকা স্ট্রিং, যা স্ক্রিনে দেখানো হবে।

## কোড রান করার ধাপ

### ১. ফাইল তৈরি
- আপনার IDE (VS Code, PyCharm, Jupyter) খুলুন।
- নতুন ফাইল তৈরি করুন: `hello.py`
- কোড লিখুন:
  ```python
  print("Hello, World!")
  ```

### ২. রান করুন
- **VS Code/Terminal:**
  ```bash
  python hello.py
  ```
- **PyCharm:** Run বাটন ক্লিক করুন।
- **Jupyter Notebook:** নতুন সেলে কোড লিখে "Run" চাপুন।

### ৩. আউটপুট
```
Hello, World!
```

## বাস্তব উদাহরণ ও অনুপ্রেরণা

ধরা যাক, আপনি নতুন একটি ভাষা শিখছেন। "Hello, World!" হচ্ছে সেই প্রথম কথা, যা আপনি নতুন ভাষায় বলছেন। এটি ঠিক যেমন শিশুরা প্রথম "মা" বা "বাবা" বলে, তেমনি প্রোগ্রামিংয়ে "Hello, World!"।

## print ফাংশনের আরও ব্যবহার

`print()` শুধু টেক্সট নয়, ভ্যারিয়েবল, গণনা, এবং একাধিক আইটেমও দেখাতে পারে।

### উদাহরণ ১: ভ্যারিয়েবল
```python
name = "Anis"
print("Hello,", name)
```

### উদাহরণ ২: গণনা
```python
print(5 + 7)
```

### উদাহরণ ৩: একাধিক আইটেম
```python
print("Python", 2025, True)
```

## ভুল ও সমাধান

- ভুল: কোটেশন ভুল হলে
  ```python
  print(Hello, World!)  # ভুল
  ```
  সমাধান: স্ট্রিং কোটেশনের মধ্যে রাখতে হবে
  ```python
  print("Hello, World!")
  ```

## অনুশীলন

১. নিজের নাম দিয়ে "Hello, [আপনার নাম]!" প্রিন্ট করুন।
২. দুটি সংখ্যা যোগ করে প্রিন্ট করুন।
৩. একাধিক ভিন্ন টাইপের ডাটা (স্ট্রিং, সংখ্যা, বুলিয়ান) একসাথে প্রিন্ট করুন।

## উপসংহার

"Hello, World!" দিয়ে শুরু হলেও, এটি পাইথনের সিনট্যাক্স, ফাংশন, এবং আউটপুট বোঝার প্রথম ধাপ। পরবর্তী পাঠে আমরা ভ্যারিয়েবল, ডাটা টাইপ ও টাইপ কনভার্সন নিয়ে আলোচনা করব।
