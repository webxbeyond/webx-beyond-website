Postman হলো API টেস্টিং, ডকুমেন্টেশন ও শেয়ারিংয়ের জন্য সবচেয়ে জনপ্রিয় টুল। কালেকশন (Collection) ফিচার দিয়ে একাধিক API রিকোয়েস্ট, টেস্ট ও ডকুমেন্টেশন একসাথে সংরক্ষণ ও শেয়ার করা যায়।

## Postman-এর সুবিধা
- ইন্টারফেসে সহজে API রিকোয়েস্ট তৈরি ও টেস্ট
- কালেকশন দিয়ে প্রজেক্টভিত্তিক API গ্রুপিং
- অটোমেটেড টেস্ট, স্ক্রিপ্টিং ও এনভায়রনমেন্ট সাপোর্ট
- Mock Server, Monitoring, Documentation ফিচার

## কালেকশন (Collection) কী?
- একাধিক API রিকোয়েস্ট, টেস্ট, স্ক্রিপ্ট ও ডকুমেন্টেশন একসাথে সংরক্ষণ
- টিমে শেয়ার ও ভার্সন কন্ট্রোল সহজ

## উদাহরণ: কালেকশন তৈরি
1. New > Collection
2. রিকোয়েস্ট অ্যাড করুন (GET, POST, ইত্যাদি)
3. টেস্ট স্ক্রিপ্ট, এনভায়রনমেন্ট ভ্যারিয়েবল অ্যাড করুন
4. Export/Share করুন

## অটোমেটেড টেস্টিং
- Test ট্যাবে JavaScript দিয়ে রেসপন্স চেক করুন
```js
pm.test("Status code is 200", function () {
    pm.response.to.have.status(200);
});
```

## ডকুমেন্টেশন ও শেয়ারিং
- কালেকশন থেকে অটো-ডকুমেন্টেশন জেনারেট
- Public URL দিয়ে টিম/ক্লায়েন্টকে শেয়ার

## বেস্ট প্র্যাকটিস
- প্রতিটি API রিকোয়েস্টে টেস্ট স্ক্রিপ্ট লিখুন
- এনভায়রনমেন্ট ভ্যারিয়েবল ব্যবহার করুন (dev, prod)
- কালেকশন নিয়মিত আপডেট ও শেয়ার করুন

## টিপস
- Postman CLI (newman) দিয়ে CI/CD-তে টেস্ট অটোমেট করুন
- Mock Server ফিচার দিয়ে ফেক রেসপন্স টেস্ট করুন
- কালেকশন ডকুমেন্টেশন Public URL-এ শেয়ার করুন
