আনসিবল-এ "ইনভেন্টরি" হচ্ছে সেই তালিকা, যেখানে কোন কোন সার্ভারে কাজ হবে তা লেখা থাকে। ইনভেন্টরি দুই ধরনের—স্ট্যাটিক ও ডাইনামিক।

## ১. স্ট্যাটিক ইনভেন্টরি

স্ট্যাটিক ইনভেন্টরি হলো সাধারণত একটি ফাইল (INI, YAML, বা JSON), যেখানে সার্ভারের IP, গ্রুপ, হোস্টনেম লিখে রাখা হয়।

### উদাহরণ (INI):
```ini
[web]
192.168.1.10
192.168.1.11

[db]
192.168.1.20
```

### উদাহরণ (YAML):
```yaml
all:
  hosts:
    web1: 192.168.1.10
    web2: 192.168.1.11
  children:
    db:
      hosts:
        db1: 192.168.1.20
```

### বাস্তব উদাহরণ:

ধরা যাক, আপনার অফিসে ৫টি সার্ভার আছে, IP ফিক্সড। আপনি inventory ফাইলে লিখে রাখলেন—web, db, cache।

### অ্যানালজি:

স্ট্যাটিক ইনভেন্টরি হলো স্কুলের ছাত্রদের রোল নম্বরের তালিকা—একবার লিখে রাখলে, বারবার পরিবর্তন হয় না।

## ২. ডাইনামিক ইনভেন্টরি

ডাইনামিক ইনভেন্টরি হলো এমন inventory, যা অটোমেটিকভাবে ক্লাউড, API, স্ক্রিপ্ট, বা অন্য সোর্স থেকে সার্ভারের তালিকা নিয়ে আসে।

### উদাহরণ:
- AWS EC2 instance, GCP VM, Azure VM—সবকিছু API দিয়ে inventory-তে যোগ হয়
- Custom Python/JSON script দিয়ে inventory তৈরি

### বাস্তব উদাহরণ:

আপনার কোম্পানির AWS-এ ৫০টি VM আছে, প্রতিদিন নতুন VM যোগ হয়, পুরাতন ডিলিট হয়। ডাইনামিক inventory দিয়ে আনসিবল অটোমেটিকভাবে সব VM-কে টার্গেট করতে পারে।

### অ্যানালজি:

ডাইনামিক ইনভেন্টরি হলো স্কুলের ছাত্রদের উপস্থিতির তালিকা—প্রতিদিন পরিবর্তন হয়, নতুন ছাত্র যোগ হয়, কেউ বাদ পড়ে।

## ৩. ডাইনামিক ইনভেন্টরি কনফিগারেশন

ডাইনামিক inventory-র জন্য আনসিবল-এ প্লাগইন, স্ক্রিপ্ট, বা API ব্যবহার করা হয়।

### AWS EC2 Plugin Example:
```ini
[defaults]
inventory = ./aws_ec2.yaml
```

`aws_ec2.yaml` ফাইলে AWS credentials, region, filters ইত্যাদি থাকে।

### Custom Script Example:
```bash
ansible-inventory -i myscript.py --list
```

## ৪. কখন কোনটা ব্যবহার করবেন?

- **স্ট্যাটিক ইনভেন্টরি:** ছোট, স্থায়ী environment, IP/host কম পরিবর্তন হয়
- **ডাইনামিক ইনভেন্টরি:** বড়, ক্লাউড environment, VM/host নিয়মিত পরিবর্তন হয়

## সংক্ষেপে

স্ট্যাটিক inventory সহজ, ছোট environment-এ উপযোগী। ডাইনামিক inventory বড়, ক্লাউড environment-এ must-have। আনসিবল-এ দুই ধরনের inventory configure করা যায়, environment অনুযায়ী সঠিকটা বেছে নিন।
