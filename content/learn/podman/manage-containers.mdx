পডম্যান কনটেইনার চালানো মাত্রই শুরু হয়। কিন্তু একবার কনটেইনার চলে গেলে, আমাদের সেটাকে নিয়ন্ত্রণ করার প্রয়োজন হয়। এই লেসনে আমরা শিখব কীভাবে **কনটেইনার শুরু, বন্ধ, রিস্টার্ট, মুছে ফেলা এবং লগ দেখা যায়।**

## ১. কনটেইনার স্টার্ট করা

ধরা যাক তুমি একটি কনটেইনার তৈরি করেছ, কিন্তু এখনো সেটা চালু হয়নি। কনটেইনার চালু করার জন্য:

```bash
podman start <container_name_or_id>
````

উদাহরণ:

```bash
podman start mynginx
```

> **টিপ:** কনটেইনার নাম বা আইডি জানা দরকার। `podman ps -a` দিয়ে সব কনটেইনার দেখতে পারো।

## ২. কনটেইনার স্টপ করা

চলমান কনটেইনার বন্ধ করতে:

```bash
podman stop <container_name_or_id>
```

উদাহরণ:

```bash
podman stop mynginx
```

> **বিশ্লেষণ:** এটি কনটেইনারকে নিরাপদভাবে বন্ধ করে। আর যদি فورসibly বন্ধ করতে চাও, `podman kill <container_name_or_id>` ব্যবহার করা যায়।

## ৩. কনটেইনার রিস্টার্ট করা

যদি কনটেইনার আবার শুরু করতে চাও:

```bash
podman restart <container_name_or_id>
```

এটি স্বয়ংক্রিয়ভাবে কনটেইনার বন্ধ করে আবার চালু করবে।

## ৪. কনটেইনার রিমুভ করা (মুছে ফেলা)

যদি আর কনটেইনারের দরকার না থাকে:

```bash
podman rm <container_name_or_id>
```

> **নোট:** কনটেইনার বন্ধ থাকতে হবে। চালু অবস্থায় থাকলে আগে `stop` করতে হবে।
> **ফোর্স মুছে ফেলার জন্য:**

```bash
podman rm -f <container_name_or_id>
```

## ৫. কনটেইনার লগ দেখা

চলমান বা বন্ধ কনটেইনারের লগ দেখার জন্য:

```bash
podman logs <container_name_or_id>
```

উদাহরণ:

```bash
podman logs mynginx
```

> **ফলো লগ:** যদি রিয়েল-টাইম লগ দেখতে চাও,
> `podman logs -f mynginx` ব্যবহার করো।

## ৬. সহজ উপমা

তুমি ভাবতে পারো কনটেইনার হচ্ছে **একটি ছোট দোকান**।

* **Start** → দোকান খোলা
* **Stop** → দোকান বন্ধ করা
* **Restart** → দোকান বন্ধ করে আবার খোলা
* **Remove** → দোকান সম্পূর্ণ ভেঙে ফেলা
* **Logs** → দোকানের কার্যকলাপ রেকর্ড দেখা

এই উপমা মনে রাখলে কনটেইনার ম্যানেজমেন্ট সহজ মনে হবে।

## ৭. প্রাকটিক্যাল এক্সারসাইজ

steps:
  - একটি নতুন কনটেইনার চালু করো, উদাহরণস্বরূপ `nginx`।
  - কনটেইনারটি চালু হওয়ার পর, তার লগ দেখো।
  - তারপর কনটেইনার বন্ধ করো।
  - কনটেইনার আবার চালু করো (restart করো)।
  - শেষে কনটেইনার মুছে ফেলো।
