# কনটেইনার চালানো (Foreground, Background, Interactive)

কনটেইনার চালানো হলো পডম্যানের সবচেয়ে মৌলিক কাজ। এটি এমনভাবে কাজ করে যেন আপনি আপনার অ্যাপ বা সার্ভিসকে একটি আলাদা পরিবেশে, হোস্ট সিস্টেমকে না ছাড়াই চালাতে পারেন। কনটেইনার চালানোর তিনটি প্রধান মোড আছে:

1. **Foreground Mode**  
2. **Background Mode**  
3. **Interactive Mode**

এবার প্রতিটা মোড বিস্তারিতভাবে দেখা যাক।

---

## ১. Foreground মোডে কনটেইনার চালানো

Foreground মোডে কনটেইনার চালানো মানে হলো কনটেইনারের আউটপুট সরাসরি আপনার টার্মিনালে দেখা যাবে এবং আপনি তাৎক্ষণিকভাবে কন্ট্রোল করতে পারবেন। এটি সাধারণত ডিবাগ বা পরীক্ষা করার জন্য ব্যবহার করা হয়।  

```bash
podman run alpine echo "হ্যালো পডম্যান"
````

**ব্যাখ্যা:**

* `alpine` হলো কনটেইনার ইমেজ।
* `echo "হ্যালো পডম্যান"` হলো সেই কমান্ড যা কনটেইনারের মধ্যে রান করবে।
* টার্মিনালে ফলাফল সরাসরি দেখা যাবে।

**Analogy:** এটি এমন, যেন আপনি রান্নাঘরে সরাসরি রান্না করছেন এবং সমস্ত প্রসেস চোখের সামনে ঘটছে।

---

## ২. Background মোডে কনটেইনার চালানো (Detached Mode)

Background মোডে কনটেইনার চালালে এটি পেছনের দিকে চলে যাবে এবং আপনি তাৎক্ষণিকভাবে টার্মিনাল থেকে কন্ট্রোল করতে পারবেন না। এটি সার্ভার বা ব্যাকগ্রাউন্ড সার্ভিসের জন্য আদর্শ।

```bash
podman run -d --name mynginx nginx
```

**ব্যাখ্যা:**

* `-d` মানে detached mode, অর্থাৎ ব্যাকগ্রাউন্ডে চালানো।
* `--name mynginx` কনটেইনারের একটি নাম দেয়।
* `nginx` হলো ইমেজ নাম।

**পরিদর্শন:**

* চলমান কনটেইনার দেখতে:

```bash
podman ps
```

**Analogy:** এটি এমন, যেন আপনি রান্না শুরু করে অন্য কাজ করতে পারেন, রান্না স্বয়ংক্রিয়ভাবে চলছে, আপনাকে প্রতিনিয়ত নজর রাখতে হবে না।

---

## ৩. Interactive মোডে কনটেইনার চালানো

Interactive মোডে কনটেইনার চালালে আপনি সরাসরি কনটেইনারের শেল বা প্রোগ্রামে প্রবেশ করতে পারেন। এটি প্রায়শই পরীক্ষামূলক বা রিয়েল-টাইম ইন্টারঅ্যাকশনের জন্য ব্যবহার করা হয়।

```bash
podman run -it alpine sh
```

**ব্যাখ্যা:**

* `-i` মানে interactive mode।
* `-t` মানে টার্মিনাল allocate করা।
* `sh` হলো shell।

এবার আপনি কনটেইনারের ভিতরে কমান্ড চালাতে পারবেন:

```bash
# apk update
# apk add vim
```

**Analogy:** এটি এমন, যেন আপনি একটি নতুন রান্নাঘরে ঢুকে নিজেই সব কিছু হাতে রান্না করছেন এবং নিয়ন্ত্রণ করছেন।

---

## প্র্যাকটিক্যাল এক্সারসাইজ

১. একটি নতুন কনটেইনার চালান যেটি foreground মোডে "Hello Podman" মেসেজ দেখায়।
২. একটি nginx কনটেইনার ব্যাকগ্রাউন্ডে চালান এবং নিশ্চিত করুন এটি চালু আছে।
৩. একটি alpine কনটেইনার interactive মোডে চালান এবং ভিতরে `ls` ও `echo "Podman Test"` চালান।
