## পরিচিতি

পড লাইফসাইকেল ম্যানেজমেন্ট মানে হলো **পডের জন্ম থেকে মৃত্যু পর্যন্ত পুরো জীবনচক্র নিয়ন্ত্রণ করা**।  
Podman পড তৈরি, চালনা, স্টপ করা, পুনরায় শুরু করা এবং মুছে ফেলার মাধ্যমে এটি সম্ভব।  

**অনালজি:**  
পডকে একটি গাড়ির মতো ভাবুন।  
- গাড়ি তৈরি (create) → পড তৈরি।  
- চালানো (start) → পড চালু।  
- থামানো (stop) → পড বন্ধ।  
- পুনরায় চালু (restart) → পড আবার চালু।  
- ডিলিট (remove) → গাড়ি বিক্রি বা বাতিল।  

---

## ১. পড তৈরি করা

```bash
podman pod create --name lifepod -p 8085:80
````

* `lifepod` নামের পড তৈরি হলো।
* এখন পড **Created** স্টেটে আছে।

### ১.১ স্ট্যাটাস দেখা

```bash
podman pod ps -a
```

* `STATUS` কলামে `Created` দেখাবে।

---

## ২. পড চালানো (Start)

```bash
podman pod start lifepod
```

* পড এবং এর সব কনটেইনার চালু হবে।
* `podman pod ps` কমান্ড দিয়ে `Running` স্ট্যাটাস দেখা যাবে।

---

## ৩. পড বন্ধ করা (Stop)

```bash
podman pod stop lifepod
```

* পডের সব কনটেইনার বন্ধ হবে।
* `podman pod ps -a` → `Exited` স্ট্যাটাস দেখাবে।

---

## ৪. পড পুনরায় চালু করা (Restart)

```bash
podman pod restart lifepod
```

* পড আবার চালু হবে।
* চলমান অবস্থায় সব কনটেইনার পুনরায় শুরু হবে।

---

## ৫. পড মুছে ফেলা (Remove)

```bash
podman pod rm lifepod
```

* পড এবং এর সব কনটেইনার পুরোপুরি ডিলিট হবে।
* মুছে ফেলার আগে পড অবশ্যই বন্ধ থাকতে হবে।

**Optional:**

```bash
podman pod rm -f lifepod
```

* `-f` ফ্ল্যাগ দিয়ে চালু থাকা পডও জোরপূর্বক মুছে ফেলা যায়।

---

## ৬. পড লাইফসাইকেল কমান্ড সংক্ষিপ্ত চেকলিস্ট

| ক্রিয়া      | কমান্ড               | স্টেট পরিবর্তন |
| ------------ | -------------------- | -------------- |
| তৈরি         | `podman pod create`  | Created        |
| চালানো       | `podman pod start`   | Running        |
| বন্ধ করা     | `podman pod stop`    | Exited         |
| পুনরায় চালু | `podman pod restart` | Running        |
| মুছে ফেলা    | `podman pod rm`      | Deleted        |

---

## ব্যবহারিক উদাহরণ

**চেষ্টা করুন:**

1. একটি পড তৈরি করুন `lifecyclepod`।
2. পড চালু করুন এবং `podman pod ps` দিয়ে পরীক্ষা করুন।
3. পড বন্ধ করুন।
4. পুনরায় চালু করুন।
5. পড মুছে ফেলুন।

**কমান্ড উদাহরণ:**

```bash
podman pod create --name lifecyclepod -p 8086:80
podman pod start lifecyclepod
podman pod ps
podman pod stop lifecyclepod
podman pod restart lifecyclepod
podman pod rm lifecyclepod
```

## উপসংহার

পডের লাইফসাইকেল ম্যানেজমেন্ট বোঝা হলে আপনি:

* পড তৈরি, চালনা, স্টপ, পুনরায় চালু এবং মুছে ফেলা সম্পূর্ণভাবে নিয়ন্ত্রণ করতে পারবেন।
* মাল্টি-কনটেইনার পডের কার্যকারিতা আরও দক্ষভাবে পরিচালনা করতে পারবেন।
