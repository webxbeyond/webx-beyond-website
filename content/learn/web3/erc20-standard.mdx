# ERC-20 টোকেন স্ট্যান্ডার্ড — ফাঞ্জিবল টোকেনের ভিত্তি ও বাস্তব ব্যবহার

## পরিচিতি

ERC-20 হলো—ইথেরিয়াম ব্লকচেইনে ফাঞ্জিবল (Fungible) টোকেন তৈরির জন্য সবচেয়ে জনপ্রিয় ও স্ট্যান্ডার্ড কন্ট্রাক্ট। Google বা Netflix যেমন ইউজারদের জন্য পয়েন্ট/ক্রেডিট সিস্টেম ব্যবহার করে, ঠিক তেমনই ERC-20 টোকেন দিয়ে Web3-এ ডিজিটাল কারেন্সি, পয়েন্ট, বা রিওয়ার্ড তৈরি করা যায়। এই পাঠে জানব—ERC-20 কী, কেন দরকার, এবং কিভাবে ব্যবহার করতে হয়।

## Core Concepts

### ১. ERC-20 কী?
- "Ethereum Request for Comment" (ERC) — টোকেন স্ট্যান্ডার্ড
- Fungible Token: প্রতিটি টোকেন একে অপরের সমান
- Standard Functions: transfer, approve, balanceOf, totalSupply
- Interoperability: Exchange, Wallet, dApp

### ২. Workflow
1. Contract লিখুন (Solidity)
2. ERC-20 Interface Implement করুন
3. Deploy করুন (Testnet/Mainnet)
4. Token Transfer, Approve, Spend করুন
5. Exchange, Wallet-এ Integrate করুন

### ৩. Solidity ERC-20 Example
```solidity
pragma solidity ^0.8.0;

interface IERC20 {
    function totalSupply() external view returns (uint256);
    function balanceOf(address account) external view returns (uint256);
    function transfer(address recipient, uint256 amount) external returns (bool);
    function approve(address spender, uint256 amount) external returns (bool);
    function allowance(address owner, address spender) external view returns (uint256);
    function transferFrom(address sender, address recipient, uint256 amount) external returns (bool);
}
```

### ৪. Real-life Use Cases
- **Stablecoin:** USDT, USDC
- **Exchange Token:** BNB, UNI
- **Reward Point:** Loyalty Program
- **Gaming Currency:** In-game Token

### ৫. Analogies
- ERC-20—"ডিজিটাল টাকা" (Digital Money)
- transfer—"বিকাশে টাকা পাঠানো"
- approve—"ক্রেডিট কার্ডে লিমিট দেওয়া"

### ৬. Common Pitfalls
- Contract bug—Token loss
- Approve/transfer mistake—Double spend
- Low gas—Transaction fail
- No event emit—Wallet integration সমস্যা

### ৭. Best Practices
- OpenZeppelin ERC-20 Implementation ব্যবহার করুন
- Testnet-এ Deploy ও Test করুন
- Event emit করুন (Transfer, Approval)
- Security Audit করুন

### ৮. Industry Trends
- Wrapped Token (WBTC, WETH)
- Cross-chain ERC-20
- Automated Market Maker (AMM)
- Token Burn/Mint Feature

## উপসংহার

ERC-20 টোকেন—Web3 ও DeFi ইকোসিস্টেমের ভিত্তি। ডিজিটাল কারেন্সি, পয়েন্ট, ও রিওয়ার্ড—সবকিছু ERC-20 দিয়ে সহজে তৈরি ও ব্যবস্থাপনা করা যায়। Best Practices ও Industry Trends অনুসরণ করুন—তাহলেই নিরাপদ ও সফল টোকেন তৈরি সম্ভব।
