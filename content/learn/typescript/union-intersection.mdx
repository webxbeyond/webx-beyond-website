## পরিচিতি

ধরা যাক, আপনি একটি রেস্টুরেন্টে গেছেন। মেনুতে আছে "চিকেন অথবা বিফ"—আপনি যেকোনো একটি নিতে পারেন (Union)। আবার, "চিকেন + বিফ"—দুইটাই একসাথে নিতে পারেন (Intersection)। TypeScript-এ Union ও Intersection টাইপ ঠিক এমন—একটি ভ্যারিয়েবল একাধিক টাইপের হতে পারে, অথবা একাধিক টাইপ একসাথে থাকতে পারে।

## অ্যানালজি

"রেস্টুরেন্টের মেনু"—Union মানে, যেকোনো একটি; Intersection মানে, দুইটাই একসাথে।

## বিস্তারিত ব্যাখ্যা

### Union টাইপ (|)
- একাধিক টাইপের মধ্যে যেকোনো একটি হতে পারে
- উদাহরণ:
  ```typescript
  let value: string | number;
  value = "hello"; // ঠিক আছে
  value = 42; // ঠিক আছে
  value = true; // ভুল
  ```
- সাধারণত API, ফাংশন, বা ডাটা মডেল flexible করার জন্য ব্যবহার হয়

### Intersection টাইপ (&)
- একাধিক টাইপ একসাথে থাকতে হবে
- উদাহরণ:
  ```typescript
  type Person = { name: string };
  type Employee = { employeeId: number };
  type Staff = Person & Employee;
  const staff: Staff = { name: "Rahim", employeeId: 101 };
  ```
- Complex object তৈরি বা multiple feature combine করার জন্য ব্যবহার হয়

### কখন কোনটা ব্যবহার করবেন?
- Union: যখন ভিন্ন ভিন্ন টাইপের মধ্যে যেকোনো একটি দরকার
- Intersection: যখন একাধিক টাইপের সবগুলো দরকার

## মূল পয়েন্ট

- Union (|): যেকোনো একটি টাইপ
- Intersection (&): সবগুলো টাইপ একসাথে
- Flexible ও type-safe কোডের জন্য গুরুত্বপূর্ণ
