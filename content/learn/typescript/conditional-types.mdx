## পরিচিতি

ধরা যাক, আপনি একটি স্কুলে শিক্ষক। ছাত্রের নম্বর দেখে সিদ্ধান্ত নেন—পাস না ফেল। TypeScript-এ "Conditional টাইপ" ঠিক এমন—টাইপের উপর ভিত্তি করে নতুন টাইপ নির্ধারণ। আর "infer" দিয়ে টাইপ থেকে কোনো অংশ বের করে আনা যায়।

## অ্যানালজি

"স্কুলের রেজাল্ট"—নম্বর দেখে সিদ্ধান্ত (পাস/ফেল)। Conditional টাইপ মানে, টাইপের উপর ভিত্তি করে নতুন টাইপ; infer মানে, টাইপ থেকে কোনো অংশ বের করা।

## বিস্তারিত ব্যাখ্যা

### Conditional টাইপ
- টাইপের উপর ভিত্তি করে নতুন টাইপ নির্ধারণ
- Syntax: `T extends U ? X : Y`
- উদাহরণ:
  ```typescript
  type IsString<T> = T extends string ? "Yes" : "No";
  type A = IsString<string>; // "Yes"
  type B = IsString<number>; // "No"
  ```
- Generic টাইপের জন্য খুবই গুরুত্বপূর্ণ

### infer কী?
- টাইপ থেকে কোনো অংশ বের করে আনা
- সাধারণত Conditional টাইপের সাথে ব্যবহার হয়
- উদাহরণ:
  ```typescript
  type ReturnType<T> = T extends (...args: any[]) => infer R ? R : any;
  function foo(): number { return 42; }
  type Result = ReturnType<typeof foo>; // number
  ```

### কখন ব্যবহার করবেন?
- যখন টাইপের উপর ভিত্তি করে সিদ্ধান্ত নিতে হবে
- যখন টাইপ থেকে কোনো অংশ বের করতে হবে

## মূল পয়েন্ট

- Conditional টাইপ: টাইপের উপর ভিত্তি করে নতুন টাইপ
- infer: টাইপ থেকে কোনো অংশ বের করা
- Advanced টাইপ ম্যানিপুলেশনের জন্য গুরুত্বপূর্ণ
