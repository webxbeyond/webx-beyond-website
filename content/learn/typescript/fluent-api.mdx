## মূল ধারণা

### ১. Type-safe Builder Pattern

- **সংজ্ঞা:** Step-by-step configuration—type-safe object build
- **Usage:**
  ```typescript
  class UserBuilder {
    private user: { name?: string; age?: number } = {};
    setName(name: string) {
      this.user.name = name;
      return this;
    }
    setAge(age: number) {
      this.user.age = age;
      return this;
    }
    build() {
      return this.user;
    }
  }

  const user = new UserBuilder().setName("Anis").setAge(25).build();
  ```

#### সহজ এনালজি

- "Builder pattern—একটি কনস্ট্রাকশন লাইন—step-by-step item add করে final product তৈরি হয়।"

### ২. Fluent API Design

- **সংজ্ঞা:** Chainable method—readable, expressive API
- **Usage:**
  ```typescript
  class Query {
    private query: string = "";
    select(fields: string) {
      this.query += `SELECT ${fields} `;
      return this;
    }
    from(table: string) {
      this.query += `FROM ${table} `;
      return this;
    }
    where(condition: string) {
      this.query += `WHERE ${condition} `;
      return this;
    }
    build() {
      return this.query.trim();
    }
  }

  const sql = new Query().select("*").from("users").where("age > 18").build();
  ```

#### সহজ এনালজি

- "Fluent API—একটি conversation—একটি প্রশ্নের পর আরেকটি প্রশ্ন, সবশেষে উত্তর।"
