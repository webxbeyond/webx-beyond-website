## পরিচিতি

ধরা যাক, আপনি একটি বাড়ি বানাচ্ছেন—প্রথমে ভিত্তি, তারপর দেয়াল, তারপর ছাদ, তারপর দরজা-জানালা। প্রতিটি ধাপ ঠিকঠাক না হলে বাড়ি টেকসই হবে না। TypeScript-এ Type-safe Builders ও Fluent API ডিজাইন ঠিক এমনই—প্রতিটি ধাপে টাইপ সেফটি নিশ্চিত করে, যাতে ভুল হলে সঙ্গে সঙ্গে ধরা যায়।

## অ্যানালজি

Builder Pattern ও Fluent API-কে ভাবুন "লেগো ব্লক" হিসেবে—প্রতিটি ব্লক নির্দিষ্ট নিয়মে বসাতে হয়, ভুল ব্লক বসালে গঠন ভেঙে যায়। TypeScript-এর টাইপ সেফটি এই ভুলগুলো আগেই ধরতে সাহায্য করে।

## বিস্তারিত ব্যাখ্যা

### Type-safe Builders
- Builder Pattern: ধাপে ধাপে অবজেক্ট তৈরি করার পদ্ধতি
- TypeScript-এ টাইপ সেফটি দিয়ে প্রতিটি ধাপের সঠিকতা নিশ্চিত করা যায়
- ভুল ধাপে বা ভুল টাইপ দিলে কম্পাইল টাইমেই error

### Fluent API
- Fluent API: চেইন করে মেথড কল করা যায়
- প্রতিটি মেথড আগের স্টেট ধরে রাখে, ফলে কোড পড়তে ও লিখতে সহজ
- টাইপস্ক্রিপ্টে টাইপ ইনফারেন্স ও জেনেরিকস দিয়ে চেইনিং আরও শক্তিশালী

### কোড উদাহরণ

```typescript
// Type-safe Builder
class BurgerBuilder {
  private bun: string = "";
  private patty: string = "";
  private sauces: string[] = [];

  setBun(bun: string): this {
    this.bun = bun;
    return this;
  }
  setPatty(patty: string): this {
    this.patty = patty;
    return this;
  }
  addSauce(sauce: string): this {
    this.sauces.push(sauce);
    return this;
  }
  build(): { bun: string; patty: string; sauces: string[] } {
    return {
      bun: this.bun,
      patty: this.patty,
      sauces: this.sauces,
    };
  }
}

const burger = new BurgerBuilder()
  .setBun("Sesame")
  .setPatty("Chicken")
  .addSauce("Mayo")
  .addSauce("Ketchup")
  .build();

// Fluent API
class Query {
  private fields: string[] = [];
  select(...fields: string[]): this {
    this.fields.push(...fields);
    return this;
  }
  build(): string {
    return `SELECT ${this.fields.join(", ")}`;
  }
}

const query = new Query().select("name", "age").build();
```

## সুবিধা

- টাইপ সেফটি: ভুল ধাপে error
- কোড রিডেবিলিটি: চেইনিং সহজ
- API ডিজাইন: ব্যবহারকারী বান্ধব
- জটিল অবজেক্ট/কনফিগারেশন সহজে তৈরি

## ব্যবহার

- Form builder
- Query builder
- UI component builder
- Complex configuration

## গুরুত্বপূর্ণ পয়েন্ট

- জেনেরিকস ও টাইপ ইনফারেন্স ব্যবহার করুন
- প্রতিটি মেথডে `this` রিটার্ন করুন
- ভুল টাইপ বা ভুল অর্ডার হলে error দেখান

## অনুশীলন

1. একটি FormBuilder ক্লাস লিখুন, যেখানে চেইন করে ফিল্ড যোগ করা যায় এবং build করলে টাইপ সেফ অবজেক্ট পাওয়া যায়।
2. একটি Fluent API তৈরি করুন, যেখানে চেইনিং ছাড়া build করা যায় না।
3. ভুল টাইপ দিলে TypeScript কীভাবে error দেয়, দেখে নিন।
