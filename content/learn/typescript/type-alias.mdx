## পরিচিতি

ধরা যাক, আপনি একটি স্কুলের ছাত্রদের তথ্য সংরক্ষণ করছেন। প্রতিটি ছাত্রের নাম, বয়স, ক্লাস—সবকিছু আলাদাভাবে লিখতে হয়। কিন্তু যদি একই ধরনের তথ্য বারবার লিখতে হয়, তাহলে সময় ও জায়গা নষ্ট হয়। TypeScript-এ "অবজেক্ট টাইপ" দিয়ে আপনি ডাটা স্ট্রাকচার নির্ধারণ করতে পারেন, আর "টাইপ অ্যালিয়াস" দিয়ে বারবার একই টাইপ সহজে ব্যবহার করতে পারেন।

## অ্যানালজি

"ছাত্রের তথ্যের ফরম"—প্রতিবার নতুন ছাত্রের জন্য একই ফরম ব্যবহার করা যায়। অবজেক্ট টাইপ হচ্ছে সেই ফরমের কাঠামো, আর টাইপ অ্যালিয়াস হচ্ছে ফরমের নাম—যা বারবার ব্যবহার করা যায়।

## বিস্তারিত ব্যাখ্যা

- **অবজেক্ট টাইপ**
  - একাধিক প্রপার্টি/ফিল্ড নিয়ে ডাটা স্ট্রাকচার
  - উদাহরণ: `{ name: string, age: number, class: string }`
  - কোড: `let student: { name: string; age: number; class: string } = { name: "Anis", age: 25, class: "Ten" };`

- **টাইপ অ্যালিয়াস**
  - টাইপের জন্য নাম নির্ধারণ
  - বারবার একই টাইপ ব্যবহার সহজ
  - কোড:
    ```typescript
    type Student = {
      name: string;
      age: number;
      class: string;
    };
    let s: Student = { name: "Anis", age: 25, class: "Ten" };
    ```

- **Nested ও Complex টাইপ**
  - টাইপ অ্যালিয়াসে nested অবজেক্ট, অ্যারে, ফাংশন টাইপও রাখা যায়
  - কোড:
    ```typescript
    type School = {
      students: Student[];
      address: string;
    };
    ```

- **কেন ব্যবহার করবেন?**
  - কোড পরিষ্কার ও পুনঃব্যবহারযোগ্য
  - বড় প্রজেক্টে টাইপ ম্যানেজমেন্ট সহজ

## মূল পয়েন্ট

- অবজেক্ট টাইপ দিয়ে ডাটা স্ট্রাকচার নির্ধারণ
- টাইপ অ্যালিয়াস দিয়ে বারবার একই টাইপ সহজে ব্যবহার
- কোড পরিষ্কার, পুনঃব্যবহারযোগ্য ও স্কেলেবল হয়
