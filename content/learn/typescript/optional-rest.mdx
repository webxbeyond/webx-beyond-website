## মূল ধারণা

### ১. Optional Parameter

- **Parameter-এ `?`—value না দিলে undefined হবে।**
- **Usage:**
  ```typescript
  function greet(name: string, title?: string): string {
    return title ? `${title} ${name}` : name;
  }

  greet("Anis"); // "Anis"
  greet("Sara", "Dr."); // "Dr. Sara"
  ```

#### সহজ এনালজি

- "Optional parameter—একটি gift box—চাইলে বাড়তি উপহার নিতে পারেন, না নিলেও সমস্যা নেই।"

### ২. Rest Parameter

- **Parameter-এ `...`—multiple value array আকারে handle।**
- **Usage:**
  ```typescript
  function sum(...numbers: number[]): number {
    return numbers.reduce((acc, curr) => acc + curr, 0);
  }

  sum(1, 2, 3); // 6
  sum(5, 10); // 15
  ```

#### সহজ এনালজি

- "Rest parameter—একটি ব্যাগ—যত খুশি জিনিস রাখতে পারেন, সব একসাথে নিয়ে কাজ করা যায়।"

### ৩. Practical Example

```typescript
function createUser(name: string, age?: number, ...roles: string[]): object {
  return { name, age, roles };
}

const user1 = createUser("Anis");
const user2 = createUser("Sara", 30, "admin", "editor");
```
