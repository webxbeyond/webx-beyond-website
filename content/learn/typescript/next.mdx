# Next.js ও API রাউটসের সাথে TypeScript

## প্রস্তাবনা

Google-এর Search UI, Netflix-এর Streaming Dashboard, Uber-এর Real-time Map—সবখানে scalable, maintainable, এবং type-safe fullstack app দরকার। Next.js ও API routes-এ TypeScript—modern web app-এ industry-standard, যা robust, readable, এবং type-safe UI ও backend তৈরি করে।

## কেন গুরুত্বপূর্ণ?

- **Developer Experience:** Autocomplete, error checking—productivity boost

## মূল ধারণা

### ১. Next.js-এ TypeScript

- **সংজ্ঞা:** Next.js—React-based framework—type-safe UI ও API build করা যায়
- **Usage:**
  ```typescript
  // pages/index.tsx
  type Props = { name: string };
  export default function Home({ name }: Props) {
    return <div>Hello, {name}</div>;
  }
  ```

#### সহজ এনালজি

- "Next.js-এ TypeScript—একটি traffic controller—UI ও API data type-check করে।"

### ২. API Routes-এ TypeScript

- **সংজ্ঞা:** API route—type-safe request/response handle করা যায়
- **Usage:**
  ```typescript
  // pages/api/user.ts
  import type { NextApiRequest, NextApiResponse } from 'next';

  export default function handler(req: NextApiRequest, res: NextApiResponse) {
    const id: string = req.query.id as string;
    res.status(200).json({ id });
  }
  ```

#### সহজ এনালজি

- "API route—একটি security gate—সব request/response type-check করে।"

### ৩. Data Fetch ও Type Safety

- **সংজ্ঞা:** Data fetch—type-safe API call করা যায়
- **Usage:**
  ```typescript
  type User = { id: string; name: string };

  async function fetchUser(id: string): Promise<User> {
    const res = await fetch(`/api/user?id=${id}`);
    return res.json();
  }
  ```

### ৪. কখন ব্যবহার করবেন?

- UI/API/data—type-safe handle করতে
- Large-scale app—easy refactor ও maintain করতে
- Developer experience—boost করতে

### ৫. Common Pitfalls

- Wrong type—unexpected bug/error
- Optional/nullable param—type miss করলে runtime bug
- API response—type mismatch

### ৬. Best Practices

- Props/API/data—clear type define করুন
- Optional/nullable—properly handle করুন
- Documentation/readme—clear রাখুন
- Team/project standard—follow করুন
