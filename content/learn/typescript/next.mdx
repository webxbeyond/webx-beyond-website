## পরিচিতি

ধরা যাক, আপনি একটি রেস্টুরেন্ট চালাচ্ছেন—রান্নাঘর (backend), সার্ভার (API), এবং কাস্টমার (frontend) সবকিছুই সঠিকভাবে পরিচালনা করতে হয়। Next.js-এ TypeScript ব্যবহার করলে পুরো "রেস্টুরেন্ট"—UI, API, ডাটা—সবকিছুর টাইপ সেফটি নিশ্চিত হয়।

## অ্যানালজি

Next.js-এ TypeScript ব্যবহারকে ভাবুন "রেস্টুরেন্টের ম্যানেজমেন্ট সফটওয়্যার" হিসেবে—প্রতিটি অর্ডার, রান্না, ডেলিভারি, বিলিং—সবকিছুর তথ্য ও প্রসেস টাইপ সেফ, তাই ভুল কমে যায়।

## বিস্তারিত ব্যাখ্যা

### ১. Next.js কম্পোনেন্ট টাইপিং
- পেজ, লেআউট, কম্পোনেন্ট—সবকিছুর Props টাইপ নির্ধারণ
- Server-side ও Client-side—উভয় জায়গায় টাইপ সেফটি

```typescript
interface ProductProps {
  id: string;
  name: string;
}

export default function Product({ id, name }: ProductProps) {
  return <div>{name} (ID: {id})</div>;
}
```

### ২. API Route টাইপিং
- API route-এর request, response, body—সবকিছুর টাইপ নির্ধারণ
- Edge API ও Serverless function—সবকিছুর টাইপ সেফটি

```typescript
import type { NextApiRequest, NextApiResponse } from "next";

export default function handler(req: NextApiRequest, res: NextApiResponse) {
  const { id } = req.query;
  res.status(200).json({ id });
}
```

### ৩. Data Fetching ও getServerSideProps
- getServerSideProps, getStaticProps—সবকিছুর return টাইপ নির্ধারণ
- ডাটা ফেচিং ও পেজ জেনারেশন টাইপ সেফ

```typescript
import type { GetServerSideProps } from "next";

export const getServerSideProps: GetServerSideProps = async (context) => {
  // ...fetch data
  return {
    props: { name: "Anis" },
  };
};
```

## সুবিধা

- টাইপ সেফটি: UI ও API বাগ কমে যায়
- কোড রিডেবিলিটি ও মেইনটেনেবিলিটি বাড়ে
- IDE auto-complete ও refactor সহজ হয়
- বড় প্রজেক্টে স্কেলিং সহজ

## ব্যবহার

- Component, Page, API Route—সবকিছুর টাইপিং
- Data fetching ও SSR/SSG-তে টাইপ সেফটি
- Third-party library integration

## গুরুত্বপূর্ণ পয়েন্ট

- Props, API request/response-এর টাইপ নির্ধারণ করুন
- Data fetching ও getServerSideProps-এ টাইপ ব্যবহার করুন
- Error handling ও validation-এ টাইপ ব্যবহার করুন

## অনুশীলন

1. Next.js-এ একটি Product কম্পোনেন্ট লিখুন, যেখানে Props টাইপিং আছে
2. একটি API route লিখুন, যেখানে request ও response টাইপিং আছে
3. getServerSideProps-এ টাইপ সেফ ডাটা ফেচিং করুন
