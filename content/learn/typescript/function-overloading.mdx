## মূল ধারণা

### ১. Function Overloading কী?

- **Function—multiple signature—different input/output—handle।**
- **Usage:**
  ```typescript
  function greet(name: string): string;
  function greet(name: string, title: string): string;
  function greet(name: string, title?: string): string {
    return title ? `${title} ${name}` : name;
  }

  greet("Anis"); // "Anis"
  greet("Sara", "Dr."); // "Dr. Sara"
  ```

#### সহজ এনালজি

- "Function overloading—একটি সুইস আর্মি নাইফ—একই টুল দিয়ে অনেক কাজ করা যায়, input/output অনুযায়ী।"

### ২. Practical Example

```typescript
function add(a: number, b: number): number;
function add(a: string, b: string): string;
function add(a: any, b: any): any {
  return a + b;
}

add(2, 3); // 5
add("Hello, ", "World!"); // "Hello, World!"
```
