## মূল ধারণা

### ১. কাস্টম টাইপ ডেফিনিশন (Custom Type Definition)

- **সংজ্ঞা:** JS library/function/object—type definition—TypeScript-এ `.d.ts` file লিখে declare করা হয়।
- **Usage:**
  ```typescript
  // my-lib.d.ts
  declare module "my-lib" {
    export function greet(name: string): string;
    export interface User {
      id: number;
      name: string;
    }
  }
  ```
  ```typescript
  // main.ts
  import { greet, User } from "my-lib";
  const user: User = { id: 1, name: "Anis" };
  greet(user.name); // Type-safe
  ```

### ২. Integration Workflow

1. JS library/function/object—type requirement clear করুন

3. Type definition—property/function/interface/class—declare করুন
4. Type-safe integration/documentation—maintain করুন

### ৩. Common Pitfalls

- Type definition miss—runtime bug
- Wrong type definition—unexpected value/error
- `.d.ts` file—miss করলে—type error
