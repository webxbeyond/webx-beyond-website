## পরিচিতি

ধরা যাক, আপনি একটি কেক বানাচ্ছেন—কেকের উপর আইসিং, চকলেট, ফল, নানা সাজসজ্জা যোগ করছেন। মূল কেক ঠিকই আছে, কিন্তু সাজসজ্জা দিয়ে কেকের চেহারা ও স্বাদ বদলে যাচ্ছে। TypeScript-এ ডেকোরেটর ঠিক এমনই—ক্লাস, মেথড, প্রপার্টি ইত্যাদির উপর "ম্যাজিক" যোগ করে, তাদের আচরণ পরিবর্তন বা বাড়িয়ে দেয়।

## অ্যানালজি

ডেকোরেটরকে ভাবুন "গিফট র‍্যাপিং" হিসেবে—মূল উপহার ঠিকই আছে, কিন্তু বাহারি র‍্যাপিং দিয়ে সেটি আরও আকর্ষণীয় ও বিশেষ হয়ে যায়।

## বিস্তারিত ব্যাখ্যা

- ডেকোরেটর হচ্ছে একটি ফাংশন, যা ক্লাস, মেথড, প্রপার্টি, বা প্যারামিটারকে "র‍্যাপ" করে
- এটি টাইপস্ক্রিপ্টের experimental ফিচার (TypeScript 5.x+ এ আরও শক্তিশালী)
- ডেকোরেটর ব্যবহার করতে হলে `experimentalDecorators` ফ্ল্যাগ `tsconfig.json`-এ true করতে হয়
- ডেকোরেটর মূলত মেটা-প্রোগ্রামিং—কোডের আচরণ runtime-এ পরিবর্তন করা যায়

### ডেকোরেটর টাইপ
- **ক্লাস ডেকোরেটর**: পুরো ক্লাসকে র‍্যাপ করে
- **মেথড ডেকোরেটর**: ক্লাসের মেথডকে র‍্যাপ করে
- **প্রপার্টি ডেকোরেটর**: ক্লাসের প্রপার্টিকে র‍্যাপ করে
- **প্যারামিটার ডেকোরেটর**: মেথডের প্যারামিটারকে র‍্যাপ করে

### কোড উদাহরণ

```typescript
// ডেকোরেটর ফাংশন
function Logger(constructor: Function) {
  console.log("Class created:", constructor.name);
}

@Logger
class User {
  constructor(public name: string) {}
}
// Output: Class created: User

// মেথড ডেকোরেটর
function LogMethod(target: any, propertyKey: string, descriptor: PropertyDescriptor) {
  const original = descriptor.value;
  descriptor.value = function (...args: any[]) {
    console.log(`Calling ${propertyKey} with`, args);
    return original.apply(this, args);
  };
}

class Calculator {
  @LogMethod
  add(a: number, b: number) {
    return a + b;
  }
}
const calc = new Calculator();
calc.add(2, 3); // Output: Calling add with [2, 3]
```

## সুবিধা

- কোডের behavior সহজে extend/modify করা যায়
- Logging, validation, authorization ইত্যাদি সহজে implement করা যায়
- Framework/library-তে powerful API তৈরি করা যায়

## ব্যবহার

- Angular, NestJS, MobX ইত্যাদি ফ্রেমওয়ার্কে ডেকোরেটর heavily ব্যবহৃত
- Custom validation, logging, caching
- Dependency injection

## গুরুত্বপূর্ণ পয়েন্ট

- ডেকোরেটর experimental, তাই সব ব্রাউজারে/রানটাইমে সাপোর্ট নাও থাকতে পারে
- `tsconfig.json`-এ `experimentalDecorators: true` লাগবে
- ডেকোরেটর ফাংশন সাধারণত তিনটি আর্গুমেন্ট নেয়: target, propertyKey, descriptor

## অনুশীলন

1. একটি ক্লাস ডেকোরেটর লিখুন, যা ক্লাস তৈরি হলে কনসোল-এ "Created" দেখায়
2. একটি মেথড ডেকোরেটর লিখুন, যা মেথড কল হলে আর্গুমেন্ট ও রিটার্ন ভ্যালু লগ করে
3. `tsconfig.json`-এ experimentalDecorators true করে দেখুন
