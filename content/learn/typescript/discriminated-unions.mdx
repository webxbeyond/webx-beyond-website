## পরিচিতি

ধরা যাক, আপনি একটি ট্রাফিক সিগন্যাল দেখছেন—লাল, হলুদ, সবুজ। প্রতিটি রঙের অর্থ আলাদা, এবং সিগন্যাল দেখে আপনি সহজেই বুঝতে পারেন কী করতে হবে। TypeScript-এ "Literal টাইপ" মানে, নির্দিষ্ট মান; আর "Discriminated Unions" মানে, একাধিক টাইপের মধ্যে পার্থক্য করার সহজ উপায়।

## অ্যানালজি

"ট্রাফিক সিগন্যাল"—প্রতিটি রঙের নির্দিষ্ট অর্থ (Literal)। সিগন্যালের রঙ দেখে আপনি সিদ্ধান্ত নেন (Discriminated Union)।

## বিস্তারিত ব্যাখ্যা

### Literal টাইপ
- নির্দিষ্ট মানের টাইপ
- উদাহরণ:
  ```typescript
  let direction: "left" | "right";
  direction = "left"; // ঠিক আছে
  direction = "up"; // ভুল
  ```
- সাধারণত enum-এর বিকল্প হিসেবে ব্যবহার হয়

### Discriminated Unions
- একাধিক টাইপের মধ্যে পার্থক্য করার উপায়
- সাধারণত একটি "discriminant" property থাকে
- উদাহরণ:
  ```typescript
  type Circle = { kind: "circle", radius: number };
  type Square = { kind: "square", side: number };
  type Shape = Circle | Square;

  function area(shape: Shape): number {
    switch (shape.kind) {
      case "circle":
        return Math.PI * shape.radius * shape.radius;
      case "square":
        return shape.side * shape.side;
    }
  }
  ```
- TypeScript সহজেই বুঝতে পারে কোন টাইপ ব্যবহার হচ্ছে

### কখন ব্যবহার করবেন?
- যখন নির্দিষ্ট মান দরকার
- যখন একাধিক টাইপের মধ্যে পার্থক্য করতে হবে

## মূল পয়েন্ট

- Literal টাইপ: নির্দিষ্ট মান
- Discriminated Unions: পার্থক্য করার property
- Type-safe ও readable কোডের জন্য গুরুত্বপূর্ণ
