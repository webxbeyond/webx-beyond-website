## মূল ধারণা

### ১. Jest কী?

- **সংজ্ঞা:** Jest—popular JavaScript/TypeScript testing framework
- **Usage:**
  - Unit test, integration test, snapshot test
  - TypeScript support—ts-jest

#### সহজ এনালজি

- "Jest—একটি quality inspector—code-এর প্রতিটি অংশ পরীক্ষা করে।"

### ২. TypeScript-এ Jest সেটআপ

- **Step-by-step:**
  1. Install Jest, ts-jest, @types/jest
     ```bash
     npm install --save-dev jest ts-jest @types/jest
     npx ts-jest config:init
     ```
  2. Update package.json
     ```json
     "scripts": {
       "test": "jest"
     }
     ```
  3. Create test file: `sum.test.ts`
     ```typescript
     import { sum } from './sum';
     test('adds 1 + 2 = 3', () => {
       expect(sum(1, 2)).toBe(3);
     });
     ```

### ৩. Type-safe টেস্ট লিখুন

- **Usage:**
  ```typescript
  function multiply(a: number, b: number): number {
    return a * b;
  }

  test('multiply two numbers', () => {
    expect(multiply(2, 3)).toBe(6);
    // expect(multiply('2', 3)).toBe(6); // Type error
  });
  ```

#### সহজ এনালজি

- "Type-safe test—একটি checklist—সব input/output ঠিক আছে কিনা verify করে।"
