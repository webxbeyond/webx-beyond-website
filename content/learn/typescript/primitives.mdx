# প্রিমিটিভ টাইপ (string, number, boolean, null, undefined)

- **Type safety:** Data model—type mismatch, bug—early detect।
- **Maintainability:** Clear type—codebase maintain, refactor সহজ।
- **Productivity:** IntelliSense, auto-complete—fast coding।

## মূল ধারণা

### ১. String টাইপ

- **string:** Text data—name, email, message—define করতে।
- **Usage:**
  ```typescript
  let username: string = "Anis";
  let email: string = "anis@example.com";
  ```

#### সহজ এনালজি

- "string—একটি বাক্স, যেখানে শুধু লেখা (text) রাখা যায়।"

### ২. Number টাইপ

- **number:** Numeric data—age, price, rating—define করতে।
- **Usage:**
  ```typescript
  let age: number = 25;
  let price: number = 99.99;
  ```

### ৩. Boolean টাইপ

- **boolean:** True/false value—status, flag, condition—define করতে।
- **Usage:**
  ```typescript
  let isActive: boolean = true;
  let isAdmin: boolean = false;
  ```

### ৪. Null ও Undefined টাইপ

- **null:** Value নেই—intentional empty—variable/object property-এ use।
- **undefined:** Value assign হয়নি—default state—variable/object property-এ use।
- **Usage:**
  ```typescript
  let data: null = null;
  let result: undefined = undefined;
  ```

#### সহজ এনালজি

- "null—একটি খালি বাক্স, যেখানে কিছুই নেই। undefined—বাক্স আছে, কিন্তু কিছু রাখা হয়নি।"

### ৫. Practical Example

```typescript
interface User {
  name: string;
  age: number;
  isActive: boolean;
  email?: string;
  lastLogin: null | string;
}
```
