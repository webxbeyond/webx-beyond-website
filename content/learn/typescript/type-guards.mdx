## পরিচিতি

ধরা যাক, আপনি একটি ব্যাগ খুলে দেখছেন—ভেতরে বই, জামা, অথবা খাবার থাকতে পারে। আপনি আগে দেখে নেন, কী আছে, তারপর সিদ্ধান্ত নেন কী করবেন। TypeScript-এ "টাইপ গার্ড" ঠিক এমন—কোডে runtime-এ চেক করে নেয়, কোন টাইপের ডাটা আছে, তারপর সেই অনুযায়ী কাজ করে।

## অ্যানালজি

"ব্যাগের ভেতরের জিনিস চেনা"—প্রথমে দেখে নেন, তারপর সিদ্ধান্ত নেন। টাইপ গার্ড মানে, runtime-এ টাইপ চেক করে, নিরাপদে কাজ করা।

## বিস্তারিত ব্যাখ্যা

### টাইপ গার্ড কী?
- কোডে runtime-এ টাইপ চেক করার টেকনিক
- TypeScript-এ type narrowing-এর জন্য ব্যবহার হয়
- উদাহরণ:
  ```typescript
  function print(value: string | number) {
    if (typeof value === "string") {
      console.log("String value:", value);
    } else {
      console.log("Number value:", value);
    }
  }
  ```

### typeof ও instanceof
- **typeof**: primitive টাইপ চেক করার জন্য
  ```typescript
  if (typeof x === "boolean") { ... }
  ```
- **instanceof**: object/class টাইপ চেক করার জন্য
  ```typescript
  if (obj instanceof Date) { ... }
  ```

### Custom টাইপ গার্ড
- function return type: value is Type
- উদাহরণ:
  ```typescript
  function isUser(obj: any): obj is User {
    return obj && typeof obj.name === "string";
  }
  ```

### কখন ব্যবহার করবেন?
- যখন multiple টাইপ handle করতে হয়
- যখন type-safe কোড দরকার

## মূল পয়েন্ট

- টাইপ গার্ড: runtime-এ টাইপ চেক
- typeof: primitive টাইপ
- instanceof: object/class টাইপ
- Custom টাইপ গার্ড: function return type
