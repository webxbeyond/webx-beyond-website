## পরিচিতি

ধরা যাক, আপনি একটি রেস্টুরেন্টে গেছেন। মেনুতে কিছু আইটেমের পাশে "ডিফল্ট" অপশন আছে—যদি কিছু না বলেন, সেইটাই দেওয়া হবে। TypeScript-এ "ডিফল্ট জেনেরিক টাইপ" মানে, জেনেরিক টাইপে কোনো মান না দিলে, ডিফল্ট টাইপ ব্যবহার হবে।

## অ্যানালজি

"রেস্টুরেন্টের ডিফল্ট আইটেম"—কিছু না বললে, ডিফল্টটাই পাবেন। ডিফল্ট জেনেরিক টাইপ মানে, টাইপ না দিলে, ডিফল্টটাই হবে।

## বিস্তারিত ব্যাখ্যা

### ডিফল্ট জেনেরিক টাইপ কী?
- জেনেরিক টাইপে ডিফল্ট মান নির্ধারণ
- Syntax: `T = Default`
- উদাহরণ:
  ```typescript
  function identity<T = string>(value: T): T {
    return value;
  }
  identity(); // string
  identity<number>(42); // number
  ```
- Flexible API ও কোড রিডেবিলিটির জন্য গুরুত্বপূর্ণ

### Advanced Usage
- Multiple generic parameters
- Default type override
- উদাহরণ:
  ```typescript
  interface ApiResponse<T = any> {
    data: T;
    error?: string;
  }
  const res: ApiResponse = { data: "hello" };
  const res2: ApiResponse<number> = { data: 42 };
  ```

### কখন ব্যবহার করবেন?
- যখন টাইপ না দিলে ডিফল্ট দরকার
- যখন flexible ও readable API চাই
