# ফাংশন টাইপ অ্যানোটেশন ও ডিফল্ট প্যারামিটার

## প্রস্তাবনা

Google, Netflix, Uber—production app-এ function—type safety, maintainability—essential। TypeScript-এর function type annotation ও default parameter—bug কমানো, efficient workflow—industry standard।

## কেন গুরুত্বপূর্ণ?

- **Type safety:** Function parameter/return type—early error detect—bug কমে।
- **Maintainability:** Clear function signature—codebase maintain, refactor সহজ।
- **Productivity:** Default parameter—flexible function—fast coding, error avoid।

## বাস্তব উদাহরণ

- **Google:** API handler—function type annotation—type safety, maintainability।
- **Netflix:** Data processing—default parameter—flexible workflow।
- **Uber:** Ride calculation—function type annotation—bug কমানো।

## মূল ধারণা

### ১. Function Type Annotation

- **Parameter/return type—manually define করা।**
- **Usage:**
  ```typescript
  function add(a: number, b: number): number {
    return a + b;
  }

  const greet: (name: string) => string = (name) => `Hello, ${name}`;
  ```

#### সহজ এনালজি

- "Function type annotation—একটি contract—কোন input দিলে, কী output পাবেন—সব স্পষ্ট।"

### ২. Default Parameter

- **Parameter-এ default value—function call-এ value না দিলে default use হবে।**
- **Usage:**
  ```typescript
  function register(name: string, role: string = "user"): string {
    return `${name} registered as ${role}`;
  }

  register("Anis"); // "Anis registered as user"
  register("Sara", "admin"); // "Sara registered as admin"
  ```

#### সহজ এনালজি

- "Default parameter—একটি restaurant menu—order না দিলে, default dish দেয়া হয়।"

### ৩. Practical Example

```typescript
function calculateTotal(price: number, quantity: number = 1): number {
  return price * quantity;
}

const total1 = calculateTotal(100); // 100
const total2 = calculateTotal(100, 3); // 300
```
