## মূল ধারণা

### ১. Template Literal টাইপ (টেমপ্লেট লিটারাল টাইপ)

- **সংজ্ঞা:** TypeScript-এ string pattern—backtick (`) ও `${}` syntax—type-level string manipulation
- **Usage:**
  ```typescript
  type Status = "pending" | "active" | "completed";
  type StatusMessage = `ride-${Status}`;

  const msg: StatusMessage = "ride-active"; // Valid
  // const msg2: StatusMessage = "ride-cancelled"; // Error
  ```
