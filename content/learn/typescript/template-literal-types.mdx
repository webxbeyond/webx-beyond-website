## পরিচিতি

ধরা যাক, আপনি একটি নাম্বার প্লেট ডিজাইন করছেন—"AB-1234" বা "XY-5678"। প্রতিটি প্লেটের ফরম্যাট নির্দিষ্ট, কিন্তু কিছু অংশ পরিবর্তনশীল। TypeScript-এ "Template Literal টাইপ" ঠিক এমন—টাইপের মধ্যে স্ট্রিং প্যাটার্ন নির্ধারণ করা যায়, যাতে নির্দিষ্ট ফরম্যাটে ডাটা থাকে।

## অ্যানালজি

"নাম্বার প্লেট ডিজাইন"—একই ফরম্যাট, কিন্তু কিছু অংশ পরিবর্তনশীল। Template Literal টাইপ মানে, টাইপের মধ্যে স্ট্রিং প্যাটার্ন নির্ধারণ।

## বিস্তারিত ব্যাখ্যা

### Template Literal টাইপ কী?
- স্ট্রিং টাইপের মধ্যে নির্দিষ্ট প্যাটার্ন
- উদাহরণ:
  ```typescript
  type Plate = `${string}-${number}`;
  const valid: Plate = "AB-1234"; // ঠিক আছে
  const invalid: Plate = "1234-AB"; // ভুল
  ```
- API, ডাটা ভ্যালিডেশন, ও স্ট্রং টাইপড স্ট্রিং-এর জন্য ব্যবহার হয়

### Advanced Usage
- Nested template literals
- Multiple variable parts
- উদাহরণ:
  ```typescript
  type Greeting = `Hello, ${string}!`;
  const msg: Greeting = "Hello, Rahim!";
  ```

### কখন ব্যবহার করবেন?
- যখন নির্দিষ্ট স্ট্রিং ফরম্যাট দরকার
- যখন টাইপ-লেভেলে স্ট্রিং প্যাটার্ন চেক করতে হবে

## মূল পয়েন্ট

- Template Literal টাইপ: স্ট্রিং প্যাটার্ন নির্ধারণ
- Type-safe ও readable কোডের জন্য গুরুত্বপূর্ণ
- API ও ডাটা ভ্যালিডেশনে ব্যবহার হয়
