## পরিচিতি

ধরা যাক, আপনি একটি ব্যাংকের সফটওয়্যার বানাচ্ছেন—প্রতিটি ট্রান্সাকশনের তথ্য, অ্যাকাউন্ট নম্বর, ব্যালেন্স, সবকিছু সঠিকভাবে যাচাই করতে হয়। Node.js-এ TypeScript ব্যবহার করলে ব্যাকএন্ড কোডে "ভুল" বা "বাগ" কমে যায়, কারণ প্রতিটি API, ডাটা, কনফিগ—সবকিছুর টাইপ সেফটি নিশ্চিত হয়।

## অ্যানালজি

Node.js-এ TypeScript ব্যবহারকে ভাবুন "ডিজিটাল ব্যাংকিং সিস্টেম" হিসেবে—প্রতিটি ফিল্ড, ইনপুট, আউটপুট টাইপ সেফ, তাই ভুল ট্রান্সাকশন বা ডাটা এন্ট্রি কমে যায়।

## বিস্তারিত ব্যাখ্যা

### ১. Express-এ TypeScript
- API route, request, response—সবকিছুর টাইপ নির্ধারণ করা যায়
- Middleware, Controller, Service—সবকিছুর টাইপ সেফটি

```typescript
import express, { Request, Response } from "express";

const app = express();

app.get("/user/:id", (req: Request, res: Response) => {
  const id: string = req.params.id;
  res.json({ id });
});
```

### ২. Fastify-তে TypeScript
- Fastify-এর Schema ও Plugin—সবকিছুর টাইপ নির্ধারণ করা যায়
- Validation ও Serialization টাইপ সেফ

```typescript
import fastify from "fastify";

const app = fastify();

app.get<{ Params: { id: string } }>("/user/:id", (request, reply) => {
  const { id } = request.params;
  reply.send({ id });
});
```

### ৩. Service ও Model টাইপিং
- Database Model, Service Layer—সবকিছুর টাইপ নির্ধারণ
- ভুল টাইপ দিলে কম্পাইল টাইমেই error

```typescript
interface User {
  id: string;
  name: string;
}

function getUser(id: string): User {
  // ...fetch user
  return { id, name: "Anis" };
}
```

## সুবিধা

- টাইপ সেফটি: API ও ব্যাকএন্ড বাগ কমে যায়
- কোড রিডেবিলিটি ও মেইনটেনেবিলিটি বাড়ে
- IDE auto-complete ও refactor সহজ হয়
- বড় প্রজেক্টে স্কেলিং সহজ

## ব্যবহার

- Express/Fastify-তে route, middleware, service—সবকিছুর টাইপিং
- Database Model ও DTO-তে টাইপ সেফটি
- Third-party library integration

## গুরুত্বপূর্ণ পয়েন্ট

- Request, Response, Params, Body—সবকিছুর টাইপ নির্ধারণ করুন
- Service ও Model-এর টাইপ সেফটি বজায় রাখুন
- Error handling ও validation-এ টাইপ ব্যবহার করুন

## অনুশীলন

1. Express-এ একটি API route লিখুন, যেখানে request ও response টাইপিং আছে
2. Fastify-তে একটি route লিখুন, যেখানে Params টাইপিং আছে
3. Service ও Model-এর টাইপ সেফটি নিশ্চিত করুন
