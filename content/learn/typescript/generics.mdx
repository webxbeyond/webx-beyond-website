# জেনেরিকস পরিচিতি: কেন ও কিভাবে

## প্রস্তাবনা

Google, Netflix, Uber—production app-এ reusable, type-safe function/class—scalability, maintainability—essential। TypeScript-এর generics—flexible, reusable code—industry standard।

## কেন গুরুত্বপূর্ণ?

- **Reusability:** Generics—একই function/class—multiple type-এ use করা যায়।
- **Type safety:** Generic type—runtime bug কমায়, compile time-এ error detect।
- **Maintainability:** Generic code—easy refactor, long-term maintain।

## বাস্তব উদাহরণ

- **Google:** API response handler—generic function—multiple data type handle।
- **Netflix:** Data processing—generic class—flexible workflow।
- **Uber:** Ride calculation—generic function—dynamic input/output।

## মূল ধারণা

### ১. Generics কী?

- **Generics:** Function/class/interface—type parameter—flexible, reusable code।
- **Usage:**
  ```typescript
  function identity<T>(value: T): T {
    return value;
  }

  identity<string>("Anis"); // "Anis"
  identity<number>(25); // 25
  ```

#### সহজ এনালজি

- "Generics—একটি খালি বাক্স—যে কোনো ধরনের জিনিস রাখা যায়, পরে type অনুযায়ী ব্যবহার করা যায়।"

### ২. Generic Function

- **Type parameter—function signature-এ define।**
- **Usage:**
  ```typescript
  function getFirst<T>(arr: T[]): T {
    return arr[0];
  }

  getFirst<number>([1, 2, 3]); // 1
  getFirst<string>(["a", "b", "c"]); // "a"
  ```

### ৩. Generic Class

- **Type parameter—class-এ define।**
- **Usage:**
  ```typescript
  class Box<T> {
    content: T;
    constructor(value: T) {
      this.content = value;
    }
  }

  const numberBox = new Box<number>(99);
  const stringBox = new Box<string>("Hello");
  ```

### ৪. Generic Interface

- **Type parameter—interface-এ define।**
- **Usage:**
  ```typescript
  interface ApiResponse<T> {
    data: T;
    error?: string;
  }

  const userResponse: ApiResponse<{ name: string }> = {
    data: { name: "Anis" }
  };
  ```

### ৫. Industry Trends

- **Generics—reusable, type-safe code—industry standard।**
- **Flexible API/class—scalable workflow।**

### ৬. Best Practices

### ৭. সাধারণ ভুলত্রুটি

- Documentation/readme unclear—new developer confused।
