## পরিচিতি

ধরা যাক, আপনি একটি বই পড়ছেন—বইয়ের কভার, টাইটেল, লেখক, প্রকাশক, ISBN, ইত্যাদি তথ্য বইয়ের ভিতরে "লুকানো" থাকে। এই তথ্যগুলো বইয়ের মূল কনটেন্ট নয়, কিন্তু বই সম্পর্কে অনেক কিছু বলে দেয়। TypeScript-এ মেটাডাটা রিফ্লেকশন API (reflect-metadata) ঠিক এমনই—কোডের "ভিতরের তথ্য" সংগ্রহ ও ব্যবহার করতে দেয়।

## অ্যানালজি

Reflect-metadata API-কে ভাবুন "এক্স-রে" মেশিন হিসেবে—বাইরে থেকে কিছু দেখা যায় না, কিন্তু এক্স-রে করলে ভিতরের গঠন, হাড়, অর্গান দেখা যায়।

## বিস্তারিত ব্যাখ্যা

- Reflect-metadata হচ্ছে একটি লাইব্রেরি, যা TypeScript-এ ডেকোরেটর ও মেটাডাটা ব্যবহারের জন্য দরকার
- এটি ক্লাস, প্রপার্টি, মেথড, প্যারামিটার ইত্যাদির সাথে "মেটাডাটা" সংযুক্ত ও পড়তে দেয়
- মেটাডাটা মানে—কোডের "বাইরের" তথ্য, যেমন টাইপ, কাস্টম ট্যাগ, কনফিগারেশন
- ডেকোরেটর দিয়ে মেটাডাটা সেট করা যায়, পরে Reflect API দিয়ে পড়া যায়

### reflect-metadata ব্যবহার
- প্রথমে `reflect-metadata` npm থেকে ইনস্টল করতে হয়
- তারপর `import "reflect-metadata"` দিয়ে কোডে যুক্ত করতে হয়
- ডেকোরেটর দিয়ে মেটাডাটা সেট, Reflect API দিয়ে পড়া

### কোড উদাহরণ

```typescript
import "reflect-metadata";

function Type(type: string) {
  return Reflect.metadata("design:type", type);
}

class User {
  @Type("string")
  name: string;
}

// মেটাডাটা পড়া
const meta = Reflect.getMetadata("design:type", User.prototype, "name");
console.log(meta); // Output: string
```

### আরও উদাহরণ: Dependency Injection

```typescript
function Injectable(target: any) {
  Reflect.defineMetadata("injectable", true, target);
}

@Injectable
class Service {}

const isInjectable = Reflect.getMetadata("injectable", Service);
console.log(isInjectable); // Output: true
```

## সুবিধা

- কোডের "ভিতরের তথ্য" সংগ্রহ ও ব্যবহার
- Framework/library-তে powerful feature (Angular, NestJS)
- Dependency injection, validation, serialization ইত্যাদি সহজ

## ব্যবহার

- Angular, NestJS, TypeORM ইত্যাদি ফ্রেমওয়ার্কে reflect-metadata heavy ব্যবহার
- Custom validation, serialization
- Dependency injection

## গুরুত্বপূর্ণ পয়েন্ট

- reflect-metadata ব্যবহার করতে npm থেকে ইনস্টল করতে হবে
- `experimentalDecorators` ও `emitDecoratorMetadata` tsconfig-এ true করতে হবে
- সব runtime-এ সাপোর্ট নাও থাকতে পারে

## অনুশীলন

1. reflect-metadata ইনস্টল করুন এবং import করুন
2. একটি ডেকোরেটর দিয়ে প্রপার্টিতে মেটাডাটা সেট করুন
3. Reflect API দিয়ে সেই মেটাডাটা পড়ুন
