## মূল ধারণা

### ১. TypeScript Compiler (tsc) কী?

- **tsc:** TypeScript compiler—.ts file-কে .js-এ compile করে।
- **Benefit:** Type error—compile time-এ detect—runtime bug কমে।
- **Integration:** Node.js, React, Angular—tsc build process-এ use হয়।

#### সহজ এনালজি

- "tsc—একটি translator—বাংলা (TypeScript) থেকে ইংরেজি (JavaScript) করে—সবাই বুঝতে পারে।"

### ২. tsc Basic Usage

- **Single file compile:**
  ```bash
  tsc file.ts
  ```
- **Project compile (tsconfig.json):**
  ```bash
  tsc
  ```
- **Watch mode (auto compile):**
  ```bash
  tsc --watch
  ```

### ৩. Build Process Workflow

1. **Source code লিখুন (.ts file)।**
2. **tsc run করুন—type check, compile।**
3. **Output (.js file)—browser/Node.js-এ run করুন।**
4. **Error detect—fix করুন—rebuild করুন।**

### ৪. Common Compiler Flags

- **--watch:** Auto compile on file change
- **--outDir:** Output directory define
- **--rootDir:** Source directory define
- **--strict:** Maximum type safety
- **--noEmitOnError:** Error হলে output না দেয়
