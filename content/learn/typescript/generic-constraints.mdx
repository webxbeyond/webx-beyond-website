## মূল ধারণা

### ১. Generic Constraints

- **সংজ্ঞা:** Generic type parameter—specific type/property enforce—`extends` keyword দিয়ে।
- **Usage:**
  ```typescript
  function getLength<T extends { length: number }>(item: T): number {
    return item.length;
  }

  getLength([1, 2, 3]); // 3
  getLength("Anis"); // 4
  // getLength(123); // Error
  ```

#### সহজ এনালজি

- "Generic constraint—একটি ফিল্টার—শুধু নির্দিষ্ট বৈশিষ্ট্যযুক্ত জিনিসই যেতে পারে।"

### ২. extends কী?

- **সংজ্ঞা:** Generic type parameter—specific type/property inherit/extend—constraint enforce।
- **Usage:**
  ```typescript
  function merge<T extends object, U extends object>(a: T, b: U): T & U {
    return { ...a, ...b };
  }

  merge({ name: "Anis" }, { age: 25 }); // { name: "Anis", age: 25 }
  // merge(123, { age: 25 }); // Error
  ```
