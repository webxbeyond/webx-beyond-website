## পরিচিতি

ধরা যাক, আপনি একটি গাড়ি চালাচ্ছেন—গাড়ির প্রতিটি যন্ত্রাংশের সঠিক তথ্য জানা থাকলে দুর্ঘটনার ঝুঁকি কমে যায়। React-এ TypeScript ব্যবহার করলে UI ডেভেলপমেন্টে "দুর্ঘটনা" (বাগ, টাইপ mismatch) কমে যায়, কারণ প্রতিটি কম্পোনেন্ট, প্রপস, স্টেট, কনটেক্সট—সবকিছুর টাইপ সেফটি নিশ্চিত হয়।

## অ্যানালজি

React-এ TypeScript ব্যবহারকে ভাবুন "নিরাপদ গাড়ি চালানো" হিসেবে—গাড়ির প্রতিটি যন্ত্রাংশের health report আগে থেকেই জানা, তাই ভুল হলে সঙ্গে সঙ্গে সতর্কতা আসে।

## বিস্তারিত ব্যাখ্যা

### ১. কম্পোনেন্ট ও Props টাইপিং
- প্রতিটি কম্পোনেন্টের Props-এর টাইপ নির্ধারণ করা যায়
- ভুল টাইপ দিলে কম্পাইল টাইমেই error

```typescript
interface UserProps {
  name: string;
  age: number;
}

function User({ name, age }: UserProps) {
  return <div>{name} ({age})</div>;
}
```

### ২. State ও useState টাইপিং
- useState-এর initial value থেকে টাইপ ইনফার করা যায়
- চাইলে টাইপ এক্সপ্লিসিটলি দেওয়া যায়

```typescript
const [count, setCount] = useState<number>(0);
```

### ৩. Custom Hooks টাইপিং
- Custom hook-এর return ও parameter টাইপ নির্ধারণ করা যায়

```typescript
function useToggle(initial: boolean): [boolean, () => void] {
  const [state, setState] = useState(initial);
  const toggle = () => setState(s => !s);
  return [state, toggle];
}
```

### ৪. Context API টাইপিং
- Context value-এর টাইপ নির্ধারণ করা যায়
- Consumer/Provider-এ টাইপ mismatch হলে error

```typescript
interface Theme {
  color: string;
}

const ThemeContext = createContext<Theme | undefined>(undefined);

function App() {
  return (
    <ThemeContext.Provider value={{ color: "blue" }}>
      <Child />
    </ThemeContext.Provider>
  );
}

function Child() {
  const theme = useContext(ThemeContext);
  if (!theme) return null;
  return <div style={{ color: theme.color }}>Hello</div>;
}
```

## সুবিধা

- টাইপ সেফটি: UI বাগ কমে যায়
- কোড রিডেবিলিটি ও মেইনটেনেবিলিটি বাড়ে
- IDE auto-complete ও refactor সহজ হয়
- বড় প্রজেক্টে স্কেলিং সহজ

## ব্যবহার

- Props, State, Context, Reducer—সবকিছুর টাইপিং
- Custom hooks ও HOC-এ টাইপ সেফটি
- Third-party component integration

## গুরুত্বপূর্ণ পয়েন্ট

- Props ও State-এর টাইপ নির্ধারণ করুন
- Context API-তে টাইপ সেফটি বজায় রাখুন
- Custom hook ও reducer-এ টাইপ ব্যবহার করুন

## অনুশীলন

1. একটি User কম্পোনেন্ট লিখুন, যেখানে Props টাইপিং আছে
2. একটি Custom hook লিখুন, যেখানে return value টাইপ সেফ
3. Context API দিয়ে একটি থিম সিস্টেম তৈরি করুন
