## মূল ধারণা

### ১. Typed Props

- **সংজ্ঞা:** Component-এর input (props)—type-safe করা যায়
- **Usage:**
  ```typescript
  type UserProps = {
    name: string;
    age?: number;
  };

  function User({ name, age }: UserProps) {
    return <div>{name} ({age})</div>;
  }
  ```

#### সহজ এনালজি

- "Typed props—একটি contract—component কী data expect করবে, তা স্পষ্ট।"

### ২. Typed State ও Hooks

- **সংজ্ঞা:** State ও hooks—type-safe করা যায়
- **Usage:**
  ```typescript
  import { useState } from 'react';

  const [count, setCount] = useState<number>(0);
  setCount(5); // OK
  // setCount('five'); // Error
  ```

#### Example: Custom Hook
```typescript
function useUser(): [string, (name: string) => void] {
  const [name, setName] = useState<string>("");
  return [name, setName];
}
```

### ৩. Context API

- **সংজ্ঞা:** Global state—type-safe context দিয়ে manage করা যায়
- **Usage:**
  ```typescript
  import { createContext, useContext } from 'react';

  type Theme = 'light' | 'dark';
  const ThemeContext = createContext<Theme>('light');

  function App() {
    const theme = useContext(ThemeContext);
    return <div>{theme}</div>;
  }
  ```

#### সহজ এনালজি

- "Context API—একটি global address book—সব component context থেকে data নিতে পারে।"

### ৫. Common Pitfalls

- Wrong type—unexpected bug/error
- Optional/nullable prop—type miss করলে runtime bug
- Context value—type mismatch

### ৬. Best Practices
