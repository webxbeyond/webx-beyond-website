## পরিচিতি

ধরা যাক, আপনি একটি ট্রেনের কামরা গুনছেন—প্রথমে ইঞ্জিন, তারপর যাত্রীবাহী, তারপর মালবাহী, আবার যাত্রীবাহী, আবার মালবাহী... এইভাবে ট্রেনের কামরার সংখ্যা ও ধরন একেকবার একেক রকম হয়। ঠিক TypeScript-এও অনেক সময় এমন ডাইনামিক, পরিবর্তনশীল লিস্ট বা টাপল দরকার হয়, যেখানে কিছু অংশ নির্দিষ্ট, আর কিছু অংশ "যত খুশি" যোগ হতে পারে। Variadic Tuple টাইপ এই সমস্যার সমাধান দেয়।

## অ্যানালজি

Variadic Tuple টাইপকে ভাবুন "রেস্টুরেন্টের মেনু" হিসেবে—প্রথমে Starter, তারপর Main Course, তারপর যত খুশি Side Dish, শেষে Dessert। Starter/Main/Dessert নির্দিষ্ট, কিন্তু Side Dish-এর সংখ্যা ও ধরন পরিবর্তনশীল।

## বিস্তারিত ব্যাখ্যা

- Variadic Tuple টাইপ TypeScript 4.0+ এ এসেছে।
- এটি টাপল টাইপের মধ্যে "রেস্ট প্যারামিটার" (`...`) ব্যবহার করে ডাইনামিক অংশ যোগ করতে দেয়।
- সাধারণ টাপল: `[string, number, boolean]`
- Variadic Tuple: `[string, ...number[], boolean]` — এখানে মাঝখানে যত খুশি `number` থাকতে পারে।
- জেনেরিকসের সাথে ব্যবহার করলে, ফাংশন/টাইপ অনেক বেশি flexible হয়।

### কোড উদাহরণ

```typescript
// সাধারণ টাপল
const fixed: [string, number, boolean] = ["A", 42, true];

// Variadic Tuple
const variadic: [string, ...number[], boolean] = ["A", 1, 2, 3, 4, true];

// জেনেরিক ফাংশন
function concat<T extends unknown[]>(...args: T): T {
  return args;
}
const result = concat("A", 1, 2, 3, true); // result: [string, number, number, number, boolean]
```

## Variadic Tuple-এর সুবিধা

- ডাইনামিক ফাংশন প্যারামিটার টাইপিং
- Array/Arguments-এর টাইপ সেফ ম্যানিপুলেশন
- API/Library-তে flexible signature
- TypeScript-এর type inference আরও শক্তিশালী হয়

## Variadic Tuple-এর ব্যবহার

- ফাংশন ওভারলোডিং
- Arguments/Parameters-এর টাইপিং
- Array ম্যানিপুলেশন
- Complex data structure তৈরি

## গুরুত্বপূর্ণ পয়েন্ট

- Variadic Tuple শুধু টাপলেই নয়, mapped/conditional টাইপেও ব্যবহার করা যায়
- TypeScript 4.0+ লাগবে
- Type inference অনেক সহজ হয়

## অনুশীলন

1. একটি ফাংশন লিখুন, যেখানে প্রথম প্যারামিটার `string`, তারপর যত খুশি `number`, শেষে `boolean`—সবকিছুর টাইপ সেফটি নিশ্চিত করুন।
2. Variadic Tuple দিয়ে arguments-এর টাইপ চেক করুন।
3. কোডে ভুল টাইপ দিলে TypeScript কীভাবে error দেয়, দেখে নিন।
