## মূল ধারণা

### ১. Variadic Tuple টাইপ কী?

- **সংজ্ঞা:** Tuple—variable-length—type-level-এ handle করা যায়
- **Usage:**
  ```typescript
  type LogEntry<T extends unknown[]> = [string, ...T];
  const entry1: LogEntry<[number, boolean]> = ["Start", 1, true];
  const entry2: LogEntry<[string, number, boolean]> = ["End", "Done", 42, false];
  ```

#### সহজ এনালজি

- "Variadic tuple—একটি expandable box—যেখানে নির্দিষ্ট কিছু item-এর পর variable-length item রাখা যায়।"

### ২. কিভাবে কাজ করে?

- Tuple-এর মধ্যে `...T`—rest parameter-এর মতো—multiple type/value handle করে
- TypeScript—type inference—tuple-এর length/type automatically manage করে

#### Example: Function Arguments
```typescript
function log<T extends unknown[]>(...args: [string, ...T]) {
  console.log(args);
}

log("Start", 1, true);
log("End", "Done", 42, false);
```
