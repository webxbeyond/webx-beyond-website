## ভূমিকা

TypeScript কম্পাইলার শুধু কোড চেক বা ট্রান্সফর্মই করে না—কোড জেনারেশনও করতে পারে। TS Transformers দিয়ে compile time-এ কাস্টম কোড জেনারেশন সম্ভব, যেমন: API contract থেকে টাইপ/ইন্টারফেস, boilerplate code, বা অটোমেটেড ফাংশন/ক্লাস তৈরি।

## অ্যানালজি: "রোবটিক কারখানা"

ধরা যাক, একটি কারখানায় রোবট অটোমেটেডভাবে বিভিন্ন যন্ত্রাংশ তৈরি করে—নকশা অনুযায়ী, নির্দিষ্ট নিয়মে। TS Transformers-ও ঠিক তেমন—AST traversal করে, কাস্টম প্রসেসিং করে, নতুন কোড জেনারেট করে। এতে সময়, শ্রম, ও ভুল কমে যায়।

## বিস্তারিত আলোচনা

### Code Generation কী?
- Compile time-এ AST traversal করে, নতুন কোড তৈরি
- API contract, ডাটাবেস স্কিমা, বা কনফিগ থেকে টাইপ/ইন্টারফেস জেনারেট
- Boilerplate code, validation, বা utility function auto-generate
- Manual repetitive কাজ অটোমেটেড হয়

### TS Transformers দিয়ে Code Generation
- Transformer function AST traversal করে, নতুন নোড/statement যোগ করে
- ts.factory API দিয়ে নতুন ক্লাস, ফাংশন, টাইপ, ইত্যাদি তৈরি
- Compile time-এ কোডে পরিবর্তন/যোগ হয়

### উদাহরণ
```typescript
import ts from 'typescript';

function codegenTransformer(context: ts.TransformationContext) {
  return (rootNode: ts.SourceFile) => {
    function visit(node: ts.Node): ts.Node {
      // উদাহরণ: একটি নতুন ফাংশন যোগ
      if (ts.isSourceFile(node)) {
        const newFunction = ts.factory.createFunctionDeclaration(
          undefined,
          [ts.factory.createModifier(ts.SyntaxKind.ExportKeyword)],
          undefined,
          'autoGenerated',
          undefined,
          [],
          undefined,
          ts.factory.createBlock([
            ts.factory.createExpressionStatement(
              ts.factory.createCallExpression(
                ts.factory.createIdentifier('console.log'),
                undefined,
                [ts.factory.createStringLiteral('Hello from codegen!')]
              )
            )
          ], true)
        );
        return ts.factory.updateSourceFile(node, [newFunction, ...node.statements]);
      }
      return ts.visitEachChild(node, visit, context);
    }
    return ts.visitNode(rootNode, visit);
  };
}
```

### ব্যবহার
- API contract থেকে টাইপ/ইন্টারফেস জেনারেট
- ডাটাবেস স্কিমা থেকে মডেল/ভ্যালিডেশন ফাংশন তৈরি
- repetitive utility function/class auto-generate
- Boilerplate code minimize
