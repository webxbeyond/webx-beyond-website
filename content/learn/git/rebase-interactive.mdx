# ইন্টার‌্যাকটিভ রিবেস: কমিট এডিট ও স্কোয়াশ (`git rebase -i`)

ইন্টার‌্যাকটিভ রিবেস (interactive rebase) দিয়ে আপনি একাধিক কমিট একত্রে স্কোয়াশ, এডিট, রিনেম বা ডিলিট করতে পারেন। এতে ইতিহাস আরও পরিষ্কার ও সুন্দর হয়।

## ইন্টার‌্যাকটিভ রিবেস কী?
- `git rebase -i` দিয়ে নির্দিষ্ট কমিটের পর থেকে সব কমিটের তালিকা পাবেন।
- এখান থেকে কমিট স্কোয়াশ, এডিট, রিনেম, ডিলিট ইত্যাদি করতে পারবেন।

## ধাপসমূহ

1. কোন কমিটের পর থেকে রিবেস করতে চান, তার হ্যাশ বা সংখ্যা দিন:
   ```sh
   git rebase -i HEAD~4
   ```
   এতে শেষ ৪টি কমিটের তালিকা আসবে।

2. এডিটর খুলে যাবে, সেখানে:
   - `pick` মানে কমিট রেখে দিন
   - `squash` মানে আগের কমিটের সাথে একত্র করুন
   - `edit` মানে কমিট এডিট করুন
   - `reword` মানে কমিট মেসেজ পরিবর্তন করুন
   - `drop` মানে কমিট ডিলিট করুন

3. প্রয়োজনীয় পরিবর্তন করে ফাইল সেভ ও ক্লোজ করুন।

4. স্কোয়াশ বা এডিট করলে গিট নির্দেশনা দেবে:
   - স্কোয়াশ: কমিট মেসেজ একত্র করুন
   - এডিট: পরিবর্তন করে `git commit --amend` দিন, তারপর `git rebase --continue`

## উদাহরণ

- শেষ ৩টি কমিট স্কোয়াশ করতে:
  ```sh
  git rebase -i HEAD~3
  ```
  তারপর দ্বিতীয় ও তৃতীয় কমিটের পাশে `squash` লিখুন।

## সংক্ষেপে
- `git rebase -i` দিয়ে কমিট স্কোয়াশ, এডিট, রিনেম, ডিলিট করুন
- ইতিহাস পরিষ্কার ও সুন্দর রাখুন

ইন্টার‌্যাকটিভ রিবেস দিয়ে গিট ইতিহাস আরও প্রফেশনাল ও সহজবোধ্য হয়।
