# ফর্কিং ওয়ার্কফ্লো: ওপেন সোর্সে অবদান

ওপেন সোর্স প্রজেক্টে অবদান রাখতে ফর্কিং ওয়ার্কফ্লো সবচেয়ে জনপ্রিয় ও নিরাপদ পদ্ধতি।

## ফর্ক কী?
- ফর্ক মানে কারো রিপোজিটরির একটি কপি নিজের গিটহাব/গিটল্যাব অ্যাকাউন্টে নেওয়া।
- এতে মূল প্রজেক্টে সরাসরি না লিখে নিজের কপিতে কাজ করা যায়।

## ফর্কিং ওয়ার্কফ্লো ধাপসমূহ
1. **প্রজেক্ট ফর্ক করুন:**
   - গিটহাব/গিটল্যাবে মূল রিপোতে গিয়ে 'Fork' বাটন চাপুন।
2. **নিজের ফর্ক ক্লোন করুন:**
   ```sh
   git clone https://github.com/your-username/project.git
   ```
3. **নতুন ব্রাঞ্চে কাজ করুন:**
   ```sh
   git checkout -b feature-x
   # কোড পরিবর্তন করুন
   git add .
   git commit -m "নতুন ফিচার যোগ"
   ```
4. **নিজের ফর্কে পুশ করুন:**
   ```sh
   git push origin feature-x
   ```
5. **মূল রিপোতে Pull Request (PR) পাঠান:**
   - গিটহাবে গিয়ে 'Compare & pull request' বাটন চাপুন।
   - পরিবর্তনের বিবরণ লিখে PR সাবমিট করুন।

## মূল রিপোর আপডেট আনুন (upstream)
- মূল রিপোকে `upstream` নামে যোগ করুন:
  ```sh
  git remote add upstream https://github.com/original-owner/project.git
  ```
- আপডেট আনতে:
  ```sh
  git fetch upstream
  git checkout main
  git merge upstream/main
  ```

## কেন ফর্কিং ওয়ার্কফ্লো?
- মূল কোডবেস নিরাপদ থাকে
- অবদানকারীরা স্বাধীনভাবে কাজ করতে পারে
- ওপেন সোর্সে সহজ অবদান

## সংক্ষেপে
- ফর্ক করুন → ক্লোন করুন → ব্রাঞ্চে কাজ করুন → পুশ করুন → PR পাঠান
- মূল রিপোর আপডেট রাখতে `upstream` ব্যবহার করুন

এভাবে ফর্কিং ওয়ার্কফ্লো অনুসরণ করলে ওপেন সোর্সে অবদান রাখা সহজ ও নিরাপদ।
