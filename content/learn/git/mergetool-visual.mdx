# ভিজ্যুয়াল কনফ্লিক্ট সমাধানে `git mergetool` ব্যবহার

গিটে মার্জ কনফ্লিক্ট হলে টেক্সট এডিটরে ম্যানুয়ালি সমাধান করা যায়, তবে ভিজ্যুয়াল টুল দিয়ে আরও সহজে কনফ্লিক্ট ম্যানেজ করা যায়। এজন্য `git mergetool` ব্যবহার করা হয়।

## ১. mergetool কী?

`git mergetool` কমান্ড দিয়ে জনপ্রিয় ভিজ্যুয়াল টুল (যেমন: VSCode, Meld, KDiff3, Beyond Compare) দিয়ে কনফ্লিক্ট সমাধান করা যায়।

## ২. mergetool কনফিগার

### VSCode ব্যবহার করতে:
```sh
git config --global merge.tool vscode
git config --global mergetool.vscode.cmd "code --wait $MERGED"
```

### Meld ব্যবহার করতে:
```sh
git config --global merge.tool meld
```

## ৩. কনফ্লিক্ট হলে চালান:
```sh
git mergetool
```

- এতে কনফ্লিক্ট হওয়া ফাইল ভিজ্যুয়াল টুলে খুলবে।
- প্রয়োজনীয় অংশ রেখে সেভ করুন।

## ৪. সমাধানের পর
- ফাইল সেভ ও ক্লোজ করুন
- তারপর ফাইল স্টেজ করুন:
  ```sh
  git add <file>
  ```
- মার্জ কমিট দিন:
  ```sh
  git commit
  ```

## সংক্ষেপে
- `git mergetool` দিয়ে সহজে কনফ্লিক্ট সমাধান করুন
- পছন্দের ভিজ্যুয়াল টুল কনফিগার করুন

ভিজ্যুয়াল টুল ব্যবহার করলে কনফ্লিক্ট সমাধান আরও দ্রুত ও নির্ভুল হয়।
