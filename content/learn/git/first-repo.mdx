# প্রথম রিপোজিটরি তৈরি

গিটে কাজ শুরু করতে হলে আপনাকে একটি রিপোজিটরি (repository) তৈরি করতে হবে। রিপোজিটরি হলো এমন একটি ফোল্ডার, যেখানে গিট আপনার কোড ও তার ইতিহাস সংরক্ষণ করে।

## লোকাল রিপোজিটরি তৈরি (`git init`)

নতুন কোনো প্রজেক্টে গিট ব্যবহার করতে চাইলে:

1. আপনার প্রজেক্ট ফোল্ডারে যান:
   ```sh
   cd path/to/your/project
   ```
2. গিট ইনিশিয়ালাইজ করুন:
   ```sh
   git init
   ```
   এতে `.git` নামের একটি হিডেন ফোল্ডার তৈরি হবে, যেখানে গিট সব তথ্য সংরক্ষণ করবে।

## রিমোট রিপোজিটরি ক্লোন করা (`git clone`)

ইতিমধ্যে কোনো গিট রিপোজিটরি (যেমন গিটহাব, গিটল্যাব) থেকে কোড ডাউনলোড করতে চাইলে:

1. ক্লোন করুন:
   ```sh
   git clone https://github.com/username/repo-name.git
   ```
   এতে একটি নতুন ফোল্ডার তৈরি হবে এবং সব ফাইল ও ইতিহাস ডাউনলোড হবে।

## সংক্ষেপে
- নতুন প্রজেক্টে গিট চালু করতে `git init`
- অনলাইন রিপোজিটরি থেকে কোড আনতে `git clone`

---

## টিপস
- `git status` দিয়ে রিপোজিটরির অবস্থা দেখুন
- `.git` ফোল্ডার ডিলিট করলে গিট ট্র্যাকিং বন্ধ হয়ে যাবে

এভাবেই আপনি গিটে প্রথম রিপোজিটরি তৈরি ও ক্লোন করতে পারবেন।
