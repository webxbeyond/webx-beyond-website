# স্ট্যাশ প্রয়োগ, দেখা ও ম্যানেজ

`git stash` দিয়ে কাজের অগ্রগতি অস্থায়ীভাবে সংরক্ষণ করার পর, সেগুলো প্রয়োগ, দেখা ও ম্যানেজ করার জন্য কিছু কমান্ড আছে।

## স্ট্যাশ লিস্ট দেখা
```sh
git stash list
```
- সব স্ট্যাশ এন্ট্রি দেখাবে (stash@{0}, stash@{1}, ...)

## নির্দিষ্ট স্ট্যাশ প্রয়োগ
```sh
git stash apply stash@{0}
```
- সর্বশেষ স্ট্যাশ প্রয়োগ করতে শুধু `git stash apply`
- প্রয়োগের পরও স্ট্যাশ লিস্টে থাকবে

## স্ট্যাশ পপ (প্রয়োগ + ডিলিট)
```sh
git stash pop
```
- সর্বশেষ স্ট্যাশ প্রয়োগ করে লিস্ট থেকে ডিলিট করে

## নির্দিষ্ট স্ট্যাশ ডিলিট
```sh
git stash drop stash@{1}
```

## সব স্ট্যাশ ডিলিট
```sh
git stash clear
```

## স্ট্যাশ ব্রাঞ্চে প্রয়োগ
- নতুন ব্রাঞ্চে স্ট্যাশ প্রয়োগ:
  ```sh
  git stash branch feature-x stash@{0}
  ```

## টিপস
- স্ট্যাশ প্রয়োগে কনফ্লিক্ট হলে ম্যানুয়ালি ঠিক করুন
- স্ট্যাশে আনট্র্যাকড ফাইল রাখতে চাইলে: `git stash -u`

## সংক্ষেপে
- `git stash list` = সব স্ট্যাশ দেখুন
- `git stash apply`/`pop` = প্রয়োগ করুন
- `git stash drop`/`clear` = ডিলিট করুন
- `git stash branch` = নতুন ব্রাঞ্চে প্রয়োগ

এভাবে স্ট্যাশ ম্যানেজ করলে কাজের অগ্রগতি সহজে সংরক্ষণ ও পুনরুদ্ধার করা যায়।
