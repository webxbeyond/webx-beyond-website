# ইতিহাস দেখা: `git log`, `git diff`

গিটে আপনার কোডের ইতিহাস দেখা এবং পরিবর্তন তুলনা করা খুবই গুরুত্বপূর্ণ। এজন্য দুটি প্রধান কমান্ড ব্যবহার হয়: `git log` এবং `git diff`।

## ১. git log

এই কমান্ডটি দিয়ে আপনি রিপোজিটরির সমস্ত কমিটের ইতিহাস দেখতে পারবেন।

```sh
git log
```

এতে কমিট আইডি, লেখক, তারিখ ও কমিট মেসেজ দেখা যায়।

### সংক্ষিপ্তভাবে দেখতে:
```sh
git log --oneline
```

### নির্দিষ্ট ফাইলের ইতিহাস:
```sh
git log <filename>
```

## ২. git diff

এই কমান্ডটি দিয়ে আপনি বর্তমান পরিবর্তন, স্টেজড ও আনস্টেজড পরিবর্তন, বা দুইটি কমিটের মধ্যে পার্থক্য দেখতে পারবেন।

### আনস্টেজড পরিবর্তন দেখতে:
```sh
git diff
```

### স্টেজড পরিবর্তন দেখতে:
```sh
git diff --staged
```

### দুই কমিটের মধ্যে পার্থক্য:
```sh
git diff <commit1> <commit2>
```

## সংক্ষেপে
- `git log` — কমিট ইতিহাস দেখুন
- `git diff` — পরিবর্তন তুলনা করুন

এই কমান্ডগুলো ব্যবহার করে আপনি সহজেই কোডের ইতিহাস ও পরিবর্তন বিশ্লেষণ করতে পারবেন।
