## গিট ক্রেডেনশিয়াল হেল্পার কী?

গিট ক্রেডেনশিয়াল হেল্পার হলো—রিমোট রিপোজিটরিতে পুশ/পুল করার সময় ইউজারনেম/পাসওয়ার্ড বা টোকেন সংরক্ষণ ও স্বয়ংক্রিয়ভাবে ব্যবহারের টুল।

## প্রধান ক্রেডেনশিয়াল হেল্পার
- **cache**: মেমোরিতে অস্থায়ীভাবে সংরক্ষণ (ডিফল্ট ১৫ মিনিট)
- **store**: প্লেইন টেক্সটে ডিস্কে সংরক্ষণ (কম নিরাপদ)
- **manager**: Windows-এ GUI/Windows Credential Manager
- **osxkeychain**: macOS-এর জন্য
- **gpg**: GPG এনক্রিপ্টেড

## কনফিগারেশন

### ক্যাশ (cache)
```sh
git config --global credential.helper cache
# সময় বাড়াতে (৩০ মিনিট):
git config --global credential.helper 'cache --timeout=1800'
```

### স্টোর (store)
```sh
git config --global credential.helper store
```

### Windows Credential Manager
```sh
git config --global credential.helper manager
```

### macOS Keychain
```sh
git config --global credential.helper osxkeychain
```

## ক্যাশ ক্লিয়ার করা
```sh
git credential-cache exit
```

## নিরাপত্তা টিপস
- store হেল্পার প্লেইন টেক্সটে সংরক্ষণ করে—ব্যবহার না করাই ভালো
- টোকেন/পাসওয়ার্ড শেয়ার করবেন না
- পিসি শেয়ার্ড হলে ক্যাশ ক্লিয়ার করুন
