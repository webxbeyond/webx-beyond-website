# গিট ইনস্টল ও ইউজার আইডেন্টিটি কনফিগার

গিট ব্যবহার শুরু করতে হলে প্রথমে আপনার কম্পিউটারে গিট ইনস্টল করতে হবে এবং নিজের নাম ও ইমেইল কনফিগার করতে হবে।

## গিট ইনস্টল

### Windows:
1. https://git-scm.com থেকে গিট ডাউনলোড করুন।
2. ডাউনলোড করা ফাইলটি রান করুন এবং ডিফল্ট সেটিংস রেখে ইনস্টল করুন।
3. ইনস্টল শেষে Command Prompt বা Git Bash খুলুন।

### macOS:
- টার্মিনালে লিখুন:
  ```sh
  brew install git
  ```
  অথবা Xcode Command Line Tools ইনস্টল করুন:
  ```sh
  xcode-select --install
  ```

### Linux (Ubuntu):
- টার্মিনালে লিখুন:
  ```sh
  sudo apt update
  sudo apt install git
  ```

## ইনস্টলেশন যাচাই

টার্মিনালে লিখুন:
```sh
git --version
```
সফল হলে গিটের ভার্সন দেখাবে।

## ইউজার আইডেন্টিটি কনফিগার

প্রতিটি কমিটের সাথে আপনার নাম ও ইমেইল সংরক্ষিত হয়। তাই প্রথমেই এগুলো কনফিগার করা জরুরি।

টার্মিনালে লিখুন:
```sh
git config --global user.name "আপনার নাম"
git config --global user.email "আপনার ইমেইল"
```

উদাহরণ:
```sh
git config --global user.name "Anis Rahman"
git config --global user.email "anis@email.com"
```

## কনফিগারেশন যাচাই

আপনার কনফিগারেশন দেখতে:
```sh
git config --list
```

## সংক্ষেপে
- গিট ইনস্টল করুন
- নিজের নাম ও ইমেইল কনফিগার করুন
- ইনস্টল ও কনফিগারেশন যাচাই করুন

এবার আপনি গিট ব্যবহার শুরু করতে প্রস্তুত!
