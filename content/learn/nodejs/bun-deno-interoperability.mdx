Node.js-এর পরবর্তী যুগ—Bun ও Deno! Modern runtime, high performance, এবং interoperability—সব developer-এর জন্য নতুন সুযোগ।

## Bun ও Deno কী?
- Bun: Fast JS runtime, built-in bundler, transpiler, test runner
- Deno: Secure, modern JS/TS runtime, built-in TypeScript, permission system

### Analogy
"যেন Node.js-এর নতুন ভার্সন—একটা গাড়ি, Bun turbo engine, Deno security lock!"

## Interoperability কেন?
- Legacy code reuse
- Ecosystem leverage
- Modern feature adopt

## Example: Bun-এ Node.js API
```js
// Bun supports many Node.js APIs
import { readFileSync } from 'fs';
const data = readFileSync('./file.txt', 'utf8');
console.log(data);
```

## Example: Deno-তে Node.js Module Import
```ts
// Deno supports npm modules
import express from 'npm:express';
const app = express();
app.get('/', (req, res) => res.send('Hello from Deno!'));
app.listen(3000);
```
