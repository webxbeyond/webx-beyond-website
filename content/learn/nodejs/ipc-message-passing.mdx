Node.js-এ IPC (Inter-Process Communication) ও Message Passing হচ্ছে একাধিক process/thread-এর মধ্যে ডেটা আদান-প্রদানের জন্য অপরিহার্য টুল।

## IPC কী?

IPC হচ্ছে একাধিক process-এর মধ্যে ডেটা, কমান্ড, বা সিগন্যাল আদান-প্রদানের টেকনিক। Node.js-এ Child Process, Cluster, Worker Thread—সবখানে IPC ব্যবহৃত হয়।

## Message Passing

Message Passing হচ্ছে process/thread-এর মধ্যে "message" পাঠানো ও গ্রহণ করার পদ্ধতি।

### Child Process-এ Message Passing
```js
// parent.js
const { fork } = require('child_process');
const child = fork('child.js');
child.on('message', msg => console.log('From child:', msg));
child.send('Hello Child');

// child.js
process.on('message', msg => {
  process.send(msg + ' World!');
});
```

### Worker Thread-এ Message Passing
```js
const { Worker, isMainThread, parentPort } = require('worker_threads');
if (isMainThread) {
  const worker = new Worker(__filename);
  worker.on('message', msg => console.log('From worker:', msg));
  worker.postMessage('Hello Worker');
} else {
  parentPort.on('message', msg => {
    parentPort.postMessage(msg + ' World!');
  });
}
```

## Cluster Module-এ IPC

Cluster module-এ master ও worker-এর মধ্যে IPC হয়।

```js
const cluster = require('cluster');
if (cluster.isMaster) {
  const worker = cluster.fork();
  worker.on('message', msg => console.log('From worker:', msg));
  worker.send('Hello Cluster');
} else {
  process.on('message', msg => {
    process.send(msg + ' World!');
  });
}
```

## Real Life Analogy

IPC হচ্ছে "ইন্টারকম"—একাধিক রুমে বসে থাকা মানুষ একে অপরকে বার্তা পাঠায়। Message Passing হচ্ছে "চিঠি"—একজন লিখে পাঠায়, অন্যজন পড়ে উত্তর দেয়।

## Practical Example: Parallel Computation with IPC

```js
// parent.js
const { fork } = require('child_process');
const child = fork('child.js');
child.send({ num: 21 });
child.on('message', res => console.log('Result:', res));

// child.js
process.on('message', msg => {
  let result = msg.num * 2;
  process.send(result);
});
```
