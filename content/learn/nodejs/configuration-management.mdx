বড় স্কেল Node.js অ্যাপে environment, secret, feature toggle, external service—সবকিছু manage করতে configuration management অপরিহার্য।

## কনফিগারেশন ম্যানেজমেন্ট কী?

Configuration management হচ্ছে app-এর behavior/environment external file/variable দিয়ে control করা—hardcoded value নয়।

### Analogy
Hardcoded config হচ্ছে "রান্নার salt আগে থেকেই মিশানো"—সব খাবারে একই স্বাদ। Config management হচ্ছে "খাবার পরিবেশনের সময় salt আলাদা দেওয়া"—প্রয়োজনে adjust করা যায়।

## কেন Configuration Management?
- Environment-specific config (dev/staging/prod)
- Secret management (API key, password)
- Feature toggle
- Easy deployment/rollback

## Node.js-এ Configuration Management

### ১. Environment Variable
```sh
PORT=3000 NODE_ENV=production node app.js
```
```js
const port = process.env.PORT || 3000;
const env = process.env.NODE_ENV || 'development';
```

### ২. .env File (dotenv)
```sh
npm install dotenv
```
```env
PORT=3000
DB_URL=mongodb://localhost:27017/app
```
```js
require('dotenv').config();
console.log(process.env.DB_URL);
```

### ৩. Config File (JSON/YAML)
```json
{
  "dev": { "db": "mongodb://localhost/dev" },
  "prod": { "db": "mongodb://localhost/prod" }
}
```
```js
const config = require('./config.json');
const env = process.env.NODE_ENV || 'dev';
console.log(config[env].db);
```

### ৪. Centralized Config (Vault, AWS Parameter Store)
- External service থেকে config fetch করুন
- Secret rotation, audit, access control

## Best Practices
- Never commit secret/config file to git
- Use environment variable for secret
- Validate config at startup
- Use config schema (Joi, convict)
- Document config options

## Real World Example
- **Twelve-Factor App:** Strict config management
- **Kubernetes:** ConfigMap, Secret

## Performance Tips
- Config cache করুন
- Dynamic config reload handle করুন
- Sensitive config encrypt করুন

## উপসংহার

Configuration management Node.js-এ scalable, secure ও flexible deployment নিশ্চিত করে। Production-এ environment variable, external config service ও validation best practices অনুসরণ করুন। পরবর্তী পাঠে Environment Segregation নিয়ে আলোচনা হবে।
