## for, while, do-while

প্রোগ্রামিংয়ে কোনো কাজ বারবার করতে হলে লুপ ব্যবহার করা হয়। লুপ কোডকে সংক্ষিপ্ত, রিডেবল ও দক্ষ করে তোলে। এই অধ্যায়ে আমরা for, while, do-while লুপের ব্যবহার, গঠন, পার্থক্য, উদাহরণ ও ভালো অভ্যাস জানব।

---

## ১. লুপ কী?

- লুপ মানে কোনো কোড ব্লক বারবার চালানো
- নির্দিষ্ট শর্ত পর্যন্ত বা নির্দিষ্ট সংখ্যক বার

---

## ২. for লুপ

**for লুপ** সাধারণত নির্দিষ্ট সংখ্যক বার পুনরাবৃত্তির জন্য ব্যবহৃত হয়।

### Python:
```python
for i in range(5):
    print(i)
# আউটপুট: 0 1 2 3 4
```

### C:
```c
for (int i = 0; i < 5; i++) {
    printf("%d\n", i);
}
```

---

## ৩. while লুপ

**while লুপ** কোনো শর্ত সত্য থাকলে বারবার চলে।

### Python:
```python
i = 0
while i < 5:
    print(i)
    i += 1
```

### C:
```c
int i = 0;
while (i < 5) {
    printf("%d\n", i);
    i++;
}
```

---

## ৪. do-while লুপ

**do-while লুপ** (C, C++, Java ইত্যাদি):
- প্রথমে কোড ব্লক একবার চলে, তারপর শর্ত চেক হয়
- Python-এ নেই, তবে while True + break দিয়ে অনুকরণ করা যায়

### C:
```c
int i = 0;
do {
    printf("%d\n", i);
    i++;
} while (i < 5);
```

### Python (অনুকরণ):
```python
i = 0
while True:
    print(i)
    i += 1
    if i >= 5:
        break
```

---

## ৫. নেস্টেড লুপ

একটি লুপের ভিতরে আরেকটি লুপ চালানো যায়।

### উদাহরণ:
```python
for i in range(3):
    for j in range(2):
        print(i, j)
```

---

## ৬. ইনফিনিট লুপ

শর্ত কখনো মিথ্যা না হলে লুপ অনন্তকাল চলে—একে ইনফিনিট লুপ বলে।

### উদাহরণ:
```python
while True:
    print("চলছেই...")
    # break দিয়ে থামাতে হয়
```

---

## ৭. লুপ কন্ট্রোল: break, continue

- **break:** লুপ থামিয়ে দেয়
- **continue:** বর্তমান ইটারেশন বাদ দিয়ে পরের ইটারেশনে যায়

### উদাহরণ:
```python
for i in range(5):
    if i == 3:
        break
    print(i)  # 0 1 2

for i in range(5):
    if i == 3:
        continue
    print(i)  # 0 1 2 4
```

---

## ৮. অনুশীলন

১. ১ থেকে ১০ পর্যন্ত সংখ্যার যোগফল বের করুন (for/while লুপে)
২. নেস্টেড লুপ দিয়ে গুণের টেবিল (multiplication table) প্রিন্ট করুন
৩. ইনফিনিট লুপ থেকে break দিয়ে বের হওয়ার উদাহরণ দিন

---

## ৯. ভালো অভ্যাস

- লুপের শর্ত ও ভেরিয়েবল ঠিকভাবে ব্যবহার করুন
- ইনফিনিট লুপ এড়িয়ে চলুন (প্রয়োজন ছাড়া)
- নেস্টেড লুপ কম ব্যবহার করুন (পারফরম্যান্সের জন্য)
- break/continue বুঝে ব্যবহার করুন

---

## ১০. উপসংহার

লুপ ছাড়া প্রোগ্রামিং অসম্পূর্ণ। for, while, do-while লুপ ভালোভাবে জানলে যেকোনো পুনরাবৃত্তিমূলক কাজ সহজে করা যায়।
