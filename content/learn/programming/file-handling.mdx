ফাইল হ্যান্ডলিং হলো প্রোগ্রামিংয়ের একটি গুরুত্বপূর্ণ অংশ, যার মাধ্যমে টেক্সট ও বাইনারি ফাইল থেকে ডাটা পড়া ও লেখা যায়। এই অধ্যায়ে আমরা ফাইল হ্যান্ডলিংয়ের ধারণা, টেক্সট ও বাইনারি ফাইলের পার্থক্য, বিভিন্ন ভাষায় উদাহরণ, ভালো অভ্যাস ও অনুশীলন জানব।

---

## ১. ফাইল হ্যান্ডলিং কী?

- ফাইল থেকে ডাটা পড়া (read)
- ফাইলে ডাটা লেখা (write)
- ফাইল তৈরি, মুছে ফেলা, আপডেট করা

---

## ২. টেক্সট ফাইল বনাম বাইনারি ফাইল

| বৈশিষ্ট্য         | টেক্সট ফাইল         | বাইনারি ফাইল         |
|-------------------|---------------------|----------------------|
| ডাটা ফরম্যাট      | ASCII/Unicode টেক্সট| 0/1 (বাইনারি)        |
| পড়া-লেখা         | সহজ                | নির্দিষ্ট ফরম্যাট দরকার|
| উদাহরণ            | .txt, .csv, .md     | .jpg, .png, .exe     |

---

## ৩. Python-এ ফাইল পড়া ও লেখা

### টেক্সট ফাইল পড়া:
```python
with open('data.txt', 'r', encoding='utf-8') as f:
    content = f.read()
    print(content)
```

### টেক্সট ফাইলে লেখা:
```python
with open('output.txt', 'w', encoding='utf-8') as f:
    f.write('Hello, World!')
```

### বাইনারি ফাইল পড়া:
```python
with open('image.png', 'rb') as f:
    data = f.read()
```

### বাইনারি ফাইলে লেখা:
```python
with open('copy.png', 'wb') as f:
    f.write(data)
```

---

## ৪. JavaScript (Node.js)-এ ফাইল পড়া ও লেখা

### টেক্সট ফাইল পড়া:
```js
const fs = require('fs');
const data = fs.readFileSync('data.txt', 'utf8');
console.log(data);
```

### টেক্সট ফাইলে লেখা:
```js
fs.writeFileSync('output.txt', 'Hello, World!');
```

### বাইনারি ফাইল পড়া:
```js
const img = fs.readFileSync('image.png');
```

### বাইনারি ফাইলে লেখা:
```js
fs.writeFileSync('copy.png', img);
```

## ৫. ফাইল হ্যান্ডলিংয়ের ভালো অভ্যাস

- ফাইল বন্ধ করা (with/open context ব্যবহার করুন)
- এক্সসেপশন হ্যান্ডলিং করুন
- ফাইলের path ও encoding স্পষ্ট রাখুন
- বড় ফাইল chunk করে পড়ুন
