## try, catch, finally

এক্সসেপশন হ্যান্ডলিং হলো প্রোগ্রামিংয়ের এমন একটি কৌশল, যার মাধ্যমে কোডে কোনো এরর বা অপ্রত্যাশিত ঘটনা ঘটলে সেটি সুন্দরভাবে ধরা, সমাধান ও প্রয়োজনীয় পদক্ষেপ নেওয়া যায়। এই অধ্যায়ে আমরা এক্সসেপশন হ্যান্ডলিংয়ের ধারণা, প্রয়োজনীয়তা, try-catch-finally ব্লকের ব্যবহার, বিভিন্ন ভাষায় উদাহরণ, ভালো অভ্যাস ও অনুশীলন জানব।

---

## ১. এক্সসেপশন ও এরর পার্থক্য

- **এরর:** সাধারণত কোডে এমন সমস্যা, যা প্রোগ্রাম থামিয়ে দেয় (যেমন: সিনট্যাক্স, রানটাইম, লজিক)
- **এক্সসেপশন:** রানটাইমে বিশেষ ধরনের এরর, যাকে হ্যান্ডল করা যায়

---

## ২. এক্সসেপশন হ্যান্ডলিংয়ের প্রয়োজনীয়তা

- প্রোগ্রাম ক্র্যাশ না করে সুন্দরভাবে এরর মেসেজ দেখানো
- ইউজার এক্সপেরিয়েন্স ভালো রাখা
- ডিবাগিং সহজ করা

---

## ৩. try-catch-finally কী?

- **try:** সন্দেহজনক কোড ব্লক যেখানে এরর হতে পারে
- **catch/except:** এরর হলে যেটা চালানো হয়
- **finally:** এরর হোক বা না হোক, সবশেষে চালানো হয়

---

## ৪. উদাহরণ

### Python:
```python
try:
    x = int(input("Enter a number: "))
    y = 10 / x
except ValueError:
    print("সংখ্যা দিন!")
except ZeroDivisionError:
    print("শূন্য দিয়ে ভাগ করা যায় না!")
finally:
    print("সমাপ্ত!")
```

### JavaScript:
```js
try {
  let x = parseInt(prompt("Enter a number:"));
  let y = 10 / x;
} catch (e) {
  if (e instanceof TypeError) {
    alert("ভুল টাইপ!");
  } else {
    alert("এরর: " + e.message);
  }
} finally {
  alert("সমাপ্ত!");
}
```

---

## ৫. একাধিক এক্সসেপশন হ্যান্ডলিং

- Python-এ except ব্লক একাধিকবার ব্যবহার করা যায়
- JavaScript-এ catch ব্লকে instanceof দিয়ে চেক করা যায়

---

## ৬. কাস্টম এক্সসেপশন

- নিজস্ব এক্সসেপশন ক্লাস তৈরি করে নির্দিষ্ট এরর হ্যান্ডল করা যায়

### Python:
```python
class MyError(Exception):
    pass
try:
    raise MyError("নিজস্ব এরর")
except MyError as e:
    print(e)
```

---

## ৭. ভালো অভ্যাস

- শুধুমাত্র সন্দেহজনক কোড try ব্লকে রাখুন
- নির্দিষ্ট এক্সসেপশন ধরুন, সবকিছু except-এ ধরবেন না
- finally ব্লকে ক্লিনআপ (ফাইল বন্ধ, রিসোর্স মুক্ত) করুন
- এরর মেসেজ ইউজার-ফ্রেন্ডলি রাখুন

---

## ৮. অনুশীলন

১. Python-এ ইনপুট নিয়ে ZeroDivisionError ও ValueError হ্যান্ডল করুন
২. JavaScript-এ try-catch-finally দিয়ে ইউজার ইনপুট হ্যান্ডল করুন
৩. কাস্টম এক্সসেপশন তৈরি ও ব্যবহার করুন

---

## ৯. উপসংহার

এক্সসেপশন হ্যান্ডলিংয়ের মাধ্যমে প্রোগ্রাম আরও নির্ভরযোগ্য, ইউজার-ফ্রেন্ডলি ও মেইনটেইনেবল হয়। ভালোভাবে try-catch-finally ব্যবহার করলে এরর ম্যানেজমেন্ট সহজ হয়।
