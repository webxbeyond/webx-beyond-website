## অঙ্ক, লজিক, তুলনা, বিটওয়াইজ

প্রোগ্রামিংয়ের অন্যতম গুরুত্বপূর্ণ অংশ হলো অপারেটর। অপারেটর দিয়ে ভেরিয়েবলের মান পরিবর্তন, তুলনা, লজিক্যাল সিদ্ধান্ত, বিট লেভেল অপারেশন ইত্যাদি করা যায়। এই অধ্যায়ে আমরা অঙ্ক, লজিক, তুলনা ও বিটওয়াইজ অপারেটর নিয়ে বিস্তারিত জানব।

---

## ১. অঙ্ক অপারেটর (Arithmetic Operators)

অঙ্ক অপারেটর দিয়ে গাণিতিক হিসাব করা হয়।

| অপারেটর | অর্থ           | উদাহরণ (Python) |
|----------|----------------|-----------------|
| +        | যোগ            | `a + b`         |
| -        | বিয়োগ          | `a - b`         |
| *        | গুণ            | `a * b`         |
| /        | ভাগ            | `a / b`         |
| %        | ভাগশেষ (mod)  | `a % b`         |
| **       | ঘাত (power)    | `a ** b`        |
| //       | পূর্ণসংখ্যা ভাগ| `a // b`        |

### উদাহরণ:
```python
a = 10
b = 3
print(a + b)  # 13
print(a / b)  # 3.333...
print(a % b)  # 1
```

---

## ২. তুলনা অপারেটর (Comparison/Relational Operators)

তুলনা অপারেটর দিয়ে দুটি মানের মধ্যে সম্পর্ক নির্ধারণ করা হয়। ফলাফল হয় `True` বা `False`।

| অপারেটর | অর্থ           | উদাহরণ (Python) |
|----------|----------------|-----------------|
| ==       | সমান           | `a == b`        |
| !=       | অসমান          | `a != b`        |
| >        | বড়             | `a > b`         |
| `<`        | ছোট            | `a < b`         |
| >=       | বড় বা সমান     | `a >= b`        |
| `<=`       | ছোট বা সমান    | `a <= b`        |

### উদাহরণ:
```python
x = 5
y = 8
print(x == y)  # False
print(x < y)   # True
```

---

## ৩. লজিক্যাল অপারেটর (Logical Operators)

লজিক্যাল অপারেটর দিয়ে একাধিক শর্ত একত্রে যাচাই করা হয়।

| অপারেটর | অর্থ           | উদাহরণ (Python) |
|----------|----------------|-----------------|
| and      | এবং            | `a and b`       |
| or       | অথবা           | `a or b`        |
| not      | না             | `not a`         |

### উদাহরণ:
```python
x = True
y = False
print(x and y)  # False
print(x or y)   # True
print(not x)    # False
```

---

## ৪. বিটওয়াইজ অপারেটর (Bitwise Operators)

বিটওয়াইজ অপারেটর দিয়ে বিট লেভেলে অপারেশন করা হয়।

| অপারেটর | অর্থ           | উদাহরণ (Python) |
|----------|----------------|-----------------|
| &        | বিটওয়াইজ AND   | `a & b`         |
| |        | বিটওয়াইজ OR    | `a | b`         |
| ^        | বিটওয়াইজ XOR   | `a ^ b`         |
| ~        | বিটওয়াইজ NOT   | `~a`            |
| `<<`       | বাম শিফট       | `a << n`        |
| >>       | ডান শিফট       | `a >> n`        |

### উদাহরণ:
```python
a = 5      # 0101
b = 3      # 0011
print(a & b)   # 1 (0001)
print(a | b)   # 7 (0111)
print(a ^ b)   # 6 (0110)
print(~a)      # -6
print(a << 1)  # 10 (1010)
print(a >> 1)  # 2 (0010)
```

## ৫. ইনক্রিমেন্ট ও ডিক্রিমেন্ট অপারেটর

কিছু ভাষায় (যেমন C, C++, Java) ভেরিয়েবলের মান এক ধাপে বাড়ানো বা কমানোর জন্য `++` ও `--` অপারেটর আছে।

```c
int x = 5;
x++; // x = 6
x--; // x = 5
```

Python-এ সরাসরি নেই, তবে `x += 1` বা `x -= 1` ব্যবহার করা হয়।

## ৬. অ্যাসাইনমেন্ট অপারেটর (Assignment Operators)

ভেরিয়েবলে মান বসানোর জন্য ব্যবহৃত হয়।

| অপারেটর | অর্থ           | উদাহরণ (Python) |
|----------|----------------|-----------------|
| =        | মান বসানো      | `x = 10`        |
| +=       | যোগ করে বসানো | `x += 5`        |
| -=       | বিয়োগ করে বসানো| `x -= 2`        |
| *=       | গুণ করে বসানো | `x *= 3`        |
| /=       | ভাগ করে বসানো | `x /= 4`        |
| %=       | ভাগশেষ বসানো  | `x %= 3`        |

## ৭. অপারেটরের ব্যবহারিক উদাহরণ

### উদাহরণ ১: শর্ত নির্ধারণ
```python
age = 18
if age >= 18:
    print("বয়স্ক")
else:
    print("অপ্রাপ্তবয়স্ক")
```

### উদাহরণ ২: বিটওয়াইজ অপারেশন
```python
a = 12  # 1100
b = 5   # 0101
print(a & b)  # 4 (0100)
```
