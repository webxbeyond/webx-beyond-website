রিফ্যাক্টরিং হলো কোডের কার্যকারিতা না বদলে তার গঠন, রিডেবিলিটি ও মেইনটেনেবিলিটি উন্নত করা। অপটিমাইজেশন হলো কোডকে আরও দ্রুত, কম মেমরি বা কম রিসোর্সে চালানোর কৌশল। এই অধ্যায়ে আমরা রিফ্যাক্টরিং ও বেসিক অপটিমাইজেশনের ধারণা, কৌশল, উদাহরণ, ভালো অভ্যাস ও অনুশীলন জানব।

---

## ১. রিফ্যাক্টরিং কী?

- কোডের কার্যকারিতা অপরিবর্তিত রেখে গঠন, স্টাইল, রিডেবিলিটি উন্নত করা
- বাগ ফিক্স, ফিচার অ্যাড, টিমওয়ার্ক সহজ হয়

---

## ২. রিফ্যাক্টরিংয়ের কৌশল

1. অর্থবহ নামকরণ
2. বড় ফাংশন ভেঙে ছোট করা
3. ডুপ্লিকেট কোড সরানো
4. ডেড কোড/অপ্রয়োজনীয় অংশ মুছে ফেলা
5. কমেন্ট ও ডকুমেন্টেশন আপডেট
6. কনসিসটেন্ট স্টাইল

---

## ৩. অপটিমাইজেশন কী?

- কোড, অ্যালগরিদম, ডাটা স্ট্রাকচার আরও কার্যকর করা
- পারফরম্যান্স, মেমরি, রিসোর্স ব্যবহারে উন্নতি

---

## ৪. বেসিক অপটিমাইজেশনের কৌশল

1. অ্যালগরিদম ও ডাটা স্ট্রাকচার নির্বাচন
2. লুপ অপ্টিমাইজেশন
3. অপ্রয়োজনীয় ক্যালকুলেশন এড়িয়ে চলা
4. মেমরি লিক প্রতিরোধ
5. Lazy loading, caching

---

## ৫. উদাহরণ

### রিফ্যাক্টরিং (Python):
```python
# খারাপ কোড
def calc(a, b, c): return a+b+c
# ভালো কোড
def calculate_sum(a, b, c):
    """তিন সংখ্যার যোগফল ফেরত দেয়"""
    return a + b + c
```

### অপটিমাইজেশন (Python):
```python
# খারাপ কোড
result = []
for i in range(10000):
    result.append(i*i)
# ভালো কোড
result = [i*i for i in range(10000)]
```

- Premature optimization এড়িয়ে চলুন
- টিম স্টাইলগাইড ফলো করুন
