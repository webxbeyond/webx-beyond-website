স্ট্রিম হলো ডাটা প্রসেসিংয়ের একটি শক্তিশালী ধারণা, যেখানে ডাটা টুকরো টুকরো করে (continuous flow) পাঠানো, পড়া বা প্রসেস করা হয়। বড় ফাইল, নেটওয়ার্ক ডাটা, লাইভ ভিডিও/অডিও, রিয়েল-টাইম ডাটা প্রসেসিংয়ে স্ট্রিম অপরিহার্য। এই অধ্যায়ে আমরা স্ট্রিমের ধারণা, প্রকারভেদ, বিভিন্ন ভাষায় উদাহরণ, ভালো অভ্যাস ও অনুশীলন জানব।

---

## ১. স্ট্রিম কী?

- ডাটা একসাথে না পাঠিয়ে, ছোট ছোট অংশে পাঠানো
- মেমরি ব্যবহারে দক্ষ
- ইনপুট স্ট্রিম (পড়া), আউটপুট স্ট্রিম (লেখা)

---

## ২. স্ট্রিমের ব্যবহারক্ষেত্র

- বড় ফাইল পড়া/লেখা
- নেটওয়ার্ক ডাটা ট্রান্সফার
- লাইভ ভিডিও/অডিও স্ট্রিমিং
- রিয়েল-টাইম ডাটা প্রসেসিং

---

## ৩. Python-এ স্ট্রিম

### ফাইল স্ট্রিম:
```python
with open('large.txt', 'r', encoding='utf-8') as f:
    for line in f:
        print(line.strip())
```

### বাইনারি স্ট্রিম:
```python
with open('image.png', 'rb') as f:
    chunk = f.read(1024)
    while chunk:
        # প্রসেস করুন
        chunk = f.read(1024)
```

---

## ৪. Node.js-এ স্ট্রিম

### রিডেবল স্ট্রিম:
```js
const fs = require('fs');
const readStream = fs.createReadStream('large.txt', 'utf8');
readStream.on('data', chunk => {
  console.log(chunk);
});
```

### রাইটেবল স্ট্রিম:
```js
const writeStream = fs.createWriteStream('output.txt');
writeStream.write('Hello, Stream!\n');
writeStream.end();
```

### পাইপিং স্ট্রিম:
```js
const readStream = fs.createReadStream('input.txt');
const writeStream = fs.createWriteStream('output.txt');
readStream.pipe(writeStream);
```

---

## ৫. স্ট্রিমের ভালো অভ্যাস

- বড় ফাইল chunk করে প্রসেস করুন
- স্ট্রিম ক্লোজ করতে ভুলবেন না
- এরর হ্যান্ডলিং করুন (error event)
- মেমরি ব্যবহারে সতর্ক থাকুন

---

## ৬. অনুশীলন

১. Python-এ বড় ফাইল chunk করে পড়ুন
২. Node.js-এ স্ট্রিম দিয়ে ফাইল কপি করুন
৩. স্ট্রিমের সুবিধা ও সতর্কতা লিখুন

---

## ৭. উপসংহার

স্ট্রিম ব্যবহার করলে বড় ডাটা, লাইভ ডাটা ও রিয়েল-টাইম প্রসেসিং সহজ ও দক্ষ হয়। দক্ষভাবে স্ট্রিম ব্যবহার করলে সফটওয়্যার আরও শক্তিশালী ও স্কেলেবল হয়।
