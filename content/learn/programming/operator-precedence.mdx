প্রোগ্রামিংয়ে এক্সপ্রেশন লিখতে গেলে অনেক সময় একাধিক অপারেটর একসাথে ব্যবহার করতে হয়। কোন অপারেটর আগে কাজ করবে, কোনটা পরে—এটা নির্ধারণ করে অপারেটর প্রাধান্য (precedence) ও অ্যাসোসিয়েটিভিটি (associativity)। এই অধ্যায়ে আমরা এগুলো বিস্তারিতভাবে জানব।

---

## ১. অপারেটর প্রাধান্য (Operator Precedence) কী?

**অপারেটর প্রাধান্য** হলো কোন অপারেটর আগে এক্সিকিউট হবে, তার নিয়ম।

- উচ্চ প্রাধান্যের অপারেটর আগে কাজ করে
- নিম্ন প্রাধান্যের অপারেটর পরে কাজ করে

### উদাহরণ:
```python
result = 2 + 3 * 4
print(result)  # 14
```
এখানে `*` অপারেটরের প্রাধান্য `+` থেকে বেশি, তাই আগে গুণ হবে: `3 * 4 = 12`, তারপর `2 + 12 = 14`

---

## ২. অপারেটর অ্যাসোসিয়েটিভিটি (Associativity) কী?

**অ্যাসোসিয়েটিভিটি** নির্ধারণ করে, একই প্রাধান্যের একাধিক অপারেটর এক্সপ্রেশনে থাকলে কোন দিক থেকে অপারেশন হবে—বাম থেকে ডানে (left-to-right) নাকি ডান থেকে বামে (right-to-left)।

### উদাহরণ:
```python
x = 10 - 5 - 2
print(x)  # 3
```
এখানে `-` অপারেটরের অ্যাসোসিয়েটিভিটি বাম থেকে ডানে, তাই `(10 - 5) - 2 = 3`

## ৩. অপারেটর প্রাধান্য টেবিল (Python উদাহরণ)

| প্রাধান্য | অপারেটর                       | অ্যাসোসিয়েটিভিটি |
|-----------|-------------------------------|------------------|
| সর্বোচ্চ  | `()` (parentheses)            | -                |
|           | `**` (power)                  | right-to-left    |
|           | `+`, `-` (unary)              | right-to-left    |
|           | `*`, `/`, `//`, `%`           | left-to-right    |
|           | `+`, `-` (binary)             | left-to-right    |
|           | `<<`, `>>`                    | left-to-right    |
|           | `&`                           | left-to-right    |
|           | `^`                           | left-to-right    |
|           | `|`                           | left-to-right    |
|           | `==`, `!=`, `>`, `<`, `>=`, `<=` | left-to-right |
|           | `not`                         | right-to-left    |
|           | `and`                         | left-to-right    |
|           | `or`                          | left-to-right    |
| সর্বনিম্ন | `=` (assignment), `+=`, `-=`  | right-to-left    |

- জটিল এক্সপ্রেশনে ব্র্যাকেট ব্যবহার করুন
- এতে কোড পড়া সহজ হয় এবং ভুল কমে

### উদাহরণ:
```python

```

## ৬. উদাহরণ ও অনুশীলন

### উদাহরণ ১:
```python
x = 2 + 3 * 4 ** 2
# ** আগে, তারপর *, তারপর +
# 4 ** 2 = 16
# 3 * 16 = 48
# 2 + 48 = 50
print(x)  # 50
```

### উদাহরণ ২:
```python
x = 100 / 10 * 2
# / ও * এর প্রাধান্য সমান, অ্যাসোসিয়েটিভিটি বাম থেকে ডানে
# (100 / 10) * 2 = 10 * 2 = 20
print(x)  # 20
```
