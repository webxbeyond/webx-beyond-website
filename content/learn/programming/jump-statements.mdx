## break, continue, return

প্রোগ্রামিংয়ে লুপ ও ফাংশনের প্রবাহ নিয়ন্ত্রণের জন্য জাম্প স্টেটমেন্ট ব্যবহৃত হয়। এগুলোর মধ্যে সবচেয়ে গুরুত্বপূর্ণ হলো break, continue ও return। এই অধ্যায়ে আমরা এগুলোর ব্যবহার, গঠন, পার্থক্য, উদাহরণ ও ভালো অভ্যাস জানব।

---

## ১. break স্টেটমেন্ট

**break** দিয়ে লুপ বা switch/case ব্লক একদম থামিয়ে দেওয়া যায়।

### ব্যবহার:
- লুপের ভিতরে কোনো শর্তে break দিলে, লুপ সঙ্গে সঙ্গে শেষ হয়
- switch/case-এ case ব্লক শেষ করতে

### উদাহরণ (Python):
```python
for i in range(10):
    if i == 5:
        break
    print(i)  # 0 1 2 3 4
```

### উদাহরণ (C):
```c
for (int i = 0; i < 10; i++) {
    if (i == 5) break;
    printf("%d\n", i);
}
```

---

## ২. continue স্টেটমেন্ট

**continue** দিয়ে লুপের বর্তমান ইটারেশন বাদ দিয়ে পরের ইটারেশনে চলে যাওয়া যায়।

### ব্যবহার:
- কোনো শর্তে কোড ব্লকের বাকি অংশ স্কিপ করে পরের ইটারেশনে যেতে

### উদাহরণ (Python):
```python
for i in range(5):
    if i == 2:
        continue
    print(i)  # 0 1 3 4
```

### উদাহরণ (C):
```c
for (int i = 0; i < 5; i++) {
    if (i == 2) continue;
    printf("%d\n", i);
}
```

---

## ৩. return স্টেটমেন্ট

**return** দিয়ে ফাংশন থেকে মান ফেরত পাঠানো বা ফাংশন এক্সিকিউশন থামানো যায়।

### ব্যবহার:
- ফাংশনের এক্সিকিউশন শেষ করতে
- কোনো মান ফেরত দিতে

### উদাহরণ (Python):
```python
def add(a, b):
    return a + b

result = add(3, 4)
print(result)  # 7
```

### উদাহরণ (C):
```c
int add(int a, int b) {
    return a + b;
}
```

## ৪. break, continue ও return-এর পার্থক্য

| স্টেটমেন্ট | কোথায় ব্যবহৃত | কী করে                |
|------------|---------------|-----------------------|
| break      | লুপ, switch   | ব্লক একদম থামায়       |
| continue   | লুপ           | বর্তমান ইটারেশন বাদ   |
| return     | ফাংশন         | ফাংশন থামায়, মান ফেরত|

---

## ৫. নেস্টেড লুপে break ও continue

- break দিলে শুধু যে লুপে আছে সেটাই থামে
- নেস্টেড লুপে বাইরের লুপ থামাতে লেবেল (C, Java) বা ফ্ল্যাগ ব্যবহার করা হয়

### উদাহরণ (Python):
```python
for i in range(3):
    for j in range(3):
        if j == 1:
            break
        print(i, j)
# আউটপুট: 0 0, 1 0, 2 0
```

---

## ৬. অনুশীলন

১. break দিয়ে ১ থেকে ১০ পর্যন্ত সংখ্যার মধ্যে ৫ পর্যন্ত প্রিন্ট করুন
২. continue দিয়ে ১ থেকে ৫ পর্যন্ত সংখ্যার মধ্যে ৩ বাদে সব প্রিন্ট করুন
৩. return দিয়ে দুটি সংখ্যার যোগফল ফেরত দেওয়া ফাংশন লিখুন

---

## ৭. ভালো অভ্যাস

- break/continue বুঝে ব্যবহার করুন, বেশি ব্যবহার এড়িয়ে চলুন
- return সবসময় ফাংশনের শেষে ব্যবহার করুন (যদি সম্ভব হয়)
- জটিল নেস্টেড লুপে লজিক পরিষ্কার রাখুন

---

## ৮. উপসংহার

break, continue, return—এই জাম্প স্টেটমেন্টগুলো প্রোগ্রামিংয়ের প্রবাহ নিয়ন্ত্রণে অত্যন্ত গুরুত্বপূর্ণ। এগুলো ভালোভাবে জানলে কোড আরও দক্ষ ও রিডেবল হয়।
