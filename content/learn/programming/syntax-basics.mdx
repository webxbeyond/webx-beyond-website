## কীওয়ার্ড, আইডেন্টিফায়ার, অপারেটর

প্রোগ্রামিং ভাষার মৌলিক কাঠামো গঠনের জন্য সিনট্যাক্স, কীওয়ার্ড, আইডেন্টিফায়ার ও অপারেটর অত্যন্ত গুরুত্বপূর্ণ। এই টিউটোরিয়ালে আমরা এগুলোর সংজ্ঞা, বৈশিষ্ট্য, উদাহরণ ও ব্যবহারিক দিক নিয়ে বিস্তারিত আলোচনা করব।

## সিনট্যাক্স (Syntax) কী?

সিনট্যাক্স হচ্ছে কোনো প্রোগ্রামিং ভাষার গঠনগত নিয়মাবলি—কোড কীভাবে লিখতে হবে, কোন শব্দ বা চিহ্ন কোথায় বসবে, কোন কমান্ড কিভাবে গঠিত হবে ইত্যাদি।

- ভুল সিনট্যাক্স মানে কোড কম্পাইল/রান হবে না (Syntax Error)।
- প্রতিটি ভাষার নিজস্ব সিনট্যাক্স আছে।

### উদাহরণ (Python):
```python
print("Hello, World!")
```

### উদাহরণ (JavaScript):
```js
console.log("Hello, World!");
```

## কীওয়ার্ড (Keyword)

কীওয়ার্ড হচ্ছে ভাষার সংরক্ষিত শব্দ, যেগুলো বিশেষ অর্থ বহন করে এবং নির্দিষ্ট কাজের জন্য ব্যবহৃত হয়। এগুলো ভ্যারিয়েবল বা ফাংশনের নাম হিসেবে ব্যবহার করা যায় না।

### সাধারণ কীওয়ার্ডের উদাহরণ
- **Python:** if, else, for, while, def, return, import
- **JavaScript:** var, let, const, if, else, function, return
- **C:** int, float, if, else, return, for, while

### বৈশিষ্ট্য
- ভাষা নির্ধারিত (reserved)
- সিনট্যাক্স গঠনে অপরিহার্য
- সাধারণত ছোট হাতের অক্ষরে

### ভুল উদাহরণ
```python
def = 5  # ভুল, কারণ 'def' কীওয়ার্ড
```

## আইডেন্টিফায়ার (Identifier)

আইডেন্টিফায়ার হচ্ছে প্রোগ্রামে ব্যবহৃত নাম—যেমন ভ্যারিয়েবল, ফাংশন, ক্লাস, অবজেক্ট ইত্যাদির নাম।

### বৈশিষ্ট্য
- অক্ষর (a-z, A-Z), সংখ্যা (0-9), আন্ডারস্কোর (_) ব্যবহার করা যায়
- সংখ্যা দিয়ে শুরু করা যায় না
- কীওয়ার্ড ব্যবহার করা যায় না
- কেস-সেন্সিটিভ (Python, Java, JS)

### উদাহরণ
```python
name = "Anis"      # ভ্যারিয়েবল
get_age()           # ফাংশন
StudentClass        # ক্লাস
```

### ভুল উদাহরণ
```python
2name = "abc"   # ভুল, সংখ্যা দিয়ে শুরু
if = 10         # ভুল, কীওয়ার্ড
```

## অপারেটর (Operator)

অপারেটর হচ্ছে বিশেষ চিহ্ন বা শব্দ, যা ভ্যালু বা এক্সপ্রেশন নিয়ে নির্দিষ্ট অপারেশন করে।

### প্রধান অপারেটরের ধরন
- **Arithmetic:** `+`, `-`, `*`, `/`, `%`, `**` (গাণিতিক)
- **Assignment:** `=`, `+=`, `-=`, `*=`, `/=`
- **Comparison:** `==`, `!=`, `>`, `<`, `>=`, `<=`
- **Logical:** and, or, not (Python); `&&`, `||`, `!` (JS, C)
- **Bitwise:** `&`, `|`, `^`, `~`, `<<`, `>>`
- **Membership:** in, not in (Python)
- **Identity:** is, is not (Python)

### উদাহরণ
```python
x = 5 + 3      # + (Arithmetic)
x *= 2         # *= (Assignment)
if x > 10:     # > (Comparison)
    print("Big")
```

### অপারেটর ওভারলোডিং
কিছু ভাষায় (যেমন Python, C++) অপারেটর ওভারলোড করা যায়—অর্থাৎ, নিজের মতো করে অপারেটরের ব্যবহার নির্ধারণ করা যায়।

## সিনট্যাক্স, কীওয়ার্ড, আইডেন্টিফায়ার ও অপারেটরের সম্পর্ক

- সিনট্যাক্স নির্ধারণ করে কীভাবে কীওয়ার্ড, আইডেন্টিফায়ার ও অপারেটর একত্রে ব্যবহার হবে।
- ভুল সিনট্যাক্স বা ভুল কীওয়ার্ড/আইডেন্টিফায়ার ব্যবহার করলে কোড চলবে না।

## বিভিন্ন ভাষায় পার্থক্য

- **Python:** সহজ সিনট্যাক্স, কম কীওয়ার্ড, ইন্ডেন্টেশন গুরুত্বপূর্ণ
- **C:** সেমিকোলন, কার্লি ব্রেস, বেশি অপারেটর
- **JavaScript:** ফাংশন ডিফাইন, ভ্যারিয়েবল ডিক্লেয়ারেশনে ভিন্নতা

## উপসংহার

সিনট্যাক্স, কীওয়ার্ড, আইডেন্টিফায়ার ও অপারেটর—প্রোগ্রামিং ভাষার মৌলিক ভিত্তি। এগুলো ভালোভাবে না জানলে কোনো ভাষায় দক্ষতা অর্জন সম্ভব নয়।
