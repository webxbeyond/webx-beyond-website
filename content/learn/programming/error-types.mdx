## সিনট্যাক্স, রানটাইম, লজিক

প্রোগ্রামিংয়ে এরর বা ত্রুটি একটি সাধারণ বিষয়। এরর বিভিন্ন ধরনের হতে পারে—সিনট্যাক্স, রানটাইম ও লজিক। এই অধ্যায়ে আমরা এই তিন ধরনের এররের বৈশিষ্ট্য, উদাহরণ, কারণ, প্রতিরোধ ও সমাধান নিয়ে বিস্তারিত আলোচনা করব।

---

## ১. এরর কী?

- প্রোগ্রাম চলাকালীন বা লেখার সময় যে সমস্যা দেখা দেয়, তাকে এরর বলে
- এরর কোড এক্সিকিউশন থামিয়ে দিতে পারে বা ভুল ফলাফল দিতে পারে

---

## ২. সিনট্যাক্স এরর (Syntax Error)

- কোড লেখার সময় ভাষার নিয়ম না মানলে হয়
- কম্পাইলার/ইন্টারপ্রিটার সঙ্গে সঙ্গে ধরতে পারে
- প্রোগ্রাম রানই হয় না

### উদাহরণ (Python):
```python
print("Hello World!"
# বন্ধনী না থাকায় সিনট্যাক্স এরর
```

### উদাহরণ (JavaScript):
```js
if (x > 5) {
  console.log("Hi");
// বন্ধনী না থাকলে সিনট্যাক্স এরর
```

---

## ৩. রানটাইম এরর (Runtime Error)

- কোড সঠিকভাবে লেখা, কিন্তু এক্সিকিউশনের সময় সমস্যা
- ইনপুট, ডিভাইড বাই জিরো, ফাইল না পাওয়া ইত্যাদি
- প্রোগ্রাম চলার সময় থেমে যায়

### উদাহরণ (Python):
```python
x = 10 / 0  # ZeroDivisionError
```

### উদাহরণ (JavaScript):
```js
let arr = [1,2,3];
console.log(arr[5].toString()); // Cannot read property
```

---

## ৪. লজিক এরর (Logic Error)

- কোড চলে, কিন্তু কাঙ্ক্ষিত ফলাফল দেয় না
- ভুল অ্যালগরিদম, ভুল শর্ত, ভুল লুপ ইত্যাদি
- সবচেয়ে কঠিন ধরতে

### উদাহরণ (Python):
```python
def is_even(n):
    return n % 2 == 1  # ভুল: আসলে n % 2 == 0 হওয়া উচিত
```

### উদাহরণ (JavaScript):
```js
function add(a, b) {
  return a - b; // ভুল: যোগের বদলে বিয়োগ
}
```

## ৫. এররের কারণ

- টাইপো, ভুল সিনট্যাক্স
- ইনপুটের ভুল ধরন
- ভুল অ্যালগরিদম
- ফাইল/নেটওয়ার্ক সমস্যা

## ৬. এরর প্রতিরোধ ও সমাধান

- কোড রিভিউ ও টেস্টিং
- ইনপুট ভ্যালিডেশন
- এক্সসেপশন হ্যান্ডলিং
- ডিবাগিং টুল ব্যবহার
