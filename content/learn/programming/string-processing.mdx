স্ট্রিং ও টেক্সট প্রসেসিং প্রোগ্রামিংয়ের অত্যন্ত গুরুত্বপূর্ণ অংশ। ডাটা বিশ্লেষণ, ইউজার ইনপুট, ফাইল প্রসেসিং, ওয়েব ডেভেলপমেন্ট—সবখানেই স্ট্রিং নিয়ে কাজ করতে হয়। এই অধ্যায়ে আমরা স্ট্রিং-এর ধারণা, গঠন, ম্যানিপুলেশন, কমন অপারেশন, রেগুলার এক্সপ্রেশন, বিভিন্ন ভাষায় ব্যবহার, সমস্যা ও ভালো অভ্যাস জানব।

---

## ১. স্ট্রিং কী?

- অক্ষর, সংখ্যা, চিহ্নের সিকোয়েন্স (text data)
- সাধারণত ইমিউটেবল (Python, Java), কিছু ভাষায় মিউটেবল (C)

### Python:
```python
s = "Hello, World!"
```

### C:
```c
char s[] = "Hello, World!";
```

---

## ২. স্ট্রিং অপারেশন

### দৈর্ঘ্য (Length):
```python
len(s)
```

### সাবস্ট্রিং (Substring):
```python
s[0:5]  # 'Hello'
```

### কনক্যাটেনেশন (Concatenation):
```python
s1 + s2
```

### রিপিট (Repeat):
```python
"ha" * 3  # 'hahaha'
```

### খোঁজা (Search):
```python
"World" in s  # True
```

### রিপ্লেস (Replace):
```python
s.replace("World", "Python")
```

### স্প্লিট (Split):
```python
s.split(",")
```

### ট্রিম (Trim):
```python
s.strip()
```

---

## ৩. স্ট্রিং ইটারেশন ও ম্যানিপুলেশন

### Python:
```python
for c in s:
    print(c)
```

### C:
```c
for (int i = 0; s[i] != '\0'; i++) {
    printf("%c\n", s[i]);
}
```

---

## ৪. রেগুলার এক্সপ্রেশন (Regular Expression)

- প্যাটার্ন দিয়ে টেক্সট খোঁজা, মডিফাই, ভ্যালিডেশন
- Python: re মডিউল

### উদাহরণ:
```python
import re
text = "My email is test@example.com"
pattern = r"[\w.-]+@[\w.-]+\.\w+"
match = re.search(pattern, text)
if match:
    print(match.group())  # test@example.com
```

---

## ৫. স্ট্রিং ফরম্যাটিং

### Python:
```python
name = "Rahim"
age = 25
msg = f"Name: {name}, Age: {age}"
```

### C:
```c
char name[] = "Rahim";
printf("Name: %s\n", name);
```

---

## ৬. স্ট্রিং ইমিউটেবিলিটি

- Python, Java-তে স্ট্রিং ইমিউটেবল (পরিবর্তন করা যায় না)
- C-তে মিউটেবল (char array)

---

## ৭. ইউনিকোড ও মাল্টিল্যাঙ্গুয়েজ টেক্সট

- Python 3-এ স্ট্রিং ইউনিকোড সাপোর্টেড
- C-তে wchar_t, Java-তে UTF-16

---

## ৮. সমস্যা ও সতর্কতা

- স্ট্রিং কনক্যাটেনেশনে পারফরম্যান্স সমস্যা (বড় লুপে)
- ইনপুট ভ্যালিডেশন না করলে সিকিউরিটি ইস্যু
- এনকোডিং সমস্যা (Unicode vs ASCII)

---

## ৯. ভালো অভ্যাস

- স্ট্রিং ফরম্যাটিং ব্যবহার করুন
- ইনপুট ভ্যালিডেশন করুন
- বড় টেক্সট প্রসেসিংয়ে স্ট্রিমিং/বাফার ব্যবহার করুন
- ইউনিকোড সাপোর্ট নিশ্চিত করুন

---

## ১০. অনুশীলন

১. Python-এ ইউজার ইনপুট নিয়ে রিভার্স স্ট্রিং প্রিন্ট করুন
২. রেগুলার এক্সপ্রেশন দিয়ে ইমেইল খুঁজুন
৩. C-তে স্ট্রিং কনক্যাটেনেশনের উদাহরণ দিন

---

## ১১. উপসংহার

স্ট্রিং ও টেক্সট প্রসেসিং জানা থাকলে ডাটা বিশ্লেষণ, ইউজার ইন্টারঅ্যাকশন, ফাইল প্রসেসিং—সবকিছু সহজ হয়।
