স্ট্যান্ডার্ড ইনপুট (stdin) ও আউটপুট (stdout) হলো প্রোগ্রামিংয়ের মৌলিক ধারণা, যার মাধ্যমে প্রোগ্রাম বাইরের ডাটা নেয় ও আউটপুট দেয়। এই অধ্যায়ে আমরা স্ট্যান্ডার্ড ইনপুট-আউটপুট কী, কিভাবে কাজ করে, বিভিন্ন ভাষায় উদাহরণ, ভালো অভ্যাস ও অনুশীলন জানব।

---

## ১. স্ট্যান্ডার্ড ইনপুট (stdin) কী?

- ইউজার বা বাইরের সোর্স থেকে ডাটা নেওয়ার উপায়
- সাধারণত কীবোর্ড ইনপুট

### উদাহরণ (Python):
```python
name = input("তোমার নাম কী? ")
print("স্বাগতম,", name)
```

### উদাহরণ (JavaScript - Node.js):
```js
const readline = require('readline').createInterface({
  input: process.stdin,
  output: process.stdout
});
readline.question('তোমার নাম কী? ', name => {
  console.log(`স্বাগতম, ${name}`);
  readline.close();
});
```

---

## ২. স্ট্যান্ডার্ড আউটপুট (stdout) কী?

- প্রোগ্রাম থেকে বাইরের দিকে ডাটা পাঠানো
- সাধারণত স্ক্রিনে প্রিন্ট

### উদাহরণ (Python):
```python
print("Hello, World!")
```

### উদাহরণ (JavaScript):
```js
console.log("Hello, World!");
```

---

## ৩. স্ট্যান্ডার্ড এরর (stderr)

- এরর বা ওয়ার্নিং আলাদা আউটপুট চ্যানেলে পাঠানো

### উদাহরণ (Python):
```python
import sys
print("এটি এরর", file=sys.stderr)
```

### উদাহরণ (JavaScript):
```js
console.error("এটি এরর");
```

---

## ৪. ফাইল রিডাইরেকশন ও পাইপ

- ইনপুট/আউটপুট ফাইল বা অন্য প্রোগ্রামে পাঠানো যায়
- Shell-এ `>`, `<`, | ব্যবহার

### উদাহরণ (Bash):
```bash
echo "Hello" > output.txt  # আউটপুট ফাইলে
python script.py < input.txt  # ইনপুট ফাইল থেকে
cat file.txt | grep "data"  # পাইপ
```

## ৫. ভালো অভ্যাস

- ইনপুট ভ্যালিডেশন করুন
- আউটপুট ফরম্যাট স্পষ্ট রাখুন
- এরর আউটপুট আলাদা রাখুন

---

## ৬. অনুশীলন

১. Python-এ ইউজার ইনপুট নিয়ে গুণফল বের করুন
২. Node.js-এ ইনপুট নিয়ে আউটপুট দেখান
৩. Bash-এ আউটপুট ফাইলে রিডাইরেক্ট করুন

## ৭. উপসংহার

স্ট্যান্ডার্ড ইনপুট-আউটপুট প্রোগ্রামিংয়ের ভিত্তি। দক্ষভাবে ব্যবহার করলে প্রোগ্রাম আরও ইন্টারঅ্যাকটিভ ও শক্তিশালী হয়।
