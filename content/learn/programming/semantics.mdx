# সিনট্যাক্সের অর্থ

প্রোগ্রামিং ভাষার সিনট্যাক্স বলে দেয় কোড কীভাবে লিখতে হবে, আর সেমান্টিক্স বলে দেয় সেই কোডের অর্থ বা কার্যকারিতা কী। অর্থাৎ, সেমান্টিক্স হচ্ছে—কোডের আড়ালে লুকিয়ে থাকা অর্থ, লজিক ও আচরণ।

## সেমান্টিক্স (Semantics) কী?

সেমান্টিক্স হচ্ছে কোনো প্রোগ্রামিং ভাষার স্টেটমেন্ট, এক্সপ্রেশন, ব্লক ইত্যাদির অর্থ, অর্থাৎ—কোড কী করবে, কীভাবে আচরণ করবে, কোন কন্ডিশনে কী হবে ইত্যাদি।

- একই সিনট্যাক্স, ভিন্ন সেমান্টিক্স হতে পারে (ভিন্ন ভাষায়)
- সঠিক সিনট্যাক্স মানেই সঠিক অর্থ নয়

### উদাহরণ
```python
x = 5 + 3
```
- সিনট্যাক্স: অ্যাসাইনমেন্ট স্টেটমেন্ট
- সেমান্টিক্স: x-এ ৮ সংরক্ষণ হবে

```python
if x = 5:
    print("Hi")
```
- সিনট্যাক্স: ভুল (Python-এ = অ্যাসাইনমেন্ট, == তুলনা)
- সেমান্টিক্স: ভুল অর্থ, তাই সিনট্যাক্স এরর

## সিনট্যাক্স বনাম সেমান্টিক্স

| দিক                | সিনট্যাক্স                | সেমান্টিক্স                |
|--------------------|--------------------------|----------------------------|
| সংজ্ঞা             | গঠনগত নিয়ম              | অর্থ ও আচরণ                |
| চেক করে           | কম্পাইলার/ইন্টারপ্রেটার  | রানটাইম/কম্পাইলার          |
| ভুল হলে            | Syntax Error             | Logic/Semantic Error       |
| উদাহরণ             | `if (x > 0) { ... }`       | `x > 0` হলে ব্লক এক্সিকিউট    |

## সেমান্টিক্সের ধরন

1. **স্ট্যাটিক সেমান্টিক্স:**
   - কম্পাইল টাইমে চেক হয়
   - যেমন: টাইপ চেকিং, ভ্যারিয়েবল ডিক্লেয়ারেশন
   - উদাহরণ: C-তে টাইপ মিসম্যাচ
2. **ডাইনামিক সেমান্টিক্স:**
   - রানটাইমে চেক হয়
   - যেমন: ডিভাইড বাই জিরো, নাল রেফারেন্স
   - উদাহরণ: Python-এ ZeroDivisionError

## সেমান্টিক এরর কী?

- কোডের সিনট্যাক্স ঠিক, কিন্তু অর্থ বা লজিক ভুল
- উদাহরণ:
```python
x = 10
y = 0
z = x / y   # সিনট্যাক্স ঠিক, কিন্তু রানটাইমে এরর (ZeroDivisionError)
```

## বিভিন্ন ভাষায় সেমান্টিক্স

- **Python:** ডাইনামিক টাইপিং, ইন্ডেন্টেশন গুরুত্বপূর্ণ
- **C:** স্ট্যাটিক টাইপিং, টাইপ মিসম্যাচে কম্পাইল টাইম এরর
- **JavaScript:** টাইপ কনভার্সন, কোয়ের্সিভ সেমান্টিক্স

## সেমান্টিক্সের গুরুত্ব

- কোডের অর্থ ও কার্যকারিতা বোঝা
- বাগ ও লজিক্যাল এরর খুঁজে বের করা
- নিরাপদ ও নির্ভরযোগ্য সফটওয়্যার ডেভেলপমেন্ট

## সেমান্টিক্স চেকিং টুল

- **লিন্টার:** কোডের লজিক্যাল ভুল চেক করে (ESLint, Pylint)
- **টাইপ চেকার:** টাইপ সংক্রান্ত সেমান্টিক্স চেক (TypeScript, MyPy)

## উপসংহার

সিনট্যাক্স শেখার পাশাপাশি সেমান্টিক্স বোঝা আরও গুরুত্বপূর্ণ—কারণ, কোড শুধু ঠিকভাবে লেখা নয়, ঠিকভাবে কাজ করাও জরুরি।
