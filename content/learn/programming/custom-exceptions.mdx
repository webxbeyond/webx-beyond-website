কাস্টম এক্সসেপশন হলো নিজের প্রয়োজন অনুযায়ী নতুন এক্সসেপশন ক্লাস তৈরি করা, যাতে নির্দিষ্ট এরর বা বিশেষ পরিস্থিতি সুন্দরভাবে হ্যান্ডল করা যায়। এই অধ্যায়ে আমরা কাস্টম এক্সসেপশন কী, কেন প্রয়োজন, কীভাবে তৈরি ও ব্যবহার করতে হয়, বিভিন্ন ভাষায় উদাহরণ, ভালো অভ্যাস ও অনুশীলন জানব।

---

## ১. কাস্টম এক্সসেপশন কী?

- প্রোগ্রামিং ভাষার ডিফল্ট এক্সসেপশন ছাড়াও নিজের মতো এক্সসেপশন তৈরি
- নির্দিষ্ট এরর বা কন্ডিশন হ্যান্ডল করা সহজ হয়

---

## ২. কেন কাস্টম এক্সসেপশন দরকার?

- স্পষ্ট ও অর্থবহ এরর মেসেজ
- বড় প্রজেক্টে নির্দিষ্ট এরর আলাদা করে ধরা
- ডিবাগিং ও মেইনটেনেন্স সহজ

---

## ৩. Python-এ কাস্টম এক্সসেপশন

### কিভাবে তৈরি করবেন:
```python
class NegativeNumberError(Exception):
    def __init__(self, value):
        self.value = value
        super().__init__(f"নেগেটিভ সংখ্যা: {value}")

def square_root(x):
    if x < 0:
        raise NegativeNumberError(x)
    return x ** 0.5

try:
    print(square_root(-5))
except NegativeNumberError as e:
    print(e)
```

---

## ৪. JavaScript-এ কাস্টম এক্সসেপশন

### কিভাবে তৈরি করবেন:
```js
class NegativeNumberError extends Error {
  constructor(value) {
    super(`নেগেটিভ সংখ্যা: ${value}`);
    this.name = "NegativeNumberError";
  }
}

function squareRoot(x) {
  if (x < 0) {
    throw new NegativeNumberError(x);
  }
  return Math.sqrt(x);
}

try {
  console.log(squareRoot(-5));
} catch (e) {
  if (e instanceof NegativeNumberError) {
    console.log(e.message);
  }
}
```
