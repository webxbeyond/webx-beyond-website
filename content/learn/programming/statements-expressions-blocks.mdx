প্রোগ্রামিং ভাষার মৌলিক গঠন বোঝার জন্য স্টেটমেন্ট, এক্সপ্রেশন ও ব্লকের ধারণা অত্যন্ত গুরুত্বপূর্ণ। এগুলো ছাড়া কোনো কার্যকর কোড লেখা সম্ভব নয়। এই টিউটোরিয়ালে আমরা এগুলোর সংজ্ঞা, পার্থক্য, উদাহরণ ও ব্যবহারিক দিক নিয়ে বিস্তারিত আলোচনা করব।

## এক্সপ্রেশন (Expression) কী?

এক্সপ্রেশন হচ্ছে এমন কোডের অংশ, যা একটি ভ্যালু রিটার্ন করে। এটি ভ্যারিয়েবল, অপারেটর, ফাংশন কল, কনস্ট্যান্ট ইত্যাদি নিয়ে গঠিত হতে পারে।

### উদাহরণ
```python
5 + 3           # 8
x * 2           # x-এর মানের দ্বিগুণ
get_age()       # ফাংশন কলের রিটার্ন ভ্যালু
"Hello" + name # স্ট্রিং কনক্যাট
```

- এক্সপ্রেশন সবসময় একটি ভ্যালু দেয়।
- এক্সপ্রেশনকে অন্য এক্সপ্রেশনের অংশ হিসেবে ব্যবহার করা যায়।

## স্টেটমেন্ট (Statement) কী?

স্টেটমেন্ট হচ্ছে এমন কোডের অংশ, যা কোনো অ্যাকশন বা নির্দেশনা দেয়। এটি এক বা একাধিক এক্সপ্রেশন নিয়ে গঠিত হতে পারে।

### উদাহরণ
```python
x = 5 + 3           # অ্যাসাইনমেন্ট স্টেটমেন্ট
print("Hello")      # ফাংশন কল স্টেটমেন্ট
if x > 10:          # কন্ডিশনাল স্টেটমেন্ট
    print("Big")
```

- স্টেটমেন্ট কোডের কার্যকরী ইউনিট
- স্টেটমেন্ট এক্সপ্রেশনও থাকতে পারে, আবার নাও থাকতে পারে

### স্টেটমেন্টের ধরন
- অ্যাসাইনমেন্ট: `x = 10`
- কন্ডিশনাল: `if`, `else`, `switch`
- লুপ: `for`, `while`
- ফাংশন ডেফিনিশন: `def`, `function`
- জাম্প: `break`, `continue`, `return`

## ব্লক (Block) কী?

ব্লক হচ্ছে একাধিক স্টেটমেন্টের সমষ্টি, যা একসাথে গ্রুপ করা হয়। সাধারণত কার্লি ব্রেস `{}` (C, JS) বা ইন্ডেন্টেশন (Python) দিয়ে ব্লক নির্ধারণ করা হয়।

### উদাহরণ (Python):
```python
if x > 0:
    print("Positive")
    x = x - 1
```

### উদাহরণ (JavaScript):
```js
if (x > 0) {
  console.log("Positive");
  x = x - 1;
}
```

- ব্লক সাধারণত কন্ট্রোল স্ট্রাকচার (if, for, while, function) এর ভেতরে ব্যবহৃত হয়
- ব্লকের মধ্যে একাধিক স্টেটমেন্ট থাকতে পারে

## এক্সপ্রেশন ও স্টেটমেন্টের পার্থক্য

| বৈশিষ্ট্য         | এক্সপ্রেশন                | স্টেটমেন্ট                |
|-------------------|--------------------------|--------------------------|
| উদ্দেশ্য           | ভ্যালু রিটার্ন            | অ্যাকশন/নির্দেশনা         |
| উদাহরণ            | 5 + 3, x * 2             | x = 5, if x > 0: ...     |
| কম্পোজেবল        | হ্যাঁ (অন্য এক্সপ্রেশনে)  | না (সবসময় নয়)           |
| রানটাইম ইফেক্ট    | ভ্যালু তৈরি              | কার্যকরী পরিবর্তন         |

## ব্লকের গুরুত্ব

- কোড সংগঠিত ও পড়তে সহজ হয়
- স্কোপ নির্ধারণ (ভ্যারিয়েবল, ফাংশন)
- কন্ট্রোল ফ্লো (if, for, while, function)

## বিভিন্ন ভাষায় পার্থক্য

- **Python:** ইন্ডেন্টেশন দিয়ে ব্লক
- **C, Java, JS:** কার্লি ব্রেস `{}` দিয়ে ব্লক
- **Ruby:** `do ... end`

## উপসংহার

স্টেটমেন্ট, এক্সপ্রেশন ও ব্লক—প্রোগ্রামিং ভাষার গঠনগত ভিত্তি। এগুলো ভালোভাবে না জানলে কোনো ভাষায় দক্ষতা অর্জন সম্ভব নয়।
