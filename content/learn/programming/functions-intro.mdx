প্রোগ্রামিংয়ে ফাংশন (Function) হলো কোডের পুনঃব্যবহারযোগ্য ব্লক, যা নির্দিষ্ট কাজ সম্পাদন করে। ফাংশন ব্যবহারে কোড ছোট, পরিষ্কার ও মডুলার হয়। এই অধ্যায়ে আমরা ফাংশনের উদ্দেশ্য, সুবিধা, ধরন, গঠন, উদাহরণ ও ভালো অভ্যাস জানব।

---

## ১. ফাংশন কী?

- ফাংশন হলো নির্দিষ্ট কাজের জন্য লেখা কোড ব্লক
- একাধিকবার ডাকা যায় (reuse)
- ইনপুট (parameter/argument) নিতে পারে, আউটপুট (return value) দিতে পারে

### উদাহরণ (Python):
```python
def greet():
    print("Hello, World!")

greet()  # ফাংশন কল
```

### উদাহরণ (C):
```c
void greet() {
    printf("Hello, World!\n");
}

greet();
```

---

## ২. ফাংশন ব্যবহারের উদ্দেশ্য

- কোড পুনঃব্যবহার (reuse)
- কোড ছোট ও মডুলার রাখা
- জটিল কাজ ছোট ছোট অংশে ভাগ করা
- ডিবাগ ও টেস্ট সহজ করা

---

## ৩. ফাংশনের ধরন

### ১. বিল্ট-ইন ফাংশন
- ভাষায় আগে থেকেই থাকা (যেমন: print, len, printf, scanf)

### ২. ইউজার-ডিফাইন্ড ফাংশন
- প্রোগ্রামার নিজে তৈরি করে

### ৩. পিওর ও ইম্পিওর ফাংশন
- **পিওর:** একই ইনপুটে সবসময় একই আউটপুট (side-effect নেই)
- **ইম্পিওর:** side-effect থাকতে পারে (যেমন: ফাইল লেখা, গ্লোবাল ভেরিয়েবল পরিবর্তন)

### ৪. রিকার্সিভ ফাংশন
- ফাংশন নিজেকে নিজে কল করে

---

## ৪. ফাংশনের গঠন

### Python:
```python
def function_name(parameters):
    # কোড ব্লক
    return value
```

### C:
```c
return_type function_name(parameters) {
    // কোড ব্লক
    return value;
}
```

---

## ৫. ফাংশন কল ও আর্গুমেন্ট

- ফাংশন ডাকার সময় আর্গুমেন্ট পাঠানো হয়
- ফাংশন ইনপুট নেয়, আউটপুট দেয়

### উদাহরণ (Python):
```python
def add(a, b):
    return a + b

result = add(3, 4)
print(result)  # 7
```

---

## ৬. ফাংশনের সুবিধা

- কোড পুনঃব্যবহারযোগ্য
- কোড ছোট ও পরিষ্কার
- টেস্ট ও ডিবাগ সহজ
- বড় প্রজেক্টে টিমওয়ার্ক সহজ

## ৭. অনুশীলন

১. Python-এ দুটি সংখ্যার গুণফল ফেরত দেওয়া ফাংশন লিখুন
২. C-তে ফাংশন দিয়ে দুটি সংখ্যার যোগফল বের করুন
৩. রিকার্সিভ ফাংশনের উদাহরণ দিন

---

## ৮. ভালো অভ্যাস

- ফাংশনের নাম অর্থবহ দিন
- এক ফাংশনে এক কাজ রাখুন
- ইনপুট ও আউটপুট স্পষ্ট রাখুন
- প্রয়োজন ছাড়া গ্লোবাল ভেরিয়েবল ব্যবহার এড়িয়ে চলুন

---

## ৯. উপসংহার

ফাংশন ছাড়া প্রোগ্রামিং অসম্পূর্ণ। ফাংশনের উদ্দেশ্য, ধরন ও ভালো অভ্যাস জানা থাকলে কোড আরও দক্ষ, পরিষ্কার ও রক্ষণযোগ্য হয়।
