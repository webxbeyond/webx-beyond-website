“হ্যালো ওয়ার্ল্ড” প্রোগ্রাম হলো যেকোনো প্রোগ্রামিং ভাষা শেখার প্রথম ধাপ। এতে সাধারণত স্ক্রিনে “Hello, World!” আউটপুট দেখানো হয়। এই অধ্যায়ে আমরা বিভিন্ন জনপ্রিয় ভাষায় “হ্যালো ওয়ার্ল্ড” প্রোগ্রামের উদাহরণ, সিনট্যাক্সের পার্থক্য, বিশ্লেষণ ও অনুশীলন জানব।

---

## ১. C
```c
#include <stdio.h>
int main() {
    printf("Hello, World!\n");
    return 0;
}
```

## ২. C++
```cpp
#include <iostream>
int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
```

## ৩. Python
```python
print("Hello, World!")
```

## ৪. Java
```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

## ৫. JavaScript
```js
console.log("Hello, World!");
```

## ৬. Go
```go
package main
import "fmt"
func main() {
    fmt.Println("Hello, World!")
}
```

## ৭. Rust
```rust
fn main() {
    println!("Hello, World!");
}
```

## ৮. PHP
```php
<?php
echo "Hello, World!";
```

## ৯. Ruby
```ruby
puts "Hello, World!"
```

## ১০. Bash
```bash
echo "Hello, World!"
```

|--------------|-------------------|---------------------|--------|
