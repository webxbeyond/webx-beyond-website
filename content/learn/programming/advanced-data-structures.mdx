## হ্যাশম্যাপ, ট্রি, গ্রাফ

অ্যাডভান্সড ডাটা স্ট্রাকচার ডাটা ম্যানিপুলেশন ও অ্যালগরিদম ডিজাইনে অত্যন্ত গুরুত্বপূর্ণ। এই অধ্যায়ে আমরা হ্যাশম্যাপ (HashMap), ট্রি (Tree), ও গ্রাফ (Graph) নিয়ে বিস্তারিত জানব—তাদের গঠন, ব্যবহার, উদাহরণ, তুলনা ও ভালো অভ্যাসসহ।

---

## ১. হ্যাশম্যাপ (HashMap)

- কী-ভ্যালু (key-value) পেয়ার ডাটা স্ট্রাকচার
- দ্রুত ইনসার্ট, ডিলিট ও সার্চ
- Python-এ dict, Java-তে HashMap

### Python:
```python
d = {"name": "Rahim", "age": 25}
print(d["name"])  # Rahim
d["city"] = "Dhaka"
```

### Java:
```java
import java.util.HashMap;
HashMap<String, Integer> map = new HashMap<>();
map.put("a", 1);
System.out.println(map.get("a"));
```

### ব্যবহার:
- ডাটাবেস ইনডেক্সিং
- ক্যাশিং
- কাউন্টিং

---

## ২. ট্রি (Tree)

- হায়ারার্কিকাল ডাটা স্ট্রাকচার
- রুট (root), নোড (node), লিফ (leaf)
- বাইনারি ট্রি, BST, AVL, হিপ, ইত্যাদি

### বাইনারি ট্রি (Binary Tree):
- প্রতিটি নোডের সর্বোচ্চ দুইটি চাইল্ড

### Python (বেসিক):
```python
class Node:
    def __init__(self, data):
        self.data = data
        self.left = None
        self.right = None

root = Node(1)
root.left = Node(2)
root.right = Node(3)
```

### C (বেসিক):
```c
struct Node {
    int data;
    struct Node* left;
    struct Node* right;
};
```

### ব্যবহার:
- ফাইল সিস্টেম
- পার্সিং
- সার্চ অ্যালগরিদম

---

## ৩. গ্রাফ (Graph)

- নোড (vertex) ও এজ (edge) নিয়ে গঠিত
- ডাটা ও তাদের সম্পর্ক প্রকাশে ব্যবহৃত
- ডাইরেক্টেড/আনডাইরেক্টেড, ওয়েটেড/আনওয়েটেড

### Python (adjacency list):
```python
graph = {
    0: [1, 2],
    1: [0, 2],
    2: [0, 1, 3],
    3: [2]
}
```

### C (adjacency matrix):
```c
#define V 4
int graph[V][V] = {
    {0, 1, 1, 0},
    {1, 0, 1, 0},
    {1, 1, 0, 1},
    {0, 0, 1, 0}
};
```

### ব্যবহার:
- সোশ্যাল নেটওয়ার্ক
- রোড নেটওয়ার্ক
- ওয়েব ক্রলিং

## ৪. তুলনা

| স্ট্রাকচার | গঠন | অ্যাক্সেস | ব্যবহার |
|------------|------|----------|---------|
| হ্যাশম্যাপ | কী-ভ্যালু | দ্রুত | ইনডেক্সিং, ক্যাশ |
| ট্রি       | হায়ারার্কিকাল | দ্রুত (BST) | সার্চ, পার্সিং |
| গ্রাফ      | নোড-এজ | জটিল | নেটওয়ার্ক, রিলেশন |

---

## ৫. ভালো অভ্যাস

- প্রয়োজন অনুযায়ী ডাটা স্ট্রাকচার বাছাই করুন
- হ্যাশম্যাপে কোলিশন হ্যান্ডলিং বুঝে ব্যবহার করুন
- ট্রি ও গ্রাফ ট্রাভার্সাল (DFS, BFS) শিখুন
- মেমোরি ব্যবস্থাপনায় সতর্ক থাকুন
