# Rate Limiting ও Circuit Breaking

## পরিচিতি

Modern app-এ reliability ও abuse protection—দুইটাই must-have। Traefik-এর rate limiting ও circuit breaking middleware দিয়ে traffic control, abuse prevention, এবং backend stability automate করা যায়। এই পাঠে জানব, rate limiting ও circuit breaking কী, কেন দরকার, কীভাবে Traefik-এ সেটআপ করবেন, এবং real-world example ও pitfalls।

---

## Rate Limiting Middleware

- **Rate Limiting**—একই IP/user থেকে নির্দিষ্ট সময়ের মধ্যে কতগুলো request আসতে পারবে, তার সীমা নির্ধারণ করা
- DDoS (Distributed Denial of Service), brute-force, বা bot abuse ঠেকাতে must-have

### Config Example

```yaml
http:
  middlewares:
    rate-limit:
      rateLimit:
        average: 100
        burst: 50
```

- **average:** প্রতি সেকেন্ডে কতটি request
- **burst:** short time-এ কতটি extra request tolerate করবে

### Real-life Use Case
Netflix—API endpoint-এ rate limit enforce করে, abuse ও overload ঠেকাতে

---

## Circuit Breaking Middleware

- **Circuit Breaking**—backend service fail হলে, auto block/redirect করে, overload ও cascading failure ঠেকায়
- Microservices, API gateway, বা distributed app-এ must-have

### Config Example

```yaml
http:
  services:
    my-service:
      loadBalancer:
        servers:
          - url: "http://app1:80"
        healthCheck:
          path: "/health"
          interval: "10s"
          timeout: "2s"
        circuitBreaker:
          expression: "NetworkErrorRatio() > 0.5"
```

- **expression:** failure condition—যেমন network error ratio

### Real-life Use Case
Google—API call fail হলে circuit breaker trigger, cascading failure ঠেকায়

---

## Common Pitfalls (সাধারণ ভুল)

- **Rate limit ভুল config:** Legitimate user block হতে পারে
- **Circuit breaker miss:** Backend overload হলে downtime বাড়তে পারে
- **Monitoring miss:** Trigger condition না বুঝলে false positive

---

## Best Practices

- Rate limit value realistic রাখুন—user experience খারাপ না হয়
- Circuit breaker expression logically define করুন
- Monitoring/logging setup করুন—trigger condition validate করুন
- Config change/staging-এ test করুন
