## পরিচিতি

রিঅ্যাক্ট শেখার প্রথম ধাপ হলো একটি সঠিক ডেভেলপমেন্ট এনভায়রনমেন্ট তৈরি করা। এটি এমন একটি জায়গা যেখানে আপনি সহজে কোড লিখতে, পরীক্ষা করতে এবং আপনার অ্যাপ চালাতে পারবেন।

এই পাঠে আমরা শিখব কিভাবে Node.js এবং npm ইনস্টল করতে হয়, এবং কীভাবে Vite বা Create React App (CRA) ব্যবহার করে একটি নতুন রিঅ্যাক্ট প্রজেক্ট শুরু করবেন।

## Node.js এবং npm কী?

### সহজ ভাষায়—একটি উপমা

ধরা যাক, আপনি একটি রেসিপি বই পড়ছেন।
- **Node.js** হলো সেই শেফ, যিনি রান্না করেন (জাভাস্ক্রিপ্ট চালান)।
- **npm** হলো সেই বাজার, যেখানে আপনি রান্নার জন্য প্রয়োজনীয় উপকরণ (লাইব্রেরি) কিনতে পারেন।

### Node.js
Node.js একটি জাভাস্ক্রিপ্ট রানটাইম যা আপনাকে সার্ভার-সাইডে জাভাস্ক্রিপ্ট চালাতে দেয়। এটি রিঅ্যাক্ট প্রজেক্ট চালানোর জন্য প্রয়োজন।

### npm
npm (Node Package Manager) হলো একটি টুল, যা আপনাকে বিভিন্ন প্যাকেজ (লাইব্রেরি) ইনস্টল করতে সাহায্য করে।

## Node.js এবং npm ইনস্টলেশন

### ধাপ ১: Node.js ডাউনলোড করুন
1. [Node.js অফিসিয়াল ওয়েবসাইটে](https://nodejs.org) যান।
2. LTS (Long Term Support) সংস্করণটি ডাউনলোড করুন।

### ধাপ ২: ইনস্টল করুন
1. ডাউনলোড করা ফাইলটি চালান।
2. নির্দেশনা অনুসরণ করে ইনস্টলেশন সম্পন্ন করুন।

### ধাপ ৩: যাচাই করুন
ইনস্টলেশন সফল হয়েছে কিনা তা যাচাই করতে টার্মিনালে নিচের কমান্ড দিন:
```bash
node -v
npm -v
```

আপনি যদি Node.js এবং npm-এর সংস্করণ দেখতে পান, তাহলে ইনস্টলেশন সফল।

## Vite এবং CRA কী?

### Vite
Vite হলো একটি আধুনিক টুল, যা দ্রুত এবং সহজে রিঅ্যাক্ট প্রজেক্ট তৈরি করতে সাহায্য করে। এটি দ্রুত ডেভেলপমেন্ট সার্ভার এবং হট রিলোডিং সাপোর্ট করে।

### Create React App (CRA)
CRA হলো রিঅ্যাক্ট টিমের তৈরি একটি টুল, যা আপনাকে দ্রুত একটি রিঅ্যাক্ট প্রজেক্ট শুরু করতে দেয়। এটি সহজ এবং নতুনদের জন্য উপযুক্ত।

## Vite দিয়ে রিঅ্যাক্ট প্রজেক্ট শুরু করা

### ধাপ ১: একটি নতুন প্রজেক্ট তৈরি করুন
```bash
npm create vite@latest my-react-app --template react
```

### ধাপ ২: ডিপেন্ডেন্সি ইনস্টল করুন
```bash
cd my-react-app
npm install
```

### ধাপ ৩: ডেভেলপমেন্ট সার্ভার চালান
```bash
npm run dev
```

### ধাপ ৪: ব্রাউজারে অ্যাক্সেস করুন
টার্মিনালে দেওয়া URL ব্রাউজারে খুলুন।

## CRA দিয়ে রিঅ্যাক্ট প্রজেক্ট শুরু করা

### ধাপ ১: একটি নতুন প্রজেক্ট তৈরি করুন
```bash
npx create-react-app my-react-app
```

### ধাপ ২: প্রজেক্ট ফোল্ডারে যান
```bash
cd my-react-app
```

### ধাপ ৩: ডেভেলপমেন্ট সার্ভার চালান
```bash
npm start
```

### ধাপ ৪: ব্রাউজারে অ্যাক্সেস করুন
আপনার ডিফল্ট ব্রাউজারে অ্যাপটি খুলে যাবে।

২. Vite ব্যবহার করে একটি নতুন রিঅ্যাক্ট প্রজেক্ট তৈরি করুন।
