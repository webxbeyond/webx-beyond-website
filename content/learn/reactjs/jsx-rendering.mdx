## পরিচিতি

আপনি কি কখনো ভেবেছেন, রিঅ্যাক্ট কোড দেখতে HTML-এর মতো হলেও এটি আসলে কীভাবে কাজ করে? এর উত্তর হলো JSX। JSX (JavaScript XML) হলো রিঅ্যাক্টের একটি বিশেষ সিনট্যাক্স, যা জাভাস্ক্রিপ্ট এবং HTML-এর মিশ্রণ। এটি রিঅ্যাক্টে UI তৈরি করার একটি সহজ এবং কার্যকর উপায়।

এই পাঠে আমরা শিখব JSX কী, এটি কীভাবে কাজ করে, এবং রেন্ডারিং-এর বেসিক ধারণা।

## JSX কী?

### সহজ ভাষায়—একটি উপমা

ধরা যাক, আপনি একটি চিঠি লিখছেন।
- **HTML** হলো সেই চিঠির ভাষা।
- **JSX** হলো সেই ভাষার একটি নতুন রূপ, যেখানে আপনি একই চিঠিতে ইংরেজি এবং বাংলা মিশিয়ে লিখতে পারেন।

JSX হলো জাভাস্ক্রিপ্টের একটি এক্সটেনশন, যা HTML-এর মতো দেখতে। এটি জাভাস্ক্রিপ্ট কোডের মধ্যে HTML লিখতে দেয়।

### উদাহরণ:
```jsx
const element = <h1>হ্যালো, রিঅ্যাক্ট!</h1>;
```

উপরের কোডটি দেখতে HTML-এর মতো হলেও এটি আসলে জাভাস্ক্রিপ্ট।

### কেন JSX ব্যবহার করবেন?
- এটি কোড পড়া এবং লেখা সহজ করে।
- এটি রিঅ্যাক্ট কম্পোনেন্ট তৈরি করতে কার্যকর।
- এটি জাভাস্ক্রিপ্টের শক্তি এবং HTML-এর সরলতা একত্রিত করে।

## JSX কীভাবে কাজ করে?

JSX সরাসরি ব্রাউজারে কাজ করে না। এটি Babel-এর মতো একটি ট্রান্সপাইলার ব্যবহার করে জাভাস্ক্রিপ্টে রূপান্তরিত হয়।

### উদাহরণ:
JSX:
```jsx
const element = <h1>হ্যালো, রিঅ্যাক্ট!</h1>;
```

রূপান্তরিত জাভাস্ক্রিপ্ট:
```javascript
const element = React.createElement('h1', null, 'হ্যালো, রিঅ্যাক্ট!');
```

## রেন্ডারিং কী?

রেন্ডারিং হলো সেই প্রক্রিয়া, যেখানে রিঅ্যাক্ট আপনার JSX কোডকে DOM-এ রূপান্তরিত করে এবং ব্রাউজারে দেখায়।

### ReactDOM.render
ReactDOM.render() হলো সেই ফাংশন, যা JSX কোডকে DOM-এ রেন্ডার করে।

### উদাহরণ:
```jsx
import React from 'react';
import ReactDOM from 'react-dom';

const element = <h1>হ্যালো, রিঅ্যাক্ট!</h1>;
ReactDOM.render(element, document.getElementById('root'));
```

উপরের কোডটি একটি `<h1>` ট্যাগকে DOM-এ রেন্ডার করবে।

## JSX-এর নিয়ম

### ১. একটি প্যারেন্ট এলিমেন্ট
JSX-এ সবসময় একটি প্যারেন্ট এলিমেন্ট থাকতে হবে।

ভুল:
```jsx
return (
  <h1>হ্যালো</h1>
  <h2>বিশ্ব</h2>
);
```

সঠিক:
```jsx
return (
  <div>
    <h1>হ্যালো</h1>
    <h2>বিশ্ব</h2>
  </div>
);
```

### ২. ক্লাস নয়, className
JSX-এ HTML-এর `class` অ্যাট্রিবিউটের পরিবর্তে `className` ব্যবহার করতে হয়।

### ৩. জাভাস্ক্রিপ্ট এক্সপ্রেশন
JSX-এ `{}` এর মধ্যে জাভাস্ক্রিপ্ট এক্সপ্রেশন ব্যবহার করা যায়।

উদাহরণ:
```jsx
const name = 'রিঅ্যাক্ট';
const element = <h1>হ্যালো, {name}!</h1>;
```

## অনুশীলন

### কাজ:

১. একটি নতুন রিঅ্যাক্ট প্রজেক্ট তৈরি করুন।

২. নিচের JSX কোডটি লিখুন এবং রেন্ডার করুন:
```jsx
const element = (
  <div>
    <h1>রিঅ্যাক্ট শেখা শুরু করুন</h1>
    <p>এটি একটি চমৎকার লাইব্রেরি।</p>
  </div>
);
ReactDOM.render(element, document.getElementById('root'));
```

৩. JSX-এর নিয়ম অনুসরণ করে একটি কাস্টম কম্পোনেন্ট তৈরি করুন।
