## পরিচিতি

রিঅ্যাক্টের সবচেয়ে গুরুত্বপূর্ণ অংশ হলো "কম্পোনেন্ট"। কম্পোনেন্ট হলো একটি ছোট কোড ব্লক, যা UI-এর একটি নির্দিষ্ট অংশ তৈরি করে। এটি পুনরায় ব্যবহারযোগ্য এবং বড় অ্যাপ্লিকেশনকে ছোট ছোট অংশে ভাগ করতে সাহায্য করে।

এই পাঠে আমরা শিখব রিঅ্যাক্ট কম্পোনেন্ট কী, ফাংশন এবং ক্লাস কম্পোনেন্টের মধ্যে পার্থক্য, এবং কোনটি কখন ব্যবহার করবেন।

## রিঅ্যাক্ট কম্পোনেন্ট কী?

### সহজ ভাষায়—একটি উপমা

ধরা যাক, আপনি একটি বাড়ি তৈরি করছেন।
- **কম্পোনেন্ট** হলো সেই বাড়ির ইট। প্রতিটি ইট ছোট, কিন্তু একসাথে মিলিয়ে একটি বড় বাড়ি তৈরি হয়।

রিঅ্যাক্টে, কম্পোনেন্ট হলো UI-এর ছোট ছোট অংশ, যা একসাথে মিলিয়ে পুরো অ্যাপ তৈরি করে।

### উদাহরণ:
```jsx
function Welcome() {
  return <h1>স্বাগতম!</h1>;
}
```

উপরের কোডটি একটি ফাংশন কম্পোনেন্ট, যা একটি `<h1>` ট্যাগ রেন্ডার করে।

## ফাংশন কম্পোনেন্ট

ফাংশন কম্পোনেন্ট হলো একটি জাভাস্ক্রিপ্ট ফাংশন, যা JSX রিটার্ন করে। এটি সহজ এবং নতুনদের জন্য উপযুক্ত।

### উদাহরণ:
```jsx
function Greeting() {
  return <h1>হ্যালো, রিঅ্যাক্ট!</h1>;
}
```

### বৈশিষ্ট্য:
- সহজ এবং সংক্ষিপ্ত
- React Hooks ব্যবহার করতে পারে
- নতুন রিঅ্যাক্ট সংস্করণে বেশি ব্যবহৃত

## ক্লাস কম্পোনেন্ট

ক্লাস কম্পোনেন্ট হলো একটি ES6 ক্লাস, যা React.Component থেকে এক্সটেন্ড করে। এটি state এবং lifecycle মেথড ব্যবহার করতে পারে।

### উদাহরণ:
```jsx
class Greeting extends React.Component {
  render() {
    return <h1>হ্যালো, রিঅ্যাক্ট!</h1>;
  }
}
```

### বৈশিষ্ট্য:
- state এবং lifecycle মেথড সাপোর্ট করে
- পুরনো রিঅ্যাক্ট কোডে বেশি ব্যবহৃত

## ফাংশন বনাম ক্লাস কম্পোনেন্ট

| বৈশিষ্ট্য              | ফাংশন কম্পোনেন্ট             | ক্লাস কম্পোনেন্ট             |
|-----------------------|-----------------------------|-----------------------------|
| সিনট্যাক্স             | সহজ এবং সংক্ষিপ্ত            | জটিল এবং দীর্ঘ               |
| state                | React Hooks ব্যবহার করে      | this.state ব্যবহার করে       |
| পারফরম্যান্স           | দ্রুত                         | তুলনামূলক ধীর                |
| ব্যবহারের প্রবণতা      | নতুন কোডে বেশি ব্যবহৃত       | পুরনো কোডে বেশি ব্যবহৃত      |

## কোনটি কখন ব্যবহার করবেন?

### ফাংশন কম্পোনেন্ট ব্যবহার করুন যদি:
- আপনি নতুন রিঅ্যাক্ট সংস্করণ ব্যবহার করছেন।
- আপনার কম্পোনেন্টে state বা lifecycle মেথড দরকার নেই।
- আপনি React Hooks ব্যবহার করতে চান।

### ক্লাস কম্পোনেন্ট ব্যবহার করুন যদি:
- আপনি পুরনো রিঅ্যাক্ট কোড মেইনটেইন করছেন।
- আপনার কম্পোনেন্টে state বা lifecycle মেথড দরকার।

## অনুশীলন

### কাজ:

১. একটি ফাংশন কম্পোনেন্ট তৈরি করুন, যা একটি `<h1>` ট্যাগ রেন্ডার করে।

২. একটি ক্লাস কম্পোনেন্ট তৈরি করুন, যা একটি `<h1>` ট্যাগ রেন্ডার করে।

৩. ফাংশন এবং ক্লাস কম্পোনেন্টের মধ্যে পার্থক্য লিখুন।

৪. React Hooks ব্যবহার করে একটি stateful ফাংশন কম্পোনেন্ট তৈরি করুন।
