# useState দিয়ে স্টেট ম্যানেজমেন্ট

## পরিচিতি

রিঅ্যাক্ট অ্যাপ্লিকেশনে ডেটা ম্যানেজ করার জন্য স্টেট একটি গুরুত্বপূর্ণ ধারণা। স্টেট হলো একটি ডেটা স্ট্রাকচার, যা কম্পোনেন্টের ভেতরে থাকে এবং সময়ের সাথে পরিবর্তিত হতে পারে।

এই পাঠে আমরা শিখব `useState` হুক কী, এটি কীভাবে কাজ করে, এবং এটি ব্যবহার করে স্টেট ম্যানেজ করার পদ্ধতি।

## স্টেট কী?

### সহজ ভাষায়—একটি উপমা

ধরা যাক, আপনি একটি ঘড়ি তৈরি করছেন।
- **ঘড়ির সময়** হলো স্টেট। এটি পরিবর্তিত হয়, কিন্তু ঘড়ি (কম্পোনেন্ট) একই থাকে।

স্টেট হলো সেই ডেটা, যা একটি কম্পোনেন্টের ভেতরে থাকে এবং UI-তে পরিবর্তন আনে।

## useState হুক কী?

`useState` হলো একটি রিঅ্যাক্ট হুক, যা ফাংশন কম্পোনেন্টে স্টেট যোগ করতে দেয়। এটি একটি আর্রে রিটার্ন করে, যেখানে:
1. প্রথম আইটেম হলো বর্তমান স্টেট।
2. দ্বিতীয় আইটেম হলো স্টেট আপডেট করার ফাংশন।

### উদাহরণ:
```jsx
import React, { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>বর্তমান সংখ্যা: {count}</p>
      <button onClick={() => setCount(count + 1)}>বাড়ান</button>
    </div>
  );
}
```

উপরের কোডে:
- `count` হলো বর্তমান স্টেট।
- `setCount` হলো স্টেট আপডেট করার ফাংশন।
- `useState(0)` দিয়ে স্টেটের প্রাথমিক মান `0` সেট করা হয়েছে।

## useState কীভাবে কাজ করে?

### ধাপ ১: useState ইমপোর্ট করুন
```jsx
import React, { useState } from 'react';
```

### ধাপ ২: স্টেট ডিক্লেয়ার করুন
```jsx
const [state, setState] = useState(initialValue);
```

### ধাপ ৩: স্টেট ব্যবহার করুন
```jsx
<p>{state}</p>
```

### ধাপ ৪: স্টেট আপডেট করুন
```jsx
setState(newValue);
```

## স্টেট আপডেট করার নিয়ম

1. **স্টেট ইমিউটেবল:** সরাসরি স্টেট পরিবর্তন করবেন না।
   ভুল:
   ```jsx
   state = newValue;
   ```
   সঠিক:
   ```jsx
   setState(newValue);
   ```

2. **স্টেট আপডেট অ্যাসিনক্রোনাস:** স্টেট আপডেট করার পরপরই নতুন মান পাওয়া যাবে না।

3. **স্টেট ম্যানেজমেন্ট সহজ রাখুন:** স্টেট যতটা সম্ভব সরল রাখুন।

## useState-এর ব্যবহার

### সংখ্যা বাড়ানো:
```jsx
function Increment() {
  const [count, setCount] = useState(0);

  return (
    <button onClick={() => setCount(count + 1)}>
      সংখ্যা বাড়ান: {count}
    </button>
  );
}
```

### টেক্সট আপডেট করা:
```jsx
function TextUpdater() {
  const [text, setText] = useState('প্রথম টেক্সট');

  return (
    <div>
      <input
        type="text"
        value={text}
        onChange={(e) => setText(e.target.value)}
      />
      <p>আপনার টেক্সট: {text}</p>
    </div>
  );
}
```

### টগল করা:
```jsx
function Toggle() {
  const [isOn, setIsOn] = useState(false);

  return (
    <button onClick={() => setIsOn(!isOn)}>
      {isOn ? 'চালু' : 'বন্ধ'}
    </button>
  );
}
```

## অনুশীলন

### কাজ:

১. একটি `Counter` কম্পোনেন্ট তৈরি করুন, যা একটি সংখ্যা বাড়াবে এবং কমাবে।

২. একটি `Toggle` কম্পোনেন্ট তৈরি করুন, যা চালু এবং বন্ধ অবস্থার মধ্যে পরিবর্তন করবে।

৩. একটি `Form` কম্পোনেন্ট তৈরি করুন, যা ইনপুট ফিল্ডের মান স্টেট দিয়ে ম্যানেজ করবে।

## উপসংহার

`useState` হুক রিঅ্যাক্টে স্টেট ম্যানেজ করার একটি সহজ এবং কার্যকর উপায়। এটি ফাংশন কম্পোনেন্টে ডায়নামিক ডেটা যোগ করতে সাহায্য করে। পরবর্তী পাঠে আমরা শিখব ইভেন্ট হ্যান্ডলিং সম্পর্কে।
