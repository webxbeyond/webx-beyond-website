## পরিচিতি

রিঅ্যাক্ট কম্পোনেন্টের মধ্যে ডেটা পাস করার জন্য Props (Properties) ব্যবহার করা হয়। এটি একটি কম্পোনেন্টকে আরও ডাইনামিক এবং পুনরায় ব্যবহারযোগ্য করে তোলে।

এই পাঠে আমরা শিখব Props কী, এটি কীভাবে কাজ করে, এবং Children কীভাবে Props-এর একটি বিশেষ অংশ হিসেবে কাজ করে।

## Props কী?

### সহজ ভাষায়—একটি উপমা

ধরা যাক, আপনি একটি রেস্তোরাঁয় গেছেন এবং একটি পিৎজা অর্ডার করেছেন।
- **পিৎজা** হলো কম্পোনেন্ট।
- **আপনার পছন্দ অনুযায়ী টপিংস** হলো Props।

Props হলো সেই ডেটা, যা একটি কম্পোনেন্টের বাইরে থেকে পাঠানো হয়। এটি Immutable, অর্থাৎ কম্পোনেন্ট এটি পরিবর্তন করতে পারে না।

### উদাহরণ:
```jsx
function Welcome(props) {
  return <h1>স্বাগতম, {props.name}!</h1>;
}

<Welcome name="রিঅ্যাক্ট" />
```

উপরের কোডে, `name` একটি Prop, যা `Welcome` কম্পোনেন্টে পাঠানো হয়েছে।

## Props কীভাবে কাজ করে?

Props একটি অবজেক্ট হিসেবে কম্পোনেন্টে পাস হয়।

### উদাহরণ:
```jsx
function Greeting(props) {
  console.log(props);
  return <h1>হ্যালো, {props.name}!</h1>;
}

<Greeting name="বিশ্ব" />
```

উপরের কোডে, `props` অবজেক্টে `{ name: "বিশ্ব" }` থাকবে।

## Children কী?

### সহজ ভাষায়—একটি উপমা

ধরা যাক, আপনি একটি গিফট বক্স কিনেছেন।
- **গিফট বক্স** হলো কম্পোনেন্ট।
- **বক্সের ভেতরের উপহার** হলো Children।

Children হলো সেই JSX, যা একটি কম্পোনেন্টের ওপেনিং এবং ক্লোজিং ট্যাগের মধ্যে থাকে।

### উদাহরণ:
```jsx
function Box(props) {
  return <div className="box">{props.children}</div>;
}

<Box>
  <p>এটি একটি গিফট বক্স।</p>
</Box>
```

উপরের কোডে, `<p>এটি একটি গিফট বক্স।</p>` হলো `Box` কম্পোনেন্টের Children।

## Props এবং Children-এর ব্যবহার

### Props ব্যবহার:
```jsx
function Button(props) {
  return <button>{props.label}</button>;
}

<Button label="ক্লিক করুন" />
```

### Children ব্যবহার:
```jsx
function Card(props) {
  return <div className="card">{props.children}</div>;
}

<Card>
  <h2>শিরোনাম</h2>
  <p>এটি একটি কার্ড।</p>
</Card>
```

## Props এবং State-এর মধ্যে পার্থক্য

| বৈশিষ্ট্য              | Props                         | State                         |
|-----------------------|------------------------------|------------------------------|
| উৎস                  | প্যারেন্ট কম্পোনেন্ট          | কম্পোনেন্ট নিজেই             |
| পরিবর্তনশীলতা         | Immutable                    | Mutable                      |
| ব্যবহার               | ডেটা পাস করার জন্য            | ডেটা ম্যানেজ করার জন্য        |

## অনুশীলন

### কাজ:

১. একটি `Profile` কম্পোনেন্ট তৈরি করুন, যা Props ব্যবহার করে নাম এবং বয়স দেখাবে।

২. একটি `Container` কম্পোনেন্ট তৈরি করুন, যা Children ব্যবহার করে যেকোনো কন্টেন্ট রেন্ডার করবে।

৩. Props এবং State-এর মধ্যে পার্থক্য লিখুন।

## উপসংহার

Props এবং Children রিঅ্যাক্ট কম্পোনেন্টের মধ্যে ডেটা পাস করার একটি গুরুত্বপূর্ণ উপায়। এগুলো ব্যবহার করে আপনি আরও ডাইনামিক এবং পুনরায় ব্যবহারযোগ্য কম্পোনেন্ট তৈরি করতে পারবেন। পরবর্তী পাঠে আমরা শিখব `useState` দিয়ে স্টেট ম্যানেজমেন্ট।
