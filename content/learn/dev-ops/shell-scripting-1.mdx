শেল স্ক্রিপ্টিংয়ের প্রথম ধাপে আমরা শিখব—কীভাবে স্ক্রিপ্ট লেখা হয়, কোন কোন সিনট্যাক্স ব্যবহার হয়, এবং কীভাবে ভ্যারিয়েবল, কন্ডিশন, লুপ ইত্যাদি ব্যবহার করা যায়।

## স্ক্রিপ্টের গঠন

প্রতিটি শেল স্ক্রিপ্ট শুরু হয় `shebang` দিয়ে, যা বলে দেয় কোন শেল দিয়ে স্ক্রিপ্টটি চালানো হবে।

```bash
#!/bin/bash
```

এরপর এক বা একাধিক কমান্ড, ভ্যারিয়েবল, কন্ডিশন, লুপ ইত্যাদি লেখা হয়।

## ভ্যারিয়েবল

শেল স্ক্রিপ্টে ভ্যারিয়েবল ডিক্লেয়ার করা খুব সহজ।

```bash
name="Anis"
echo $name
```

- ভ্যারিয়েবল ডিক্লেয়ার: `name="Anis"`
- ভ্যালু ব্যবহার: `$name`

## ইউজার ইনপুট

ইউজার থেকে ইনপুট নিতে `read` কমান্ড ব্যবহার হয়।

```bash
echo "আপনার নাম লিখুন: "
read username
echo "স্বাগতম, $username!"
```

## কন্ডিশন (if-else)

কন্ডিশন চেক করতে `if`, `else` ব্যবহার হয়।

```bash
num=10
if [ $num -gt 5 ]
then
  echo "Number is greater than 5"
else
  echo "Number is 5 or less"
fi
```

## লুপ (for, while)

### for loop
```bash
for i in 1 2 3 4 5
do
  echo "Number: $i"
done
```

### while loop
```bash
count=1
while [ $count -le 5 ]
do
  echo "Count: $count"
  count=$((count+1))
done
```

## রিয়েল লাইফ উদাহরণ

### Example 1: ফাইলের লিস্ট প্রিন্ট করা
```bash
#!/bin/bash
for file in /home/user/*.txt
do
  echo "File: $file"
done
```

### Example 2: ইউজার থেকে ইনপুট নিয়ে ফোল্ডার তৈরি
```bash
#!/bin/bash
echo "নতুন ফোল্ডারের নাম দিন: "
read foldername
mkdir $foldername
echo "$foldername ফোল্ডার তৈরি হয়েছে!"
```

## কমেন্ট

স্ক্রিপ্টে কমেন্ট লিখতে `#` ব্যবহার হয়।

```bash
# এটি একটি কমেন্ট
```

## স্ক্রিপ্ট রান করার নিয়ম

1. স্ক্রিপ্ট লিখুন (myscript.sh)
2. executable করুন: `chmod +x myscript.sh`
3. চালান: `./myscript.sh`

## শেষ কথা

এই পর্বে শেল স্ক্রিপ্টিংয়ের বেসিক কনসেপ্ট ও সিনট্যাক্স শিখলেন। পরবর্তী পর্বে আরও অ্যাডভান্সড কনসেপ্ট, ফাংশন, এবং এরর হ্যান্ডলিং নিয়ে আলোচনা হবে।
