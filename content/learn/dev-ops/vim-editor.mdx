VIM হলো লিনাক্সের সবচেয়ে জনপ্রিয় ও শক্তিশালী টেক্সট এডিটর। এটি CLI-তে চলে, তাই সার্ভার, স্ক্রিপ্ট, কনফিগ ফাইল এডিট করতে VIM অপরিহার্য।

## VIM কী?

VIM (Vi IMproved) হলো Vi-এর উন্নত সংস্করণ। এটি ফাস্ট, কাস্টমাইজেবল, এবং প্রায় সব লিনাক্স ডিস্ট্রোতে বিল্ট-ইন।

## কেন VIM শিখবেন?

- **Remote Editing:** SSH দিয়ে রিমোট সার্ভারে ফাইল এডিট
- **Speed:** GUI এডিটরের চেয়ে দ্রুত
- **Automation:** স্ক্রিপ্ট, কনফিগ, লগ—সবকিছু CLI-তে
- **Resource Friendly:** কম রিসোর্সে চলে

## VIM চালু করা

```bash
vim filename.txt
```

## VIM-এর মোড

- **Normal Mode:** কমান্ড চালানো (ডিফল্ট)
- **Insert Mode:** লেখা/এডিট (i চাপুন)
- **Visual Mode:** সিলেক্ট
- **Command Mode:** : দিয়ে কমান্ড (যেমন :wq)

## বেসিক কমান্ড

| কমান্ড | কাজ |
|--------|-----|
| i      | ইনসার্ট মোডে যান |
| Esc    | নরমাল মোডে ফিরুন |
| :w     | সেভ |
| :q     | কুইট |
| :wq    | সেভ ও কুইট |
| :q!    | ফোর্স কুইট |
| dd     | লাইন ডিলিট |
| yy     | লাইন কপি |
| p      | পেস্ট |
| u      | আনডু |
| /text  | টেক্সট খুঁজুন |

## বাস্তব উদাহরণ

ধরা যাক, আপনি `/etc/nginx/nginx.conf` ফাইল এডিট করতে চান:
```bash
sudo vim /etc/nginx/nginx.conf
```
- i চাপুন, এডিট করুন
- Esc চাপুন
- :wq লিখে সেভ ও কুইট করুন

## VIM কাস্টমাইজেশন

- `.vimrc` ফাইল দিয়ে কাস্টমাইজ
- থিম, প্লাগিন, শর্টকাট যোগ করুন

## DevOps-এ VIM

- কনফিগ ফাইল এডিট
- স্ক্রিপ্ট/লগ/ইনভাইরোনমেন্ট ফাইল
- রিমোট সার্ভারে কাজ

## শুরু করার জন্য

- vim filename.txt দিয়ে ফাইল খুলুন
- i, Esc, :wq, dd, yy, p, u, /text কমান্ড প্র্যাকটিস করুন
- `.vimrc` দিয়ে কাস্টমাইজ করুন
