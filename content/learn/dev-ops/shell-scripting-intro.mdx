শেল স্ক্রিপ্টিং হচ্ছে কম্পিউটারকে নির্দিষ্ট কিছু কাজ স্বয়ংক্রিয়ভাবে করানোর জন্য কমান্ডের একটি ধারাবাহিকতা লিখে রাখা। এটি মূলত লিনাক্স বা ইউনিক্স সিস্টেমে ব্যবহৃত হয়, যেখানে আমরা একাধিক কমান্ড একসাথে লিখে, একটি ফাইলের মাধ্যমে চালাতে পারি।

## শেল স্ক্রিপ্টিং কেন দরকার?

ধরুন, আপনাকে প্রতিদিন ১০টা ফাইল কপি করতে হয়, ৫টা ফোল্ডার ডিলিট করতে হয়, অথবা সার্ভারে লগ চেক করতে হয়। যদি প্রতিদিন একই কাজ বারবার করতে হয়, তাহলে হাতে করে করা সময়সাপেক্ষ এবং ভুলের সম্ভাবনা বেশি। শেল স্ক্রিপ্টিং দিয়ে আপনি এই কাজগুলো একবার লিখে রেখে, বারবার চালাতে পারবেন।

### রিয়েল লাইফ উদাহরণ:

- **ব্যাকআপ স্ক্রিপ্ট:** প্রতিদিন রাত ১২টায় আপনার ডাটাবেজের ব্যাকআপ নিতে হয়। শেল স্ক্রিপ্ট লিখে সেটি অটোমেট করতে পারেন।
- **লগ ক্লিনআপ:** সার্ভারে পুরনো লগ ফাইল ডিলিট করতে হয়। শেল স্ক্রিপ্ট দিয়ে নির্দিষ্ট সময় পরপর অটোমেটিক ডিলিট করা যায়।
- **বাল্ক ফাইল প্রসেসিং:** ১০০০টি ইমেজ ফাইলকে রিসাইজ করতে হবে। শেল স্ক্রিপ্ট দিয়ে একবারেই সব ফাইল প্রসেস করা যায়।

## শেল স্ক্রিপ্টের বেসিক ধারণা

শেল স্ক্রিপ্ট আসলে সাধারণ টেক্সট ফাইল, যেখানে একাধিক কমান্ড একসাথে লেখা থাকে।

```bash
#!/bin/bash
# এই স্ক্রিপ্টটি Hello World প্রিন্ট করবে

echo "Hello World"
```

- `#!/bin/bash` : এটি বলে দেয়, কোন শেল দিয়ে স্ক্রিপ্টটি চালানো হবে।
- `echo "Hello World"` : স্ক্রিপ্টের মধ্যে কমান্ড, যা "Hello World" প্রিন্ট করবে।

## কিভাবে শেল স্ক্রিপ্ট তৈরি ও চালাবেন?

1. **ফাইল তৈরি করুন:**
   ```bash
   nano myscript.sh
   ```
2. **কমান্ড লিখুন:**
   ```bash
   #!/bin/bash
   echo "Backup Started"
   cp /home/user/data.txt /home/user/backup/
   echo "Backup Completed"
   ```
3. **ফাইলটি executable করুন:**
   ```bash
   chmod +x myscript.sh
   ```
4. **স্ক্রিপ্ট চালান:**
   ```bash
   ./myscript.sh
   ```

## শেল স্ক্রিপ্টিং দিয়ে কী কী করা যায়?

- ফাইল/ফোল্ডার কপি, মুভ, ডিলিট
- সার্ভার মনিটরিং
- অটোমেটেড ডিপ্লয়মেন্ট
- ডাটাবেজ ব্যাকআপ
- লগ প্রসেসিং
- ইউজার ম্যানেজমেন্ট

## শেল স্ক্রিপ্টিং শেখার সুবিধা

- **অটোমেশন:** সময় বাঁচে, ভুল কম হয়
- **স্কেলেবিলিটি:** বড় বড় কাজ সহজে করা যায়
- **ডেভঅপস:** সার্ভার, ডিপ্লয়মেন্ট, মনিটরিং—সব জায়গায় দরকার

## শেষ কথা

শেল স্ক্রিপ্টিং ডেভঅপসের অন্যতম গুরুত্বপূর্ণ স্কিল। এটি আপনাকে অটোমেশন, দ্রুত কাজ, এবং সার্ভার ম্যানেজমেন্টে দক্ষ করে তুলবে। পরবর্তী পাঠে শেল স্ক্রিপ্টের কনসেপ্ট ও সিনটেক্স নিয়ে বিস্তারিত আলোচনা করা হবে।
