এই পর্বে আমরা শেল স্ক্রিপ্টিংয়ের আরও অ্যাডভান্সড কনসেপ্ট নিয়ে আলোচনা করব—ফাংশন, এরর হ্যান্ডলিং, আর্গুমেন্ট, এবং স্ক্রিপ্ট ডিবাগিং।

## ফাংশন

শেল স্ক্রিপ্টে ফাংশন ব্যবহার করে কোডকে আরও মডুলার ও পুনঃব্যবহারযোগ্য করা যায়।

```bash
#!/bin/bash
say_hello() {
  echo "Hello, $1!"
}

say_hello "Anis"
say_hello "DevOps"
```

- `say_hello()` : ফাংশন ডিক্লেয়ার
- `$1` : ফাংশনে পাঠানো প্রথম আর্গুমেন্ট

## স্ক্রিপ্টে আর্গুমেন্ট

স্ক্রিপ্টে আর্গুমেন্ট পাঠাতে `$1`, `$2`, ... ব্যবহার হয়।

```bash
#!/bin/bash
echo "First argument: $1"
echo "Second argument: $2"
```

স্ক্রিপ্ট চালানোর সময়:
```bash
./myscript.sh value1 value2
```

## এরর হ্যান্ডলিং

স্ক্রিপ্টে এরর হ্যান্ডলিং করতে `if`, `exit`, এবং `||` ব্যবহার হয়।

```bash
#!/bin/bash
cp /path/to/source.txt /path/to/dest.txt || {
  echo "কপি করতে ব্যর্থ!"; exit 1
}
```

## স্ক্রিপ্ট ডিবাগিং

ডিবাগিংয়ের জন্য স্ক্রিপ্ট চালানোর সময় `-x` ফ্ল্যাগ ব্যবহার করা যায়:

```bash
bash -x myscript.sh
```

এতে প্রতিটি কমান্ড এক্সিকিউশনের সময় স্ক্রিনে দেখাবে।

## রিয়েল লাইফ উদাহরণ

### Example 1: ফাইল এক্সটেনশন চেক করে প্রসেস করা
```bash
#!/bin/bash
process_file() {
  if [[ $1 == *.txt ]]; then
    echo "$1 is a text file."
  else
    echo "$1 is not a text file."
  fi
}

for file in *
do
  process_file "$file"
done
```

### Example 2: ব্যাকআপ স্ক্রিপ্টে এরর হ্যান্ডলিং
```bash
#!/bin/bash
backup_dir="/backup"
mkdir -p $backup_dir || { echo "Backup dir তৈরি হয়নি!"; exit 1; }
cp /home/user/data.txt $backup_dir || { echo "কপি ব্যর্থ!"; exit 1; }
echo "Backup সফল!"
```

## স্ক্রিপ্টে কমান্ড লাইনের আউটপুট ব্যবহার

কোনো কমান্ডের আউটপুট ভ্যারিয়েবলে রাখতে:

```bash
current_date=$(date)
echo "আজকের তারিখ: $current_date"
```

## শেষ কথা

এই পর্বে শিখলেন—ফাংশন, আর্গুমেন্ট, এরর হ্যান্ডলিং, ডিবাগিং, এবং কমান্ড আউটপুট ব্যবহার। এগুলো শেল স্ক্রিপ্টিংকে আরও শক্তিশালী ও প্রফেশনাল করে তোলে।
