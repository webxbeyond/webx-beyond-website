লিনাক্সে সফটওয়্যার ইনস্টল, আপডেট, এবং রিমুভ করার জন্য প্যাকেজ ম্যানেজার ব্যবহার হয়। Windows-এ .exe ফাইল ডাউনলোড করে ইনস্টল করতে হয়, কিন্তু লিনাক্সে এক লাইনের কমান্ডেই সবকিছু করা যায়।

## প্যাকেজ ম্যানেজার কী?

প্যাকেজ ম্যানেজার হলো এমন একটি টুল, যা সফটওয়্যার (প্যাকেজ) খুঁজে, ডাউনলোড, ইনস্টল, আপডেট, এবং রিমুভ করতে সাহায্য করে।

## জনপ্রিয় প্যাকেজ ম্যানেজার

| ডিস্ট্রো   | প্যাকেজ ম্যানেজার | কমান্ড |
|------------|-------------------|--------|
| Ubuntu/Debian | apt, dpkg        | sudo apt install nginx |
| CentOS/RHEL   | yum, dnf         | sudo yum install nginx |
| Arch Linux    | pacman           | sudo pacman -S nginx |

## সফটওয়্যার ইনস্টল, আপডেট, রিমুভ

- **ইনস্টল:**
  - `sudo apt install git`
  - `sudo yum install git`
- **আপডেট:**
  - `sudo apt update && sudo apt upgrade`
  - `sudo yum update`
- **রিমুভ:**
  - `sudo apt remove git`
  - `sudo yum remove git`

## প্যাকেজ খোঁজা

- `apt search docker`
- `yum search docker`

## বাস্তব উদাহরণ

ধরা যাক, আপনাকে Docker ইনস্টল করতে হবে:
```bash
sudo apt update
sudo apt install docker.io
```

## প্যাকেজ ম্যানেজার ফিচার

- **Dependency Management:** প্রয়োজনীয় লাইব্রেরি অটো-ইনস্টল
- **Repositories:** সফটওয়্যার স্টোরেজ
- **Security:** সফটওয়্যার ভেরিফাইড
- **Automation:** স্ক্রিপ্টে ব্যবহার করা যায়

## DevOps-এ প্যাকেজ ম্যানেজার

- সার্ভার/VM-এ টুলস ইনস্টল
- অটোমেশন স্ক্রিপ্টে সফটওয়্যার সেটআপ
- কনফিগারেশন ম্যানেজমেন্ট

## শুরু করার জন্য

- apt, yum, dnf, pacman কমান্ড প্র্যাকটিস করুন
- ইনস্টল, আপডেট, রিমুভ, সার্চ কমান্ড ব্যবহার করুন
- স্ক্রিপ্টে প্যাকেজ ম্যানেজার ব্যবহার করুন
