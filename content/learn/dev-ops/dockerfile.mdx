# Dockerfile: কাস্টম কন্টেইনার ইমেজ তৈরির মূল চাবিকাঠি

Dockerfile হচ্ছে এমন একটি টেক্সট ফাইল, যেখানে আপনি কন্টেইনার ইমেজ তৈরির জন্য প্রয়োজনীয় instruction, dependency, config—all define করতে পারেন। এই টিউটোরিয়ালে Dockerfile-এর কনসেপ্ট, কমান্ড, বাস্তব উদাহরণ, এবং Best Practices নিয়ে বিস্তারিত আলোচনা করব।

---

## Dockerfile কী?

- Dockerfile হচ্ছে ইমেজ তৈরির রেসিপি
- Instruction-based: FROM, RUN, COPY, CMD, EXPOSE, ENV, etc.
- Build command: `docker build -t my-app:1.0 .`

---

## Dockerfile Basic Structure

**Example:**

```Dockerfile
FROM node:18
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
EXPOSE 3000
CMD ["npm", "start"]
```

---

## Common Instructions

| Instruction | Description                       |
|-------------|-----------------------------------|
| FROM        | Base image                        |
| WORKDIR     | Working directory set             |
| COPY        | File/folder copy                  |
| RUN         | Command execute (build/install)   |
| EXPOSE      | Port expose                      |
| ENV         | Environment variable set          |
| CMD         | Default command/container start   |
| ENTRYPOINT  | Main process define               |

---

## Build & Run Workflow

1. Dockerfile লিখুন
2. Build image: `docker build -t my-app:1.0 .`
3. Run container: `docker run -d -p 3000:3000 my-app:1.0`

---

## Multi-stage Build

- Production-ready, minimal image
- Example:
```Dockerfile
FROM node:18 AS build
WORKDIR /app
COPY . .
RUN npm install && npm run build

FROM node:18-alpine
WORKDIR /app
COPY --from=build /app/dist ./dist
CMD ["node", "dist/server.js"]
```

## বাস্তব উদাহরণ

**Case Study:**
একটি SaaS কোম্পানি production-ready Node.js app-এর জন্য multi-stage Dockerfile ব্যবহার করে—build, test, deploy—all automated, minimal, secure image তৈরি হয়।

---

## Best Practices

- Minimal base image ব্যবহার করুন
- Unnecessary file/image layer avoid করুন
- Secrets/config `.env` ফাইলে রাখুন
- Healthcheck configure করুন
- Multi-stage build ব্যবহার করুন
- Caching optimize করুন

## Conclusion

Dockerfile দিয়ে কাস্টম কন্টেইনার ইমেজ build, automation, এবং DevOps workflow সহজ হয়। Practice ও automation script-এ Dockerfile ব্যবহার করুন—আপনার container management next level হবে!
