## পরিচিতি

ইন্টারনেট ব্যবহার করতে গেলে DNS, IP Address, এবং Hosting—এই তিনটি বিষয় জানা খুব জরুরি। এগুলো প্রযুক্তির "মূল রাস্তা"।

### সহজ উদাহরণ
ধরা যাক, আপনি একটি শহরে চিঠি পাঠাতে চান। চিঠি পাঠাতে দরকার ঠিকানা (IP Address), ঠিকানার নাম (DNS), আর চিঠি রাখার জায়গা (Hosting)।

## IP Address কী?

- প্রতিটি ডিভাইসের অনন্য "ডিজিটাল ঠিকানা"
- যেমন: 192.168.1.1, 8.8.8.8
- IP Address ছাড়া ডিভাইস ইন্টারনেটে যুক্ত হতে পারে না
- IPv4 (পুরাতন), IPv6 (নতুন, বেশি ঠিকানা)

## DNS কী?

- DNS (Domain Name System) হলো—নাম থেকে ঠিকানা খুঁজে দেয়
- google.com টাইপ করলে DNS সেটাকে IP Address-এ রূপান্তর করে
- DNS ছাড়া ওয়েবসাইট খুঁজে পাওয়া কঠিন

## Hosting কী?

- Hosting হলো—ওয়েবসাইট, ফাইল, বা অ্যাপ রাখার "ডিজিটাল জায়গা"
- Server-এ ওয়েবসাইট রাখা হয়
- Hosting ছাড়া ওয়েবসাইট অনলাইনে দেখা যাবে না
- Shared, VPS, Cloud—বিভিন্ন ধরনের Hosting

## কেন গুরুত্বপূর্ণ?

- IP Address ছাড়া ডিভাইস "নামহীন"
- DNS ছাড়া ওয়েবসাইট খুঁজে পাওয়া কঠিন
- Hosting ছাড়া ওয়েবসাইট/অ্যাপ "অনুপস্থিত"

## প্র্যাকটিকাল এক্সারসাইজ

১. নিজের ডিভাইসের IP Address খুঁজে বের করুন
২. ব্রাউজারে google.com টাইপ করুন—DNS কীভাবে কাজ করে, চিন্তা করুন
৩. ফ্রি Hosting-এ একটি ফাইল/ওয়েবসাইট আপলোড করুন (যেমন: Netlify, GitHub Pages)
৪. DNS Lookup Tool ব্যবহার করে একটি ওয়েবসাইটের IP Address দেখুন

## উপসংহার

DNS, IP Address, Hosting—এই তিনটি বিষয় প্রযুক্তি ব্যবহারে "ভিত্তি"। এগুলো শেখা মানে ইন্টারনেটের "মূল রাস্তা" বুঝে নেওয়া।
