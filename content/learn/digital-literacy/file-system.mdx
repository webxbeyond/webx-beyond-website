## পরিচিতি

কম্পিউটার বা মোবাইল ব্যবহার করতে গেলে ফাইল, ফোল্ডার, এবং তাদের সঠিক ব্যবস্থাপনা জানা খুব জরুরি। ফাইল সিস্টেম, এক্সটেনশন, এবং অর্গানাইজেশন—এই তিনটি বিষয় প্রযুক্তি ব্যবহারে দক্ষতা বাড়ায়।

### সহজ উদাহরণ
ধরা যাক, আপনার ঘরে অনেক বই আছে। যদি বইগুলো এলোমেলোভাবে ছড়িয়ে থাকে, তাহলে প্রয়োজনের সময় খুঁজে পাওয়া কঠিন। কিন্তু যদি বইগুলো আলাদা আলাদা শেলফে, বিষয়ভিত্তিক সাজানো থাকে, তাহলে খুঁজে পাওয়া সহজ। ফাইল অর্গানাইজেশনও ঠিক তেমন।

## ফাইল সিস্টেম কী?

- ফাইল সিস্টেম হলো—কম্পিউটারে ফাইল সংরক্ষণ, খোঁজা, এবং ব্যবস্থাপনার নিয়ম
- Windows-এ NTFS, macOS-এ APFS, Linux-এ EXT4
- ফাইল, ফোল্ডার, ডিরেক্টরি—সব ফাইল সিস্টেমের অংশ

## ফাইল এক্সটেনশন কী?

- ফাইলের শেষে থাকা .docx, .jpg, .pdf, .mp3 ইত্যাদি
- এক্সটেনশন দেখে ফাইলের ধরন বোঝা যায়
- .txt = টেক্সট, .jpg = ছবি, .mp3 = অডিও
- সঠিক সফটওয়্যার ছাড়া ফাইল খুলবে না

## ফাইল অর্গানাইজেশন

- ফোল্ডার তৈরি করে বিষয়ভিত্তিক ফাইল সাজানো
- নামকরণে তারিখ, বিষয়, বা প্রজেক্ট যুক্ত করা
- অপ্রয়োজনীয় ফাইল ডিলিট করা
- ব্যাকআপ রাখা
- Cloud Storage (Google Drive, Dropbox) ব্যবহার

## কেন গুরুত্বপূর্ণ?

- প্রয়োজনের সময় ফাইল দ্রুত খুঁজে পাওয়া যায়
- ডেটা নিরাপদ থাকে
- কাজের গতি বাড়ে
- ভুল বা ডুপ্লিকেট ফাইল কমে

## প্র্যাকটিকাল এক্সারসাইজ

১. নিজের কম্পিউটারে নতুন ফোল্ডার তৈরি করুন—বিষয়ভিত্তিক ফাইল সাজান।
২. ফাইলের এক্সটেনশন দেখে ধরন চিনুন।
৩. অপ্রয়োজনীয় ফাইল ডিলিট করুন এবং ব্যাকআপ নিন।
৪. Cloud Storage-এ একটি ফাইল আপলোড করুন।

## উপসংহার

ফাইল সিস্টেম, এক্সটেনশন, এবং অর্গানাইজেশন—এই তিনটি বিষয় শেখা মানে প্রযুক্তি ব্যবহারে "স্মার্ট" ও "দক্ষ" হওয়া।
