## সিড থেকে কী জেনারেশন

ক্রিপ্টো ওয়ালেটের নিরাপত্তা ও ব্যবহারযোগ্যতার অন্যতম গুরুত্বপূর্ণ স্তম্ভ হচ্ছে "সিড ফ্রেইজ"। এই সিড ফ্রেইজের মাধ্যমে কীভাবে আপনার ওয়ালেটের সকল অ্যাক্সেস ও চাবি তৈরি হয়, তার পেছনের প্রযুক্তি ও স্ট্যান্ডার্ড হচ্ছে BIP39। চলুন, সহজ ভাষায় BIP39 কী, কেন গুরুত্বপূর্ণ, এবং কীভাবে এটি কাজ করে—তা জানি।

## BIP39 কী?
BIP39 (Bitcoin Improvement Proposal 39) হচ্ছে একটি স্ট্যান্ডার্ড, যা ক্রিপ্টো ওয়ালেটের জন্য Mnemonic Phrase (সিড ফ্রেইজ) তৈরির নিয়ম নির্ধারণ করে। Mnemonic Phrase মানে—একটি নির্দিষ্ট সংখ্যক সাধারণ শব্দ (সাধারণত ১২, ১৮, বা ২৪টি)—যা আপনার ওয়ালেটের সকল চাবি (key) পুনরুদ্ধার করতে পারে।

### উদাহরণ:
ধরা যাক, আপনার সিড ফ্রেইজ:
```
cat mango river moon tiger apple cloud drum fish lamp tree sun
```
এই ১২টি শব্দই আপনার ওয়ালেটের "master key"।

## কেন Mnemonic Phrase দরকার?
- **সহজে মনে রাখা যায়:** সংখ্যা বা এলোমেলো অক্ষরের চেয়ে শব্দ-based phrase সহজে মনে রাখা যায়।
- **ব্যাকআপ ও রিকভারি:** ওয়ালেট হারিয়ে গেলে বা নতুন ডিভাইসে সেটআপ করতে চাইলে এই phrase দিয়েই সব restore করা যায়।
- **একাধিক অ্যাকাউন্ট:** একই সিড ফ্রেইজ থেকে একাধিক অ্যাকাউন্ট/অ্যাড্রেস তৈরি করা যায় (HD Wallets)।


## BIP39: পূর্ণাঙ্গ জেনারেশন প্রসেস
BIP39 Mnemonic Phrase তৈরির পুরো অ্যালগরিদম:


### ১. Entropy (র‍্যান্ডম ডেটা) তৈরি
- সাধারণত 128, 160, 192, 224, বা 256 বিটের র‍্যান্ডম সংখ্যা তৈরি হয়।

#### কিভাবে Entropy তৈরি হয়?
- Entropy মানে হচ্ছে একদম এলোমেলো (random) বিট/সংখ্যা, যা কম্পিউটার বা ডিভাইসের "random number generator" থেকে আসে।
- আধুনিক ওয়ালেট/লাইব্রেরি সাধারণত OS-level cryptographic random generator ব্যবহার করে (যেমন: Python-এ `os.urandom`, JavaScript-এ `crypto.getRandomValues`)।
- এই এলোমেলো বিটই Mnemonic phrase-এর ভিত্তি।

#### উদাহরণ (Python):
```python
import os
# 128-bit entropy
entropy = os.urandom(16) # 16 bytes = 128 bits
print(entropy.hex())
```

#### উদাহরণ (JavaScript):
```js
const entropy = window.crypto.getRandomValues(new Uint8Array(16)); // 128 bits
console.log(Array.from(entropy).map(b=>b.toString(16).padStart(2,'0')).join(''));
```

> **নোট:** বেশি entropy মানে বেশি নিরাপত্তা।


### ২. Checksum যোগ
- SHA256 দিয়ে entropy-এর checksum বের করা হয়।
- checksum-এর প্রথম N বিট (N = entropy length / 32) entropy-এর শেষে যোগ করা হয়।

### ৩. Bit Grouping ও Word Mapping
- পুরো বিটস্ট্রিংকে 11-বিট করে ভাগ করা হয়।
- প্রতিটি 11-বিট একটি শব্দের index (BIP39 wordlist-এ) নির্দেশ করে।
- যেমন, 128 বিট entropy + 4 বিট checksum = 132 বিট → 12টি শব্দ (12 x 11 = 132)

### ৪. Mnemonic Phrase
- এই index-গুলো দিয়ে BIP39 wordlist থেকে Mnemonic Phrase তৈরি হয়।

### ৫. Seed Derivation
- Mnemonic phrase + (ঐচ্ছিক) passphrase নিয়ে PBKDF2 (HMAC-SHA512) অ্যালগরিদমে 2048 রাউন্ড চালিয়ে 512-বিট seed তৈরি হয়।

### উদাহরণ: Seed Phrase ক্যালকুলেশন

ধরা যাক, 128-বিট entropy:  
`00000010 11010101 10101010 11110000 10101010 11001100 11110000 10101010 11001100 11110000 10101010 11001100 11110000 10101010 11001100 11110000`

১. SHA256 দিয়ে checksum বের করুন (প্রথম 4 বিট):  
`1010`

২. Entropy + checksum:  
`...11110000 1010`

৩. 11-বিট করে ভাগ করুন:  
`00000010110`, `10101101010`, ... (মোট 12টি গ্রুপ)

৪. প্রতিটি গ্রুপের index অনুযায়ী wordlist থেকে শব্দ নিন:  
`abandon`, `ability`, ... (মোট 12টি শব্দ)

এভাবেই Mnemonic Phrase তৈরি হয়।

### Python Demo: পূর্ণ অ্যালগরিদম

```python
import os, hashlib
from mnemonic import Mnemonic

# 128-bit entropy
entropy = os.urandom(16)
checksum = hashlib.sha256(entropy).hexdigest()
# checksum bits
checksum_bits = bin(int(checksum, 16))[2:].zfill(256)[:4]
entropy_bits = bin(int.from_bytes(entropy, 'big'))[2:].zfill(128)
full_bits = entropy_bits + checksum_bits

# 11-bit group
words_idx = [int(full_bits[i:i+11], 2) for i in range(0, len(full_bits), 11)]
mnemo = Mnemonic("english")
mnemonic = ' '.join([mnemo.wordlist[idx] for idx in words_idx])
seed = mnemo.to_seed(mnemonic, passphrase="")
print("Mnemonic:", mnemonic)
print("Seed:", seed.hex())
```

### অ্যানালজি
- যেমন লকারের মাস্টার চাবি তৈরি হয় নির্দিষ্ট নিয়মে, Mnemonic Phrase-ও নির্দিষ্ট অ্যালগরিদমে তৈরি হয়—এতে নিরাপত্তা ও পুনরুদ্ধার সহজ হয়।

### Python Demo: Mnemonic Phrase থেকে Seed
```python
from mnemonic import Mnemonic

mnemo = Mnemonic("english")
words = "cat mango river moon tiger apple cloud drum fish lamp tree sun"
seed = mnemo.to_seed(words, passphrase="")
print(seed.hex())
```

## বাস্তব উদাহরণ ও অ্যানালজি
- **লকারের মাস্টার চাবি:** যেমন একটি মাস্টার চাবি দিয়ে সব লকার খোলা যায়, তেমনি সিড ফ্রেইজ দিয়ে আপনার সব অ্যাকাউন্ট/অ্যাড্রেস restore করা যায়।
- **পাসপোর্টের মতো:** পাসপোর্ট হারালে নতুন পাসপোর্ট করতে হয়, কিন্তু সিড ফ্রেইজ থাকলে যেকোনো ওয়ালেট restore করা যায়।

## নিরাপত্তা টিপস
- **সিড ফ্রেইজ কখনো ডিজিটালভাবে সংরক্ষণ করবেন না (স্ক্রিনশট, ক্লাউড, ইমেইল)।**
- **কাগজে লিখে নিরাপদ স্থানে রাখুন।**
- **কাউকে শেয়ার করবেন না।**
- **ফিশিং/স্ক্যাম থেকে সতর্ক থাকুন—কেউ যদি সিড ফ্রেইজ চায়, সে আপনার ফান্ড চুরি করতে পারে।**
