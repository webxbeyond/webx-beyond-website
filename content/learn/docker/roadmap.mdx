### ২. Recent & Upcoming Features

- **BuildKit:** Faster, secure, efficient image build—default enforced।
- **Docker Compose v2:** Improved CLI, YAML, multi-platform support।
- **Containerd integration:** Lightweight runtime—Kubernetes compatibility।
- **Security enhancement:** Image signing, SBOM, vulnerability scan—default enforced।
- **Desktop improvements:** Resource monitoring, troubleshooting, GUI enhancement।
- **Multi-arch build:** ARM/x86—single workflow—cross-platform deploy।
