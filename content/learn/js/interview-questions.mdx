জাভাস্ক্রিপ্ট ইন্টারভিউ—বেসিক থেকে অ্যাডভান্সড প্রশ্ন, বাস্তব উদাহরণ, ও ব্যাখ্যা নিয়ে প্রস্তুতি নেয়া যায়। এখানে কিছু গুরুত্বপূর্ণ প্রশ্ন, উত্তর, ও টিপস দেয়া হয়েছে।

---

## ১. বেসিক প্রশ্ন

### প্রশ্ন: জাভাস্ক্রিপ্টে `var`, `let`, `const` এর পার্থক্য কী?
**উত্তর:**
- `var` function scoped, redeclare/overwrite করা যায়
- `let` block scoped, redeclare করা যায় না
- `const` block scoped, value change করা যায় না

---

### প্রশ্ন: ক্লোজার কী?
**উত্তর:**
- Inner function outer scope-এর variable access করতে পারে—এটাই closure

---

### প্রশ্ন: `this` কীওয়ার্ড কী?
**উত্তর:**
- Function call context—object, global, arrow function-এ ভিন্ন

---

## ২. অ্যাডভান্সড প্রশ্ন

### প্রশ্ন: ইভেন্ট লুপ কীভাবে কাজ করে?
**উত্তর:**
- Call stack, task queue, microtask queue—async code handle করে

---

### প্রশ্ন: প্রমিজ ও async/await-এর পার্থক্য?
**উত্তর:**
- Promise chaining, async/await—cleaner syntax, error handling সহজ

---

### প্রশ্ন: প্রোটোটাইপ ইনহেরিটেন্স কী?
**উত্তর:**
- Object-এর prototype chain—property/method inherit
