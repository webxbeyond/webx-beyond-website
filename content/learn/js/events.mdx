ওয়েব ডেভেলপমেন্টে ইভেন্ট হ্যান্ডলিং মানে—ইউজার কীভাবে পেজের সাথে "কথা বলে" তা নিয়ন্ত্রণ করা। এই পাঠে আমরা শিখব, কীভাবে ইনলাইন ইভেন্ট, আধুনিক `addEventListener`, এবং ইভেন্টের propagation (বুবলিং, ক্যাপচারিং) কাজ করে।

---

## ১. ইভেন্ট কী?

ইভেন্ট হলো—ইউজার বা ব্রাউজার থেকে আসা "ঘটনা"। যেমন: ক্লিক, টাইপ, স্ক্রল, লোড ইত্যাদি।

### উদাহরণ:
- বাটন ক্লিক
- ইনপুটে টাইপ
- পেজ লোড

---

## ২. ইনলাইন ইভেন্ট হ্যান্ডলিং

HTML-এ সরাসরি ইভেন্ট অ্যাট্রিবিউট দিয়ে হ্যান্ডল করা যায়।

### উদাহরণ:
```html
<button onclick="alert('ক্লিক হয়েছে!')">Click Me</button>
```

#### অ্যানালজি:
এটা অনেকটা দরজায় বেল বাজালে সরাসরি ঘন্টা বাজে—কোনো মধ্যস্থতাকারী নেই।

#### সমস্যা:
- কোড ও লজিক একসাথে থাকায় মেইনটেইন করা কঠিন
- একাধিক ইভেন্ট অ্যাসাইন করা যায় না

---

## ৩. আধুনিক addEventListener

জাভাস্ক্রিপ্ট দিয়ে DOM এলিমেন্টে ইভেন্ট অ্যাসাইন করা যায়।

### উদাহরণ:
```js
const btn = document.querySelector('button');
btn.addEventListener('click', function() {
  alert('বাটন ক্লিক হয়েছে!');
});
```

#### অ্যানালজি:
এটা অনেকটা দরজায় বেল বাজালে আগে একজন রিসেপশনিস্ট দেখে, তারপর ঘন্টা বাজে—মধ্যস্থতাকারী আছে, তাই আরও ফিচার যোগ করা যায়।

#### সুবিধা:
- একাধিক ইভেন্ট অ্যাসাইন করা যায়
- কোড আলাদা রাখা যায়
- ইভেন্ট রিমুভ করা যায়: `removeEventListener`

## ৪. ইভেন্ট বুবলিং ও ক্যাপচারিং

### ইভেন্ট বুবলিং:
ইভেন্ট প্রথমে "সবচেয়ে ভেতরের" এলিমেন্টে ট্রিগার হয়, তারপর বাইরের দিকে যায়।

### উদাহরণ:
```html
<div id="parent">
  <button id="child">Click</button>
</div>
```
```js
document.getElementById('parent').addEventListener('click', () => {
  alert('Parent Clicked');
});
document.getElementById('child').addEventListener('click', () => {
  alert('Child Clicked');
});
```
- বাটন ক্লিক করলে আগে Child, পরে Parent ইভেন্ট চলে।

#### অ্যানালজি:
এটা অনেকটা ছোট বাক্সের ভেতর বড় বাক্স—ছোটটা খুললে বড়টাও খুলে যায়।

### ইভেন্ট ক্যাপচারিং:
ইভেন্ট প্রথমে "সবচেয়ে বাইরের" এলিমেন্টে ট্রিগার হয়, তারপর ভেতরের দিকে যায়।

### উদাহরণ:
```js
document.getElementById('parent').addEventListener('click', () => {
  alert('Parent Captured');
}, true); // ক্যাপচার মোড
```

## ৫. ইভেন্ট স্টপ/প্রিভেন্ট

### ইভেন্ট স্টপ:
```js
document.getElementById('child').addEventListener('click', (e) => {
  e.stopPropagation(); // বুবলিং বন্ধ
});
```

### ডিফল্ট অ্যাকশন বন্ধ:
```js
document.querySelector('form').addEventListener('submit', (e) => {
  e.preventDefault(); // ফর্ম সাবমিট বন্ধ
});
```

## ৬. ব্যবহারিক উদাহরণ

### ১. মেনু ড্রপডাউন:

- বাইরের এলিমেন্টে ক্লিক করলে মেনু বন্ধ

### ২. ফর্ম ভ্যালিডেশন:
- সাবমিট ইভেন্টে ডাটা চেক

## ৭. চ্যালেঞ্জ

- একটি বাটন ও ডিভ তৈরি করুন—বাটন ক্লিক করলে ডিভের কালার পরিবর্তন করুন (addEventListener)।
- বুবলিং ও ক্যাপচারিং ডেমো দেখান—দুইটি এলিমেন্টে ইভেন্ট অ্যাসাইন করুন।
