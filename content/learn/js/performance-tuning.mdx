## ১. Performance Tuning

### কী?
- কোড, ডাটা, ও UI দ্রুত কাজ করানো

### টিপস:
- Unused code/remove করুন
- Loop optimization (for vs forEach)
- Heavy computation web worker-এ পাঠান
- Memory leak avoid করুন

### উদাহরণ:
```js
// Loop optimization
for(let i=0; i<arr.length; i++) {
  // ...
}
```

---

## ২. Bundle Optimization

### কী?
- Final JS/CSS/asset bundle ছোট ও efficient করা

### টিপস:
- Tree shaking: unused export/remove
- Minification: code ছোট করা
- Code splitting: route/component অনুযায়ী bundle
- Compression: gzip/brotli

### উদাহরণ:
```js
// Dynamic import
const Chart = await import('./Chart');
```
