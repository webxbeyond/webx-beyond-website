জাভাস্ক্রিপ্ট (JavaScript) হলো আধুনিক ওয়েব ডেভেলপমেন্টের প্রাণ। এটি এমন একটি ভাষা, যা ওয়েব পেজকে শুধু স্ট্যাটিক নয়, ইন্টারঅ্যাকটিভ ও ডাইনামিক করে তোলে। কিন্তু এই ভাষার জন্ম, বিবর্তন, এবং স্ট্যান্ডার্ডাইজেশন নিয়ে অনেকেরই স্পষ্ট ধারণা নেই। চলুন, ইতিহাসের পাতায় চোখ রাখি এবং ECMAScript স্ট্যান্ডার্ডের গুরুত্ব বুঝি।

## ইতিহাসের শুরু: নেটস্কেপ ও ব্রেনডন আইচ

১৯৯৫ সালে, নেটস্কেপ (Netscape) ব্রাউজারের জন্য ব্রেনডন আইচ মাত্র ১০ দিনে একটি স্ক্রিপ্টিং ভাষা তৈরি করেন। প্রথমে এর নাম ছিল Mocha, পরে LiveScript, এবং অবশেষে JavaScript নামে পরিচিত হয়।

### কেন "Java"-এর সাথে মিল?

তখন Java ভাষা খুব জনপ্রিয় ছিল। নেটস্কেপ চেয়েছিল তাদের নতুন স্ক্রিপ্টিং ভাষার জনপ্রিয়তা বাড়াতে, তাই "Java" নামটি ব্যবহার করে। কিন্তু JavaScript এবং Java একেবারেই আলাদা ভাষা—শুধু নামের মিল!

## জাভাস্ক্রিপ্টের বিবর্তন

শুরুতে জাভাস্ক্রিপ্ট ছিল খুবই সীমিত। শুধু ব্রাউজারে ছোটখাটো ইন্টারঅ্যাকশন, ফর্ম ভ্যালিডেশন, এবং অ্যানিমেশনেই ব্যবহৃত হতো। কিন্তু সময়ের সাথে সাথে:

- **AJAX**: ওয়েব পেজ রিলোড না করেই ডেটা লোড করার সুযোগ এনে দেয়।
- **Node.js**: সার্ভার-সাইডে জাভাস্ক্রিপ্ট চালানোর পথ খুলে দেয়।
- **ফ্রেমওয়ার্ক ও লাইব্রেরি**: React, Angular, Vue ইত্যাদি জাভাস্ক্রিপ্টকে অ্যাপ ডেভেলপমেন্টের মূলধারা বানিয়ে দেয়।

## ECMAScript: স্ট্যান্ডার্ডাইজেশনের গল্প

জাভাস্ক্রিপ্টের জনপ্রিয়তা বাড়ার সাথে সাথে, বিভিন্ন ব্রাউজারে ভাষার ইমপ্লিমেন্টেশন ভিন্ন হতে শুরু করে। এই সমস্যা সমাধানে, ১৯৯৭ সালে ECMA International "ECMAScript" নামে একটি স্ট্যান্ডার্ড তৈরি করে।

- **ECMAScript**: এটি জাভাস্ক্রিপ্টের অফিসিয়াল স্পেসিফিকেশন।
- **JavaScript**: ECMAScript-এর ইমপ্লিমেন্টেশন (ব্রাউজার, Node.js ইত্যাদি)।

### ECMAScript সংস্করণ

- **ES3 (1999)**: প্রথম বড় আপডেট।
- **ES5 (2009)**: নতুন ফিচার—`strict mode`, `JSON`, `Array methods`।
- **ES6/ES2015**: আধুনিক জাভাস্ক্রিপ্ট—`let`, `const`, `arrow function`, `class`, `Promise`, `modules`।
- **ESNext**: প্রতি বছর নতুন ফিচার যোগ হয়।

## বাস্তব উদাহরণ: ECMAScript স্ট্যান্ডার্ডের গুরুত্ব

ধরা যাক, আপনি একটি কোড লিখলেন:

```js
let name = "Anis";
console.log(name);
```

ECMAScript স্ট্যান্ডার্ডের কারণে, এই কোড Chrome, Firefox, Edge, Safari—সব ব্রাউজারে একইভাবে কাজ করবে।

## জাভাস্ক্রিপ্টের ভবিষ্যৎ

প্রতি বছর TC39 কমিটি নতুন ফিচার প্রস্তাব করে, পরীক্ষা করে, এবং স্ট্যান্ডার্ডে যুক্ত করে। ফলে, জাভাস্ক্রিপ্ট ক্রমাগত আধুনিক হচ্ছে।

## উপসংহার

জাভাস্ক্রিপ্ট শুধু একটি ভাষা নয়—এটি ওয়েবের প্রাণ। এর ইতিহাস, বিবর্তন, এবং ECMAScript স্ট্যান্ডার্ডের গল্প জানা মানে, আপনি ওয়েব ডেভেলপমেন্টের গভীরে প্রবেশ করলেন।

### Analogies & Real-life Example

- **ECMAScript = Recipe Book**: যেমন একটি রেসিপি বই দেখে সবাই একইভাবে রান্না করতে পারে, ECMAScript স্ট্যান্ডার্ড দেখে সব ব্রাউজার জাভাস্ক্রিপ্ট "রান" করে।
- **JavaScript = Actual Dish**: ব্রাউজার বা Node.js সেই রেসিপি অনুসারে "ডিশ" তৈরি করে।

### প্রশ্নোত্তর

- **JavaScript ও ECMAScript কি এক?**
  - ECMAScript হলো স্ট্যান্ডার্ড, JavaScript হলো তার ইমপ্লিমেন্টেশন।
- **JavaScript ও Java কি এক?**
  - না, শুধু নামের মিল।
