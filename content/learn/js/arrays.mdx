জাভাস্ক্রিপ্টে অ্যারে ডাটা প্রসেসিং, ট্রান্সফর্ম, ও ফিল্টার করার জন্য কিছু শক্তিশালী মেথড আছে—map, filter, reduce, স্প্রেড/ডেস্ট্রাকচারিং। এই পাঠে আমরা এগুলোর ব্যবহার, বাস্তব উদাহরণ, এবং পারফরম্যান্স টিপস শিখব।

---

## ১. map: ডাটা ট্রান্সফর্ম

map মেথড দিয়ে অ্যারের প্রতিটি আইটেমকে নতুনভাবে প্রসেস করে নতুন অ্যারে তৈরি করা যায়।

### উদাহরণ:
```js
const numbers = [1, 2, 3, 4];
const doubled = numbers.map(n => n * 2);
console.log(doubled); // [2, 4, 6, 8]
```

#### অ্যানালজি:
এটা অনেকটা—একটি তালিকা থেকে প্রতিটি আইটেমকে নতুন রূপে সাজানো।

---

## ২. filter: ডাটা ফিল্টার

filter দিয়ে নির্দিষ্ট শর্তে অ্যারের আইটেম বাছাই করা যায়।

### উদাহরণ:
```js
const ages = [12, 18, 25, 30];
const adults = ages.filter(age => age >= 18);
console.log(adults); // [18, 25, 30]
```

#### অ্যানালজি:
এটা অনেকটা—একটি ঝাঁঝরি দিয়ে শুধু বড় দানা রেখে ছোটগুলো বাদ দেওয়া।

---

## ৩. reduce: ডাটা কম্পোজ/সমাহার

reduce দিয়ে অ্যারের সব আইটেমকে একত্রিত করে একটি ভ্যালু তৈরি করা যায়—sum, average, object, ইত্যাদি।

### উদাহরণ:
```js
const nums = [1, 2, 3, 4];
const sum = nums.reduce((acc, curr) => acc + curr, 0);
console.log(sum); // 10
```

#### অ্যানালজি:
এটা অনেকটা—সব উপাদান একসাথে মিশিয়ে একটি নতুন জিনিস তৈরি করা।

---

## ৪. স্প্রেড ও ডেস্ট্রাকচারিং

### স্প্রেড:
```js
const arr1 = [1, 2];
const arr2 = [3, 4];
const combined = [...arr1, ...arr2];
console.log(combined); // [1, 2, 3, 4]
```

### ডেস্ট্রাকচারিং:
```js
const [first, second] = [10, 20, 30];
console.log(first); // 10
console.log(second); // 20
```

#### অ্যানালজি:
স্প্রেড—দুইটি বাক্সের জিনিস একসাথে ছড়িয়ে দেওয়া।
ডেস্ট্রাকচারিং—বড় বাক্স থেকে নির্দিষ্ট জিনিস বের করে নেওয়া।

---

## ৫. Practical Tips
- map/filter/reduce চেইন করে complex ডাটা প্রসেস করা যায়
- স্প্রেড দিয়ে অ্যারে কপি/মার্জ সহজ
- ডেস্ট্রাকচারিং দিয়ে কোড সংক্ষিপ্ত ও readable

---

## ৬. Security & Performance
- Sensitive ডাটা প্রসেস করার সময় filter/map-এ validation করুন
- reduce-এ initial value ভুল হলে bug হতে পারে
- বড় অ্যারে প্রসেসে পারফরম্যান্স impact হতে পারে—chunk/limit ব্যবহার করুন

---

## ৭. চ্যালেঞ্জ
- একটি অ্যারে থেকে শুধু even সংখ্যা বের করুন (filter)
- map দিয়ে প্রতিটি নামকে uppercase করুন
- reduce দিয়ে অ্যারের sum বের করুন
- স্প্রেড দিয়ে দুইটি অ্যারে মার্জ করুন
