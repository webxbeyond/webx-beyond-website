জাভাস্ক্রিপ্ট শেখার মূল ভিত্তি হলো এর সিনট্যাক্স ও ডাটা টাইপস। এই পাঠে আমরা দেখব কীভাবে ভ্যারিয়েবল ডিক্লেয়ার করা হয়, কোন টাইপের ডাটা ব্যবহার হয়, এবং কোন সিনট্যাক্সে কোড লেখা হয়।

## ১. ভ্যারিয়েবল ডিক্লেয়ারেশন: `var`, `let`, `const`

### `var`
- পুরনো পদ্ধতি, ES5 পর্যন্ত ব্যবহৃত।
- ফাংশন স্কোপড, ব্লক স্কোপড নয়।
- রিডিক্লেয়ার ও রিএসাইন করা যায়।

```js
var name = "Anis";
name = "Rahim";
var name = "Karim";
```

### `let`
- ES6 থেকে এসেছে।
- ব্লক স্কোপড ({} এর মধ্যে সীমাবদ্ধ)।
- রিএসাইন করা যায়, কিন্তু রিডিক্লেয়ার করা যায় না।

```js
let age = 25;
age = 30;
// let age = 40; // Error
```

### `const`
- ES6 থেকে এসেছে।
- ব্লক স্কোপড।
- রিডিক্লেয়ার বা রিএসাইন করা যায় না।
- শুধু একবার ভ্যালু সেট করা যায়।

```js
const country = "Bangladesh";
// country = "India"; // Error
// const country = "Nepal"; // Error
```

#### Analogies:
- `var` = পুরনো চাবি, যেটা সব দরজা খুলে (scope leak)।
- `let` = আধুনিক চাবি, শুধু নির্দিষ্ট দরজা খুলে (block scope)।
- `const` = একবারই লক করা যায়, পরে আর খোলা যায় না (immutable)।

## ২. ডাটা টাইপস: প্রিমিটিভ ও রেফারেন্স

### প্রিমিটিভ টাইপস
- **String**: "Hello"
- **Number**: 42
- **Boolean**: true/false
- **Null**: কোনো ভ্যালু নেই
- **Undefined**: ডিক্লেয়ার হয়েছে, ভ্যালু নেই
- **Symbol**: ইউনিক আইডেন্টিফায়ার
- **BigInt**: বড় সংখ্যা

```js
let str = "Bangla";
let num = 2025;
let isActive = true;
let nothing = null;
let notSet;
let sym = Symbol("id");
let big = 12345678901234567890n;
```

### রেফারেন্স টাইপস
- ‍`**Object**: { name: "Anis" }‍‍‍`
- `**Array**: [1, 2, 3]`
- `**Function**: function() {}`

## ৩. টাইপ চেকিং

```js
typeof "Bangla"; // "string"
typeof 42; // "number"
typeof true; // "boolean"
typeof null; // "object" (জাভাস্ক্রিপ্টের বিখ্যাত bug)
typeof undefined; // "undefined"
typeof Symbol(); // "symbol"
typeof 123n; // "bigint"
typeof [1,2,3]; // "object"
typeof function(){}; // "function"
```

## ৪. কোড লেখার বেসিক সিনট্যাক্স

- **স্টেটমেন্ট**: এক লাইনে একটি কাজ
- **সেমিকোলন**: শেষের চিহ্ন (অনেক সময় অটো)
- **কোমেন্ট**: কোডে নোট

```js
// এটা এক লাইনের কমেন্ট
/* এটা
   মাল্টিলাইন কমেন্ট */
let x = 5; // স্টেটমেন্ট
```

## ৫. বাস্তবিক উদাহরণ

```js
let student = {
  name: "Anis",
  age: 22,
  isActive: true
};

const PI = 3.1416;
let scores = [90, 85, 88];
```

## ৬. সাধারণ ভুল ও সমাধান

- **`var` দিয়ে ব্লক স্কোপ আশা করা:**
  - `var` ব্লক স্কোপড নয়, ফাংশন স্কোপড।
- **`const` দিয়ে রিএসাইন:**
  - `const` একবারই ভ্যালু সেট করা যায়।
- **`typeof null` "object" দেখায়:**
  - এটি জাভাস্ক্রিপ্টের পুরনো bug।

## উপসংহার

ভ্যারিয়েবল ডিক্লেয়ারেশন ও ডাটা টাইপস ভালোভাবে আয়ত্ত করলে, জাভাস্ক্রিপ্ট শেখার ভিত্তি মজবুত হয়।
