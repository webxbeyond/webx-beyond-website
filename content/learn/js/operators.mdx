জাভাস্ক্রিপ্টে অপারেটর ও এক্সপ্রেশন হলো কোডের প্রাণ। অপারেটর দিয়ে ডাটা প্রসেস, তুলনা, সিদ্ধান্ত, এবং এক্সপ্রেশন দিয়ে ফলাফল তৈরি হয়। এই পাঠে আমরা দেখব প্রধান অপারেটর, তাদের ব্যবহার, এবং বাস্তব উদাহরণ।

## ১. তুলনা অপারেটর (Comparison Operators)

- **`==`**: ভ্যালু সমান কিনা (টাইপ কনভার্সন করে)
- **`===`**: ভ্যালু ও টাইপ দুটোই সমান কিনা
- **`!=`**: সমান নয়
- **`!==`**: ভ্যালু বা টাইপ সমান নয়
- **`>`**: বড়
- **`<`**: ছোট
- **`>=`**: বড় বা সমান
- **`<=`**: ছোট বা সমান

```js
5 == "5"; // true
5 === "5"; // false
10 > 5; // true
3 <= 2; // false
```

#### Analogies:
- `==` = "নাম এক, পোশাক ভিন্ন হলেও চলবে"
- `===` = "নাম ও পোশাক দুটোই এক হতে হবে"

## ২. লজিকাল অপারেটর (Logical Operators)

- **`&&`**: AND (দুটো শর্তই সত্য হলে)
- **`||`**: OR (যেকোনো একটি শর্ত সত্য হলে)
- **`!`**: NOT (বিপরীত)

```js
true && false; // false
true || false; // true
!true; // false
```

### বাস্তব উদাহরণ:

```js
let age = 20;
let hasID = true;
if (age >= 18 && hasID) {
  console.log("Entry allowed");
}
```

## ৩. টারনারি অপারেটর (Ternary Operator)

- সংক্ষিপ্ত if-else
- সিনট্যাক্স: `condition ? valueIfTrue : valueIfFalse`

```js
let isMember = true;
let price = isMember ? 100 : 150;
console.log(price); // 100
```

#### Analogies:
- "যদি বৃষ্টি হয়? ছাতা নাও : না হলে ছাতা লাগবে না"

## ৪. এক্সপ্রেশন (Expressions)

- এক্সপ্রেশন মানে এমন কোড, যা কোনো ভ্যালু রিটার্ন করে
- উদাহরণ: `2 + 3`, `name === "Anis"`, `isActive && hasID`

## ৫. অন্যান্য অপারেটর

- **Arithmetic Operators**: `+`, `-`, `*`, `/`, `%`, `**`
- **Assignment Operators**: `=`, `+=`, `-=`, `*=`, `/=`, `%=`
- **String Concatenation**: `+`

```js
let x = 5;
x += 2; // x = x + 2 => 7
let greeting = "Hello" + " " + "World"; // "Hello World"
```

## ৬. সাধারণ ভুল ও সমাধান

- **`==` vs `===`**: টাইপ কনভার্সন নিয়ে বিভ্রান্তি
- **Multiple conditions:**
  - `if (a && b || c)` — অপারেটর প্রাধান্য বুঝতে হবে
- **Falsy values:**
  - `0`, `""`, `null`, `undefined`, `false` — সব Falsy

## ৭. বাস্তবিক চ্যালেঞ্জ

```js
let score = 85;
let grade = score >= 80 ? "A" : score >= 70 ? "B" : "C";
console.log(grade); // "A"
```

## উপসংহার

অপারেটর ও এক্সপ্রেশন ভালোভাবে আয়ত্ত করলে, জাভাস্ক্রিপ্টে সিদ্ধান্ত, ডাটা প্রসেসিং, এবং কোডিং সহজ হয়।
