জাভাস্ক্রিপ্টে ড্র্যাগ-ড্রপ UI—ইউজার ইন্টারঅ্যাকশন, ফাইল আপলোড, এলিমেন্ট অর্ডার চেঞ্জ, ও ভিজ্যুয়াল ফিডব্যাকের জন্য জনপ্রিয়। এখানে একটি বাস্তব উদাহরণ, অ্যানালজি, ও সিকিউরিটি টিপস দেয়া হয়েছে।

---

## ১. ড্র্যাগ-ড্রপ কী?

ইউজার মাউস দিয়ে একটি এলিমেন্ট ধরে অন্য জায়গায় নিয়ে যেতে পারে—drag & drop।

#### অ্যানালজি:
এটা অনেকটা—একটি ডেস্কে বই সরানো, যেখানে বই এক জায়গা থেকে অন্য জায়গায় রাখা যায়।

---

## ২. Basic ড্র্যাগ-ড্রপ উদাহরণ

```html
<ul id="list">
  <li draggable="true">Item 1</li>
  <li draggable="true">Item 2</li>
  <li draggable="true">Item 3</li>
</ul>
<script>
const list = document.getElementById('list');
let dragged;
list.addEventListener('dragstart', e => {
  dragged = e.target;
});
list.addEventListener('dragover', e => {
  e.preventDefault();
});
list.addEventListener('drop', e => {
  e.preventDefault();
  if (e.target.tagName === 'LI' && dragged !== e.target) {
    list.insertBefore(dragged, e.target.nextSibling);
  }
});
</script>
```
