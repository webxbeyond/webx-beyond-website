নেটওয়ার্কিং-এর জগতে OSI ও TCP/IP মডেল দুটি সবচেয়ে গুরুত্বপূর্ণ রেফারেন্স মডেল, যা ডেটা কিভাবে এক ডিভাইস থেকে অন্য ডিভাইসে যায় তা বোঝাতে সাহায্য করে। এখানে দুইটি মডেলের স্তর, কাজ, পার্থক্য ও ব্যবহারিক দিক নিয়ে বিস্তারিত আলোচনা করা হলো।

## OSI মডেল (Open Systems Interconnection Model)

**সংজ্ঞা:**
ISO কর্তৃক তৈরি ৭-স্তরের একটি রেফারেন্স মডেল, যা নেটওয়ার্ক কমিউনিকেশনকে বিভিন্ন লেয়ারে ভাগ করে।

### OSI-এর ৭টি স্তর

| স্তর | নাম (বাংলা) | কাজ |
|-----|-------------|-----|
| ৭ | Application | ইউজার অ্যাপ্লিকেশন (HTTP, FTP, Email) |
| ৬ | Presentation | ডেটা ফরম্যাটিং, এনক্রিপশন, কম্প্রেশন |
| ৫ | Session | সেশন কন্ট্রোল, লগইন, লগআউট |
| ৪ | Transport | ডেটা ট্রান্সপোর্ট, TCP/UDP, ফ্লো/এরর কন্ট্রোল |
| ৩ | Network | রাউটিং, IP অ্যাড্রেসিং |
| ২ | Data Link | ফ্রেমিং, MAC, সুইচিং |
| ১ | Physical | কেবল, সিগন্যাল, NIC |

## TCP/IP মডেল

**সংজ্ঞা:**
ইন্টারনেটের জন্য ব্যবহৃত ৪-স্তরের একটি প্র্যাকটিক্যাল মডেল।

### TCP/IP-এর ৪টি স্তর

| স্তর | নাম (বাংলা) | OSI ম্যাপিং | কাজ |
|-----|-------------|-------------|-----|
| ৪ | Application | ৭,৬,৫ | HTTP, FTP, DNS, Email |
| ৩ | Transport | ৪ | TCP, UDP |
| ২ | Internet | ৩ | IP, Routing |
| ১ | Network Access | ২,১ | Ethernet, MAC, Physical |

```
[Transport]

```

## OSI বনাম TCP/IP: তুলনামূলক টেবিল

| বৈশিষ্ট্য | OSI | TCP/IP |
|-----------|-----|--------|
| স্তর সংখ্যা | ৭ | ৪ |
| উন্নয়ন | থিওরেটিক্যাল | প্র্যাকটিক্যাল |
| ব্যবহৃত | রেফারেন্স, শিক্ষা | ইন্টারনেট, বাস্তব |
| প্রোটোকল | নির্দিষ্ট নয় | নির্দিষ্ট (TCP, IP, HTTP, ইত্যাদি) |
| উদাহরণ | Token Ring, ATM | Ethernet, IP, TCP, UDP |

---

## বাস্তব উদাহরণ

- **ওয়েব ব্রাউজিং:**
  - Application: HTTP
  - Transport: TCP
  - Internet: IP
  - Network Access: Ethernet/Wi-Fi
- **ইমেইল:**
  - Application: SMTP/IMAP/POP3
  - Transport: TCP
  - Internet: IP
  - Network Access: Ethernet/Wi-Fi

---

## উপসংহার

OSI ও TCP/IP মডেল বুঝলে নেটওয়ার্কিং-এর প্রতিটি স্তরে কী হচ্ছে, কোথায় সমস্যা হতে পারে, এবং কোন প্রোটোকল কোন স্তরে কাজ করে—এসব সহজে বোঝা যায়। বাস্তবে TCP/IP মডেল বেশি ব্যবহৃত হলেও, OSI মডেল শিক্ষার জন্য অপরিহার্য।
