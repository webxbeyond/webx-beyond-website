Wireshark হলো সবচেয়ে জনপ্রিয় ওপেন সোর্স প্যাকেট অ্যানালাইজার, যা নেটওয়ার্ক ট্রাফিক ক্যাপচার ও বিশ্লেষণে ব্যবহৃত হয়। এটি নেটওয়ার্ক ট্রাবলশুটিং, নিরাপত্তা অডিট, প্রোটোকল বিশ্লেষণ ও শিক্ষার জন্য অপরিহার্য টুল।

---

## প্যাকেট স্নিফিং কী?

- নেটওয়ার্কে চলমান ডেটা প্যাকেট ক্যাপচার ও বিশ্লেষণ
- ট্রাফিকের সোর্স, ডেস্টিনেশন, প্রোটোকল, কনটেন্ট দেখা যায়
- সমস্যা শনাক্ত, নিরাপত্তা অডিট, প্রোটোকল শেখার জন্য ব্যবহৃত

---

## Wireshark-এর প্রধান ফিচার

- লাইভ ট্রাফিক ক্যাপচার ও সেভ
- ১০০০+ প্রোটোকল সাপোর্ট
- ফিল্টারিং, সার্চ, কালারিং
- গ্রাফিক্যাল UI, ডিটেইলড প্যাকেট ভিউ
- এক্সপোর্ট/ইম্পোর্ট (pcap, txt, csv)

---

## Wireshark ইনস্টলেশন (Windows & Linux)

```bash
# Ubuntu
sudo apt update
sudo apt install wireshark

# Windows
# https://www.wireshark.org/download.html
```

---

## Wireshark দিয়ে প্যাকেট ক্যাপচার

1. Wireshark ওপেন করুন
2. ইন্টারফেস (Ethernet/Wi-Fi) সিলেক্ট করুন
3. Start বাটনে ক্লিক করুন
4. লাইভ ট্রাফিক দেখা যাবে
5. Stop বাটনে ক্লিক করে ক্যাপচার সেভ করুন

---

## ডিসপ্লে ফিল্টার উদাহরণ

| ফিল্টার            | কাজ                                 |
|--------------------|-------------------------------------|
| ip.addr == 8.8.8.8 | নির্দিষ্ট আইপি ট্রাফিক               |
| tcp.port == 80     | HTTP ট্রাফিক                        |
| http                | শুধুমাত্র HTTP প্যাকেট              |
| dns                 | শুধুমাত্র DNS প্যাকেট               |
| tcp.flags.syn == 1 | SYN প্যাকেট (TCP handshake)         |

## ব্যবহারিক দিক

- নেটওয়ার্ক সমস্যা শনাক্ত (লেটেন্সি, প্যাকেট লস)
- নিরাপত্তা অডিট (অস্বাভাবিক ট্রাফিক, আক্রমণ)
- প্রোটোকল শেখা ও বিশ্লেষণ
- পাসওয়ার্ড/সেন্সিটিভ ডেটা ক্যাপচার (শুধুমাত্র অনুমোদিত পরিবেশে)

## চ্যালেঞ্জ ও সীমাবদ্ধতা

- এনক্রিপ্টেড ট্রাফিক বিশ্লেষণ কঠিন
- বড় নেটওয়ার্কে ডেটা বিশ্লেষণ জটিল
- অনুমতি ছাড়া প্যাকেট স্নিফিং আইনত অপরাধ
