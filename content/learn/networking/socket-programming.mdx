নেটওয়ার্কিং-এ ডেটা আদান-প্রদানের জন্য সকেট প্রোগ্রামিং একটি অপরিহার্য টপিক। সকেটের মাধ্যমে দুই বা ততোধিক ডিভাইস/অ্যাপ্লিকেশন TCP/IP নেটওয়ার্কে একে অপরের সাথে যোগাযোগ করতে পারে। এখানে সকেটের ধারণা, কাজের পদ্ধতি, প্রোটোকল, উদাহরণ ও ব্যবহারিক দিক নিয়ে বিস্তারিত আলোচনা করা হলো।

---

## সকেট কী?

- সকেট হলো একটি সফটওয়্যার ইন্টারফেস, যা অ্যাপ্লিকেশন ও নেটওয়ার্কের মধ্যে ডেটা পাঠানো ও গ্রহণের পথ তৈরি করে
- IP অ্যাড্রেস + পোর্ট নম্বর = সকেট
- TCP ও UDP উভয় প্রোটোকলে সকেট ব্যবহৃত হয়

---

## সকেট প্রোগ্রামিং-এর ধাপ

1. **সার্ভার:**
   - সকেট তৈরি (socket())
   - IP ও পোর্টে বাইন্ড (bind())
   - কানেকশন শোনা (listen())
   - ক্লায়েন্টের কানেকশন গ্রহণ (accept())
   - ডেটা পাঠানো/গ্রহণ (send()/recv())
   - কানেকশন বন্ধ (close())

2. **ক্লায়েন্ট:**
   - সকেট তৈরি (socket())
   - সার্ভারে কানেক্ট (connect())
   - ডেটা পাঠানো/গ্রহণ (send()/recv())
   - কানেকশন বন্ধ (close())

---

## TCP সকেট প্রোগ্রামিং (Python উদাহরণ)

### সার্ভার
```python
import socket
s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
s.bind(('0.0.0.0', 12345))
s.listen(1)
conn, addr = s.accept()
data = conn.recv(1024)
conn.sendall(b'Hello, Client!')
conn.close()
```

### ক্লায়েন্ট
```python
import socket
s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
s.connect(('127.0.0.1', 12345))
s.sendall(b'Hello, Server!')
data = s.recv(1024)
print(data)
s.close()
```

## UDP সকেট প্রোগ্রামিং (Python উদাহরণ)

### সার্ভার
```python
import socket
s = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)
s.bind(('0.0.0.0', 12345))
data, addr = s.recvfrom(1024)
s.sendto(b'Hello, UDP Client!', addr)
```

### ক্লায়েন্ট
```python
import socket
s = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)
s.sendto(b'Hello, UDP Server!', ('127.0.0.1', 12345))
data, addr = s.recvfrom(1024)
print(data)
```

## ব্যবহারিক দিক
- ওয়েব সার্ভার, চ্যাট অ্যাপ, গেম, ফাইল ট্রান্সফার ইত্যাদিতে সকেট ব্যবহৃত হয়
- ফায়ারওয়াল ও নেটওয়ার্ক কনফিগারেশনে পোর্ট নম্বর গুরুত্বপূর্ণ
- সকেট প্রোগ্রামিংয়ের মাধ্যমে কাস্টম নেটওয়ার্ক অ্যাপ্লিকেশন তৈরি করা যায়

---

## সংক্ষেপে
- সকেট = IP + পোর্ট
- TCP: কানেকশন-ওরিয়েন্টেড, রিলায়েবল
- UDP: কানেকশনলেস, দ্রুত
- সকেট প্রোগ্রামিং নেটওয়ার্ক অ্যাপ্লিকেশন ডেভেলপমেন্টের মূল ভিত্তি

নেটওয়ার্কিং ক্যারিয়ার ও সফটওয়্যার ডেভেলপমেন্টে সকেট প্রোগ্রামিং জানা অপরিহার্য।
