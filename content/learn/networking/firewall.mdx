## প্যাকেট ফিল্টারিং, স্টেটফুল, নেক্সট-জেন গভীর বিশ্লেষণ

ফায়ারওয়াল হলো এমন একটি নেটওয়ার্ক সিকিউরিটি ডিভাইস বা সফটওয়্যার, যা ইনকামিং ও আউটগোয়িং ট্রাফিক নিয়ন্ত্রণ করে এবং অননুমোদিত অ্যাক্সেস প্রতিরোধ করে। ফায়ারওয়াল নেটওয়ার্ক নিরাপত্তার প্রথম স্তর হিসেবে কাজ করে।

---

## ফায়ারওয়ালের প্রধান কাজ

- অনুমোদিত ট্রাফিক অনুমোদন ও অননুমোদিত ট্রাফিক ব্লক
- নেটওয়ার্ক সেগমেন্টেশন
- লগিং ও মনিটরিং
- নেটওয়ার্ক আক্রমণ প্রতিরোধ

---

## ফায়ারওয়ালের ধরন

| ধরন                | বৈশিষ্ট্য ও ব্যবহার                      |
|--------------------|-----------------------------------------|
| প্যাকেট ফিল্টারিং   | প্যাকেটের হেডার (IP, Port, Protocol) দেখে সিদ্ধান্ত নেয়; দ্রুত, বেসিক |
| স্টেটফুল ইনস্পেকশন | কানেকশন ট্র্যাক করে; ট্রাফিকের স্টেট দেখে সিদ্ধান্ত নেয়; বেশি নিরাপদ |
| নেক্সট-জেন (NGFW)  | অ্যাপ্লিকেশন, ইউজার, কনটেন্ট-ভিত্তিক ফিল্টার; IPS, SSL, DPI ইন্টিগ্রেটেড |

---

## প্যাকেট ফিল্টারিং ফায়ারওয়াল

- OSI Layer 3/4 (IP, TCP/UDP)
- Source/Destination IP, Port, Protocol দেখে সিদ্ধান্ত
- দ্রুত, কিন্তু অ্যাডভান্সড আক্রমণ ধরতে পারে না
- উদাহরণ: Access Control List (ACL), iptables (Linux)

---

## স্টেটফুল ইনস্পেকশন ফায়ারওয়াল

- কানেকশন টেবিল মেইনটেইন করে
- ট্রাফিকের স্টেট (NEW, ESTABLISHED, RELATED) দেখে সিদ্ধান্ত
- অধিক নিরাপত্তা
- উদাহরণ: pf (OpenBSD), stateful iptables, Windows Firewall

---

## নেক্সট-জেনারেশন ফায়ারওয়াল (NGFW)

- অ্যাপ্লিকেশন-লেভেল ফিল্টারিং (Layer 7)
- Deep Packet Inspection (DPI)
- Intrusion Prevention System (IPS) ইন্টিগ্রেটেড
- SSL/TLS ডিক্রিপশন
- ইউজার ও কনটেন্ট-ভিত্তিক নিয়ন্ত্রণ
- উদাহরণ: Palo Alto, Fortinet, Cisco Firepower

---

## ফায়ারওয়াল কনফিগারেশন (iptables উদাহরণ)

```bash
# ইনস্টলেশন (Ubuntu)
sudo apt update
sudo apt install iptables

# ইনকামিং SSH (22) অনুমোদন
sudo iptables -A INPUT -p tcp --dport 22 -j ACCEPT

# ইনকামিং HTTP (80) অনুমোদন
sudo iptables -A INPUT -p tcp --dport 80 -j ACCEPT

# সব ইনকামিং ট্রাফিক ডিফল্টভাবে ব্লক
sudo iptables -P INPUT DROP
```

## চ্যালেঞ্জ ও সীমাবদ্ধতা

- ভুল কনফিগারেশন হলে বৈধ ট্রাফিক ব্লক হতে পারে
- NGFW তুলনামূলক ব্যয়বহুল
- SSL ডিক্রিপশন ও DPI পারফরম্যান্স কমাতে পারে

## সংক্ষেপে
- ফায়ারওয়াল নেটওয়ার্ক নিরাপত্তার প্রথম স্তর
- প্যাকেট ফিল্টারিং, স্টেটফুল, নেক্সট-জেন—তিন ধরনেরই ব্যবহার আছে
- অ্যাডভান্সড আক্রমণ প্রতিরোধে NGFW অপরিহার্য

ফায়ারওয়াল ছাড়া নিরাপদ নেটওয়ার্ক কল্পনা করা যায় না।
