## Core Concepts

### ১. Storage vs Memory
- **Storage:** Persistent, Expensive (High Gas)
- **Memory:** Temporary, Cheap (Low Gas)
- **Best Practice:** Storage Write কম, Memory Use বেশি

### ২. Data Structure Selection
- **Array:** Simple, Sequential Data
- **Mapping:** Key-Value, Fast Lookup
- **Struct:** Composite Data, Efficient Grouping
- **উদাহরণ:**
```solidity
struct User {
    uint id;
    string name;
}
mapping(address => User) public users;
```

### ৩. Packed Storage
- **Definition:** Multiple Small Variable একসাথে Store করা
- **Technique:** uint8, bool, address এক struct-এ
- **উদাহরণ:**
```solidity
struct Packed {
    uint8 a;
    bool b;
    address c;
}
Packed public packedData;
```

### ৪. Minimize Storage Write
- **Technique:** Batch Update, Conditional Write
- **উদাহরণ:**
```solidity
function batchUpdate(address[] memory addrs, uint8 value) public {
    for (uint i = 0; i < addrs.length; i++) {
        users[addrs[i]].id = value;
    }
}
```

### ৫. Efficient Loop
- **Technique:** Loop Unrolling, Minimal Iteration
- **উদাহরণ:**
```solidity
function sum(uint[] memory arr) public pure returns (uint total) {
    for (uint i = 0; i < arr.length; i++) {
        total += arr[i];
    }
}
```

### ৬. কেন গুরুত্বপূর্ণ?

### ৮. Common Pitfalls

- Unoptimized Loop
- Excessive Storage Write
