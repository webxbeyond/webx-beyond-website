### ১. Hardhat কী?
- **Definition:** Ethereum Smart Contract Development-এর জন্য Modern Framework
- **ব্যবহার:** Compile, Deploy, Test, Debug
- **Features:**
    - Local Blockchain
    - Plugin Support
    - Fast Testing

### ২. Unit Testing
- **Definition:** Contract-এর প্রতিটি Function/Logic আলাদাভাবে Test করা
- **ব্যবহার:** Bug Detection, Security Assurance
- **উদাহরণ:**
```javascript
const { expect } = require("chai");
describe("MyContract", function () {
  it("should return correct value", async function () {
    const MyContract = await ethers.getContractFactory("MyContract");
    const contract = await MyContract.deploy();
    expect(await contract.value()).to.equal(42);
  });
});
```

### ৩. কেন গুরুত্বপূর্ণ?
- Contract Reliability
