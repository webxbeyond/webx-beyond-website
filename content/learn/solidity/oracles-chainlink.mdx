# Oracle ইন্টিগ্রেশন ও Chainlink — Solidity-তে External Data Access

## প্রস্তাবনা

Google বা Uber-এর মতো বড় টেক কোম্পানিতে Real-time Data Access, API Integration ও External Service ব্যবহার হয়—Business Logic ও User Experience উন্নত করতে। Solidity Smart Contract-এও Oracle Integration—External Data (Price, Weather, API) Access—Industry Standard। Chainlink—Most Popular Oracle Network—Secure, Reliable ও Decentralized Data Feed দেয়।

## Core Concepts

### ১. Oracle কী?
- **Definition:** Blockchain-এর বাইরে থেকে Data এনে Contract-এ ব্যবহার করার পদ্ধতি
- **ব্যবহার:** Price Feed, Weather Data, API Integration
- **Challenges:** Trust, Security, Reliability

### ২. Chainlink
- **Definition:** Decentralized Oracle Network, Secure Data Feed
- **Features:**
    - Multiple Data Source
    - Decentralized Node
    - Secure & Reliable
- **উদাহরণ:**
```solidity
import "@chainlink/contracts/src/v0.8/interfaces/AggregatorV3Interface.sol";
contract PriceConsumer {
    AggregatorV3Interface internal priceFeed;
    constructor(address _feed) {
        priceFeed = AggregatorV3Interface(_feed);
    }
    function getLatestPrice() public view returns (int) {
        (,int price,,,) = priceFeed.latestRoundData();
        return price;
    }
}
```
