### ১. Mainnet Deployment
- **Definition:** Ethereum-এর Real Blockchain-এ Contract Deploy করা
- **ব্যবহার:** Real Fund, Real User, Production Launch
- **Workflow:**
    1. Contract Compile (Hardhat/Remix)
    2. Mainnet Account Setup (MetaMask)
    3. Sufficient Ether সংগ্রহ
    4. Contract Deploy (Hardhat/Remix)
    5. Transaction Monitor (Etherscan)

### ২. Gas Optimization
- **Definition:** Contract Deploy ও Transaction-এ Gas Cost কমানো
- **Techniques:**
    - Efficient Data Structure
    - Minimize Storage Write
    - Loop Optimization
    - Batch Processing
    - Use Constants/Immutables
- **উদাহরণ:**
```solidity
uint256 public constant MAX_SUPPLY = 10000;
function batchTransfer(address[] memory recipients, uint amount) public {
    for (uint i = 0; i < recipients.length; i++) {
        // Efficient loop, minimal storage write
        _transfer(msg.sender, recipients[i], amount);
    }
}
```
