# Yield Farming লজিক — Solidity-তে Automated Passive Income

## প্রস্তাবনা

Google বা Uber-এর মতো বড় টেক কোম্পানিতে Automated Service, Resource Optimization ও User Incentive ব্যবহৃত হয়—Efficiency ও Engagement বাড়াতে। DeFi-তে Yield Farming—User-রা Token Deposit করে Automated Reward পায়—Passive Income ও Liquidity Provision নিশ্চিত হয়। Solidity-তে Yield Farming Logic—Uniswap, Yearn Finance-এর মতো Industry Standard।

## Core Concepts

### ১. Yield Farming কী?
- **Definition:** User-রা Token Deposit করে Automated Reward Earn করে
- **ব্যবহার:** Liquidity Provision, Passive Income, Incentive Program
- **Popular Protocols:** Uniswap, Yearn Finance, SushiSwap

### ২. Reward Calculation
- **Technique:** Proportional, Time-based, Multi-token
- **Features:**
    - Automated Distribution
    - Pool Share
    - Bonus/Incentive
- **উদাহরণ:**
```solidity
contract YieldFarm {
    mapping(address => uint) public deposits;
    mapping(address => uint) public rewards;
    uint public totalDeposits;
    function deposit(uint amount) public {
        deposits[msg.sender] += amount;
        totalDeposits += amount;
        // Reward Logic
    }
    function claimReward() public {
        uint reward = calculateReward(msg.sender);
        rewards[msg.sender] = 0;
        // Transfer Logic
    }
    function calculateReward(address user) public view returns (uint) {
        // Proportional Reward
        return (deposits[user] * 100) / totalDeposits;
    }
}
```

### ৩. Incentive Program
- **Technique:** Bonus, Multi-token Reward, Time-based Incentive
- **Features:**
    - User Engagement
    - Liquidity Boost
    - Automated Bonus
