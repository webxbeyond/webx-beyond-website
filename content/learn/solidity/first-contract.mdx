### ১. স্মার্ট কন্ট্রাক্ট কী?
- **ডিজিটাল চুক্তিপত্র:** Blockchain-এ স্বয়ংক্রিয়ভাবে চলা কোড
- **ব্যবহার:** Token, NFT, DeFi, DAO, Supply Chain

### ২. HelloWorld কন্ট্রাক্ট লেখা
```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

contract HelloWorld {
    string public greet = "Hello, Solidity!";
}
```
- `contract` কীওয়ার্ড দিয়ে শুরু
- ভেতরে ভেরিয়েবল ও ফাংশন

### ৩. Remix IDE-তে কম্পাইল ও ডিপ্লয়
1. https://remix.ethereum.org খুলুন
2. "New File" > `HelloWorld.sol` লিখুন
3. কোড লিখুন, Compile করুন
4. Deploy বাটনে ক্লিক করুন
5. Contract Address ও Function দেখুন

## Quick Reference: Remix IDE-তে Deploy

1. Remix IDE খুলুন
2. নতুন ফাইল তৈরি করুন: `HelloWorld.sol`
3. নিচের কোড লিখুন:
```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

contract HelloWorld {
    string public greet = "Hello, Solidity!";
}
```
4. Compile করুন
5. Deploy করুন
6. Contract Address ও Function দেখুন
