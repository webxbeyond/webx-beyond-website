### ১. payable Function
- **Ether Receive/Transfer-এর জন্য:** Function-এ payable Modifier
- **ব্যবহার:** Deposit, Crowdfunding, Token Sale
- **উদাহরণ:**
```solidity
function deposit() public payable {
    // Ether Deposited
}
```

### ২. fallback Function
- **Unknown Call Handle:** Contract-এ Function না থাকলে Call Handle
- **ব্যবহার:** Logging, Error Handling, Ether Receive (pre-0.6.0)
- **উদাহরণ:**
```solidity
fallback() external payable {
    // Handle Unknown Call
}
```

### ৩. receive Function
- **Direct Ether Receive:** Only Ether Receive, No Data
- **উদাহরণ:**
```solidity
receive() external payable {
    // Ether Received
}
```

### ৪. পার্থক্য ও গুরুত্ব
| বিষয়         | payable Function   | fallback Function   | receive Function  |
|--------------|-------------------|--------------------|-------------------|
| Use Case     | Ether Handling    | Unknown Call       | Direct Ether      |
| Data         | Yes/No            | Yes/No             | No                |

### ৫. কেন গুরুত্বপূর্ণ?
- Contract Flexibility
- Ether Receive
- Error Handling

### ৬. বাস্তব উদাহরণ
- Crowdfunding Contract: deposit() payable
- dApp Payment: receive() external payable
- Logging Contract: fallback() external

### ৭. Common Pitfalls
- Non-payable Function-এ Ether Send
- fallback-এ Logic ভুল
- receive ও fallback Confusion

### ৮. Best Practices
- Proper Modifier ব্যবহার করুন
- Minimal Logic in fallback/receive
- Error Handling
- Security Audit

### ৯. Industry Trends
- Custom Fallback Pattern
- Advanced Ether Handling
- Layer 2 Payment Integration

## উদাহরণ ও এনালজি
- payable—"বিক্রেতা টাকা নিতে পারবে"
- fallback—"অজানা কলের জন্য প্রস্তুত"
- receive—"শুধু টাকা নেওয়ার জন্য"

## Quick Reference

- payable: `function name() public payable {}`
- fallback: `fallback() external payable {}`
- receive: `receive() external payable {}`

## উপসংহার

payable Function, fallback ও receive—Solidity-তে Contract-এর Ether Receive, Flexibility ও Error Handling-এর জন্য অপরিহার্য। সঠিকভাবে ব্যবহার করলে Contract নিরাপদ, কার্যকর ও Maintainable হয়। ভুলত্রুটি এড়িয়ে, Best Practices অনুসরণ করুন—তাহলেই Solidity-তে Ether Handling ও Contract Flexibility-তে দক্ষতা অর্জন করবেন।
