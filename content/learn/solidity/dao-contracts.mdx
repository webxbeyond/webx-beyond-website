# DAO স্মার্ট কন্ট্রাক্ট — Solidity-তে Decentralized Organization Design

## প্রস্তাবনা

Google বা Uber-এর মতো বড় টেক কোম্পানিতে Decision Making, Resource Management ও User Participation—সবকিছু Hierarchical ও Centralized ভাবে হয়। DeFi-তে DAO (Decentralized Autonomous Organization)—User-রা Collective Decision, Resource Control ও Automated Governance করতে পারে—Central Authority ছাড়াই। Solidity-তে DAO Contract—MakerDAO, MolochDAO-এর মতো Industry Standard।

## Core Concepts

### ১. DAO কী?
- **Definition:** Decentralized Organization, Smart Contract-এ Automated Governance
- **ব্যবহার:** Collective Decision, Fund Management, Voting
- **Popular Protocols:** MakerDAO, MolochDAO, Aragon

### ২. Membership & Governance
- **Technique:** Token-based Membership, Voting Power
- **Features:**
    - Proposal Creation
    - Voting
    - Execution
- **উদাহরণ:**
```solidity
contract DAO {
    mapping(address => bool) public members;
    struct Proposal {
        string description;
        uint voteCount;
        bool executed;
    }
    Proposal[] public proposals;
    function addMember(address user) public {
        members[user] = true;
    }
    function createProposal(string memory desc) public {
        require(members[msg.sender], "Not a member");
        proposals.push(Proposal(desc, 0, false));
    }
    function vote(uint proposalId) public {
        require(members[msg.sender], "Not a member");
        proposals[proposalId].voteCount++;
    }
    function execute(uint proposalId) public {
        require(proposals[proposalId].voteCount > 1, "Not enough votes");
        proposals[proposalId].executed = true;
        // Execution Logic
    }
}
```
