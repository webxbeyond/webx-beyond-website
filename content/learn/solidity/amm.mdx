# AMM (Automated Market Maker) কন্ট্রাক্ট — Solidity-তে Decentralized Exchange Design

## প্রস্তাবনা

Google বা Uber-এর মতো বড় টেক কোম্পানিতে Real-time Matching, Pricing Algorithm ও Automated Service ব্যবহৃত হয়—Efficiency ও Scalability বাড়াতে। DeFi-তে AMM (Automated Market Maker)—Decentralized Exchange-এর জন্য—Liquidity, Price Discovery ও Trading Automation নিশ্চিত করে। Solidity-তে AMM Contract—Uniswap, SushiSwap-এর মতো Industry Standard।

## Core Concepts

### ১. AMM কী?
- **Definition:** Order Book ছাড়াই Liquidity Pool ও Algorithmic Pricing-এ Token Swap
- **ব্যবহার:** Decentralized Exchange, Liquidity Provision, Automated Trading
- **Popular AMM:** Uniswap, SushiSwap, Balancer

### ২. Liquidity Pool
- **Definition:** User-রা Token Deposit করে Pool তৈরি করে
- **Features:**
    - Swap Fee
    - Pool Share
    - Impermanent Loss
- **উদাহরণ:**
```solidity
contract AMM {
    mapping(address => uint) public balancesA;
    mapping(address => uint) public balancesB;
    uint public reserveA;
    uint public reserveB;
    function addLiquidity(uint amountA, uint amountB) public {
        balancesA[msg.sender] += amountA;
        balancesB[msg.sender] += amountB;
        reserveA += amountA;
        reserveB += amountB;
    }
    function swapAtoB(uint amountA) public {
        uint amountB = getAmountOut(amountA, reserveA, reserveB);
        require(reserveB >= amountB, "Insufficient liquidity");
        reserveA += amountA;
        reserveB -= amountB;
        // Transfer logic
    }
    function getAmountOut(uint amountIn, uint reserveIn, uint reserveOut) public pure returns (uint) {
        // Uniswap formula
        return (amountIn * reserveOut) / (reserveIn + amountIn);
    }
}
```
