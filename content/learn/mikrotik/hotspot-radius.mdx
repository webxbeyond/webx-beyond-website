মাইক্রোটিক হটস্পট ব্যবস্থাপনায় সোশ্যাল লগইন ও এক্সটার্নাল Radius ইন্টিগ্রেশন ব্যবহার করে ইউজার অথেন্টিকেশন আরও সহজ, নিরাপদ ও ফ্লেক্সিবল করা যায়। এতে ইউজাররা Facebook, Google, ইত্যাদি সোশ্যাল প্ল্যাটফর্ম দিয়ে লগইন করতে পারে এবং বড় নেটওয়ার্কে এক্সটার্নাল Radius সার্ভার দিয়ে ইউজার ম্যানেজমেন্ট করা যায়।

## সোশ্যাল লগইন কী?

সোশ্যাল লগইন মানে ইউজাররা Facebook, Google, Twitter ইত্যাদি অ্যাকাউন্ট দিয়ে হটস্পটে লগইন করতে পারবে। সাধারণত কাস্টম লগইন পেজে সোশ্যাল বাটন যুক্ত করতে হয় এবং থার্ড-পার্টি API ব্যবহার করতে হয়।

### সোশ্যাল লগইন ইমপ্লিমেন্টেশন (বেসিক ধারণা)
- কাস্টম login.html-এ Facebook/Google Login বাটন যুক্ত করুন
- থার্ড-পার্টি OAuth সার্ভিস (যেমন: Wifidog, Antamedia, IronWifi) ব্যবহার করুন
- ইউজার অথেন্টিকেশন সফল হলে MikroTik-এ Login API কল করুন

> নোট: MikroTik-এ ডিফল্টভাবে সোশ্যাল লগইন নেই, তাই থার্ড-পার্টি গেটওয়ে বা API ব্যবহার করতে হয়।

## এক্সটার্নাল Radius ইন্টিগ্রেশন

Radius (Remote Authentication Dial-In User Service) হলো একটি এক্সটার্নাল অথেন্টিকেশন সার্ভার, যা বড় নেটওয়ার্ক, ISP, অফিসে ইউজার ম্যানেজমেন্ট ও অথেন্টিকেশন সহজ করে।

### Radius সার্ভার কনফিগারেশন (MikroTik)

#### Step 1: Radius সার্ভার সেটআপ (যেমন: FreeRADIUS)
- FreeRADIUS ইনস্টল ও কনফিগার করুন
- ইউজার ও পাসওয়ার্ড এন্ট্রি দিন

#### Step 2: MikroTik-এ Radius যুক্ত করুন
```shell
/radius add service=hotspot address=192.168.1.200 secret=radiusSecret
/ip hotspot profile set default use-radius=yes
```

#### Step 3: User Authentication
- ইউজার লগইন করলে MikroTik Radius সার্ভারে অথেন্টিকেশন চায়
- Radius সার্ভার ইউজার ভ্যালিড হলে এক্সেস দেয়

## বাস্তব কেস স্টাডি

ধরা যাক, একটি বিশ্ববিদ্যালয়ে হাজারো ইউজার। FreeRADIUS দিয়ে ইউজার অথেন্টিকেশন, ব্যান্ডউইথ কন্ট্রোল, এক্সপায়ারি—সবকিছুই কেন্দ্রীয়ভাবে ম্যানেজ করা যায়।

## টিপস
- Radius সার্ভার ও MikroTik-এর টাইম সিঙ্ক ঠিক রাখুন
- Secret ও IP ঠিকমতো সেট করুন
- লগ মনিটর করুন
- থার্ড-পার্টি সোশ্যাল লগইন API ব্যবহার করলে ডকুমেন্টেশন অনুসরণ করুন

## উপসংহার

সোশ্যাল লগইন ও এক্সটার্নাল Radius ইন্টিগ্রেশন ব্যবহার করে হটস্পট ইউজার অথেন্টিকেশন আরও স্মার্ট, স্কেলেবল ও নিরাপদ করা যায়। বড় নেটওয়ার্ক, ISP, পাবলিক ওয়াইফাই—সবখানেই এই প্রযুক্তি অত্যন্ত কার্যকর।
