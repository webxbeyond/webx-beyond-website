মাইক্রোটিক RouterOS-এ ব্যাকআপ ও কনফিগারেশন পরিবর্তন অটোমেশন হলো নেটওয়ার্ক ম্যানেজমেন্ট, রিকভারি, এবং সিকিউরিটির জন্য অত্যন্ত গুরুত্বপূর্ণ। স্ক্রিপ্টিং ও Scheduler-এর মাধ্যমে আপনি স্বয়ংক্রিয়ভাবে ব্যাকআপ নিতে, কনফিগারেশন পরিবর্তন করতে, এবং ইমেইল/FTP-তে ব্যাকআপ পাঠাতে পারেন। এই টিউটোরিয়ালে আমরা ব্যাকআপ ও কনফিগারেশন পরিবর্তন অটোমেশন-এর কনফিগারেশন, স্ক্রিপ্ট, এবং বাস্তব উদাহরণ নিয়ে বিস্তারিত আলোচনা করব।

---

## ব্যাকআপ কেন দরকার?

- রাউটার/নেটওয়ার্ক ডিভাইসের কনফিগারেশন রিকভারি
- সিকিউরিটি ও অডিটিং
- কনফিগারেশন পরিবর্তন ট্র্যাকিং

---

## ব্যাকআপ স্ক্রিপ্টিং

### ১. ব্যাকআপ ফাইল তৈরি
```shell
/system backup save name=auto-backup
/export file=auto-export
```
- `.backup`: RouterOS binary backup
- `.rsc`: Human-readable export

### ২. FTP/Email-এ ব্যাকআপ পাঠানো
#### FTP Example
```shell
/tool fetch address=192.168.88.200 src-path=auto-backup.backup user=ftpuser password=ftppass upload=yes
```
#### Email Example
```shell
/tool e-mail send to="admin@example.com" subject="Router Backup" file=auto-backup.backup
```

---

## Scheduled Backup (Scheduler)

### Scheduler Example
```shell
/system scheduler add name=daily-backup interval=1d on-event="/system backup save name=auto-backup; /export file=auto-export; /tool e-mail send to=\"admin@example.com\" subject=\"Router Backup\" file=auto-backup.backup"
```
- `interval`: কতদিন পরপর ব্যাকআপ
- `on-event`: ব্যাকআপ স্ক্রিপ্ট

---

## কনফিগারেশন পরিবর্তন অটোমেশন

- Script দিয়ে কনফিগারেশন পরিবর্তন, Restore, Import
- Scheduled Task দিয়ে নির্দিষ্ট সময়ে পরিবর্তন

### Config Import Example
```shell
/import file=auto-export.rsc
```

---

## Winbox-এ ব্যাকআপ ও অটোমেশন

1. **Files > Backup/Export**: ম্যানুয়ালি ব্যাকআপ/এক্সপোর্ট নিন
2. **System > Scripts/Scheduler**: Script ও Scheduler Add করুন
3. **Tool > Fetch/E-mail**: FTP/Email-এ ব্যাকআপ পাঠান

---

## বাস্তব কেস স্টাডি

### কেস ১: ISP-তে প্রতিদিন স্বয়ংক্রিয় ব্যাকআপ
- Scheduler + Script + Email/FTP ব্যবহার করুন

### কেস ২: কনফিগারেশন পরিবর্তন ট্র্যাকিং
- Export Script + Scheduled Task ব্যবহার করুন

### কেস ৩: Disaster Recovery
- Backup File Restore করুন

---

## টিপস ও বেস্ট প্র্যাকটিস
- ব্যাকআপ ফাইল নিরাপদ স্থানে সংরক্ষণ করুন
- Scheduled Backup Enable রাখুন
- Email/FTP-তে ব্যাকআপ পাঠান
- কনফিগারেশন পরিবর্তন লগ রাখুন
- Restore করার আগে ফাইল যাচাই করুন

---

## উপসংহার

ব্যাকআপ ও কনফিগারেশন পরিবর্তন অটোমেশন ব্যবহার করলে নেটওয়ার্ক ম্যানেজমেন্ট, রিকভারি, এবং সিকিউরিটি অনেক সহজ ও শক্তিশালী হয়। মাইক্রোটিকের বিল্ট-ইন স্ক্রিপ্টিং ও Scheduler ব্যবহার করে সহজেই এই সুবিধা পাওয়া যায়।
