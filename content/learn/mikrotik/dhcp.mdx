মাইক্রোটিক রাউটারে DHCP (Dynamic Host Configuration Protocol) হলো এমন একটি প্রযুক্তি, যার মাধ্যমে নেটওয়ার্কের ডিভাইসগুলোকে স্বয়ংক্রিয়ভাবে IP, Gateway, DNS ইত্যাদি অ্যাসাইন করা যায়। DHCP Relay ও Reservation-এর মাধ্যমে বড় নেটওয়ার্কে আরও ফ্লেক্সিবল কনফিগারেশন সম্ভব। এই টিউটোরিয়ালে আমরা DHCP সার্ভার, রিলে ও রিজার্ভেশন নিয়ে বিস্তারিত আলোচনা করব।

---

## DHCP সার্ভার কী ও কেন দরকার?

- স্বয়ংক্রিয়ভাবে IP, Gateway, DNS অ্যাসাইন
- ম্যানুয়াল কনফিগারেশন ঝামেলা কমানো
- বড় নেটওয়ার্কে দ্রুত ডিভাইস কানেকশন

---

## DHCP সার্ভার কনফিগারেশন

### ১. DHCP Pool তৈরি

```shell
/ip pool add name=dhcp-pool ranges=192.168.88.10-192.168.88.100
```

### ২. DHCP সার্ভার তৈরি

```shell
/ip dhcp-server add name=dhcp1 interface=ether2 address-pool=dhcp-pool lease-time=1d disabled=no
```

### ৩. DHCP Network সেটিং

```shell
/ip dhcp-server network add address=192.168.88.0/24 gateway=192.168.88.1 dns-server=8.8.8.8,8.8.4.4
```

---

## DHCP রিলে (Relay) কী ও কবে ব্যবহার করবেন?

- যখন একাধিক সাবনেট/ভিন্ন নেটওয়ার্কে একটি DHCP সার্ভার থেকে IP দিতে হবে
- DHCP Relay মূলত DHCP Discover প্যাকেট ফরওয়ার্ড করে

### DHCP Relay কনফিগারেশন

```shell
/ip dhcp-relay add name=relay1 interface=ether3 dhcp-server=192.168.88.1
```

- `interface`: ক্লায়েন্ট নেটওয়ার্ক
- `dhcp-server`: মূল DHCP সার্ভারের IP

---

## DHCP রিজার্ভেশন (Static Lease)

- নির্দিষ্ট MAC Address-এর জন্য নির্দিষ্ট IP বরাদ্দ
- গুরুত্বপূর্ণ ডিভাইস (যেমন: সার্ভার, প্রিন্টার) এর জন্য ব্যবহার করুন

### Static Lease কনফিগারেশন

```shell
/ip dhcp-server lease add mac-address=AA:BB:CC:DD:EE:FF address=192.168.88.50 server=dhcp1
```

- `mac-address`: টার্গেট ডিভাইসের MAC
- `address`: নির্দিষ্ট IP
- `server`: DHCP সার্ভারের নাম

---

## Winbox-এ DHCP কনফিগারেশন

1. **IP > Pool**: Pool তৈরি করুন
2. **IP > DHCP Server**: DHCP Server ও Network সেট করুন
3. **IP > DHCP Relay**: Relay কনফিগার করুন
4. **IP > DHCP Server > Leases**: Static Lease অ্যাড করুন

---

## বাস্তব কেস স্টাডি

### কেস ১: অফিসে ৫০টি পিসি, স্বয়ংক্রিয় IP অ্যাসাইন
- DHCP Server + Pool ব্যবহার করুন

### কেস ২: একাধিক ব্রাঞ্চে একক DHCP সার্ভার
- DHCP Relay ব্যবহার করুন

### কেস ৩: প্রিন্টার/সার্ভারের জন্য নির্দিষ্ট IP
- Static Lease ব্যবহার করুন

---

## টিপস ও বেস্ট প্র্যাকটিস
- Pool-এর রেঞ্জ ও Network ঠিকভাবে দিন
- Static Lease-এর জন্য MAC Address সঠিকভাবে সংগ্রহ করুন
- DHCP Relay ব্যবহার করলে DHCP Server-এর IP ঠিকভাবে দিন
- Lease Time নেটওয়ার্কের চাহিদা অনুযায়ী নির্ধারণ করুন

---

## উপসংহার

DHCP সার্ভার, রিলে ও রিজার্ভেশন কনফিগারেশন করলে নেটওয়ার্ক ম্যানেজমেন্ট সহজ হয় এবং ডিভাইস কানেকশন দ্রুত ও নির্ভরযোগ্য হয়। মাইক্রোটিকের বিল্ট-ইন টুল ও কনফিগারেশন ব্যবহার করে সহজেই এই সুবিধা পাওয়া যায়।
