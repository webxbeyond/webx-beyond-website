মাইক্রোটিক RouterOS API ব্যবহার করে এক্সটার্নাল অ্যাপ্লিকেশন, স্ক্রিপ্ট, ও টুলস দ্বারা রাউটার কনফিগারেশন, মনিটরিং, ও অটোমেশন করা যায়। API ইন্টিগ্রেশন নেটওয়ার্ক অটোমেশন, কাস্টম ড্যাশবোর্ড, এবং থার্ড-পার্টি টুলসের জন্য অত্যন্ত গুরুত্বপূর্ণ।

---

## RouterOS API কী?

- RouterOS API একটি TCP ভিত্তিক ইন্টারফেস, যা এক্সটার্নাল ক্লায়েন্ট থেকে রাউটার কনফিগার ও মনিটর করতে দেয়।
- Winbox, WebFig ছাড়াও API দিয়ে প্রোগ্রামেটিক কনফিগারেশন সম্ভব।

## API Enable ও Basic Configuration

### ১. API Enable করুন
```shell
/ip service enable api
```

### ২. API Port পরিবর্তন
```shell
/ip service set api port=8728
```

### ৩. API Service Security
- Strong Password ব্যবহার করুন
- IP Service Binding করুন
- Firewall দিয়ে API Port Protect করুন

## API Client Libraries

- **Python:** `routeros-api`, `librouteros`
- **PHP:** `routeros-api`
- **Node.js:** `node-routeros`
- **Go:** `go-routeros`

---

## Python দিয়ে API Example

```python
from routeros_api import RouterOsApiPool

api_pool = RouterOsApiPool('192.168.88.1', username='admin', password='password', port=8728)
api = api_pool.get_api()

# Interface List
interfaces = api.get_resource('interface').get()
for iface in interfaces:
    print(iface['name'])

api_pool.disconnect()
```

---

## Node.js দিয়ে API Example

```js
const { RouterOSClient } = require('node-routeros');
const client = new RouterOSClient({
  host: '192.168.88.1',
  user: 'admin',
  password: 'password',
  port: 8728
});

client.connect().then(async () => {
  const interfaces = await client.menu('/interface').getAll();
  interfaces.forEach(i => console.log(i.name));
  client.close();
});
```

## API দিয়ে কী কী করা যায়?

## Security Best Practices
- Strong Password ও User Permission
- API Port Change ও Firewall Protect
- Service Binding
- TLS/SSL (API-SSL service)
