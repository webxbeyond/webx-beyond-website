মাইক্রোটিক RouterOS-এ স্ক্রিপ্টিং হলো অটোমেশন, কাস্টম টাস্ক, এবং ডাইনামিক কনফিগারেশনের জন্য অত্যন্ত শক্তিশালী টুল। স্ক্রিপ্টিং-এর মাধ্যমে আপনি রাউটারের বিভিন্ন কাজ স্বয়ংক্রিয়ভাবে পরিচালনা, কাস্টম লজিক ইমপ্লিমেন্ট, এবং নেটওয়ার্ক ম্যানেজমেন্ট সহজ করতে পারেন। এই টিউটোরিয়ালে আমরা RouterOS স্ক্রিপ্টিং-এর বেসিক, সিনট্যাক্স, উদাহরণ, এবং বাস্তব কেস স্টাডি নিয়ে বিস্তারিত আলোচনা করব।

---

## স্ক্রিপ্টিং কী ও কেন দরকার?

- রাউটারের কাজ অটোমেট করা
- কাস্টম টাস্ক, কনফিগারেশন, মনিটরিং
- Scheduled Task, Netwatch, Event-based Automation

---

## স্ক্রিপ্টিং সিনট্যাক্স ও বেসিক

- RouterOS স্ক্রিপ্টিং Bash/CLI-এর মতো
- ভ্যারিয়েবল, লুপ, কন্ডিশন, ফাংশন সাপোর্ট

### ভ্যারিয়েবল
```shell
:local myVar "Hello World"
:log info $myVar
```

### If Condition
```shell
:if ($status = "up") do={
  :log info "Interface is UP"
}
```

### For Loop
```shell
:for i from=1 to=5 do={
  :log info $i
}
```

---

## স্ক্রিপ্ট তৈরি ও রান

### ১. Script তৈরি
```shell
/system script add name=backup-script source=":log info 'Backup Started'"
```

### ২. Script রান
```shell
/system script run backup-script
```

### ৩. Winbox-এ Script তৈরি ও রান
- **System > Scripts**: নতুন Script Add করুন
- **Run** বাটনে ক্লিক করুন

---

## Scheduled Task (Scheduler)

- নির্দিষ্ট সময়ে Script রান

### Scheduler Example
```shell
/system scheduler add name=daily-backup interval=1d on-event=backup-script
```

---

## স্ক্রিপ্টিং দ্বারা অটোমেশন ও কাস্টম টাস্ক

- Backup, Config Change, Email/SMS Alert, Interface Control, Firewall Rule Change ইত্যাদি

### Email Alert Example
```shell
/tool e-mail send to="admin@example.com" subject="Alert" body="Backup Completed"
```

### Interface Control Example
```shell
/interface disable ether2
/interface enable ether2
```

---

## বাস্তব কেস স্টাডি

### কেস ১: রাউটারের Scheduled Backup
- Script + Scheduler ব্যবহার করুন

### কেস ২: Interface Down হলে Email Alert
- Netwatch + Script + Email API ব্যবহার করুন

### কেস ৩: Firewall Rule Change Automation
- Script দিয়ে Rule Add/Remove করুন

---

## টিপস ও বেস্ট প্র্যাকটিস
- Script-এর Source ও Logic ভালোভাবে টেস্ট করুন
- Scheduled Task-এর Interval ও Event ঠিকভাবে দিন
- Script-এর Output ও Logs মনিটর করুন
- Security ও Access Control নিশ্চিত করুন

---

## উপসংহার

RouterOS স্ক্রিপ্টিং ব্যবহার করলে রাউটারের কাজ অটোমেট, কাস্টমাইজ, এবং ডাইনামিকভাবে পরিচালনা করা যায়। মাইক্রোটিকের বিল্ট-ইন স্ক্রিপ্টিং ও Scheduler ব্যবহার করে সহজেই নেটওয়ার্ক অটোমেশন ইমপ্লিমেন্ট করা যায়।
