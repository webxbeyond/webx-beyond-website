## ufw, iptables দিয়ে লিনাক্স সিকিউরিটি

ফায়ারওয়াল মানে—নেটওয়ার্ক ট্রাফিক ফিল্টার করে সিস্টেমকে নিরাপদ রাখা। এখানে আমরা ufw (Uncomplicated Firewall) ও iptables—দুইটি টুলের বেসিক থেকে অ্যাডভান্সড—সবকিছু হাতে-কলমে শিখব।

---

## ১. ফায়ারওয়াল কী ও কেন?

- ইনকামিং/আউটগোয়িং ট্রাফিক কন্ট্রোল
- অননুমোদিত অ্যাক্সেস, অ্যাটাক, ভাইরাস প্রতিরোধ

---

## ২. ufw: সহজ ফায়ারওয়াল

### ২.১. ইনস্টল ও এনাবল
```bash
sudo apt install ufw
sudo ufw enable
```

### ২.২. বেসিক কমান্ড
```bash
sudo ufw status
sudo ufw allow 22/tcp
sudo ufw allow 'Nginx Full'
sudo ufw deny 23
sudo ufw delete allow 80
sudo ufw disable
```

### ২.৩. ডিফল্ট পলিসি
```bash
sudo ufw default deny incoming
sudo ufw default allow outgoing
```

### ২.৪. নির্দিষ্ট IP/রেঞ্জ
```bash
sudo ufw allow from 192.168.1.0/24 to any port 22
```

---

## ৩. iptables: অ্যাডভান্সড ফায়ারওয়াল

### ৩.১. বেসিক কনসেপ্ট
- চেইন: INPUT, OUTPUT, FORWARD
- টেবিল: filter, nat, mangle
- রুল: ট্রাফিক কন্ট্রোল

### ৩.২. রুল দেখা
```bash
sudo iptables -L -n -v
```

### ৩.৩. রুল যোগ/ডিলিট
```bash
sudo iptables -A INPUT -p tcp --dport 22 -j ACCEPT
sudo iptables -A INPUT -p tcp --dport 80 -j ACCEPT
sudo iptables -A INPUT -j DROP
sudo iptables -D INPUT -p tcp --dport 80 -j ACCEPT
```

### ৩.৪. রুল সেভ ও রিস্টোর
```bash
sudo iptables-save > /etc/iptables/rules.v4
sudo iptables-restore < /etc/iptables/rules.v4
```

---

## ৪. ফায়ারওয়াল টেস্ট ও লগিং

- পোর্ট চেক: `sudo ufw status numbered`, `sudo iptables -L`
- লগ: `/var/log/ufw.log`, `/var/log/kern.log`

---

## ৫. ফায়ারওয়াল বেস্ট প্র্যাকটিস

- শুধুমাত্র দরকারি পোর্ট ওপেন
- SSH পোর্ট চেঞ্জ/রেস্ট্রিক্ট
- রেট লিমিটিং
- ফায়ারওয়াল রুল টেস্ট করুন
- ব্যাকআপ নিন

---

## ৬. কমন সমস্যা ও সমাধান

- "Connection refused": পোর্ট ব্লক, ফায়ারওয়াল চেক
- রুল কনফ্লিক্ট: অর্ডার চেক করুন
- রুল সেভ ভুলে গেলে রিবুটে হারিয়ে যাবে

---

## ৭. বাস্তব উদাহরণ: ওয়েব সার্ভার সিকিউর

```bash
sudo ufw allow 80/tcp
sudo ufw allow 443/tcp
sudo ufw allow from 192.168.1.0/24 to any port 22
sudo ufw enable
```

---

## ৮. চ্যালেঞ্জ: নিজে চেষ্টা করুন

১. SSH ছাড়া সব পোর্ট ব্লক করুন
২. নির্দিষ্ট IP-কে শুধু ওয়েব অ্যাক্সেস দিন
৩. iptables দিয়ে কাস্টম রুল সেট করুন
