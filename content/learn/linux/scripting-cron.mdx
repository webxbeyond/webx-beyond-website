# cron দিয়ে শিডিউলিং: লিনাক্সে অটোমেটেড টাস্ক

cron হলো লিনাক্সের বিল্ট-ইন শিডিউলার, যার মাধ্যমে নির্দিষ্ট সময়ে স্বয়ংক্রিয়ভাবে স্ক্রিপ্ট বা কমান্ড চালানো যায়। এখানে আমরা cron-এর বেসিক থেকে অ্যাডভান্সড—সবকিছু হাতে-কলমে শিখব।

---

## ১. cron কী ও কেন?

cron (time-based job scheduler) দিয়ে আপনি নির্দিষ্ট সময়ে, দিনে, সপ্তাহে, মাসে, এমনকি মিনিটে কোনো টাস্ক অটোমেট করতে পারেন।

---

## ২. cron সার্ভিস ও কমান্ড

- cron সার্ভিস চালু: `sudo systemctl start cron` (Debian/Ubuntu)
- cron সার্ভিস চালু: `sudo systemctl start crond` (RHEL/CentOS)
- cron সার্ভিস স্ট্যাটাস: `systemctl status cron`

---

## ৩. ক্রনজব কীভাবে কাজ করে?

প্রতিটি ইউজারের জন্য আলাদা ক্রনজব থাকে। এগুলো `crontab` ফাইলে সংরক্ষিত।

- নিজের ক্রনজব দেখুন: `crontab -l`
- ক্রনজব এডিট করুন: `crontab -e`
- নির্দিষ্ট ইউজারের ক্রনজব: `sudo crontab -u username -e`

---

## ৪. ক্রনজবের সিনট্যাক্স

```
* * * * * command-to-execute
| | | | |
| | | | +----- সপ্তাহের দিন (০-৭; ০/৭=রবিবার)
| | | +------- মাস (১-১২)
| | +--------- মাসের দিন (১-৩১)
| +----------- ঘণ্টা (০-২৩)
+------------- মিনিট (০-৫৯)
```

---

## ৫. cron-এ সাধারণ উদাহরণ

- প্রতি মিনিটে: `* * * * * echo "Hello"`
- প্রতি দিন রাত ২টায়: `0 2 * * * /home/anis/backup.sh`
- প্রতি সোমবার সকাল ৭টায়: `0 7 * * 1 /home/anis/report.sh`
- প্রতি মাসের ১ তারিখ: `0 0 1 * * /home/anis/monthly.sh`

---

## ৬. shell স্ক্রিপ্ট cron-এ চালানো

```bash
# mybackup.sh
#!/bin/bash
tar -czf $HOME/backup-$(date +%F).tar.gz $HOME/Documents
```

ক্রনজব:
```
0 3 * * * /home/anis/mybackup.sh
```

---

## ৭. আউটপুট ও লগিং

- আউটপুট ফাইলে সংরক্ষণ: `* * * * * /path/script.sh >> /tmp/script.log 2>&1`
- শুধুমাত্র error: `* * * * * /path/script.sh 2>> /tmp/error.log`

---

## ৮. স্পেশাল টাইম স্ট্রিং

- `@reboot` : সিস্টেম বুট হলে
- `@daily` : প্রতিদিন
- `@hourly` : প্রতি ঘণ্টা
- `@weekly` : প্রতি সপ্তাহে
- `@monthly` : প্রতি মাসে

```bash
@reboot /home/anis/startup.sh
@daily /home/anis/cleanup.sh
```

---

## ৯. এনভায়রনমেন্ট ভ্যারিয়েবল

cron-এ অনেক সময় PATH বা অন্য ভ্যারিয়েবল সেট করতে হয়:
```
PATH=/usr/bin:/bin:/usr/local/bin
SHELL=/bin/bash
```

---

## ১০. cron-এ কমন সমস্যা ও সমাধান

- স্ক্রিপ্টে absolute path ব্যবহার করুন
- স্ক্রিপ্ট executable করুন: `chmod +x script.sh`
- স্ক্রিপ্টে শে-বাং (`#!/bin/bash`) দিন
- আউটপুট/এরর লগ চেক করুন

---

## ১১. বাস্তব উদাহরণ: অটো-লগ ক্লিনআপ

```bash
#!/bin/bash
find $HOME/logs -type f -name "*.log" -mtime +7 -delete
```

ক্রনজব:
```
0 1 * * * /home/anis/clean-logs.sh
```

---
