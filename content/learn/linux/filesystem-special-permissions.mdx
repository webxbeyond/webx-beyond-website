## SUID, SGID, Sticky Bit

লিনাক্স ফাইল সিস্টেমে সাধারণ rwx পারমিশনের পাশাপাশি কিছু স্পেশাল পারমিশন আছে, যেগুলো সিকিউরিটি ও মাল্টি-ইউজার এনভায়রনমেন্টে খুব গুরুত্বপূর্ণ। এগুলো হচ্ছে SUID (Set User ID), SGID (Set Group ID), এবং Sticky Bit।

---

## ১. SUID (Set User ID)

SUID সেট করা থাকলে, কোনো ইউজার যখন ফাইল/প্রোগ্রাম চালায়, তখন সেটি ফাইলের মালিকের পারমিশনে রান হয়।

**কেন দরকার?**
- কিছু সিস্টেম কমান্ড (যেমন: passwd) সাধারণ ইউজারও চালাতে পারে, কারণ SUID সেট করা থাকে।

**কীভাবে চেনা যায়:**
- `ls -l` কমান্ডে owner-এর execute (x) স্থানে `s` দেখাবে (যেমন: `-rwsr-xr-x`)

**কীভাবে সেট করবেন:**

```bash
chmod u+s filename
chmod 4755 filename
```

---

## ২. SGID (Set Group ID)

SGID ফোল্ডারে সেট করলে, সেখানে তৈরি হওয়া নতুন ফাইল/ফোল্ডার ডিফল্টভাবে সেই গ্রুপের অধীনে থাকবে। ফাইলে সেট করলে, রান করার সময় গ্রুপের পারমিশনে চলবে।

**কেন দরকার?**
- টিমওয়ার্কে, শেয়ারড ফোল্ডারে সবাই যেন একই গ্রুপে ফাইল তৈরি করতে পারে

**কীভাবে চেনা যায়:**
- `ls -l` কমান্ডে group-এর execute (x) স্থানে `s` দেখাবে (যেমন: `-rwxr-sr-x`)

**কীভাবে সেট করবেন:**

```bash
chmod g+s foldername
chmod 2755 filename/foldername
```

---

## ৩. Sticky Bit

Sticky Bit সাধারণত `/tmp`-এর মতো পাবলিক ফোল্ডারে ব্যবহৃত হয়। এতে, শুধু ফাইলের মালিক বা root-ই ফাইল ডিলিট করতে পারে, অন্যরা পারে না।

**কেন দরকার?**
- সিকিউরিটি বাড়াতে, সবাই যেন অন্যের ফাইল ডিলিট না করতে পারে

**কীভাবে চেনা যায়:**
- `ls -l` কমান্ডে others-এর execute (x) স্থানে `t` দেখাবে (যেমন: `drwxrwxrwt`)

**কীভাবে সেট করবেন:**

```bash
chmod +t foldername
chmod 1777 foldername
```

---

## ৪. স্পেশাল পারমিশন একসাথে

একই ফাইলে একাধিক স্পেশাল পারমিশনও থাকতে পারে।

**Numeric (Octal) Notation:**
- SUID: 4
- SGID: 2
- Sticky: 1

**উদাহরণ:**
- `chmod 4755 file` (SUID)
- `chmod 2755 file` (SGID)
- `chmod 1755 file` (Sticky)
- `chmod 7755 file` (SUID+SGID)

---

## ৫. প্র্যাকটিক্যাল টিপস

- SUID/SGID ব্যবহারে সতর্ক থাকুন, ভুলে দিলে সিকিউরিটি রিস্ক
- Sticky Bit পাবলিক ফোল্ডারে দিন
- `ls -l` দিয়ে স্পেশাল পারমিশন চেক করুন

---

## চ্যালেঞ্জ: নিজে চেষ্টা করুন

১. একটি ফাইলে SUID দিন, রান করে পার্থক্য দেখুন
২. একটি ফোল্ডারে SGID দিন, নতুন ফাইল তৈরি করে গ্রুপ চেক করুন
৩. একটি ফোল্ডারে Sticky Bit দিন, অন্য ইউজার দিয়ে ডিলিট করার চেষ্টা করুন

## উপসংহার
