# ব্যাকআপ ও রিস্টোর: লিনাক্সে ডেটা নিরাপত্তার হাতেখড়ি

ব্যাকআপ মানে—ডেটার কপি রেখে দেয়া, যাতে হারিয়ে গেলে সহজে রিস্টোর করা যায়। এখানে আমরা ফাইল, ডিরেক্টরি, ডাটাবেস—সবকিছুর ব্যাকআপ ও রিস্টোরের বেসিক থেকে অ্যাডভান্সড—সবকিছু হাতে-কলমে শিখব।

---

## ১. ব্যাকআপ কেন জরুরি?

- হার্ডওয়্যার ফেল, ভুল ডিলিট, ভাইরাস, র‍্যানসমওয়্যার, সিস্টেম ক্র্যাশ—সবকিছুর জন্য প্রস্তুতি
- প্রফেশনাল সিস্টেম অ্যাডমিনিস্ট্রেশনের অপরিহার্য অংশ

---

## ২. ব্যাকআপের ধরন

- **ফুল ব্যাকআপ:** পুরো ডেটার কপি
- **ইনক্রিমেন্টাল:** শুধু পরিবর্তিত অংশ
- **ডিফারেনশিয়াল:** শেষ ফুল ব্যাকআপ থেকে পরিবর্তিত অংশ

---

## ৩. ফাইল ও ডিরেক্টরি ব্যাকআপ (tar, rsync)

### ৩.১. tar দিয়ে ব্যাকআপ
```bash
tar -czvf backup-$(date +%F).tar.gz /home/anis/Documents
```

### ৩.২. rsync দিয়ে ব্যাকআপ
```bash
rsync -av --delete /home/anis/Documents/ /mnt/backup/Documents/
```

---

## ৪. অটোমেটেড ব্যাকআপ (cron)

```bash
0 2 * * * /home/anis/backup.sh
```

---

## ৫. ডাটাবেস ব্যাকআপ

### ৫.১. MySQL
```bash
mysqldump -u root -p mydb > mydb.sql
```

### ৫.২. PostgreSQL
```bash
pg_dump -U postgres mydb > mydb.sql
```

---

## ৬. ব্যাকআপ রিস্টোর

### ৬.১. tar ফাইল রিস্টোর
```bash
tar -xzvf backup-2025-08-12.tar.gz -C /
```

### ৬.২. rsync দিয়ে রিস্টোর
```bash
rsync -av /mnt/backup/Documents/ /home/anis/Documents/
```

### ৬.৩. MySQL রিস্টোর
```bash
mysql -u root -p mydb < mydb.sql
```

### ৬.৪. PostgreSQL রিস্টোর
```bash
psql -U postgres mydb < mydb.sql
```

---

## ৭. ব্যাকআপ মিডিয়া ও লোকেশন

- লোকাল: অন্য পার্টিশন, এক্সটার্নাল হার্ডডিস্ক
- রিমোট: rsync/scp দিয়ে অন্য সার্ভারে
- ক্লাউড: rclone, s3cmd, gsutil

---

## ৮. ব্যাকআপ ভেরিফিকেশন ও টেস্ট

- ব্যাকআপ ফাইল চেক: `tar -tzf`, `rsync --dry-run`
- রিস্টোর টেস্ট: টেস্ট ডিরেক্টরিতে রিস্টোর

---

## ৯. এনক্রিপশন ও সিকিউরিটি

- gpg দিয়ে এনক্রিপ্ট: `gpg -c backup.tar.gz`
- ডেটা ট্রান্সফার: scp/rsync + ssh

---

## ১০. ব্যাকআপ রোটেশন ও ক্লিনআপ

- পুরনো ব্যাকআপ ডিলিট: `find /mnt/backup -type f -mtime +30 -delete`
- log/check/report: ব্যাকআপ স্ক্রিপ্টে যুক্ত করুন

---

## ১১. কমন সমস্যা ও সমাধান

- পারমিশন: sudo ব্যবহার করুন
- ডিস্ক স্পেস: `df -h` দিয়ে চেক
- ব্যাকআপ ফাইল টেস্ট করুন

---

## ১২. বাস্তব উদাহরণ: ব্যাকআপ স্ক্রিপ্ট

```bash
#!/bin/bash
tar -czf /mnt/backup/home-$(date +%F).tar.gz /home/anis
find /mnt/backup -type f -mtime +30 -delete
```
