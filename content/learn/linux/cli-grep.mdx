লিনাক্সে ফাইল, ফোল্ডার, বা আউটপুটে দ্রুত কোনো শব্দ, প্যাটার্ন, বা লাইন খুঁজে বের করার জন্য সবচেয়ে শক্তিশালী টুল হলো `grep`। এটি টার্মিনাল ইউজারদের জন্য অপরিহার্য। এখানে আমরা grep-এর বেসিক থেকে অ্যাডভান্সড ব্যবহার, অপশন, ও প্র্যাকটিক্যাল টিপস জানব।

---

## ১. grep-এর বেসিক ব্যবহার

```bash
grep 'searchword' filename.txt
```

**উদাহরণ:**

```bash
grep 'error' /var/log/syslog
```

---

## ২. ক্যাস-ইনসেনসিটিভ সার্চ

`-i` অপশন দিলে বড়/ছোট হাতের পার্থক্য করবে না।

```bash
grep -i 'linux' notes.txt
```

---

## ৩. রিকার্সিভ সার্চ (ফোল্ডার জুড়ে)

`-r` বা `-R` দিয়ে ফোল্ডার ও সাব-ফোল্ডারে খুঁজতে পারবেন।

```bash
grep -r 'TODO' ~/projects/
```

---

## ৪. লাইনের নম্বরসহ আউটপুট

`-n` অপশন দিলে কোন লাইনে প্যাটার্ন আছে দেখাবে।

```bash
grep -n 'main' script.py
```

---

## ৫. একাধিক ফাইল/প্যাটার্নে সার্চ

- `grep 'pattern' file1.txt file2.txt`
- `grep -e 'error' -e 'fail' logfile.txt`

---

## ৬. রেগুলার এক্সপ্রেশন (Regex) ব্যবহার

`grep`-এ শক্তিশালী প্যাটার্ন সার্চ করা যায়।

```bash
grep '^start' file.txt   # লাইনের শুরুতে 'start'
grep 'end$' file.txt     # লাইনের শেষে 'end'
grep '[0-9]{3,}' file.txt # তিন বা তার বেশি সংখ্যা
```

---

## ৭. আউটপুট হাইলাইট ও কালার

`--color=auto` দিলে ম্যাচ হাইলাইট হবে।

```bash
grep --color=auto 'error' logfile.txt
```

---

## ৮. ইনভার্স সার্চ (ম্যাচ না হলে দেখাবে)

`-v` অপশন দিলে যেগুলো ম্যাচ হয়নি, সেগুলো দেখাবে।

```bash
grep -v '^#' config.conf   # কমেন্ট বাদে সব লাইন
```

---

## ৯. পাইপিং-এর সাথে grep

`grep`-কে পাইপ দিয়ে অন্য কমান্ডের আউটপুটে ব্যবহার করুন।

```bash
ps aux | grep 'firefox'
dmesg | grep -i 'usb'
```

---

## ১০. প্র্যাকটিক্যাল টিপস

- `grep -r` দিয়ে কোডবেসে ফিচার/বাগ খুঁজুন
- `grep -v` দিয়ে অপ্রয়োজনীয় লাইন বাদ দিন
- `grep -n` দিয়ে দ্রুত লাইনের অবস্থান জানুন
- পাইপিং দিয়ে লাইভ আউটপুট ফিল্টার করুন

---

## চ্যালেঞ্জ: নিজে চেষ্টা করুন

১. একটি ফাইলে নির্দিষ্ট শব্দ খুঁজুন
২. ফোল্ডার জুড়ে রিকার্সিভ সার্চ করুন
৩. পাইপিং দিয়ে আউটপুট ফিল্টার করুন
৪. রেগুলার এক্সপ্রেশন দিয়ে প্যাটার্ন খুঁজুন

---

## উপসংহার

`grep` ছাড়া লিনাক্সে দ্রুত সার্চ, ফিল্টারিং, ও ডেটা প্রসেসিং কল্পনাই করা যায় না। এটি ভালোভাবে আয়ত্ত করলে আপনি টার্মিনাল মাস্টার হয়ে উঠবেন।

**প্র্যাকটিস করুন, grep-এর জাদু শিখুন!**
