# শেল স্ক্রিপ্টে ফাংশন: কোড রিপিটেশন কমান, স্ক্রিপ্টকে মডুলার করুন

ফাংশন (function) ব্যবহার করলে স্ক্রিপ্ট ছোট, পরিষ্কার ও পুনঃব্যবহারযোগ্য হয়। এখানে আমরা শেল ফাংশনের বেসিক থেকে অ্যাডভান্সড—সবকিছু হাতে-কলমে শিখব।

---

## ১. ফাংশন কী ও কেন?

ফাংশন হলো কোডের এমন একটি ব্লক, যেটা বারবার ডাকা যায়। এতে কোড রিপিটেশন কমে, স্ক্রিপ্ট পড়তে ও মেইনটেইন করতে সহজ হয়।

---

## ২. ফাংশন ডিক্লেয়ার ও কল

### ২.১. বেসিক ফাংশন
```bash
say_hello() {
  echo "হ্যালো, লিনাক্স!"
}

say_hello
```

### ২.২. বিকল্প সিনট্যাক্স
```bash
function say_hello {
  echo "হ্যালো!"
}

say_hello
```

## ৩. ফাংশনে আর্গুমেন্ট

```bash
greet() {
  echo "স্বাগতম, $1!"
}

greet "anis"
```

একাধিক আর্গুমেন্ট:
```bash
sum() {
  echo $(($1 + $2))
}

sum 5 7
```

## ৪. ফাংশন থেকে রিটার্ন ভ্যালু

```bash
get_date() {
  date
}

dt=$(get_date)
echo "আজকের তারিখ: $dt"
```

`return` দিয়ে exit status পাঠানো যায়:
```bash
is_even() {
  if [ $(($1 % 2)) -eq 0 ]; then
    return 0
  else
    return 1
  fi
}

is_even 4 && echo "জোড় সংখ্যা" || echo "বিজোড় সংখ্যা"
```

## ৫. লোকাল ভ্যারিয়েবল

```bash
myfunc() {
  local msg="লোকাল"
  echo $msg
}

myfunc
echo $msg # কিছু দেখাবে না
```

## ৬. রিকার্সিভ ফাংশন

```bash
factorial() {
  if [ $1 -le 1 ]; then
    echo 1
  else
    prev=$(factorial $(($1-1)))
    echo $(($1 * $prev))
  fi
}

factorial 5
```

## ৮. বাস্তব উদাহরণ: ফাইল কাউন্টার

```bash
count_files() {
  ext=$1
  count=$(ls *.$ext 2>/dev/null | wc -l)
  echo "$ext ফাইল: $count টি"
}

count_files txt
count_files jpg
```
