# ip a দিয়ে নেটওয়ার্ক ইন্টারফেস দেখা: আধুনিক লিনাক্স নেটওয়ার্কিং

লিনাক্সে নেটওয়ার্ক ইন্টারফেস, IP, MAC, স্টেটাস ইত্যাদি দেখতে সবচেয়ে আধুনিক ও শক্তিশালী টুল হলো `ip` কমান্ড, বিশেষ করে `ip a` (ip addr)। এখানে আমরা ip a-এর ব্যবহার, আউটপুট বিশ্লেষণ, ও প্র্যাকটিক্যাল টিপস জানব।

---

## ১. ip কমান্ড ও ip a কী?

- `ip` হলো net-tools (ifconfig) এর আধুনিক বিকল্প
- `ip a` বা `ip addr` দিয়ে সব নেটওয়ার্ক ইন্টারফেস, IP, MAC, স্টেটাস দেখা যায়

---

## ২. ip a বেসিক ব্যবহার

```bash
ip a
```
- সব ইন্টারফেসের ডিটেইল দেখাবে

### শুধু নির্দিষ্ট ইন্টারফেস:
```bash
ip a show eth0
```

### শুধু IPv4:
```bash
ip -4 a
```

### শুধু IPv6:
```bash
ip -6 a
```

---

## ৩. আউটপুট বিশ্লেষণ

```
2: eth0: <BROADCAST,MULTICAST,UP,LOWER_UP> mtu 1500
    inet 192.168.1.10/24 brd 192.168.1.255 scope global eth0
    inet6 fe80::a00:27ff:fe4e:66a1/64 scope link
    ether 08:00:27:4e:66:a1 brd ff:ff:ff:ff:ff:ff
```
- **eth0**: ইন্টারফেসের নাম
- **UP**: সক্রিয়
- **inet**: IPv4
- **inet6**: IPv6
- **ether**: MAC address
- **mtu**: Maximum Transmission Unit

---

## ৪. ইন্টারফেস অন/অফ করা

```bash
sudo ip link set eth0 down   # বন্ধ
sudo ip link set eth0 up     # চালু
```

---

## ৫. নতুন IP অ্যাসাইন

```bash
sudo ip addr add 192.168.1.50/24 dev eth0
```

## ৬. ইন্টারফেসের নাম ও স্টেট চেক

```bash
ip link
```

---

## ৭. ifconfig বনাম ip

- ifconfig পুরনো, ip আধুনিক ও বেশি ফিচারড
- ip দিয়ে একসাথে অনেক কিছু করা যায়

---

## ৮. প্র্যাকটিক্যাল টিপস

- সবসময় ip কমান্ড ব্যবহার করুন
- sudo ছাড়া শুধু দেখার জন্য ip a যথেষ্ট
- man page পড়ুন: `man ip`

## উপসংহার

`ip a`—এই কমান্ড ছাড়া আধুনিক লিনাক্স নেটওয়ার্কিং অসম্ভব। ভালোভাবে আয়ত্ত করুন, নেটওয়ার্ক ট্রাবলশুটিংয়ে এক্সপার্ট হোন।
