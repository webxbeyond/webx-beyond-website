## *, ?

লিনাক্স টার্মিনালে ওয়াইল্ডকার্ড ব্যবহার করে আপনি একসাথে অনেক ফাইল/ডিরেক্টরি নিয়ে কাজ করতে পারেন। ওয়াইল্ডকার্ড হচ্ছে বিশেষ চিহ্ন, যা এক বা একাধিক ক্যারেক্টার রিপ্রেজেন্ট করে। সবচেয়ে বেশি ব্যবহৃত ওয়াইল্ডকার্ড হলো `*` (asterisk) ও `?` (question mark)।

---

## ১. * (Asterisk): যেকোনো সংখ্যা বা ক্যারেক্টার

`*` দিয়ে যেকোনো সংখ্যা ও যেকোনো ক্যারেক্টার রিপ্রেজেন্ট করা যায়।

**উদাহরণ:**

```bash
ls *.txt           # সব .txt ফাইল
cp images/*.jpg backup/   # images ফোল্ডারের সব .jpg কপি
rm *               # সব ফাইল ডিলিট (সতর্ক থাকুন!)
```

---

## ২. ? (Question Mark): যেকোনো একটি ক্যারেক্টার

`?` দিয়ে ঠিক একটি ক্যারেক্টার রিপ্রেজেন্ট করা যায়।

**উদাহরণ:**

```bash
ls file?.txt       # file1.txt, fileA.txt, file_.txt (file10.txt না)
rm data??.csv      # data12.csv, dataAB.csv (data1.csv না)
```

---

## ৩. আরও কিছু ওয়াইল্ডকার্ড

- `[abc]` : a, b, বা c যেকোনো একটি ক্যারেক্টার
- `[a-z]` : a থেকে z পর্যন্ত যেকোনো একটি ক্যারেক্টার
- `[!0-9]` : 0-9 বাদে অন্য যেকোনো ক্যারেক্টার

**উদাহরণ:**

```bash
ls file[1-3].txt   # file1.txt, file2.txt, file3.txt
ls img[!0-9].png   # imgA.png, imgB.png (img1.png না)
```

## ৪. প্র্যাকটিক্যাল টিপস

- ওয়াইল্ডকার্ড দিয়ে দ্রুত ফাইল সিলেক্ট করুন
- সতর্ক থাকুন: `rm *` বা `rm *.ext` চালানোর আগে চেক করুন
- Tab দিয়ে অটো-কমপ্লিট ব্যবহার করুন
- স্ক্রিপ্টে ওয়াইল্ডকার্ড দিয়ে ব্যাচ প্রসেসিং করুন

## চ্যালেঞ্জ: নিজে চেষ্টা করুন

১. সব .md ফাইল কপি করুন
২. file?.txt টাইপের ফাইল খুঁজুন
৩. [a-z] ওয়াইল্ডকার্ড দিয়ে ফাইল লিস্ট করুন

## উপসংহার

ওয়াইল্ডকার্ড—বিশেষ করে `*` ও `?`—ছাড়া টার্মিনালে ব্যাচ প্রসেসিং, ফাইল ফিল্টারিং, ও স্মার্ট কাজ কল্পনাই করা যায় না। এগুলো ভালোভাবে আয়ত্ত করলে আপনার টার্মিনাল দক্ষতা অনেক বাড়বে।
