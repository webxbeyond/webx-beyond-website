# useradd ও userdel দিয়ে ইউজার ম্যানেজ

লিনাক্স সিস্টেমে নতুন ইউজার তৈরি, মুছে ফেলা, ও ইউজার ম্যানেজমেন্টের জন্য সবচেয়ে বেশি ব্যবহৃত কমান্ড হলো `useradd` ও `userdel`। এগুলো সিস্টেম অ্যাডমিনদের জন্য অপরিহার্য। এখানে আমরা useradd ও userdel-এর বেসিক থেকে অ্যাডভান্সড ব্যবহার, অপশন, ও টিপস জানব।

---

## ১. useradd: নতুন ইউজার তৈরি

`useradd` কমান্ড দিয়ে নতুন ইউজার তৈরি করা যায়।

**বেসিক ব্যবহার:**

```bash
sudo useradd username
```

এতে `/home/username` নামে হোম ডিরেক্টরি তৈরি হবে না, শুধু ইউজার এন্ট্রি হবে।

### হোম ডিরেক্টরি ও শেলসহ ইউজার তৈরি

```bash
sudo useradd -m -s /bin/bash anis
```

- `-m` : হোম ডিরেক্টরি তৈরি করবে
- `-s` : ডিফল্ট শেল নির্ধারণ

### ইউজারের জন্য পাসওয়ার্ড সেট

```bash
sudo passwd anis
```

---

## ২. userdel: ইউজার ডিলিট

`userdel` কমান্ড দিয়ে ইউজার ডিলিট করা যায়।

**বেসিক ব্যবহার:**

```bash
sudo userdel username
```

### হোম ডিরেক্টরি ও ফাইলসহ ডিলিট

```bash
sudo userdel -r anis
```

- `-r` : হোম ডিরেক্টরি ও মেইল স্পুলও ডিলিট করবে

---

## ৩. ইউজার ইনফো ও মডিফাই

- ইউজার ইনফো: `getent passwd username`
- ইউজার মডিফাই: `usermod` কমান্ড (যেমন: গ্রুপ, শেল পরিবর্তন)

```bash
sudo usermod -aG sudo anis   # anis-কে sudo গ্রুপে যোগ
sudo usermod -s /bin/zsh anis # anis-এর শেল পরিবর্তন
```

---

## ৪. প্র্যাকটিক্যাল টিপস

- ইউজার তৈরি/ডিলিট করার আগে ব্যাকআপ নিন
- sudo ছাড়া useradd/userdel চালানো যাবে না
- ইউজার ডিলিট করলে তার ফাইল/ডিরেক্টরি চেক করুন
- usermod দিয়ে ইউজার কাস্টমাইজ করুন

---

## চ্যালেঞ্জ: নিজে চেষ্টা করুন

১. নতুন ইউজার anis তৈরি করুন, হোম ডিরেক্টরি ও bash শেল দিন
২. anis-এর জন্য পাসওয়ার্ড সেট করুন
৩. anis-কে sudo গ্রুপে যোগ করুন
৪. anis ইউজার ডিলিট করুন, হোম ডিরেক্টরিও ডিলিট করুন

---

## উপসংহার

`useradd`, `userdel`, ও `usermod`—এই কমান্ডগুলো ছাড়া লিনাক্সে ইউজার ম্যানেজমেন্ট অসম্ভব। এগুলো ভালোভাবে আয়ত্ত করলে আপনি সিস্টেম অ্যাডমিন হিসেবে আরও দক্ষ হবেন।

**প্র্যাকটিস করুন, ইউজার ম্যানেজমেন্টের জাদু শিখুন!**
