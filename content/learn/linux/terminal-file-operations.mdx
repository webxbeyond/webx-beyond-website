## touch, mkdir, cp, mv, rm

লিনাক্স টার্মিনালে ফাইল ও ডিরেক্টরি (ফোল্ডার) তৈরি, কপি, মুভ, রিনেম ও ডিলিট করার জন্য কিছু কমান্ড সবচেয়ে বেশি ব্যবহৃত হয়। এখানে আমরা `touch`, `mkdir`, `cp`, `mv`, ও `rm` নিয়ে বিস্তারিত আলোচনা করব।

---

## ১. touch: নতুন ফাইল তৈরি

`touch` কমান্ড দিয়ে আপনি নতুন ফাইল তৈরি করতে পারেন।

```bash
touch filename.txt
```

একসাথে একাধিক ফাইলও তৈরি করা যায়:

```bash
touch file1.txt file2.txt file3.txt
```

**ব্যবহার:**
- ফাইল না থাকলে তৈরি হবে
- ফাইল থাকলে তার last modified time আপডেট হবে

---

## ২. mkdir: নতুন ডিরেক্টরি (ফোল্ডার) তৈরি

`mkdir` দিয়ে নতুন ফোল্ডার তৈরি করা যায়।

```bash
mkdir myfolder
```

একসাথে একাধিক ফোল্ডার:

```bash
mkdir folder1 folder2 folder3
```

সাব-ডিরেক্টরি সহ তৈরি করতে:

```bash
mkdir -p parent/child/grandchild
```

---

## ৩. cp: ফাইল ও ডিরেক্টরি কপি

`cp` দিয়ে ফাইল বা ফোল্ডার কপি করা যায়।

```bash
cp source.txt destination.txt
```

এক ফোল্ডার থেকে অন্য ফোল্ডারে:

```bash
cp file.txt /home/anis/Documents/
```

ডিরেক্টরি কপি করতে `-r` (recursive) অপশন ব্যবহার করুন:

```bash
cp -r myfolder/ backupfolder/
```

---

## ৪. mv: ফাইল/ফোল্ডার মুভ ও রিনেম

`mv` কমান্ড দিয়ে ফাইল বা ফোল্ডার এক জায়গা থেকে অন্য জায়গায় নিতে পারেন, আবার ফাইল/ফোল্ডার রিনেমও করতে পারেন।

```bash
mv oldname.txt newname.txt   # রিনেম
mv file.txt /home/anis/Desktop/   # মুভ
```

একসাথে একাধিক ফাইলও মুভ করা যায়:

```bash
mv file1.txt file2.txt /home/anis/backup/
```

---

## ৫. rm: ফাইল ও ডিরেক্টরি ডিলিট

`rm` দিয়ে ফাইল বা ফোল্ডার ডিলিট করা যায়।

```bash
rm file.txt
```

একসাথে একাধিক ফাইল:

```bash
rm file1.txt file2.txt
```

ডিরেক্টরি ডিলিট করতে `-r` (recursive) অপশন ব্যবহার করুন:

```bash
rm -r myfolder/
```

**সতর্কতা:**
- `rm` দিয়ে ডিলিট করলে সাধারণত রিকভার করা যায় না।
- `rm -rf /` কমান্ডটি কখনোই চালাবেন না! এটি পুরো সিস্টেম ডিলিট করে দেবে।

---

## প্র্যাকটিক্যাল টিপস

- `cp -i`, `mv -i`, `rm -i` ব্যবহার করলে কনফার্মেশন চাইবে, ভুলে কিছু ডিলিট/ওভাররাইট হওয়া রোধ করবে।
- Tab key দিয়ে ফাইল/ফোল্ডার নাম অটো-কমপ্লিট করুন।
- `ls` দিয়ে আগে দেখে নিন, কোন ফাইল/ফোল্ডার নিয়ে কাজ করছেন।

---

## চ্যালেঞ্জ: নিজে চেষ্টা করুন

১. একটি নতুন ফোল্ডার তৈরি করুন, তার মধ্যে ৩টি ফাইল তৈরি করুন।
২. একটি ফাইল কপি করে অন্য নামে রাখুন।
৩. একটি ফাইল রিনেম করুন।
৪. একটি ফাইল ও একটি ফোল্ডার ডিলিট করুন।

---

## উপসংহার

`touch`, `mkdir`, `cp`, `mv`, ও `rm`—এই কমান্ডগুলো ছাড়া লিনাক্সে ফাইল ও ফোল্ডার ম্যানেজমেন্ট অসম্ভব। এগুলো ভালোভাবে আয়ত্ত করলে আপনার কাজ হবে অনেক দ্রুত ও দক্ষ।

**প্র্যাকটিস করুন, সাবধান থাকুন, আর হয়ে উঠুন ফাইল ম্যানেজমেন্ট মাস্টার!**
