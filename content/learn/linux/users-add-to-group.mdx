# ইউজারকে গ্রুপে যুক্ত করা (User to Group)

লিনাক্সে ইউজারদের নির্দিষ্ট গ্রুপে যুক্ত করা খুবই গুরুত্বপূর্ণ, কারণ এতে পারমিশন ও অ্যাক্সেস কন্ট্রোল সহজ হয়। এখানে আমরা ইউজারকে গ্রুপে যুক্ত করার বিভিন্ন পদ্ধতি, কমান্ড, ও প্র্যাকটিক্যাল টিপস জানব।

---

## ১. কেন ইউজারকে গ্রুপে যুক্ত করবেন?

- নির্দিষ্ট ফাইল/ডিরেক্টরিতে একাধিক ইউজারকে একসাথে অ্যাক্সেস দিতে
- টিমওয়ার্কে পারমিশন সহজ করতে
- সফটওয়্যার/সার্ভিস ব্যবহারের জন্য নির্দিষ্ট গ্রুপে ইউজার যুক্ত করা

---

## ২. usermod কমান্ড দিয়ে ইউজারকে গ্রুপে যুক্ত করা

`usermod` কমান্ড সবচেয়ে বেশি ব্যবহৃত হয় ইউজারকে গ্রুপে যুক্ত করতে।

### বেসিক সিনট্যাক্স:

```bash
sudo usermod -aG groupname username
```

- `-a` : append (পুরাতন গ্রুপ রেখে নতুন গ্রুপ যোগ)
- `-G` : গ্রুপের নাম (একাধিক গ্রুপ হলে কমা দিয়ে দিন)

### উদাহরণ:

```bash
sudo usermod -aG devs anis
```

এতে `anis` ইউজারকে `devs` গ্রুপে যুক্ত করা হবে।

### একাধিক গ্রুপে যুক্ত করা:

```bash
sudo usermod -aG devs,admins anis
```

---

## ৩. ইউজার কোন গ্রুপে আছে, কিভাবে দেখবেন?

```bash
groups anis
```

অথবা

```bash
id anis
```

---

## ৪. নতুন গ্রুপে যুক্ত হওয়ার জন্য লগআউট/রিবুট

নতুন গ্রুপে যুক্ত হওয়ার পর ইউজারকে লগআউট/লগইন করতে হতে পারে, অথবা `su - username` দিয়ে নতুন সেশনে যেতে হবে।

---

## ৫. ইউজারকে প্রাইমারি গ্রুপ পরিবর্তন

```bash
sudo usermod -g newgroup anis
```

- `-g` : প্রাইমারি গ্রুপ পরিবর্তন

---

## ৬. /etc/group ফাইল এডিট করে ইউজার যুক্ত

`/etc/group` ফাইল ম্যানুয়ালি এডিট করেও ইউজারকে গ্রুপে যুক্ত করা যায়।

```bash
sudo nano /etc/group
```

সেখানে গ্রুপ লাইনে ইউজারনেম যোগ করুন:

```
devs:x:1002:anis,rahim
```

---

## ৭. ইউজারকে গ্রুপ থেকে সরানো

- `/etc/group` ফাইল থেকে ইউজারনেম মুছে দিন
- অথবা `gpasswd -d anis devs` কমান্ড ব্যবহার করুন:

```bash
sudo gpasswd -d anis devs
```

---

## ৮. প্র্যাকটিক্যাল টিপস

- সবসময় `-aG` ব্যবহার করুন, না হলে পুরাতন গ্রুপ মুছে যেতে পারে
- গ্রুপ চেইঞ্জের পর নতুন সেশন নিন
- ভুল হলে `/etc/group` ব্যাকআপ রাখুন
