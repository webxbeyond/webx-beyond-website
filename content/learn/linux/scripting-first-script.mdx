# প্রথম শেল স্ক্রিপ্ট লেখা: হাতে-কলমে অটোমেশন

শেল স্ক্রিপ্টিং শেখার সবচেয়ে ভালো উপায় হলো নিজে হাতে প্রথম স্ক্রিপ্ট লেখা। এখানে আমরা একদম শুরু থেকে ধাপে ধাপে প্রথম স্ক্রিপ্ট তৈরি, রান, ও কাস্টমাইজ করার পুরো প্রক্রিয়া দেখব।

---

## ১. স্ক্রিপ্ট ফাইল তৈরি করা

### ১.১. ফাইল তৈরি করুন
```bash
touch myfirstscript.sh
```

### ১.২. ফাইল ওপেন করুন
```bash
nano myfirstscript.sh
```

---

## ২. স্ক্রিপ্টের গঠন ও লেখা

প্রথম লাইনে লিখুন:
```bash
#!/bin/bash
```

এরপর কিছু কমান্ড যোগ করুন:
```bash
echo "হ্যালো, লিনাক্স!"
date
echo "আপনার হোম ডিরেক্টরি: $HOME"
```

ফাইনাল স্ক্রিপ্ট:
```bash
#!/bin/bash
# আমার প্রথম শেল স্ক্রিপ্ট

echo "হ্যালো, লিনাক্স!"
date
echo "আপনার হোম ডিরেক্টরি: $HOME"
```

## ৪. স্ক্রিপ্ট চালানো

```bash

```

## ৫. স্ক্রিপ্টে ইউজার ইনপুট

```bash
#!/bin/bash
read -p "আপনার নাম লিখুন: " name
echo "স্বাগতম, $name!"
```

## ৬. স্ক্রিপ্টে আর্গুমেন্ট ব্যবহার

```bash
#!/bin/bash
echo "প্রথম আর্গুমেন্ট: $1"
echo "সব আর্গুমেন্ট: $@"
```

রান করুন:
```bash
./myfirstscript.sh anis linux
```

## ৭. স্ক্রিপ্টে কন্ডিশন ও লুপ

### কন্ডিশন:
```bash
#!/bin/bash
num=10
if [ $num -gt 5 ]; then
  echo "$num পাঁচের চেয়ে বড়"
else
  echo "$num পাঁচের চেয়ে ছোট বা সমান"
fi
```

### লুপ:
```bash
#!/bin/bash
for file in *.txt; do
  echo "ফাইল: $file"
done
```

## ৮. স্ক্রিপ্টে error handling ও ডিবাগিং

```bash
#!/bin/bash
set -e
trap 'echo "সমস্যা হয়েছে!"' ERR
cp /source/file /dest/ || echo "কপি করতে সমস্যা!"
```

ডিবাগিং:
```bash
bash -x myfirstscript.sh
```

## ৯. বাস্তব উদাহরণ: অটো-ব্যাকআপ স্ক্রিপ্ট

```bash
#!/bin/bash
src="$HOME/Documents"
dest="$HOME/backup-$(date +%F)"
cp -r "$src" "$dest"
echo "Backup complete: $dest"
```
