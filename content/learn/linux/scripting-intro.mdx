## অটোমেশন ও পাওয়ারের শুরু

শেল স্ক্রিপ্ট হলো একাধিক কমান্ডকে ধারাবাহিকভাবে চালানোর জন্য লেখা ফাইল, যা লিনাক্সে কাজকে অটোমেট করে, সময় বাঁচায়, এবং জটিল টাস্ক সহজ করে। শেল স্ক্রিপ্টিং শেখা মানে শুধু কমান্ড জানা নয়—এটা হচ্ছে লিনাক্সকে নিজের মতো করে চালানোর ক্ষমতা অর্জন। এখানে আমরা শেল স্ক্রিপ্টের ধারণা, ব্যবহার, গঠন, ভ্যারিয়েবল, ইউজার ইনপুট, কন্ডিশন, লুপ, ফাংশন, ডিবাগিং, এবং বাস্তব উদাহরণসহ সবকিছু জানব।

---

## ১. শেল ও শেল স্ক্রিপ্টের ধারণা

- **শেল:** টার্মিনালের ইন্টারপ্রেটার (bash, zsh, sh, fish)
- **শেল স্ক্রিপ্ট:** একাধিক কমান্ডের ফাইল, যা শেল এক্সিকিউট করে
- `.sh` এক্সটেনশন সাধারণত ব্যবহৃত

## ২. কেন শেল স্ক্রিপ্ট লিখবেন?

- রিপিটেটিভ টাস্ক অটোমেট
- ব্যাকআপ, ডিপ্লয়, মনিটরিং সহজ করা
- কাস্টম টুল তৈরি
- বড় টাস্ক এক ক্লিকে চালানো

## ৩. শেল স্ক্রিপ্টের বেসিক গঠন

```bash
#!/bin/bash
# এই লাইনটি স্ক্রিপ্টের শেল নির্ধারণ করে (shebang)

# কমেন্ট
echo "হ্যালো, লিনাক্স!"
ls -l
```

- প্রথম লাইন: `#!/bin/bash` (shebang)
- কমেন্ট: `#`
- প্রতিটি লাইনে এক বা একাধিক কমান্ড

### আরও উদাহরণ: ফাইল ব্যাকআপ স্ক্রিপ্ট
```bash
#!/bin/bash
src="$HOME/Documents"
dest="$HOME/backup-$(date +%F)"
cp -r "$src" "$dest"
echo "Backup complete: $dest"
```

### স্ক্রিপ্টে error handling
```bash
#!/bin/bash
set -e  # কোনো কমান্ডে error হলে স্ক্রিপ্ট থেমে যাবে
trap 'echo "সমস্যা হয়েছে!"' ERR
cp /source/file /dest/ || echo "কপি করতে সমস্যা!"
```

## ৪. শেল স্ক্রিপ্ট রান করার নিয়ম

### ১. স্ক্রিপ্ট ফাইল তৈরি করুন:
```bash
echo -e '#!/bin/bash\necho "Hello World"' > myscript.sh
```

### ২. এক্সিকিউটেবল পারমিশন দিন:
```bash
chmod +x myscript.sh
```

### ৩. স্ক্রিপ্ট চালান:
```bash
./myscript.sh
```

### ৪. সরাসরি bash দিয়ে চালানো:
```bash
bash myscript.sh
```

## ৫. শেল স্ক্রিপ্টের মূল উপাদান

### ১. ভ্যারিয়েবল
```bash
name="Anis"
echo "স্বাগতম, $name!"
```

### ২. ইউজার ইনপুট
```bash
read -p "আপনার নাম লিখুন: " username
echo "হ্যালো, $username!"
```

### ৩. কন্ডিশন (if-else)
```bash
num=10
if [ $num -gt 5 ]; then
	echo "$num পাঁচের চেয়ে বড়"
else
	echo "$num পাঁচের চেয়ে ছোট বা সমান"
fi
```

### ৪. লুপ
```bash
for file in *.txt; do
	echo "ফাইল: $file"
done
```

### ৫. ফাংশন
```bash
greet() {
	echo "হ্যালো, $1!"
}
greet "লিনাক্স"
```

### ৬. আর্গুমেন্ট
```bash
echo "প্রথম আর্গুমেন্ট: $1"
echo "সব আর্গুমেন্ট: $@"
```

### ৭. স্ক্রিপ্ট ডিবাগিং
```bash
bash -x myscript.sh
```
এতে প্রতিটি কমান্ড এক্সিকিউশনের সময় দেখাবে।

## ৭. বাস্তব উদাহরণ

### ১. অটো-আপডেট স্ক্রিপ্ট
```bash
#!/bin/bash
sudo apt update && sudo apt upgrade -y
echo "সিস্টেম আপডেট শেষ!"
```

### ২. ফোল্ডারে কত ফাইল আছে?
```bash
#!/bin/bash
count=$(ls | wc -l)
echo "এই ফোল্ডারে মোট $count টি ফাইল আছে।"
```

### ৩. ব্যাকআপ স্ক্রিপ্ট (error handling সহ)
```bash
#!/bin/bash
set -e
src="$HOME/Documents"
dest="$HOME/backup-$(date +%F)"
trap 'echo "ব্যাকআপে সমস্যা!"' ERR
cp -r "$src" "$dest"
echo "Backup complete: $dest"
```
