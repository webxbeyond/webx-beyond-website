# man কমান্ড দিয়ে ম্যানুয়াল দেখা

লিনাক্সে হাজারো কমান্ড, টুল, ও ফাংশন আছে। প্রতিটিরই ডিটেইলস জানতে চাইলে সবচেয়ে সহজ উপায় হচ্ছে `man` (manual) কমান্ড। এটি লিনাক্সের বিল্ট-ইন হেল্প সিস্টেম, যেখানে প্রতিটি কমান্ডের জন্য ডিটেইলস ডকুমেন্টেশন থাকে।

---

## ১. man কমান্ডের বেসিক ব্যবহার

```bash
man <command>
```

**উদাহরণ:**

```bash
man ls
man cp
man grep
```

এতে ঐ কমান্ডের ডিটেইলস, অপশন, উদাহরণ, ও আরও অনেক কিছু পাবেন।

---

## ২. man পেজের গঠন

man পেজ সাধারণত নিচের অংশগুলো নিয়ে গঠিত:
- NAME: কমান্ডের নাম ও সংক্ষিপ্ত বর্ণনা
- SYNOPSIS: কিভাবে ব্যবহার করবেন
- DESCRIPTION: বিস্তারিত ব্যাখ্যা
- OPTIONS: কোন কোন অপশন আছে
- EXAMPLES: ব্যবহারিক উদাহরণ
- SEE ALSO: সম্পর্কিত কমান্ড

---

## ৩. man পেজে নেভিগেশন

- Up/Down arrow: স্ক্রল
- Space: পরবর্তী পেজ
- b: আগের পেজ
- /searchword: খুঁজুন
- n: পরবর্তী রেজাল্ট
- q: বেরিয়ে আসুন

---

## ৪. নির্দিষ্ট সেকশন থেকে man দেখা

একই নামের কমান্ড ও ফাইল/ফাংশন থাকলে সেকশন নম্বর দিয়ে স্পেসিফিক দেখাতে পারেন।

```bash
man 5 passwd   # /etc/passwd ফাইলের ম্যানুয়াল
man 1 passwd   # passwd কমান্ডের ম্যানুয়াল
```

---

## ৫. man পেজে দ্রুত খোঁজার টিপস

- `/` দিয়ে যেকোনো শব্দ খুঁজুন
- `q` দিয়ে বেরিয়ে আসুন
- `man -k <keyword>` দিয়ে কমান্ড সার্চ করুন (apropos)

```bash
man -k copy
```

---

## ৬. man ছাড়া আরও হেল্প

- `command --help` বা `command -h` দিয়েও সংক্ষিপ্ত হেল্প পাওয়া যায়
- `info <command>` দিয়ে আরও ডিটেইলস

---

## ৭. প্র্যাকটিক্যাল টিপস

- নতুন কমান্ড শিখতে man পেজ পড়ুন
- অপশন ভুলে গেলে man পেজ দেখে নিন
- man পেজে উদাহরণ খুঁজে নিন

---

## চ্যালেঞ্জ: নিজে চেষ্টা করুন

১. `man ls` চালিয়ে দেখুন, কোন কোন অপশন আছে
২. `man -k search` দিয়ে সার্চ কমান্ড খুঁজুন
৩. `man 5 passwd` ও `man 1 passwd` চালিয়ে পার্থক্য দেখুন

---

## উপসংহার

`man` কমান্ড ছাড়া লিনাক্স শেখা অসম্ভব। এটি আপনার সবচেয়ে বড় গাইড। যেকোনো কমান্ড, ফাইল, বা ফাংশন সম্পর্কে জানতে man ব্যবহার করুন।

**প্র্যাকটিস করুন, man পেজ পড়ে এক্সপার্ট হয়ে উঠুন!**
