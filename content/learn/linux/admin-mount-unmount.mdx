# মাউন্ট ও আনমাউন্ট করা: লিনাক্সে স্টোরেজ ব্যবস্থাপনার মূলকথা

লিনাক্সে ফাইল সিস্টেম, হার্ডডিস্ক, ইউএসবি, ISO—সবকিছু ব্যবহার করতে হলে মাউন্ট করতে হয়। এখানে আমরা মাউন্ট ও আনমাউন্টের বেসিক থেকে অ্যাডভান্সড—সবকিছু হাতে-কলমে শিখব।

---

## ১. মাউন্ট কী?

- মাউন্ট মানে—ডিভাইস/পার্টিশনকে ফাইল সিস্টেমে যুক্ত করা
- আনমাউন্ট মানে—সেই সংযোগ বিচ্ছিন্ন করা

---

## ২. মাউন্ট পয়েন্ট

- ফোল্ডার যেখানে ডিভাইস অ্যাক্সেসযোগ্য হবে (e.g. `/mnt`, `/media/usb`)

---

## ৩. মাউন্ট কমান্ড

```bash
sudo mount /dev/sdb1 /mnt/mydata
```

- ফাইল সিস্টেম স্পেসিফাই: `sudo mount -t ext4 /dev/sdb1 /mnt/mydata`
- ISO মাউন্ট: `sudo mount -o loop file.iso /mnt/iso`

---

## ৪. আনমাউন্ট কমান্ড

```bash
sudo umount /mnt/mydata
```

- ডিভাইস দিয়ে: `sudo umount /dev/sdb1`

---

## ৫. মাউন্টেড ডিভাইস চেক

```bash
mount | grep ^/dev
lsblk
findmnt
```

---

## ৬. অটো-মাউন্ট (fstab)

`/etc/fstab` ফাইলে এন্ট্রি দিন:
```
/dev/sdb1  /mnt/mydata  ext4  defaults  0  2
```

- টেস্ট: `sudo mount -a`

---

## ৭. ইউএসবি/এক্সটার্নাল ড্রাইভ মাউন্ট

- প্লাগ ইন করলে `/media/username/`-এ অটো-মাউন্ট
- ম্যানুয়ালি: `sudo mount /dev/sdc1 /mnt/usb`

---

## ৮. ISO/IMG ফাইল মাউন্ট

```bash
sudo mount -o loop ubuntu.iso /mnt/iso
```

---

## ৯. নেটওয়ার্ক শেয়ার (NFS, SMB) মাউন্ট

- NFS: `sudo mount -t nfs server:/share /mnt/nfs`
- SMB: `sudo mount -t cifs -o username=user //server/share /mnt/smb`

---

## ১০. কমন সমস্যা ও সমাধান

- "device is busy": ফাইল ওপেন/প্রসেস চলছে, `lsof` বা `fuser` দিয়ে চেক
- পারমিশন: sudo ব্যবহার করুন
- ফাইল সিস্টেম সাপোর্ট: `dmesg`/`journalctl` চেক করুন

---

## ১১. বাস্তব উদাহরণ: ইউএসবি মাউন্ট ও আনমাউন্ট

১. `lsblk` দিয়ে ডিভাইস চেক
২. `sudo mount /dev/sdc1 /mnt/usb`
৩. কাজ শেষে `sudo umount /mnt/usb`

---

## ১২. চ্যালেঞ্জ: নিজে চেষ্টা করুন

১. ISO ফাইল মাউন্ট করুন
২. fstab দিয়ে পার্মানেন্ট মাউন্ট করুন
৩. "device is busy" সমস্যা সমাধান করুন
