# whoami ও id দিয়ে নিজের তথ্য জানা

লিনাক্সে আপনি কোন ইউজার হিসেবে লগইন করেছেন, আপনার ইউজার আইডি (UID), গ্রুপ আইডি (GID), ও কোন কোন গ্রুপে আছেন—এসব জানতে `whoami` ও `id` কমান্ড ব্যবহার করা হয়। এগুলো সিস্টেম অ্যাডমিন, ডেভেলপার, ও সাধারণ ইউজার—সবাইর জন্যই গুরুত্বপূর্ণ।

---

## ১. whoami: আমি কে?

`whoami` কমান্ড দিয়ে আপনি জানতে পারবেন, বর্তমানে কোন ইউজার হিসেবে টার্মিনালে কাজ করছেন।

```bash
whoami
```

**উদাহরণ:**

```
anis
```

---

## ২. id: ইউজার ও গ্রুপের বিস্তারিত তথ্য

`id` কমান্ড দিয়ে আপনি নিজের UID, GID, ও কোন কোন গ্রুপে আছেন, তা জানতে পারবেন।

```bash
id
```

**উদাহরণ:**

```
uid=1000(anis) gid=1000(anis) groups=1000(anis),27(sudo),1001(devs)
```

- **uid:** ইউজার আইডি
- **gid:** প্রাইমারি গ্রুপ আইডি
- **groups:** কোন কোন গ্রুপে আছেন

---

## ৩. নির্দিষ্ট ইউজারের তথ্য

```bash
id username
```

**উদাহরণ:**

```bash
id root
```

---

## ৪. গ্রুপ চেক ও সুপারইউজার চেক

- `groups` কমান্ড দিয়ে শুধু গ্রুপ লিস্ট দেখুন
- `id -u` দিয়ে ইউজার আইডি (0 হলে root)
- `id -Gn` দিয়ে গ্রুপের নাম

---

## ৫. প্র্যাকটিক্যাল টিপস

- sudo বা root হিসেবে কাজ করছেন কিনা, তা চেক করতে `whoami` বা `id -u` দেখুন
- টিমওয়ার্কে কোন গ্রুপে আছেন, তা জানুন
- স্ক্রিপ্টে ইউজার/গ্রুপ চেক করে কন্ডিশন দিন

---

## চ্যালেঞ্জ: নিজে চেষ্টা করুন

১. নিজের ইউজার ও গ্রুপ তথ্য দেখুন
২. অন্য ইউজারের তথ্য দেখুন
৩. কোন গ্রুপে আছেন, তা চেক করুন

---

## উপসংহার

`whoami` ও `id`—এই দুই কমান্ড ছাড়া লিনাক্সে ইউজার ম্যানেজমেন্ট, সিকিউরিটি, ও টিমওয়ার্ক অসম্ভব। এগুলো ভালোভাবে আয়ত্ত করলে আপনি সিস্টেম অ্যাডমিন ও পাওয়ার ইউজার হিসেবে আরও দক্ষ হবেন।

**প্র্যাকটিস করুন, ইউজার-গ্রুপের জাদু শিখুন!**
