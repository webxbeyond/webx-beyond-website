# APT, DNF, Pacman ইত্যাদি প্যাকেজ ম্যানেজার পরিচিতি

লিনাক্সে সফটওয়্যার ইন্সটল, আপডেট ও রিমুভ করার জন্য সবচেয়ে গুরুত্বপূর্ণ টুল হলো প্যাকেজ ম্যানেজার। বিভিন্ন ডিস্ট্রোতে বিভিন্ন প্যাকেজ ম্যানেজার ব্যবহৃত হয়—APT (Debian/Ubuntu), DNF (Fedora), Pacman (Arch), Zypper (openSUSE) ইত্যাদি। এখানে আমরা জনপ্রিয় প্যাকেজ ম্যানেজারগুলোর কাজ, পার্থক্য, কমান্ড ও বেস্ট প্র্যাকটিস জানব।

---

## ১. প্যাকেজ ম্যানেজার কী?

- সফটওয়্যার ইন্সটল, আপডেট, রিমুভ ও ডিপেন্ডেন্সি ম্যানেজ করার টুল
- সফটওয়্যার রিপোজিটরি থেকে প্যাকেজ ডাউনলোড ও কনফিগার করে
- সিস্টেম আপডেট ও নিরাপত্তা নিশ্চিত করে

---

## ২. APT (Advanced Package Tool) — ডেবিয়ান/উবুন্টু

### বেসিক কমান্ড:

```bash
sudo apt update           # প্যাকেজ লিস্ট আপডেট
sudo apt upgrade          # সব প্যাকেজ আপডেট
sudo apt install vim      # নতুন প্যাকেজ ইন্সটল
sudo apt remove vim       # প্যাকেজ রিমুভ
sudo apt search nginx     # প্যাকেজ সার্চ
```

- `.deb` ফরম্যাটের প্যাকেজ ব্যবহৃত হয়
- ডিপেন্ডেন্সি অটো-ম্যানেজ

---

## ৩. DNF (Dandified YUM) — ফেডোরা/রেডহ্যাট

### বেসিক কমান্ড:

```bash
sudo dnf check-update     # আপডেট চেক
sudo dnf upgrade          # সব প্যাকেজ আপডেট
sudo dnf install vim      # নতুন প্যাকেজ ইন্সটল
sudo dnf remove vim       # প্যাকেজ রিমুভ
sudo dnf search nginx     # প্যাকেজ সার্চ
```

- `.rpm` ফরম্যাটের প্যাকেজ ব্যবহৃত হয়
- আগের YUM-এর উন্নত ভার্সন

---

## ৪. Pacman — আর্চ/ম্যানজারো

### বেসিক কমান্ড:

```bash
sudo pacman -Syu          # সিস্টেম আপডেট
sudo pacman -S vim        # প্যাকেজ ইন্সটল
sudo pacman -R vim        # প্যাকেজ রিমুভ
sudo pacman -Ss nginx     # প্যাকেজ সার্চ
```

- `.pkg.tar.zst` ফরম্যাট
- খুব দ্রুত ও মিনিমালিস্টিক

---

## ৫. Zypper — ওপেনসুসে

### বেসিক কমান্ড:

```bash
sudo zypper refresh       # রিপো রিফ্রেশ
sudo zypper update        # সব প্যাকেজ আপডেট
sudo zypper install vim   # প্যাকেজ ইন্সটল
sudo zypper remove vim    # প্যাকেজ রিমুভ
sudo zypper search nginx  # প্যাকেজ সার্চ
```

## ৭. প্যাকেজ ম্যানেজার বেস্ট প্র্যাকটিস

- সবসময় প্রথমে `update`/`refresh` চালান
- একসাথে অনেক প্যাকেজ ইন্সটল করতে পারেন: `sudo apt install vim git curl`
- অপ্রয়োজনীয় প্যাকেজ রিমুভ করুন: `sudo apt autoremove`
- কনফিগারেশন ফাইল ডিলিট করতে `purge` (APT): `sudo apt purge nginx`
- প্যাকেজ ইনফো দেখুন: `apt show nginx`, `dnf info nginx`, `pacman -Qi nginx`
