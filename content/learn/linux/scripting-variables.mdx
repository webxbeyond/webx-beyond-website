# শেল স্ক্রিপ্টে ভ্যারিয়েবল ও আর্গুমেন্ট: ডায়নামিক স্ক্রিপ্টিংয়ের হাতেখড়ি

শেল স্ক্রিপ্টিংয়ে ভ্যারিয়েবল ও আর্গুমেন্ট ব্যবহার করলে স্ক্রিপ্ট আরও শক্তিশালী, ডায়নামিক ও কাস্টমাইজেবল হয়। এখানে আমরা ব্যাসিক থেকে অ্যাডভান্সড পর্যন্ত সবকিছু হাতে-কলমে শিখব।

---

## ১. ভ্যারিয়েবল কী?

ভ্যারিয়েবল হলো এমন একটি নাম, যার মধ্যে আপনি কোনো মান (value) সংরক্ষণ করতে পারেন।

### ১.১. ভ্যারিয়েবল ডিক্লেয়ার ও ব্যবহার
```bash
name="anis"
echo $name
```

### ১.২. স্পেসের নিয়ম
ভ্যারিয়েবল অ্যাসাইনমেন্টে কোনো স্পেস থাকবে না:
```bash
num=10   # ঠিক
num = 10 # ভুল
```

---

## ২. বিল্ট-ইন ভ্যারিয়েবল

শেল কিছু বিল্ট-ইন ভ্যারিয়েবল দেয়:
- `$HOME` : ইউজারের হোম ডিরেক্টরি
- `$USER` : ইউজারনেম
- `$PWD` : বর্তমান ডিরেক্টরি
- `$SHELL` : শেলের নাম

```bash
echo $USER
echo $PWD
```

---

## ৩. ইউজার ইনপুট থেকে ভ্যারিয়েবল

```bash
read -p "আপনার নাম লিখুন: " username
echo "স্বাগতম, $username!"
```

---

## ৪. ভ্যারিয়েবল সাবস্টিটিউশন ও ডিফল্ট ভ্যালু

```bash
name=${1:-Guest}
echo "হ্যালো, $name!"
```

---

## ৫. আর্গুমেন্ট কী?

স্ক্রিপ্ট রান করার সময় আপনি কিছু মান (argument) পাঠাতে পারেন।

### ৫.১. বেসিক আর্গুমেন্ট
```bash
#!/bin/bash
echo "প্রথম আর্গুমেন্ট: $1"
echo "দ্বিতীয় আর্গুমেন্ট: $2"
```

রান করুন:
```bash
./myscript.sh anis linux
```

### ৫.২. সব আর্গুমেন্ট একসাথে
```bash
echo "সব আর্গুমেন্ট: $@"
echo "আর্গুমেন্ট সংখ্যা: $#"
```

---

## ৬. শিফট অপারেটর

একাধিক আর্গুমেন্ট প্রসেস করতে `shift` ব্যবহার করা হয়:
```bash
while [ $# -gt 0 ]; do
  echo "আর্গুমেন্ট: $1"
  shift
done
```

---

## ৭. ভ্যারিয়েবল স্কোপ: লোকাল ও গ্লোবাল

ফাংশনের মধ্যে লোকাল ভ্যারিয়েবল ব্যবহার:
```bash
myfunc() {
  local msg="Hello"
  echo $msg
}
```

---

## ৮. অ্যারে ও অ্যাসোসিয়েটিভ অ্যারে

### ৮.১. অ্যারে
```bash
arr=(apple banana cherry)
echo ${arr[0]}
echo ${arr[@]}
```

### ৮.২. অ্যাসোসিয়েটিভ অ্যারে (bash 4+)
```bash
declare -A colors
colors[red]="#FF0000"
colors[green]="#00FF00"
echo ${colors[red]}
```

## ৯. ভ্যারিয়েবল এক্সপোর্ট ও এনভায়রনমেন্ট

```bash
export MYVAR=123
bash -c 'echo $MYVAR'
```

## ১০. বাস্তব উদাহরণ: ইউজার গ্রীটিং স্ক্রিপ্ট

```bash
#!/bin/bash
name=${1:-Guest}
echo "স্বাগতম, $name!"
```
