এই পেজে একটি মিনিমাল সেলফ হোস্টিং ল্যাব / VPS সেটআপের বেসিক দেখানো হবে।

## প্রি-রিকুইজিট
- VPS বা লোকাল হোস্ট (Ubuntu 24.04)
- SSH কী সেটআপ
- নন-রুট ইউজার + sudo

## চেক
```bash
lsb_release -a
whoami
uptime
```
