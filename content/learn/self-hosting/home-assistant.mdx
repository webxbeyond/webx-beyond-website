# ভূমিকা (Introduction)

**Home Assistant** একটি ওপেন সোর্স হোম অটোমেশন প্ল্যাটফর্ম যা আপনার স্মার্ট ডিভাইসগুলো এক জায়গায় নিয়ন্ত্রণ করতে সাহায্য করে। এটি নিজে হোস্ট করলে আপনি:

- **ডেটার সম্পূর্ণ নিয়ন্ত্রণ** পাবেন
- ক্লাউড সার্ভিসের ওপর নির্ভরতা কমাবে
- কাস্টম অটোমেশন এবং ইন্টিগ্রেশন তৈরি করতে পারবেন

**প্রধান ব্যবহার ক্ষেত্রসমূহ:**

- স্মার্ট লাইট, থার্মোস্ট্যাট, এবং সেন্সর নিয়ন্ত্রণ
- কাস্টম অটোমেশন স্ক্রিপ্ট চালানো
- ডিভাইস মনিটরিং এবং রিপোর্টিং
- MQTT, Zigbee, Z-Wave ইত্যাদি ইন্টিগ্রেশন

# ইনস্টলেশন পদ্ধতি (Installation Methods)

Home Assistant সবচেয়ে সহজে Docker এর মাধ্যমে হোস্ট করা যায়। নিচে ধাপে ধাপে নির্দেশনা দেওয়া হলো।

## প্রয়োজনীয়তা

- একটি লিনাক্স সার্ভার (Ubuntu/Debian) অথবা Raspberry Pi
- Docker & Docker Compose ইনস্টল করা
- একটি `.env` ফাইল ব্যবহারের জন্য

### ধাপ ১: Docker ও Docker Compose ইনস্টল

```bash
# Docker ইনস্টলেশন
sudo apt update
sudo apt install -y docker.io docker-compose

# Docker সার্ভিস চালু ও স্বয়ংক্রিয়ভাবে শুরু করা
sudo systemctl enable docker --now
sudo systemctl status docker
```

### ধাপ ২: প্রজেক্ট ডিরেক্টরি তৈরি

```bash
mkdir -p ~/homeassistant
cd ~/homeassistant
```

### ধাপ ৩: `.env` ফাইল তৈরি

`.env` ফাইলে সংবেদনশীল ডেটা রাখা উচিত। উদাহরণ:

```bash title=".env"
HOME_ASSISTANT_UID=1000
HOME_ASSISTANT_GID=1000
```

> **টিপ:** UID এবং GID আপনার সার্ভারের ব্যবহারকারীর সাথে মেলাতে হবে। `id` কমান্ড দিয়ে দেখতে পারবেন।

### ধাপ ৪: Docker Compose ফাইল তৈরি

```yml title="docker-compose.yml"
version: '3.8'
services:
  homeassistant:
    container_name: homeassistant
    image: ghcr.io/home-assistant/home-assistant:stable
    volumes:
      - ./config:/config
    environment:
      - TZ=Asia/Dhaka
      - PUID=${HOME_ASSISTANT_UID}
      - PGID=${HOME_ASSISTANT_GID}
    restart: unless-stopped
    network_mode: host
```

### ধাপ ৫: কনফিগারেশন ডিরেক্টরি তৈরি

```bash
mkdir -p ~/homeassistant/config
```

### ধাপ ৬: Docker Compose দিয়ে Home Assistant চালানো

```bash
docker-compose up -d
```

> **নোট:** প্রথমবার চালালে Home Assistant কনফিগারেশন ডাউনলোড এবং ডিরেক্টরি তৈরি করবে। এটি কিছু সময় নিতে পারে।

### ধাপ ৭: Web UI তে এক্সেস করা

* ব্রাউজারে যান: `http://YOUR_SERVER_IP:8123`
* প্রাথমিক সেটআপ সম্পন্ন করুন এবং আপনার স্মার্ট ডিভাইস সংযুক্ত করুন

# সাধারণ সমস্যা ও সমাধান (Troubleshooting)

* **কনটেইনার শুরু হচ্ছে না:**

  ```bash
  docker logs homeassistant
  ```

  লগ চেক করুন এবং ডিরেক্টরি পারমিশন ঠিক করুন

* **পোর্ট কনফ্লিক্ট:** নিশ্চিত করুন অন্য কোনো সার্ভিস 8123 পোর্ট ব্যবহার করছে না

# টিপস ও পরবর্তী ধাপ

* **SnapShots** এবং ব্যাকআপ নিয়মিত নিন
* **Add-ons** ব্যবহার করে MQTT, Node-RED, এবং Samba সংযোগ করুন
* YAML কনফিগারেশন শিখে কাস্টম অটোমেশন তৈরি করুন
