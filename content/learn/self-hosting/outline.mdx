## ভূমিকা
**Outline** হলো একটি ওপেন-সোর্স **নলেজবেস ও ডকুমেন্টেশন টুল**। এটি মূলত **টিম কোলাবরেশন** ও **নলেজ ম্যানেজমেন্ট** এর জন্য ব্যবহৃত হয়।  

### কেন Outline ব্যবহার করবেন?
- Markdown সাপোর্টেড রিচ এডিটর।
- Google Drive, Figma, Notion ইত্যাদির মতো টিম ডকুমেন্টেশন প্ল্যাটফর্মের বিকল্প।
- Role-based access control (RBAC)।
- Slack, GitHub, Google OAuth ইত্যাদির সাথে লগইন ইন্টিগ্রেশন।
- সুন্দর ও সহজ ইউআই।

## ইনস্টলেশন পদ্ধতি

Outline চালাতে সাধারণত তিনটি জিনিস প্রয়োজন:
1. **Outline অ্যাপ (Docker)**  
2. **PostgreSQL ডাটাবেস**  
3. **Redis ক্যাশ**  

---

### ধাপ ১: প্রয়োজনীয় সফটওয়্যার ইন্সটল
```bash
sudo apt update
sudo apt install docker.io docker-compose -y
```

---

### ধাপ ২: প্রোজেক্ট ফোল্ডার তৈরি

```bash
mkdir ~/outline
cd ~/outline
```

---

### ধাপ ৩: `.env` ফাইল তৈরি করুন

```bash title=".env"
# PostgreSQL
POSTGRES_USER=outline
POSTGRES_PASSWORD=supersecret
POSTGRES_DB=outline

# Outline
SECRET_KEY=changeme_super_secret_key
UTILS_SECRET=changeme_util_key
URL=https://docs.example.com

# Redis
REDIS_URL=redis://redis:6379

# Database URL
DATABASE_URL=postgres://outline:supersecret@postgres:5432/outline

# Storage (লোকাল ফাইল সিস্টেম)
FILE_STORAGE=local
```

---

### ধাপ ৪: `docker-compose.yml` ফাইল তৈরি

```yaml title="docker-compose.yml"
services:
  postgres:
    image: postgres:13
    container_name: outline-db
    restart: unless-stopped
    environment:
      - POSTGRES_USER=${POSTGRES_USER}
      - POSTGRES_PASSWORD=${POSTGRES_PASSWORD}
      - POSTGRES_DB=${POSTGRES_DB}
    volumes:
      - ./db:/var/lib/postgresql/data

  redis:
    image: redis:6
    container_name: outline-redis
    restart: unless-stopped

  outline:
    image: outlinewiki/outline:latest
    container_name: outline-app
    restart: unless-stopped
    depends_on:
      - postgres
      - redis
    environment:
      - DATABASE_URL=${DATABASE_URL}
      - SECRET_KEY=${SECRET_KEY}
      - UTILS_SECRET=${UTILS_SECRET}
      - URL=${URL}
      - REDIS_URL=${REDIS_URL}
      - FILE_STORAGE=${FILE_STORAGE}
    ports:
      - "3000:3000"
    volumes:
      - ./data:/data
```

---

### ধাপ ৫: কন্টেইনার চালু করা

```bash
docker-compose up -d
```

সবকিছু ঠিক আছে কিনা দেখতে:

```bash
docker ps
```

## Outline এ প্রবেশ

* ব্রাউজার খুলুন → `http://your-server-ip:3000`
* প্রথমবার লগইনের জন্য OAuth প্রোভাইডার (Google, GitHub, Slack ইত্যাদি) সেট করতে হবে।

---

## Nginx Proxy Manager এর সাথে ব্যবহার

Outline কে SSL সহ ডোমেইনে চালাতে চাইলে:

1. NPM ড্যাশবোর্ডে যান → **Proxy Hosts** → **Add Proxy Host**।
2. ডোমেইন লিখুন (যেমন: `docs.example.com`)।
3. Forward Hostname/IP দিন: `outline` (অথবা সার্ভারের লোকাল আইপি)।
4. Forward Port দিন: `3000`।
5. SSL ট্যাবে গিয়ে Let's Encrypt সার্টিফিকেট সেট করুন।

## টিপস ও সমস্যা সমাধান

* **ডেটা ব্যাকআপ**:

  ```bash
  tar -czvf outline-backup.tar.gz db data
  ```
* **আপডেট করতে**:

  ```bash
  docker-compose pull
  docker-compose up -d
  ```
* **OAuth সমস্যা হলে**: `.env` ফাইলে সঠিক `URL` সেট করুন (HTTPS সহ)।
