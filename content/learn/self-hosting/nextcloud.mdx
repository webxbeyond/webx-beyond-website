## ভূমিকা
**Nextcloud** হলো একটি ওপেন-সোর্স সফটওয়্যার, যা দিয়ে আপনি নিজের **Google Drive, Dropbox বা OneDrive-এর মতো ক্লাউড স্টোরেজ** তৈরি করতে পারবেন।  

### কেন Nextcloud নিজে হোস্ট করবেন?
- নিজের ডেটার উপর সম্পূর্ণ নিয়ন্ত্রণ।
- ফ্রি ও সীমাহীন ক্লাউড স্টোরেজ (সার্ভারের ক্ষমতা অনুযায়ী)।
- ক্যালেন্ডার, কনট্যাক্ট, নোটস, টাস্ক ম্যানেজমেন্টসহ অনেক এক্সটেনশন।
- টিম কোলাবরেশন, ফাইল শেয়ারিং ও নিরাপদ সিঙ্ক্রোনাইজেশন।

## ইনস্টলেশন পদ্ধতি

সবচেয়ে সহজ উপায় হলো **Docker Compose** ব্যবহার করা।

### ধাপ ১: প্রয়োজনীয় সফটওয়্যার ইন্সটল
```bash
sudo apt update
sudo apt install docker.io docker-compose -y
```

---

### ধাপ ২: প্রোজেক্ট ফোল্ডার তৈরি

```bash
mkdir ~/nextcloud
cd ~/nextcloud
```

---

### ধাপ ৩: `.env` ফাইল তৈরি করুন

```bash title=".env"
MYSQL_ROOT_PASSWORD=supersecretroot
MYSQL_PASSWORD=supersecret
MYSQL_DATABASE=nextcloud
MYSQL_USER=nextcloud
NEXTCLOUD_ADMIN_USER=admin
NEXTCLOUD_ADMIN_PASSWORD=changeme
TZ=Asia/Dhaka
```

---

### ধাপ ৪: `docker-compose.yml` ফাইল তৈরি

```yaml title="docker-compose.yml"
services:
  db:
    image: mariadb:10.6
    container_name: nextcloud-db
    restart: unless-stopped
    command: --transaction-isolation=READ-COMMITTED --binlog-format=ROW
    volumes:
      - ./db:/var/lib/mysql
    environment:
      - MYSQL_ROOT_PASSWORD=${MYSQL_ROOT_PASSWORD}
      - MYSQL_PASSWORD=${MYSQL_PASSWORD}
      - MYSQL_DATABASE=${MYSQL_DATABASE}
      - MYSQL_USER=${MYSQL_USER}
      - TZ=${TZ}

  app:
    image: nextcloud:latest
    container_name: nextcloud-app
    restart: unless-stopped
    ports:
      - 8080:80
    links:
      - db
    volumes:
      - ./nextcloud:/var/www/html
    environment:
      - MYSQL_PASSWORD=${MYSQL_PASSWORD}
      - MYSQL_DATABASE=${MYSQL_DATABASE}
      - MYSQL_USER=${MYSQL_USER}
      - MYSQL_HOST=db
      - NEXTCLOUD_ADMIN_USER=${NEXTCLOUD_ADMIN_USER}
      - NEXTCLOUD_ADMIN_PASSWORD=${NEXTCLOUD_ADMIN_PASSWORD}
      - TZ=${TZ}
```

---

### ধাপ ৫: কন্টেইনার চালু করা

```bash
docker-compose up -d
```

সবকিছু সঠিকভাবে চলছে কিনা দেখতে:

```bash
docker ps
```

## Nextcloud এ প্রবেশ

* ব্রাউজার খুলুন → `http://your-server-ip:8080`
* `.env` ফাইলে দেওয়া **অ্যাডমিন ইউজারনেম ও পাসওয়ার্ড** দিয়ে লগইন করুন।

---

## Nginx Proxy Manager এর সাথে ব্যবহার

যদি আপনি আগে থেকেই **Nginx Proxy Manager (NPM)** সেটআপ করে থাকেন, তবে Nextcloud কে সহজেই ডোমেইনের সাথে যুক্ত করতে পারবেন।

1. NPM ড্যাশবোর্ডে যান → **Proxy Hosts** → **Add Proxy Host**।
2. ডোমেইন লিখুন (যেমন: `cloud.example.com`)।
3. Forward Hostname/IP দিন: `nextcloud-app` (অথবা সার্ভারের লোকাল আইপি)।
4. Forward Port দিন: `8080`।
5. SSL ট্যাবে গিয়ে Let's Encrypt সার্টিফিকেট সেট করুন।

## টিপস ও সমস্যা সমাধান

* **ডেটা লোকেশন পরিবর্তন করতে চাইলে**: `./nextcloud` ভলিউম অন্য কোনো ডিস্ক/ডিরেক্টরিতে মাউন্ট করুন।
* **পারফরম্যান্স বাড়াতে চাইলে**: Redis Cache এবং PHP অপ্টিমাইজেশন ব্যবহার করতে পারেন।
* **ব্যাকআপ নিতে**:

  ```bash
  tar -czvf nextcloud-backup.tar.gz nextcloud db
  ```
* **আপডেট করতে**:

  ```bash
  docker-compose pull
  docker-compose up -d
  ```
