রিভার্স প্রক্সি বহুসংখ্যক সার্ভিসকে একই সার্ভার / IP এর পেছনে একত্র করে ডোমেইন / সাবডোমেইন রাউটিং দেয়।

## কেন দরকার
- সিঙ্গেল 80/443 এ মাল্টি সার্ভিস
- SSL টার্মিনেশন
- Access লগ / সিকিউরিটি লেয়ার

## উদাহরণ (Caddyfile)
```caddyfile
cloud.example.com {
  reverse_proxy :9000
}
```

> পরবর্তী: SSL অটোমেশন।
