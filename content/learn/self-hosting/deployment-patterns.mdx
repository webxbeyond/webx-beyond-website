Compose, স্বতন্ত্র systemd unit, Git pull workflow ও CI/CD (Watchtower / Woodpecker) চিন্তাধারা।

## প্যাটার্ন তুলনা
| প্যাটার্ন | সুবিধা | সীমাবদ্ধতা |
|----------|--------|------------|
| Docker Compose | দ্রুত, মাল্টি-সার্ভিস | বড় হলে জটিল |
| Systemd + Binary | মিনিমাল ওভারহেড | ম্যানুয়াল আপডেট |
| Git Pull Deploy | খুব সহজ | অটোমেশন কম |
| CI/CD পাইপলাইন | রিপ্রোডিউসেবল | সেটআপ সময় বেশি |

> সিরিজের প্রাথমিক কাঠামো শেষ। আরও ডিপ ডাইভ শীঘ্রই।
