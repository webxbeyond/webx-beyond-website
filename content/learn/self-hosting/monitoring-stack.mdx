Prometheus + Grafana + Loki + Node Exporter মিনিমাল কম্পোজ উদাহরণ।

```yaml
version: '3.9'
services:
  prometheus:
    image: prom/prometheus
    ports: ["9090:9090"]
  grafana:
    image: grafana/grafana
    ports: ["3001:3000"]
```

> পরবর্তী: ব্যাকআপ স্ট্রাটেজি।
